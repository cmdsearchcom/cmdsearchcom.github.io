NAME

dtc-xen_domUconf_network_debian - configure a debian VPS for using network

SYNOPSIS

dtc-xen_domUconf_network_debian VPS_PATH IP_ADDR NETMASK NETWORK BROADCAST GATEWAY
 

DESCRIPTION

dtc-xen_domUconf_network_debian This shell script is a part of the dtc-xen package that is to be used by the dtc panel to manage a Xen VPS server.
 
When doing it's VPS install, dtc_reinstall_os will call a script to do the setup of the network. dtc-xen_domUconf_network_debian does the job when the VPS type is Debian (that is: works also for Ubuntu based VPS).
 
See http://www.gplhost.com/software-dtc-xen.html for updates.

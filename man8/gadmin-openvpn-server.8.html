NAME

gadmin-openvpn-server - GTK+ configuration tool for openvpn (server)
 

SYNOPSIS

gadmin-openvpn-server
 

DESCRIPTION

gadmin-openvpn-server is a fast and easy to use GTK+ administration tool for the OpenVPN server.
 

OPTIONS

gadmin-openvpn-server has no options.
 

HOMEPAGE

More information about gadmin-openvpn-server and the gadmintools project can be found at < http://www.gadmintools.org/>.

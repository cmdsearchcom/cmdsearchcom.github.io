NAME

nss_updatedb - cache name service directories in DB format

SYNOPSIS

nss_updatedb nameservice [passwd⎪group]

DESCRIPTION

The nss_updatedb program gathers entries from the specified nameservice and stores it into a local database in Berkley DB format. It can store passwd and group information.

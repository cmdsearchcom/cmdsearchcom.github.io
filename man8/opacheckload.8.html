NAME
opacheckload

Returns load information on hosts in the fabric.

Syntax
opacheckload [-f hostfile] [-h ’hosts’] [-r] [-a|-n numprocs] [-d uploaddir]

Options
--help Produces full help text.

-f hostfile
File with hosts to check. Default = /etc/sysconfig/opa/hosts

-h hosts List of hosts to check.

-r Reverses output to show the least busy hosts. Default is busiest hosts.

-n numprocs
Shows top numprocs hosts. Default is 10.

-a Shows all hosts. Default is 10.

-d upload_dir
Directory to upload loadavg to. Default is uploads.

Examples
opacheckload opacheckload -h ’arwen elrond’ HOSTS=’arwen elrond’ opacheckload

Environment Variables
HOSTS List of hosts, used if -h option not supplied.

HOSTS_FILE
File containing list of hosts, used in absence of -f and -h.

UPLOADS_DIR
Directory to upload loadavg, used in absence of -d.

FF_MAX_PARALLEL
Maximum concurrent operations.

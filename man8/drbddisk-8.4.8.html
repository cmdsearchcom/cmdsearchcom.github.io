NAME
drbddisk - Script to mark devices as primary and mount file systems

SYNOPSIS
/etc/ha.d/resource.d/drbddisk [resource] {{start} | {stop} | {status}}

INTRODUCTION
The /etc/ha.d/resource.d/drbddisk script brings the local device of resource into primary role. It is designed to be used by Heartbeat.

In order to use /etc/ha.d/resource.d/drbddisk you must define a resource, a host, and any other configuration options in the DRBD configuration file. See /etc/drbd.conf for
details. If resource is omitted, then all of the resources listed in the config file are affected.

NAME
dtc-xen_domU_gen_xen_conf - configure the Xen startup file of a VPS

DESCRIPTION
dtc-xen_domU_gen_xen_conf This shell script is a part of the dtc-xen package that is to be used by the dtc panel to manage a Xen VPS server.

When doing it’s VPS install, dtc_reinstall_os will call a script to do the setup of the xen startup configuration file.

See http://www.gplhost.com/software-dtc-xen.html for updates.

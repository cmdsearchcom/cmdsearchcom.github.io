NAME
gamatronic - Driver for Gamatronic UPS equipment

NOTE
This man page only documents the hardware-specific features of the gamatronic driver. For information about the core driver, see nutupsdrv(8).

SUPPORTED HARDWARE
Various - Rebuilt to work with Gamatronic UPS Units, but should recognize any UPS that speaks the SEC protocol at 1200-19200 bps.

EXTRA ARGUMENTS
This driver does not support any extra settings in the ups.conf(5).

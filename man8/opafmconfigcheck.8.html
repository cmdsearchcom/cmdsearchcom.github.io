NAME
opafmconfigcheck

Parses and verifies the configuration file of a Fabric Manager (FM). Displays debugging and status information.

Syntax
opafmconfigcheck [-s] [-c config_file] [-v] [-d]

Options
--help Produces full help text.

-s Enables strict check mode. This option points out inconsistencies or invalid settings in VF and multicast configurations.

-c config_file
Specifies configuration file. Default = /etc/sysconfig/opafm.xml

-v Displays debugging and status information.

-d Displays configuration checksum information.

Example
opafmconfigcheck
opafmconfigcheck -v
opafmconfigcheck -sv

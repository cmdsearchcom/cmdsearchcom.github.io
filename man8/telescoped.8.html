NAME

telescoped - control the telescope, the server

SYNOPSIS


telescoped [options, values]

DESCRIPTION

The options are as follows:
-longitude
Longitude of the mount's site in degress. Positive on east.
-latitude
Latitude of the mount's site in degress. Positive on north. -c Location of config file. Usually: nightview.conf.

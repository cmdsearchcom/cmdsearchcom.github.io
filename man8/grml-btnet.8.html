NAME
grml-btnet - program to configure ip over bluetooth

SYNOPSIS
grml-btnet [OPTIONS] <ACTION>

DESCRIPTION
grml-btnet is a program to quickly configure IP over bluetooth.

ACTIONS
help
Show the help message.

OPTIONS
-h, help
Show summary of options.

-v
Show what is going on (more v â more out).

grml-btnets(8), grml-btnetc(8)

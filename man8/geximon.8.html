NAME
geximon - a Gtk2 exim MTA monitor

SYNOPSIS
geximon

DESCRIPTION
geximon is a Gtk2 monitor program for the exim mail transfer agent. In functionality it is similar to eximon. It has all features of the original (tracking the log, managing the
mail queue, plotting, etc.) and more. It also integrates with exiwhat (allowing you to manage active exim instances), exigrep, and eximstats. The interface is intuitive, compact,
and easy to use.

OPTIONS
geximon ignores all command-line arguments.

SEE ALSO
exim(8), eximon(8), exiwhat(8), exigrep(8), eximstats(8).

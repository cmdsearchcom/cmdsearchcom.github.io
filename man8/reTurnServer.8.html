NAME

reTurnServer - reTurn TURN server for NAT traversal

SYNOPSIS

reTurnServer [OPTIONS...]
 

DESCRIPTION

reTurnServer is an open-source, free TURN server implementing the RFC 5766 TURN protocol for NAT traversal.

NAME

cfetool - Standalone UI for Long term state registry

SYNOPSIS

cfetool [COMMAND] ARGS

DESCRIPTION

cfetool allows arbitrary and periodic data to be stored and evaluated. The user interface is modeled after rrdtool, another widely used tool to store measured data.
 
The COMMAND is the name of a cfetool command, each command has its own manpage; all commands are listed in the SEE ALSO section.

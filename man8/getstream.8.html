NAME
GETSTREAM - Lightweight DVB Streaming tool

SYNOPSIS
getstream [OPTION]

DESCRIPTION
GETSTREAM is a lightweight tool that streams a full DVB-S transponder into individual Multicast groups, one per program into the local area network. symbol tables.

OPTIONS
-d Start GetStream in debug mode.

-c config-file
Mandatory path to the configuration file. See the package documentation for examples.

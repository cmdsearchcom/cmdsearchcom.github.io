NAME

sxadm - manage SX clusters and individual nodes

SYNOPSIS

sxadm <command> [OPTIONS]...

DESCRIPTION

sxadm is used to create and manage SX clusters and individual nodes. It works in two modes: local and remote.

COMMANDS

The available commands are listed below. To get more information about a specific command run 'man sxadm-COMMAND'.
node
Perform local operations on a specific node.
cluster
Perform operations on a remote cluster.

OPTIONS

-h, --help
Print help and exit
-V, --version
Print version and exit

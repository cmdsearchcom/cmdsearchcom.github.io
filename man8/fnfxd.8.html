FNFXD(8) Toshiba Hot-Keys Util FNFXD(8)

NAME

fnfxd - the daemon reacts on Fn-x combinations and hot-keys which are defined in `/etc/fnfxd/fnfxd.conf'.
 
 

DESCRIPTION


fnfx8 is a daemon to use hot-keys on toshiba laptops, it depends of acpi toshiba kernel support.
This manual page documents briefly the fnfxd commands.
This manual page was written for the Debian distribution because the original program does not have a manual page. See fnfxd README for more information.

OPTIONS

-h
Show summary of options.
-d
Enable debug output.
-n
Do not send into daemon mode.

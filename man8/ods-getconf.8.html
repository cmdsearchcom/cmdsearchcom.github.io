NAME
ods-getconf - OpenDNSSEC get configuration values

SYNOPSIS
ods-getconf [-c FILE] [-h] [-V]expr

DESCRIPTION
ods-getconf is part of the OpenDNSSEC software. With this tool, you can extract the value of a configuration option by giving the appropriate expression. For more information, go
to http://www.opendnssec.org and visit the Documentation page.

OPTIONS
-c FILE
Read configuration from file, instead of using the default.

-h Show this help.

-V Show version and exit.

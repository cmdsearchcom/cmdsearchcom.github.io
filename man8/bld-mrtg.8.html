NAME

bld-mrtg.pl - return MRTG-friendly statistics about local BLD status

SYNOPSIS

bld-mrtg.pl

DESCRIPTION

bld-mrtg.pl reads bld daemon dump files and prints statistics about the number of currently blacklisted IPs and the number of IPs added within the last 5 minutes. The default output is MRTG-friendly.

SEE ALSO

bld(8) bldread(8)

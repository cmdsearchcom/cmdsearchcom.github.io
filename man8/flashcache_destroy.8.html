NAME
flashcache_destroy - destroy an existing flashcache volume

DESCRIPTION
The purpose of the flashcache_destroy command is to Clear all meta data from cache device. Beware, this will cause data loss on the affected devices.

SYNOPSIS
flashcache_destroy cache_devname

EXAMPLE
flashcache_destroy /dev/sdc
Destroy the existing cache on /dev/sdc. All data is lost!

SEE ALSO
flashcache_create(8), flashcache_load(8)

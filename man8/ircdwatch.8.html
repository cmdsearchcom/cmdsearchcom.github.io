IRCDWATCH(8) System Manager's Manual IRCDWATCH(8)

NAME

ircdwatch - check ircds current state

SYNOPSIS

ircdwatch [--kill | --rest | --help]

DESCRIPTION

ircdwatch checks ircd's current state and starts/kills it if necessary.
--kill stop both ircdwatch and ircd
--rest stop ircdwatch but let ircd alone
--help display help text
If you start ircdwatch without any arguments, it will check if ircd runs and start it if it doesn't.

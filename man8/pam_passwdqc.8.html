NAME
pam_passwdqc â Password quality-control PAM module

SYNOPSIS
[service-name] module-type control-flag pam_passwdqc [options]

DESCRIPTION
The pam_passwdqc module is a simple password strength checking module for PAM. In addition to checking regular passwords, it offers support for passphrases and can provide ranâ
domly generated ones.

The pam_passwdqc module provides functionality for only one PAM management group: password changing. In terms of the module-type parameter, this is the âpasswordâ feature.

The pam_chauthtok() service function may ask the user for a new password, and verify that it meets certain minimum standards. If the chosen password is unsatisfactory, the service
function returns PAM_AUTHTOK_ERR.

The set of options that may be passed to the module is exactly the same as the set of options that may be specified in the /etc/passwdqc.conf file. These options are described in
passwdqc.conf(5).

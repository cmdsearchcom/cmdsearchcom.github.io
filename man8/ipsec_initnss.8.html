NAME
ipsec_initnss - Initialise the IPsec NSS database

SYNOPSIS
ipsec initnss [--nssdir /var/lib/ipsec/nss]

DESCRIPTION
Initnss initialises the NSS database where all private keys for RSA and certificate keypairs are stored. If already initialised, it will return an error. To remove an existing
IPsec NSS database, remove all the *.db files and pkcs11.txt from the NSS data directory (default: /var/lib/ipsec/nss).

SEE ALSO
ipsec.conf(5), ipsec(8), ipsec_auto(8), route(8)

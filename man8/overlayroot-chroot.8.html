NAME
overlayroot-chroot - chroot into the lower filesystem in a writable mode

SYNOPSIS
overlayroot-chroot [COMMAND [ARG]...]

DESCRIPTION
overlayroot-chroot is a program that sets up bind mounts for /proc, /run, and /sys, remounts the lower directory in an overlayfs with write capabilities and will chroot(8) into
it. Upon exit, the bind mounts are unmounted.

Any [COMMAND [ARG]...] are passed directly through to chroot(8).

SEE ALSO
chroot(8)

NAME

fence_pcmk - Part of the Pacemaker cluster resource manager

SYNOPSIS

fence_pcmk [ options]

DESCRIPTION

Helper that presents a RHCS-style interface to stonith-ng for CMAN based clusters
Should never need to use invoked by the user directly

OPTIONS

-h
usage
-n <name>
nodename
-o <string>
Action: on | off | reset (default) | stat | hostlist
-q
quiet mode
-V
version
January 2017 fence_pcmk 1.0.0

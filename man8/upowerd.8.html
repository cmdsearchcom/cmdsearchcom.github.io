NAME
upowerd - UPower Daemon

SYNOPSIS
upowerd [--help]

DESCRIPTION
upowerd provides the org.freedesktop.UPower service on the system message bus. Users or administrators should never need to start this daemon as it will be automatically started
by dbus-daemon(1) whenever an application calls into the org.freedesktop.UPower service.

OPTIONS
--help
Show help options.

AUTHOR
Written by David Zeuthen <davidz@redhat.com> with a lot of help from many others.

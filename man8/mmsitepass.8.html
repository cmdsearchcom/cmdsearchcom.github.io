NAME
mmsitepass - Set the Mailman site password, prompting from the terminal.

SYNOPSIS
mmsitepass [options] [password]

OPTIONS
-c, --listcreator

Set the list creator password instead of the site password. The list creator is authorized to create and remove lists, but does not have the total power of the site
administrator.

-h, --help

Print a small help text and exit

SECURITY
Including the admin-password on the command line is a small security risk if you have untrusted users on the system.

DESCRIPTION
The site password can be used in most if not all places that the list administrator’s password can be used, which in turn can be used in most places that a list users password
can be used.

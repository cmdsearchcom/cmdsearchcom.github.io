NAME
status-dirsrv - script for showing the status of Directory Server

SYNOPSIS
status-dirsrv [<instance>]

DESCRIPTION
status-dirsrv shows all the status of Directory Server instances on a system, or of a single instance if specified

OPTIONS
<instance>
Specifies a single Directory Server instance to show the status

USAGE
Sample usage:

status-dirsrv example

EXIT STATUS
status-dirsrv returns 0 if all Directory Server instances are running. Otherwise, it returns a positive value which represents the number of server instances which are not runâ
ning.

If a single Directory Server instance is specified, 0 is returned if the instance is running; 1 is returned if the instance is not running; 255 is returned if the instance does
not exist.

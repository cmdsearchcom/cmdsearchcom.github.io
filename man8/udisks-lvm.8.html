NAME
udisks-lvm - The udisks command line tool

SYNOPSIS
udisks-lvm [-b] [-f] COMMAND

DESCRIPTION
udisks-lvm is a helper program used by udisks to query the state of LVM2 via the lvm2app library.

OPTIONS
-h
Show help options.

-b
Print the result of COMMAND in binary form (serialized).

-f
Use an alternative LVM system directory; no lock.

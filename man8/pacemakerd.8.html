PACEMAKER(8) System Administration Utilities PACEMAKER(8)

NAME

Pacemaker - Part of the Pacemaker cluster resource manager

SYNOPSIS

pacemakerd mode [options]

DESCRIPTION

pacemakerd - Start/Stop Pacemaker

OPTIONS

-?, --help
This text
-$, --version
Version information
-V, --verbose
Increase debug output
-S, --shutdown
Instruct Pacemaker to shutdown on this machine
-F, --features
Display the full version and list of features Pacemaker was built with

Additional Options:

-f, --foreground
(Ignored) Pacemaker always runs in the foreground
-p, --pid-file=value
(Ignored) Daemon pid file location

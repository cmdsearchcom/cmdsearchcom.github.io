NAME
thin_ls - List thin volumes within a pool.

SYNOPSIS
thin_ls [options] {metadata device}

DESCRIPTION
thin_ls Displays infomation about thin volumes in a pool. Pass the metadata device on the command line, not the pool device.

This tool cannot be run on live metadata unless the --metadata-snap option is used.

OPTIONS
-o, --format
Give a comma separated list of the fields to be output. Valid fields are: DEV, MAPPED_BLOCKS, EXCLUSIVE_BLOCKS, SHARED_BLOCKS, MAPPED_SECTORS, EXCLUSIVE_SECTORS,
SHARED_SECTORS, MAPPED_BYTES, EXCLUSIVE_BYTES, SHARED_BYTES, MAPPED, EXCLUSIVE, SHARED, TRANSACTION CREATE_TIME, SNAP_TIME

--no-headers
Don’t output headers.

-m, --metadata-snap

If you want to get information out of a live pool then you will need to take a metadata snapshot and use this switch.

--verbose
Provide extra information on the mappings.

-h, --help
Print help and exit.

-V, --version
Output version information and exit.

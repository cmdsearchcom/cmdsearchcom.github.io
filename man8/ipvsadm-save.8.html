NAME
ipvsadm-save - save the IPVS table to stdout

SYNOPSIS
ipvsadm-save [-n]

DESCRIPTION
ipvsadm-save prints the IPVS table to stdout.

OPTIONS
The option that is recognized by ipvsadm-save is:

-n print out the table in numeric format.

ipvsadm(8), the LVS-HOWTO.

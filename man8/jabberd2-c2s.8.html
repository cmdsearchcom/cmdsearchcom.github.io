NAME

jabberd2-c2s - client-to-server connector

SYNOPSIS

jabberd2-c2s [-h] [-c config] [-D]

DESCRIPTION

jabberd2-c2s accepts incoming connections from clients, performs authentication and registration functions, and communicates with the session manager on their behalf.

OPTIONS

-h
Show summary of options.
-c
Alternate configuration file to use. The compiled-in default is /etc/jabberd2/c2s.xml.
-D
Print debugging output. You should configure jabberd with the --enable-debug switch to enable this.

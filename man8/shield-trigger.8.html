NAME
shield-trigger - program used with pam_shield PAM module to manage ip blocking

SYNOPSIS
shield-trigger [add|del] <IP number>

DESCRIPTION
This is a program that manages the ip blocks for pam_shield PAM module. shield-trigger is normally called by the pam_shield PAM module

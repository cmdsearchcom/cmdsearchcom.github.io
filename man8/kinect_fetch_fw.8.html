NAME
kinect_fetch_fw - download the Usb Audio Class firmware for the Kinect

SYNOPSIS
kinect_fetch_fw <firmware destdir>

DESCRIPTION
kinect_fetch_fw is a program to download the Usb Audio Class device firmware for the Microsoft Kinect.

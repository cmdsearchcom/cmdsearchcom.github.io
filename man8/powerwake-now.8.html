NAME

powerwake-now - send the powernapd daemon the 'powerwake now' signal
 
 

DESCRIPTION

When the powernapd(8) runs, it is monitoring the system, and determining when it should take the specificied powernap action.
 
If the administrator (or even other utilities or processes) could asynchronously determine that the system should 'powerwake now' from the powersave mode, the powerwake-now(8) can be used to send the powernapd(8) a signal, which will log the event and force a wake-up from the powersave mode immediately.
 
http://launchpad.net/powernap

NAME
umount.udisks2 - unmount file systems that have been mounted by udisks

DESCRIPTION
The umount.udisks2 program is a helper for the umount(8) program. Its purpose is to clean up automatically created directories created at file system mount-time. It should never
be called directly.

AUTHOR

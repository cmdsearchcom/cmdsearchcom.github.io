NAME

fiforead - test program for aprsdigi.

SYNOPSIS

fiforead filename

DESCRIPTION

Fiforead is useful for testing aprsdigi. Unix datagrams received from aprsdigi -p:unix:filename,callsign,alias... are written to stdout.

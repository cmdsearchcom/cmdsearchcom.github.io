NAME

update-geoip-database - script to download the latest version of the MaxMind GeoIP Lite database

SYNOPSIS

update-geoip-database
 

DESCRIPTION

The update-geoip-database command downloads the latest version of the MaxMind GeoIP Lite database from MaxMind webiste.
 
In particular, the following binary databases are downloaded:
—
GeoLite Country
—
GeoLite Country IPv6
—
GeoLite City
—
GeoLite ASN
and stored in the /usr/share/GeoIP directory.

geoipupdate(1), geoiplookup(1). geoiplookup6(1).

NAME
dnf.plugin.reposync - DNF reposync Plugin

Synchronize packages of a remote Yum repository to a local directory.

SYNOPSIS
dnf reposync [-p <download-path>] ...

DESCRIPTION
reposync makes local copies of remote repositories. Packages that are already present in the local directory are not downloaded again.

OPTIONS
-p <download-path>, --download-path=<download-path>
Root path under which the downloaded repositories are stored, relative to the current working directory. Defaults to the current working directory. Every downloaded reposâ
itory has a subdirectory named after its ID under this path.

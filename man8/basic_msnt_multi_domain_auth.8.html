NAME
basic_msnt_multi_domain_auth

SYNOPSIS
basic_msnt_multi_domain_auth [options]

DESCRIPTION
basic_msnt_multi_domain_auth is a Squid authenticator to check user credentials against multiple NT domains using nmblookup.

The user is expected to enter his/her credentials as domainsername or domain/username (in analogy to what MS-Proxy does).

Requires Authen::SMB from CPAN and Samba if you need to perform NETBIOS queries.

OPTIONS
--debug Write debug info to stderr.

--wins-server
Use the named WINS server.

Default: broadcast will be attempted.

--no-fqdn Some servers don’t like to be called by their fully qualified name. Define this if you wish to call them ONLY by their hostname.

--no-rdns Some servers really really want to be called by address.

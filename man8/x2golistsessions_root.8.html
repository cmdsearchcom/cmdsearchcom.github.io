NAME
x2golistsessions_root - List X2Go Sessions for System

SYNOPSIS
x2golistsessions_root [<server_hostname>]

DESCRIPTION
x2golistsessions_root shows a list of all available X2Go sessions on the server <server_hostname>. If no server hostname is given at the command line, the session list query
runs for the local system.

x2golistsessions_root must be run with root privileges.

RETURN VALUES
Same as explained in x2golistsessions(8).

x2golistsessions(8)

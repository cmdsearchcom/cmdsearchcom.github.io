DHIS-GENID(8) BSD System Manager’s Manual DHIS-GENID(8)

NAME
dhis-genid â Dynamic Host Information System client ID generation.

SYNOPSIS
dhis-genid

DESCRIPTION
dhis-genid is a tool to generate a client ID entry in the DHIS client database. When run dhis-genid will create a new entry in the client database with a unique ID.

FILES
/etc/dhis-server/dhis.db
DHIS client database. Contains the credentials of clients.

dhisd(8)

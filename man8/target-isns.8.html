NAME
target-isns - iSNS client for the Linux LIO iSCSI target

SYNOPSIS
target-isns [OPTION]

DESCRIPTION
target-isns allows to register LIO iSCSI targets to an iSNS server.

The Internet Storage Name Service (iSNS, RFC 4171) protocol makes easier to discover, manage and configure iSCSI devices.

With iSNS, iSCSI targets can be registered to a central iSNS server and initiators can be configured to discover the targets by asking the iSNS server.

OPTIONS
-i, --isns-server=HOSTNAME
Set the hostname of the iSNS server.

-p, --isns-port=PORT
Set the port of the iSNS server (default iSNS port is 3205).

-d, --debug
Increase the debugging level (implies --foreground).

-f, --foreground
Run in the foreground.

-v, --version
Print version information.

-h, --help
Print a help message.

FILES
/etc/target-isns.conf is the default configuration file.

The following parameters can be set in the configuration file:

isns_server=HOSTNAME
Set the hostname of the iSNS server.

isns_port=PORT
Set the port of the iSNS server (default iSNS port is 3205).

log_level=[info|debug]
Set the log level (default is info).

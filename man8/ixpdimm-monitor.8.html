NAME
ixpdimm-monitor - Daemon to monitor Intel(R) DIMM management events. Used by ixpdimm-cli.

SYNOPSIS
ixpdimm-monitor [-d]

DESCRIPTION
Daemon to monitor Intel(R) DIMM management events. These events can be viewed using ixpdimm-cli.

OPTION
-d Starts ixpdimm-monitor as a background daemon.

EXIT CODES
0 The operation succeeded.

1 An error occurred.

FILES
/var/lib/ixpdimm_sw/apss.dat
/var/run/ixpdimm-monitor.pid

SEE ALSO
ixpdimm-cli (8)

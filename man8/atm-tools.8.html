DOCUMENTATION

atm-tools is a package containing the basic tools that are needed for setting up, monitoring, and tuning ATM networks. All available documentation can be found in /usr/share/doc/atm-tools in Debian systems, and on the Web on: http://linux-atm.sourceforge.net.

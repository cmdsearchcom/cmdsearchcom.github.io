NAME

gutenprint-foomaticppdupdate - update Gutenprint Foomatic PPD files

SYNOPSIS

gutenprint-foomaticppdupdate VERSION

DESCRIPTION

gutenprint-foomaticppdupdate regenerates the Gutenprint Foomatic PPD files in use by CUPS, using freshly generated Foomatic PPD files as templates. The new PPD file replaces the old PPD file under /etc/cups/ppd, but all the options set in the old PPD will be merged into the new PPD, provided that they are still valid in the new file.
gutenprint-foomaticppdupdate does not restart cupsd. cupsd will need manual reloading (or send SIGHUP) once gutenprint-foomaticppdupdate has completed.
This is not intended to be used directly but instead is run automatically on package upgrade.

OPTIONS

VERSION The version number of the Gutenprint package just installed.

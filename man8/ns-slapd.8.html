NAME
nsâslapd - The main Directory Server daemon

SYNOPSIS
ns-slapd -D configdir [-d debuglevel] [-i pidlogfile] [-v] [-V]

DESCRIPTION
nsâslapd launches the LDAP Directory Server

OPTIONS
A summary of options is included below:

-v Show version of program.

-D configdir
Specifies the configuration directory pointing at the instance to be started (e.g. /etc/dirsrv/slapdâlocalhost)

-d debuglevel
Specifies the debuglevel to be used

-i pidlogfile
Specifies file where the pid of the process will be stored

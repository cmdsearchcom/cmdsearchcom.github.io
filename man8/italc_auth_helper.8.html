NAME

italc_auth_helper - iTALC PAM Authentication Helper

SYNOPSIS

italc_auth_helper

DESCRIPTION

italc_auth_helper is a PAM-su authentication helper for the iTALC service (called ica).
NOTE: italc_auth_helper has to be installed setuid root and must be available in your system's environment $PATH.

OPTIONS

No options present.

NAME
Pacemaker - Part of the Pacemaker cluster resource manager

SYNOPSIS
crm_error [options] -- rc

DESCRIPTION
crm_error - Tool for displaying the textual name or description of a reported error code

OPTIONS
-?, --help
This text

-$, --version
Version information

-V, --verbose
Increase debug output

-n, --name
Show the error’s name rather than the description. Useful for looking for sources of the error in source code

-l, --list
Show all known errors.

NAME
avcstat - Display SELinux AVC statistics

SYNOPSIS
avcstat [-c] [-f status_file] [interval]

DESCRIPTION
Display SELinux AVC statistics. If the interval parameter is specified, the program will loop, displaying updated statistics every interval seconds. Relative values are disâ
played by default.

OPTIONS
-c Display the cumulative values.

-f Specifies the location of the AVC statistics file, defaulting to /sys/fs/selinux/avc/cache_stats.

NAME
ltsp-update-kernels - copy LTSP chroot kernels to TFTP directories

SYNOPSIS
ltsp-update-kernels [OPTION] [CHROOT...]

DESCRIPTION
Copies the boot/ directory from LTSP chroots to the TFTP directories in order to make them available to PXE clients. Copying kernels from inside NBD images is also supported.
CHROOT can be a fullpath or a subdirectory of the base directory, and if it’s unset, all available chroots are processed.

OPTIONS
-b, --base[=PATH]
The LTSP base directory. Defaults to /opt/ltsp if unspecified.

-h, --help
Displays the ltsp-update-kernels help message.

--version
Output version information and exit.

AVAILABILITY
Part of the ltsp package and the latest versions are available in source form from https://launchpad.net/products/ltsp.

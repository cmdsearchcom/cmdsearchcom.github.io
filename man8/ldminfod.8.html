NAME
ldminfod â output login sessions and available locales

SYNOPSIS
ldminfod

DESCRIPTION
ldminfod will output the login sessions and locales available. it is usually run from an inetd(8) instance. it is used by ldm(1) to remotely determine available login and locale
settings.

EXAMPLE
add the following lines to /etc/inetd.conf(5):

9571 stream tcp nowait nobody /usr/sbin/tcpd /usr/sbin/ldminfod

SEE ALSO
ldm(1), inetd(8), inetd.conf(5).

AVAILABILITY
ldminfod is part of ldm package and the latest versions are available in source form from https://launchpad.net/products/ltsp.

NAME
opafastfabric

(Switch and Host) Starts the top-level Intel(R) Omni-Path Fabric Suite FastFabric Text User Interface (TUI) menu to enable setup and configuration. Refer to Intel(R) Omni-Path
Fabric Suite FastFabric User Guide

for additional details.

Syntax
opafastfabric

Options
None.

Example
#opafastfabric
#8217#160;Intel FastFabric OPA Tools
#8217#160;Version: X.X.X.X.X

#8217#160; 1) Chassis Setup/Admin
#8217#160; 2) Externally Managed Switch Setup/Admin
#8217#160; 3) Host Setup
#8217#160; 4) Host Verification/Admin
#8217#160; 5) Fabric Monitoring

#8217#160; X) Exit

NAME
newaliases - update /etc/aliases database

SYNOPSIS
newaliases

DESCRIPTION
This is a link to the ssmtp binary. It invokes /usr/sbin/sendmail with the -bi option. It is provided for compatibility with the sendmail program.

In Fedora sSMTP supports /etc/aliases as a text file so there is no need to regenerate a database.

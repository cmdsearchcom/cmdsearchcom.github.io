NAME

rad - Router Advertisement Daemon (ratools/rad)

SYNOPSIS

rad [ -s|--socket UNIXSOCKET] [ -l|--loglevel LOGLEVEL]

DESCRIPTION

rad is a fast, dynamic, multi-threading daemon for sending IPv6 Router Advertisements (RA). It is controlled via ractl.

OPTIONS

--socket
Override default path of the control socket.
--loglevel
Set logging level. Possible values are info, warning, and error.

AVAILABILITY

rad is part of the ratools framework available from https://github.com/danrl/ratools

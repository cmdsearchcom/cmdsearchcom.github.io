NAME
carrousel â perform load balancing services

SYNOPSIS
carrousel [-c config]

DESCRIPTION
Relays incoming connections to any of a number of available backend nodes. Backends register themselves using the transponder(8) program.

OPTIONS
-c config
Read configuration from config instead of the default configuration file.

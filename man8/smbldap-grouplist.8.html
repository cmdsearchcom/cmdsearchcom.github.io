NAME

smbldap-grouplist - list groups

SYNOPSIS

smbldap-grouplist [-S] [group template]

DESCRIPTION

-d Show displayName
-S Show samba SID entry
-? show the help message

EXAMPLE

smbldap-grouplist -dS
smbldap-grouplist "*ourn*"

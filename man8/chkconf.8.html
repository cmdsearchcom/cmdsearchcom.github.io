CHKCONF(8) System Manager's Manual CHKCONF(8)

NAME

chkconf - check ircd config-file

SYNOPSIS

chkconf <ircd.conf>

DESCRIPTION

chkconf checks ircd config-file <ircd.conf> for known errors.

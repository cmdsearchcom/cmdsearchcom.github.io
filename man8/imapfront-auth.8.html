NAME
imapfront-auth - IMAP Front End

SYNOPSIS
imapfront-auth program [ args ... ]

DESCRIPTION
imapfront-auth provides IMAP authentication using CVM modules. The LOGIN command is authenticated via the CVM named by $CVM_SASL_PLAIN. If authentication succeeds, it sets up
the environment and executes program. It also offers RFC 1734 compliant AUTH support through ‘‘cvm-sasl’’. The environment is set up for use with Courier IMAP.

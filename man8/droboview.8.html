DROBOVIEW(8) System Administration Utilities DROBOVIEW(8)

NAME
droboview - GUI for managing drobo units

SYNOPSIS
droboview [ <device> ]

DESCRIPTION
Starts the graphical management utility for Drobos. See drobom for detailed usage. (this is the same as the ’drobom view’)

device raw block device of a drobo. If not given, it will be searched for.

SEE ALSO
drobom(8)

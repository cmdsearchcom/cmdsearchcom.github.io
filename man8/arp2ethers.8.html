NAME
arp2ethers - convert arpwatch address database to ethers file format

SYNOPSIS
arp2ethers

DESCRIPTION
arp2ethers converts file arp.dat in the current directory into ethers(5) format on stdout. Usually arp.dat is an ethernet/ip database file generated by arpwatch(8). The arpâ
watch daemon in Debian will create different arp.dat depending on its configuration. All of them will be available at /var/lib/arpwatch/.

FILES
/var/lib/arpwatch - default directory for arp.dat
arp.dat - ethernet/ip address database

arpwatch(8), ethers(5), rarp(8), arp(8),

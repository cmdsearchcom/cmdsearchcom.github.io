NAME

sirvecole — controlaula daemon

SYNOPSIS

sirvecole

DESCRIPTION

The sirvecole daemon runs listening for orders from the teacher. When an identified teacher order is parsed, it executes it. For this to work, a real teacher in a pc identified as a teacher pc must be running ControlAula.
 

OPTIONS

This program accepts no options

NAME

kernel-packageconfig - internal handler of /etc/kernel-pkg.conf

SYNOPSIS

kernel-packageconfig

DESCRIPTION

The utility kernel-packageconfig is called internally by the kernel-package postinst script to set up the conf file /etc/kernel-pkg.conf. It is idempotent, and only modifies the file if it contains factory default values. It is not meant for general use.

OPTIONS

This utility does not have any options.

FILES

The file affected by this program is /etc/kernel-pkg.conf.

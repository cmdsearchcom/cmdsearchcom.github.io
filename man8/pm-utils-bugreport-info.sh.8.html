NAME
pm-utils-bugreport-info.sh - Print pm-utils bug report

SYNOPSIS
pm-utils-bugreport-info.sh

DESCRIPTION
pm-utils-bugreport-info.sh can be used to print pm-utils bug report including your HW configuration and pm-utils logs. This can help developers to diagnose problems regarding
suspend / hibernate.

SEE ALSO
pm-suspend(8), pm-hibernate(8)

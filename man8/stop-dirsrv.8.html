NAME

stop-dirsrv - stop script for Directory Server

SYNOPSIS

start-dirsrv [ <instance>]

DESCRIPTION

stop-dirsrv stops all Directory Server instances on a system, or only a single instance if specified

OPTIONS

<instance>
Specifies a single Directory Server instance to stop
 

USAGE

Sample usage:
stop-dirsrv example

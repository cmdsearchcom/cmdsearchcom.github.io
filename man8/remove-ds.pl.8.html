NAME
remove-ds.pl - Remove an instance of Directory Server

SYNOPSIS
remove-ds.pl [-a] [-f] [-d -d ... -d] -i instance

DESCRIPTION
Removes a Directory Server instance from the system. The instance will be shutdown and the files will be removed. The certificate database files will not be removed. The
instance configuration directory will have a .removed extension appended to it, which will contain the retained certificate database files.

OPTIONS
A summary of options is included below:

-a Removes all files and directories belonging to the instance

-f Force removal

-d Enable debugging: adding more -d will make output more verbose

-i instance
The full name of the instance to remove (e.g. slapdâexample)

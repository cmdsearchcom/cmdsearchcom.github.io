NAME
x2goterminate-desktopsharing - Cleanly Terminate Desktop Sharing for a given X2Go Session

SYNOPSIS
x2goterminate-desktopsharing <session_id>

DESCRIPTION
x2goterminate-desktopsharing allows to detect a running and associated x2godesktopsharing applet and to cleanly terminate this applet.

x2goterminate-desktopsharing is run with normal user privileges and it is used from within X2Go Server. There normally is no need to execute this command manually.

RETURN VALUES
As exitcode x2goterminate-desktopsharing returns 0 if a x2godesktopsharing process could be identified and terminated. If not, the exit code is -1.

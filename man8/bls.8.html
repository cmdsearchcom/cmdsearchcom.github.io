BLS(8) Network backup, recovery and verification BLS(8)

NAME


bls - Bacula's 'Tape LS'

SYNOPSIS

bls [options] <device-name>
 

DESCRIPTION

This manual page documents briefly the bls command.

OPTIONS

A summary of options is included below.
-?
Show version and usage of program.
-b bootstrap
Specify a bootstrap file.
-c config
Specify configuration file.
-d nn
Set debug level to nn.
-dt
Print timestamp in debug output
-e <file>
Specify exclude list file.
-i <file>
Specify include list file.
-j
List jobs.
-k
List blocks.
(no -j or -k option)
List saved files
-L
Dump label.
-p
Proceed in spite of errors.
-V volumes
Specify volume names (separated by '|').
-v
Set verbose mode.

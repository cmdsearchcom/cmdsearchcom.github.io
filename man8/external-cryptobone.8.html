NAME

external-cryptobone - Administration tool to change firewall settings and sshd configurations

SYNOPSIS

/usr/bin/external-cryptobone
 

DESCRIPTION

The program external-cryptobone is called by the GUI external-cryptobone-admin to perform administrative changes as root. It can be used to enable the external cryptobone daemon and to set or reset restrictive firewall settings as well as a hardened secure shell daemon configuration.
 

OPTIONS

none
 

FILES

/usr/bin/external-cryptobone

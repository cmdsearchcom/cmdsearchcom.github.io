NAME
ifindex - Create nodelist index

SYNOPSIS
ifindex [-h] [-x verbosity] [-I file]

DESCRIPTION
Ifindex creates a nodelist index which is used by ifcico for faster nodelist lookups.

Depending on the OPTIONS set when compiling the program, one of 3 different nodelist indices will be generated: a db- a (g)dbm- or a dbz- version. The dbz- type nodelist-index
has some advantages (smaller indices and faster index generation) over the other 2 types but is currently only in PRE-ALPHA state - you should also read the README before using
this type of nodelist-index.

OPTIONS
-h Display a short help message.

-x Verbosity
Set the debug verbosity. Verbosity may be a number from 0 to 32 to set ’on’ bits from 1 to number, or a string of letters where ’a’ = bit 1, ’b’ = bit 2, etc. up to bit
26.

-I File
Use the alternate configuration file File.

RETURN CODES
0 Successfull
64 command line usage error
65 error reading config file

DEBUG VERBOSITY
d lock files
f aka matching
g outbound directories
h call
j session handshake
k file transfer
l filelist
m nodelist flags
r modem
t nodelist index
z hydra protocol

FILES
$IFLIBDIR/config
Runtime configuration file.

nodelist.???
A list of nodes in an FTN network.

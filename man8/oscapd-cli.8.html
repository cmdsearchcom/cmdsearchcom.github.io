NAME
oscapd-cli - OpenSCAP-daemon command line interface

SYNOPSIS
oscapd-cli [-h] {eval,scan,task,task-create,status,result}

positional arguments:
{eval,task,task-create,status,result}

eval Interactive one-off evaluation of any target supported by OpenSCAP Daemon

task Show info about tasks that have already been defined. Perform operations on already defined tasks.

task-create
Create new task.

status Displays status, tasks that are planned and tasks that are being evaluated.

result Displays info about past results

optional arguments:
-h, --help
show this help message and exit

-v, --version
show program’s version number and exit

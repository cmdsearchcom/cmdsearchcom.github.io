NAME

geographiclib-get-gravity -- a GeographicLib administrative tool

DESCRIPTION

geographiclib-get-gravity downloads and installs the gravity datasets used by GeographicLib, <http://geographiclib.sourceforge.net>. For documentation, supply the -h option:

geographiclib-get-gravity -h
By default, the datasets are installed in /usr/share/GeographicLib/gravity and the script requires write access to this directory.

NAME
systemd-hostnamed.service, systemd-hostnamed - Host name bus mechanism

SYNOPSIS
systemd-hostnamed.service

/usr/lib/systemd/systemd-hostnamed

DESCRIPTION
systemd-hostnamed is a system service that may be used as a mechanism to change the system’s hostname. systemd-hostnamed is automatically activated on request and terminates
itself when it is unused.

The tool hostnamectl(1) is a command line client to this service.

See the developer documentation[1] for information about the APIs systemd-hostnamed provides.

SEE ALSO
systemd(1), hostname(5), machine-info(5), hostnamectl(1), sethostname(2)

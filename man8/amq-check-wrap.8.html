NAME
amq-check-wrap - check loopback RPC connection to amd

SYNOPSIS
amq-check-wrap

DESCRIPTION
amq-check-wrap simply checks whether or not a loopback connection to the amd service (provided by amd from the am-utils package) is authorized by the tcpd host access control
mechanism.

EXIT STATUS
0 Loopback connection to the amd RPC service is authorized by tcpd.

1 tcpd denies loopback connections to the amd RPC service.

2 amq-check-wrap was invoked with some arguments.

AUTHOR

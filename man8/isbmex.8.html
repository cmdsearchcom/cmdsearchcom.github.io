NAME
isbmex - Driver for ISBMEX UPS equipment

NOTE
This man page only documents the hardware-specific features of the isbmex driver. For information about the core driver, see nutupsdrv(8).

SUPPORTED HARDWARE
This driver supports SOLA/BASIC Mexico ISBMEX protocol UPS equipment.

EXTRA ARGUMENTS
This driver does not support any extra settings in the ups.conf(5).

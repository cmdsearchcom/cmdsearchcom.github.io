NAME
x2godbadmin - X2Go Server Database Administrator

SYNOPSIS
x2godbadmin <options>

DESCRIPTION
x2godbadmin is used to set up the initial X2Go session database and perform other maintenance tasks on the X2Go server’s database.

x2godbadmin must be run with root privileges.

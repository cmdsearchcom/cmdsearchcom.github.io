NAME
calendarserver_shell â Calendar Server interactive administrative shell

SYNOPSIS
calendarserver_shell [-f config_file]

DESCRIPTION
calendarserver_shell provides an interactive shell that can be used to view and manipulate Calendar Server data. calendarserver_shell operates directly on the back-end Calendar
Server data store, and does not connect to the Calendar Server itself.

OPTIONS
-h, --help
Displays usage information

-f, --config FILE
Use the Calendar Server configuration specified in the given file.

FILES
/etc/caldavd/caldavd.plist
The Calendar Server configuration file.

/Library/CalendarServer/Documents
The server’s document root, which is the used as the backing store for the HTTP resources on the server.

NAME
fence_legacy - Part of the Pacemaker cluster resource manager

SYNOPSIS
fence_legacy [options]

DESCRIPTION
Helper that presents a RHCS-style interface for Linux-HA stonith plugins

Should never need to use invoked by the user directly

OPTIONS
-h usage

-t <sub agent>
sub agent

-n <name>
nodename

-o <string>
Action: on | off | reset (default) | stat | hostlist

-s <stonith>
stonith command

-q quiet mode

-V version

fence_legacy 1.0.0 January 2017 FENCE_LEGACY(8)

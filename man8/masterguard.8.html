NAME
masterguard - Driver for Masterguard UPS equipment

NOTE
This man page only documents the hardware-specific features of the masterguard driver. For information about the core driver, see nutupsdrv(8).

SUPPORTED HARDWARE
This driver supports Masterguard UPS equipment.

EXTRA ARGUMENTS
CS
Cancel the shutdown procedure.

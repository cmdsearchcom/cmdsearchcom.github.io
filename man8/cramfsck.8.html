NAME
cramfsck - check a compressed ROM file system

SYNOPSIS
cramfsck [-hv] [-x dir] file

DESCRIPTION
This manual page documents briefly the cramfsck command. This manual page was written for the Debian GNU/Linux distribution (but may be used by others), because the original
program does not have a manual page.

cramfsck checks the consistency of a CramFs file system given by file. Unlike other file system checkers, it does not attempt to repair a broken file system. This is not a
deficiency as CramFs is not subject to the kind of corruption that a writable file system may endure.

cramfsck is also able to extract the contents of the file system.

OPTIONS
-h Give a brief help message.

-v Turn on verbose mode.

-x dir In addition to checking the given file system, extract its contents into dir.

AUTHOR

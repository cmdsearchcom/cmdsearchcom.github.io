NAME

dpkg-purge - mark packages for purge
 

SYNOPSIS

dpkg-purge [PACKAGE...]
 

DESCRIPTION

dpkg-purge marks package(s) to be purged.
 
It achieves this by running `dpkg --set-selections' and piping in the package name(s) followed by `purge'.
 
NOTE: This program does not install, remove, purge any package. It ONLY sets the status of the package. Any actual removal or purging is done the next time dselect, apt-get, aptitude, etc are run.
 

SEE ALSO

dpkg(1), dpkg-hold(1), dpkg-unhold(1), dpkg-remove(1)

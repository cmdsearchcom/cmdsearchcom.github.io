NAME
keytouch-editor - An editor for keytouch keyboard definition files

SYNOPSIS
keytouch-editor

DESCRIPTION
This manual page documents briefly the keytouch-editor

keytouch-editor is a program to create and edit keyboard definition files for keytouch, in order to allow that program to recognize the keystrokes for a new model of keyboard.

OPTIONS
This program has no options, and works completely from a graphical user interface.

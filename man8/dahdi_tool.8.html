NAME

dahdi_tool - Shows status of DAHDI interfaces

SYNOPSIS

dahdi_tool
 

DESCRIPTION

dahdi_tool shows the current status the DAHDI inteface cards plugged to the computer.
 
It displays values like Current Alarms, SyncSource, Tx/Rx Levels for each DAHDI interface.
 

SEE ALSO

dahdi_monitor(8), asterisk (8).

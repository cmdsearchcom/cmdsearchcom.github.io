name
sigul_server_create_db - Create a database used by a signing server

SYNOPSIS
sigul_server_create_db [OPTIONS]

DESCRIPTION
Creates a database used by a signing server.

OPTIONS
-c, --config-file PATH
Use PATH as the per-user configuration file instead of ~/.sigul/server.conf.

-v, --verbose
Be more verbose. Using this option twice enables debugging output.

EXIT STATUS
sigul_server_add_admin returns with exit status 0 on success, non-zero on error.

FILES
/etc/sigul/server.conf
A system-wide configuration file.

~/.sigul/server.conf
A per-user configuration file. Values defined in this file override the system-wide configuration file.

NAME
inventory_firmware_gui - graphically inventory and update firmware installed in hardware

SYNOPSIS
inventory_firmware_gui [-h | --help] [-c config-file ] [-o | --over section,key,value ] [ -v | --verbose ] [ -w | --warnings ] directory ]

DESCRIPTION
inventory_firmware_gui scans hardware and directories of available firmware, graphically shows the inventory, and upgrades the firmwares in that hardware.

OPTIONS
-h Print help text

-c config-file
Use the alternate system wide config-file instead of /etc/firmware/firmware.conf

-o | --over section,key,value
Override values in firmware.conf with new values as section, key, and value.

-v | --verbose
Print additional information.

-w | --warnings
Print additional warnings.

FILES
/etc/firmware/firmware.conf
The system wide configuration file.
/usr/share/firmware-tools
The location of the glade description files.

AUTHOR
firmware-tools team <firmware-tools-devel at lists.us.dell.com>

NAME

bootinfoscript - tool for debugging bootloader problems

SYNOPSIS

bootinfoscript [options]

INTRODUCTION

bootinfoscript inspects your boot environment, that is your partitions and your boot loader. It produces a report which is easy to read and easy to send to someone else who might be able to help you if you can’t fix the problem yourself.

OPTIONS

-v
Show version and exit.
-h
Show help text and exit.
-g or --gzip
Gzip report file after creation. The plain text file will be kept, too.
--this
Force bootinfoscript to run, even if you have another copy of bootinfoscript in your local directory.

SEE ALSO

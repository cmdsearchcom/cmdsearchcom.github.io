NAME

tripplitesu - Driver for Tripp-Lite SmartOnline (SU) UPS equipment

NOTE

This man page only documents the hardware-specific features of the tripplitesu driver. For information about the core driver, see nutupsdrv(8).

SUPPORTED HARDWARE

This driver supports the Tripp Lite SmartOnline family (via the serial port).

EXTRA ARGUMENTS

This driver supports the following optional settings in the ups.conf(5):
lowbatt=num
Set the low battery warning threshold in percent at which shutdown is initiated by upsmon(8). By default, the UPS may not report low battery until there are only a few seconds left. Common values are around 25—30.

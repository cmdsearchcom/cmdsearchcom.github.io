IPTOTALD(8) System Manager's Manual IPTOTALD(8)

NAME

iptotald - iptotal daemon
 

SYNOPSIS

iptotald
 

DESCRIPTION

iptotald is the daemon program for iptotal.
 
iptotal is the daemon that regularly invokes iptotal(8) to monitor traffic. It is normally started at boot from /etc/rc.
 

OPTIONS

None.
 

CONFIGURATION FILE

/etc/iptotal.conf
File that defines the configuration used to invoke iptotal(8). See iptotal_config(5) for more details.

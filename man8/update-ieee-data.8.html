NAME
update-ieee-data - update data files from IEEE Standards Association such as the OUI and IAB lists

SYNOPSIS
update-ieee-data

DESCRIPTION
update-ieee-data fetches the current version of the OUI and IAB lists from http://standards.ieee.org/regauth/oui/index.shtml and installs it, if the installed versions is older
than 5 days.

This utility requires curl, wget or libwww-perl to be installed.

OPTIONS
-q Be quiet. Do not output messages.

-f Force the update, even if the file is newer than 5 days.

FILES
/usr/share/misc/
Directory holding symbolic links to oui.txt and iab.txt

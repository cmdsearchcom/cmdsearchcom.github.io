NAME
svstat - prints the status of services monitored by supervise(8).

SYNOPSIS
svstat services

DESCRIPTION
services consists of any number of arguments, each argument naming a directory. svstat prints one human-readable line for each directory, saying whether supervise(8) is successâ
fully running in that directory, and reporting the status information maintained by supervise(8).

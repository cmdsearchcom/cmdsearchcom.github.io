NAME

rmcat - remove GNATS categories

SYNOPSIS

rmcat
category [ category... ]

DESCRIPTION

Removes any named category from the GNATS installation. Before calling rmcat, you must first remove category from the list of categories in the database. You must also reclassify or remove all problem reports associated with category (i.e., category's subdirectory must be empty).

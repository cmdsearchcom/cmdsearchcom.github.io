NAME
cloudtoserver - Tool to convert a Fedora Cloud instance to a Fedora Server instance.

DESCRIPTION
usage: cloudtoserver [-h] [-d] [-v]

optional arguments:
-h, --help
show this help message and exit

-d, --disable-cloud-service
Disable cloud-init service

-v, --version
Prints the version of the tool

SEE ALSO
The project cloudtoserver is maintained at https://github.com/fedora-cloud/cloudtoserver

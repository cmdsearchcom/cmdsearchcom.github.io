NAME

grub-mkconfig — Generate a GRUB configuration file.
 

SYNOPSIS

grub-mkconfig [-o | --output=FILE]
 

DESCRIPTION

grub-mkconfig generates a configuration file for GRUB.
 

OPTIONS

--output=FILE
Write generated output to FILE.

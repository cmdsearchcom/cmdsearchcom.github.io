runc delete - delete any resources held by the container often used with detached container

SYNOPSIS
runc delete [command options] <container-id>

Where "<container-id>" is the name for the instance of the container.

OPTIONS
--force, -f Forcibly deletes the container if it is still running (uses SIGKILL)

EXAMPLE
For example, if the container id is "ubuntu01" and runc list currently shows the status of "ubuntu01" as "stopped" the following will delete resources held for "ubuntu01" removâ
ing "ubuntu01" from the runc list of containers:

# runc delete ubuntu01

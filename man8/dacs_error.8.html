NAME

dacs_error - simple error handling utility for DACS

SYNOPSIS

dacs_error

DESCRIPTION

This program is part of the DACS suite.
The dacs_error utility is a simple Perl program that prints various state and debugging information. It is intended to be used as a very basic error handler for the AUTH_ERROR_HANDLER or ACS_ERROR_HANDLER directives (see dacs.conf(5)[1]), or customized by a DACS installation for that purpose.

OPTIONS

There are no options.

DIAGNOSTICS

The program always exits 0.

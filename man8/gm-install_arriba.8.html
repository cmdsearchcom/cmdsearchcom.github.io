NAME

gm-install_arriba - installiert den ARRIBA-Rechner für Schlaganfall/Herzinfarkt-Risiko
 

SYNOPSIS

gm-install_arriba
 

DESCRIPTION

gm-install_arriba ist ein Hilfsprogramm, um den ARRIBA-Rechner für das Schlaganfall/Herzinfarkt-Risiko zu installieren. Der Rechner selbst benötigt Java 6.
 

SEE ALSO

http://www.arriba-hausarzt.de

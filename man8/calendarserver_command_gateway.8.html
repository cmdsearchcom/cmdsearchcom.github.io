NAME
calendarserver_command_gateway â Calendar Server manager gateway

SYNOPSIS
calendarserver_command_gateway [--config file]

DESCRIPTION
calendarserver_command_gateway is a gateway between Apple OS X Server Admin and the calendar server, and is not meant to be run by end users. It processes commands in plist form
from stdin and writes results in plist form to stdout.

OPTIONS
-h, --help
Displays usage information

-f, --config FILE
Use the Calendar Server configuration specified in the given file. Defaults to /etc/caldavd/caldavd.plist.

FILES
/etc/caldavd/caldavd.plist
The Calendar Server configuration file.

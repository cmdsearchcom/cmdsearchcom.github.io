NAME
form2ufl - Convert from old FFC .form format to UFL format.

SYNOPSIS
form2ufl input.form

DESCRIPTION
Simple conversion script from old FFC .form format to UFL format.

BUGS
Send comments, questions, bug reports etc. to ufl@lists.launchpad.net.

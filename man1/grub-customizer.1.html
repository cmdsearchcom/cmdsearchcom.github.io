NAME

grub-customizer - a graphical grub2/burg editor
 

SYNOPSIS

grub-customizer
 

DESCRIPTION

Grub Customizer is an application which provides an easy way to edit the configuration of Grub 2 or BURG. The base functionality of Grub Customizer is to rename, reorder or delete the menu entries - but you can also configure settings like default entry, timeouts, backgrounds and much more.
 
Grub Customizer also helps you restoring your bootloader by providing an easy way to access the configuration of other partitions.

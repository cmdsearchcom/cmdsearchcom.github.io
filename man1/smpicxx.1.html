NAME
smpicxx - SMPI C++ wrapper compiler

SYNOPSIS
smpicxx â¦

DESCRIPTION
smpicxx is a wrapper around the C++ compiler defined when building SimGrid, adding the right flags in order to use the SimGrid library and SMPI, the MPI implementation within it.
Most of the arguments filled to smpicxx will be passed to the underlying compiler directly, except for --version which gives the SimGrid library version, and --compiler-version
which gives the C++ compiler version. Please refer to the latter documentation in case of trouble.

ENVIRONMENT VARIABLES
All environment variables taken into account by the underlying compiler will be used by it, but note that the compiler can not be modified.

NAME
spmvtest4 - SpMV kernel benchmark for sparse matrix storage formats

SYNOPSIS
spmvtest4 matrix_filename_list iter [block]

DESCRIPTION
This program inputs the matrix data from the files listed in matrix_filename_list, and computes the multiplies of matrices in available matrix storage formats and a vector (1,
..., 1)^T. The FLOPS performance is measured as the average of iter iterations. The Matrix Market format is supported for matrix_filename. If necessary, the block size of the
BSR and BSC formats can be specified by block. See Lis User Guide for more information.

EXIT STATUS
The following exit values are returned:

0 The process is normally terminated

unspecified
An error occurred

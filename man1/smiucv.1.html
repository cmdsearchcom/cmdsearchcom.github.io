NAME

smiucv - helper utility that provides an IUCV interface to the z/VM System Management API (SMAPI).

SYNOPSYS

smiucv [user] [name] <request>

DESCRIPTION

smiucv is a utility which reads requests from stdin, sends them to the SMAPI server via IUCV and writes the response to stdout. It is intended to be used with the smaclient shell command.

OPTIONAL PARAMETERS

user
Defaults to VSMREQIU
name
Defaults to DMSRSRQU.

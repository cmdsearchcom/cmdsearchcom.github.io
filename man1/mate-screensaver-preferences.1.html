NAME

mate-screensaver-preferences - Configure MATE Screensaver

SYNOPSIS

mate-screensaver-preferences [ OPTIONS]

DESCRIPTION

Manages preferences for the MATE Screensaver.

OPTIONS

This program only accepts the standard GTK options.

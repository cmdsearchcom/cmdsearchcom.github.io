NAME
papi_multiplex_cost - computes execution time costs for basic PAPI operations on multiplexed EventSets.

Synopsis
papi_cost [-m, --min < min >] [-x, --max < max >] [-k,-s]

Description
papi_multiplex_cost is a PAPI utility program that computes the min / max / mean / std. deviation of execution times for PAPI start/stop pairs and for PAPI reads on multiplexed
eventsets. This information provides the basic operating cost to a user’s program for collecting hardware counter data. Command line options control display capabilities.

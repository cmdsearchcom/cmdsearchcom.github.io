NAME
fso-gpsd - gpsd compatibility daemon for the freesmarpthone.org frameworkd

SYNOPSIS
fso-gpsd [-n] [-N] [-f device] [-P pidfile] [-D n] [-S port] [-h] [-V]

OPTIONS
-n Don’t wait for client connects to poll GPS

-N Don’t go into background

-f device
Set a GPS device name

-P pidfile
Set file to record process ID

-D integer (default 0)
Set debug level

-S integer (default 2947)
Set port for daemon

-h Display help message and terminate

-V Emit version and exit

NAME
clausetester - check formulas in models

SYNOPSIS
clausetester <interpretations-file> < <formulas-file> > <annotated-formulas-file>

DESCRIPTION
This manual page documents briefly the clausetester command.

This program takes a set of interpretations and stream of formulas. For each formula, the interpretations in which the formula is true are shown, and at the end the number of
formulas true in each interpretation is shown.

SEE ALSO
prover9(1), mace4(1).
Full documentation for clausetester is found in the prover9 manual, available on Debian systems in the prover9-doc package at /usr/share/doc/prover9-doc/manual/index.html.

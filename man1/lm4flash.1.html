USAGE:(1) User Commands USAGE:(1)

NAME

Usage: - Command-line firmware flashing tool to communicate with the Stellaris Launchpad

SYNOPSIS

lm4flash [ options] <binary-file>

DESCRIPTION

-V
Print version information
-h
Print usage information
-v
Enables verification after write
-E
Only erase blocks where binary file will be written
-S address
Write binary at the given address (in hexadecimal)
-s SERIAL
Flash device with the following serial

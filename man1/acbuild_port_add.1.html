NAME
acbuild-port-add - Add a port

SYNOPSIS
acbuild port add [OPTIONS]

DESCRIPTION
Updates the ACI to contain a port with the given name, protocol, and port. If the port already exists, its values will be changed.

OPTIONS
--count=1
Specifies a range of ports, going from PORT to PORT + count - 1

--socket-activated[=false]
Set the app to be socket activated on this/these port/ports

OPTIONS INHERITED FROM PARENT COMMANDS
--debug[=false]
Print out debug information to stderr

--modify=""
Path to an ACI to modify (ignores build context)

--no-history[=false]
Don’t add annotations with the command that was run

--work-path="."
Path to place working files in

EXAMPLE
acbuild port add https tcp 443 --socket-activated

NAME

git-annex-lock - undo unlock command

SYNOPSIS

git annex lock [path ...]

DESCRIPTION

Use this to undo an unlock command if you don't want to modify the files any longer, or have made modifications you want to discard.

OPTIONS

file matching options
The git-annex-matching-options(1) can be used to specify files to lock.

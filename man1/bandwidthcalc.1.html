NAME

bandwidthcalc - file transfer time calculator

DESCRIPTION

bandwidthcalc is a small X application written in GTK+ 2.0. It determines how long it will take to transfer a file on a given bandwidth. You can specify the bandwidth in kBit/s, kByte/s, MBit/s or MByte/s. The transfer time is output in HH:MM:SS

OPTIONS

There are no options available, because these programs runs fully as X application.

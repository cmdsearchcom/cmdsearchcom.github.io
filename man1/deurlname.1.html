NAME
deurlname - Remove URL-encoded characters from file names.

SYNOPSIS
deurlname [OPTION]... [FILE]...

DESCRIPTION
This manual page document describes the deurlname command.

deurlname will rename every file specified on the command line that has URL-encoded characters in its name. The URL-encoded characters will be translated into the corresponding
byte values. That is, %20 will become space, %41 A and so on.

OPTIONS
These programs follow the usual GNU command line syntax, with long options starting with two dashes (‘-’).

-v, --verbose
Display every rename done.

--help Show summary of options.

--version
Output version information and exit.

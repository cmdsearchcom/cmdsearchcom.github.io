ECERE-IDE(1) Ecere SDK/Integrated Development Environment ECERE-IDE(1)

NAME
ecere-ide - Ecere IDE

SYNOPSIS
ecere-ide [project] [files]*

DESCRIPTION
Launches the Ecere Integrated Environment.

A project (.epj) and/or files to be opened on startup can be specified on the command line.

SETTINGS
Settings for the IDE are stored in ~/.ecereIDE/

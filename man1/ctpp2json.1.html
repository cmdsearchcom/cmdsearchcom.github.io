CTPP2JSON(1) General Commands Manual CTPP2JSON(1)

NAME
ctpp2json â CTPP JSON file parser

SYNOPSIS
ctpp2json source.json

DESCRIPTION
ctpp2json parses data in JSON format from file source.json and prints result to standard output.

EXIT STATUS
The ctpp2json utility exits 0 on success, and >0 if an error occurs.

SEE ALSO
ctpp2c(1), ctpp2vm(1), ctpp2i(1), ctpp2-config(1)

HISTORY
An ctpp2json command appeared in CTPP 2.0.0.

DFONT2RES(1) General Commands Manual DFONT2RES(1)

NAME
dfont2res - convert Macintosh dfont font file format to Macintosh resource fork format

SYNOPSIS
dfont2res dfontfile

DESCRIPTION
The program dfont2res converts a font from the new data fork resource file (.dfont) format (as used under MacOS X) to the older resource fork format.

NAME
bisho â program to set the account information for Meego web services

SYNOPSIS
bisho

DESCRIPTION
This manual page documents briefly the bisho command.

This manual page was written for the Debian distribution because the original program does not have a manual page.

bisho is a prgram to set the account information for Meego web services. For example, it can set the Twitter account/password for Meego web services.

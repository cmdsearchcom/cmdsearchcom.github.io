NAME
gccontrol - GUI for ccontrol

SYNOPSIS
The gccontrol command takes no arguments.

DESCRIPTION
gccontrol provides a Graphical User Interface for choosing the configuration file for ccontrol to use. It works by switching the $HOME/.ccontrol/config symlink to point to
another configuration file in that directory.

SEE ALSO
ccontrol(1), ccontrol-init(1).

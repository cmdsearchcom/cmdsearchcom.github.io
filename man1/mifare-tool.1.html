NAME

mifare-tool - Low-level mifare tag command line tool based on librfid

SYNOPSIS

mifare-tool [-rlwkbh]

DESCRIPTION

mifare-tool is a command line tool for testing low-level mifare tag support in librfid.

OPTIONS

-r, --read Read a mifare sector.
-l, --loop-read
Loop reading a mifare sector.
-w, --write
Write a mifare sector.
-k, --key
Specify a mifare access key (in hex tuples).
-b, --brute-force N
Brute-force read sector N.
-h, --help
Show a help text and exit.

NAME
apertium-postlatex - This application is part of ( apertium )

This tool is part of the apertium open-source machine translation toolbox: http://www.apertium.org.

SYNOPSIS
apertium-postlatex-raw [ <input file> [ <output file> ] ]

DESCRIPTION
apertium-postlatex-raw This filter generates LaTeX code from the output of apertium-relatex command. Non-ASCII characters are generated into its native encoding, depending on the
locale of the running environment.

OPTIONS
-h, --help
Display this help.

SEE ALSO
apertium-deslatex(1), apertium-prelatex(1), apertium-relatex(1), apertium-postlatex-raw(1), lt-proc(1), apertium(1).

BUGS
Complicated constructions in LaTeX (i.e. custom defined tags) are not (yet) supported.

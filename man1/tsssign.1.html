TSSSIGN(1) User Commands TSSSIGN(1)

NAME

tsssign - manual page for tsssign v713

DESCRIPTION

sign
Runs TPM2_Sign
-hk key handle
[-pwdk password for key (default empty)] [-halg [sha1, sha256, sha384] (default sha256)] -if input message to hash and sign [-ipu public key file name to verify signature (default no verify)] [-ipem public key PEM format file name to verify signature (default no verify)]
requires -ipu
[-os signature file name] [-tk ticket file name]
-se[0-2] session handle / attributes (default PWAP)
01 continue

SEE ALSO

The full documentation for tsssign is maintained as a Texinfo manual. If the info and tsssign programs are properly installed at your site, the command
info tsssign
should give you access to the complete manual.

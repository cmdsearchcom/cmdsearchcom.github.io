NAME
i810rotate - toggle the video output on i810 hardware using i810switch(1)

SYNOPSIS
i810rotate

DESCRIPTION
This manual page documents briefly the i810rotate, command. This manual page was written for the Debian GNU/Linux distribution because the original program does not have a manâ
ual page.

i810rotate is a program that toggles the currently selected video output for i810 based video cards, as found on several laptops. It loops through three states: LCD only output,
LCD and CRT output, and CRT only output.

OPTIONS
There are none. It’s a 5 line shell script.

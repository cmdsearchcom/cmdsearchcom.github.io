NAME
hfst-foma - =Imitates xfst using foma and a wrapper script

SYNOPSIS
hfst-foma [OPTIONS...]

DESCRIPTION
Imitates xfst using foma and a wrapper script

Common options:
-h Print help message

-V Print version info

-F Parameter is HFST format name

If -F is not given, OpenFst’s tropical format will be used. The possible values for -F are { foma, openfst-tropical, openfst-log, sfst }. Input files are always treated as
UTF-8.

NAME

changepagestring - Search and replace in all PDF pages

SYNOPSIS

 changepagestring [options] infile.pdf search-regex replace-str [outfile.pdf]
 Options:
   -o --order          preserve the internal PDF ordering for output
   -v --verbose        print diagnostic messages
   -h --help           verbose help message
   -V --version        print CAM::PDF version

DESCRIPTION

Searches through all pages of a PDF file for instances of "search-regex" and inserts "replace-str". The regex should be a form that Perl understands. Note that this does not change the PDF metadata like forms and annotation. To change metadata, use instead changepdfstring.

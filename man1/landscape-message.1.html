NAME

landscape-message - Send a message to the landscape web interface

SYNOPSIS

landscape-message [options] [MESSAGE] ...

DESCRIPTION

Invoking landscape-message will cause a message to appear in the History section for this computer in the Landscape web UI.
If no MESSAGE is specified on the command line, then landscape-message will read the message from STDIN until EOF.

OPTIONS

--version
Show program's version number and exit.
-h, --help
Show this help message and exit.
-b BUS, --bus=BUS
The DBUS bus to use to send the message.

EXAMPLES

landscape-message Hello administrator

NAME
smpif90 - SMPI Fortran 90 wrapper compiler

SYNOPSIS
smpif90 â¦

DESCRIPTION
smpif90 is a wrapper around the Fortran 90 compiler defined when building SimGrid, adding the right flags in order to use the SimGrid library and SMPI, the MPI implementation
within it. Most of the arguments filled to smpif90 will be passed to the underlying compiler directly, except for --version which gives the SimGrid library version, and --comâ
piler-version which gives the F90 compiler version. Please refer to the latter documentation in case of trouble.

ENVIRONMENT VARIABLES
All environment variables taken into account by the underlying compiler will be used by it, but note that the compiler can not be modified.

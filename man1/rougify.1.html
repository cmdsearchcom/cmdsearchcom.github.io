ROUGIFY(1) User Commands ROUGIFY(1)

NAME

rougify - manual page for rougify 1.11.1

DESCRIPTION

usage: rougify [command] [args...]

where <command> is one of:

highlight
highlight code
help
print help info
style
print CSS styles
list
list available lexers
version
print the rouge version number
See `rougify help <command>` for more info.

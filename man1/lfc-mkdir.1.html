NAME
lfc-mkdir - make LFC directory in the name server

SYNOPSIS
lfc-mkdir [-m mode] [-p] path...

DESCRIPTION
lfc-mkdir creates the specified LFC directories in the name server.

This requires write permission in the parent directory. The owner ID and group ID of the new directories are set to the requestor’s real user ID and group ID, respectively.

path specifies the LFC pathname. If path does not start with /, it is prefixed by the content of the LFC_HOME environment variable.

The lfc-mkdir command has the following options:

-m specifies the mode to be used. Default mode is 777.

-p creates all the non-existing parent directories first. The mode set for the created intermediate directories is the logical difference between 0777 and the user umask but
at least 0300.

EXIT STATUS
This program returns 0 if the operation was successful or >0 if the operation failed.

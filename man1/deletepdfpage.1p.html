NAME

deletepdfpage - Remove one or more pages from a PDF

SYNOPSIS

 deletepdfpage [options] infile.pdf <pagenums> [outfile.pdf]
 Options:
   -o --order          preserve the internal PDF ordering for output
   -v --verbose        print diagnostic messages
   -h --help           verbose help message
   -V --version        print CAM::PDF version
 <pagenums> is a comma-separated list of page numbers.
      Ranges like '2-6' allowed in the list
      Example: 4-6,2,12,8-9

DESCRIPTION

Remove the specified pages from a PDF document. This may fail for very complex, annotated PDF files, for example ones that Adobe Illustrator emits.

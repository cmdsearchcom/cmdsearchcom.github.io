NAME
isdbdata - Check whether a supplied string is a known data type/identifier name

SYNOPSIS
isdbdata [-term string] [-termout toggle] -outfile outfile

isdbdata -help

OPTIONS
Input section
-term string

Required section
Additional section
-termout toggle
This option will write to file the term statement from the EDAM ontology with a name that matches the supplied string. Only the EDAM ’data’ branch (data type or identifier)
is checked. Default value: N

Output section
-outfile outfile

TSSNVREADLOCK(1) User Commands TSSNVREADLOCK(1)

NAME

tssnvreadlock - manual page for tssnvreadlock v713

DESCRIPTION

nvreadlock
Runs TPM2_NV_ReadLock
[-hia hierarchy authorization (o, p)(default index authorization)] -ha NV index handle -pwdn password for NV index (default empty)
-se[0-2] session handle / attributes (default PWAP)
01 continue

SEE ALSO

The full documentation for tssnvreadlock is maintained as a Texinfo manual. If the info and tssnvreadlock programs are properly installed at your site, the command
info tssnvreadlock
should give you access to the complete manual.

NAME
lottanzb - Usenet Downloader

SYNOPSIS
lottanzb [options] [files]

DESCRIPTION
LottaNZB aims to simplify and automate the download of files from the Usenet. You can tell LottaNZB what to download using NZB files, which are created by many Usenet search
engines. LottaNZB integrates nicely with GNOME desktops, but is not limited to them and uses the mature SABnzbd software as its foundation.

OPTIONS
[files]
List of NZB files to add to the download queue.

--version
Show program’s version number and exit.

-h, --help
Show this help message and exit.

-d --debug
Show debug messages.

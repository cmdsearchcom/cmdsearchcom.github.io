GGOBI-CONFIG(1) General Commands Manual GGOBI-CONFIG(1)

NAME

ggobi-config - ggobi compile flags and library information retrieval

SYNOPSIS

ggobi-config [--cflags] [--libs] [--xml-support]  [--xml-libs] [--xml-path] [--xml] [--ld-library-path]

DESCRIPTION

ggobi-config is a tool to obtain compile flags and library information useful for building plugins for ggobi(1).

SEE ALSO

http://www.ggobi.org, ggobi(1)

GIT-COUNT(1) Git Extras GIT-COUNT(1)

NAME

git-count - Show commit count

SYNOPSIS

git-count [--all]

DESCRIPTION

Show commit count.

OPTIONS

--all
Show commit count details.

EXAMPLES

Output commit total: 

$ git count


total 1844

$ git count --all

mtx-youless(1) youless Fetcher mtx-youless(1)

NAME
mtx-youless

SYNOPSIS
mtxrun --script youless [ OPTIONS ... ] [ FILENAMES ]

DESCRIPTION
youless Fetcher

OPTIONS
--collect
collect data from device

--nobackup
don’t backup old datafile

--nofile
don’t write data to file (for checking)

--kwh summative kwh data

--watt collected watt data

--host ip address of device

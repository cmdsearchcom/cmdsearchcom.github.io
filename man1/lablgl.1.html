NAME
lablgl - OCaml toplevel with labgl support.

SYNOPSIS
lablgl [ocaml toplevel option...]

DESCRIPTION
This manual page documents briefly the lablgl command.

This manual page was written for the Debian GNU/Linux distribution because the original program does not have a manual page.

The lablgl commands launch an OCaml toplevel with lablgl or labglut support. lablglut provides the same functionality for lablglut.

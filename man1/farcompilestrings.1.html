NAME
farcompilestrings - store compiled FSTs in a finite-state archive

SYNOPSIS
farcompilestrings [OPTIONS] [in1.txt [[in2.txt ...] out.far]]

DESCRIPTION
Compiles a set of strings as FSTs and stores them in a finite-state archive.

OPTIONS
Boolean options accept either true or false as a value, or the implicit value true if neither value is specified.

--allow_negative_labels[=BOOLEAN]
Allow negative labels (not recommended; may cause conflicts). Default: false.

--arc_type=TYPE
Set the output arc type. Default: standard.

--entry_type=TYPE
Set the entry type, either file (one FST per file) or line (one FST per line). Default: line.

--far_type=TYPE
Set the FAR file format, one of default, fst, stlist, or sttable. Default: default.

--file_list_input[=BOOLEAN]
Each input file contains a list of files to be processed. Default: false.

--fst_align[=BOOLEAN]
Write FST data aligned where appropriate. Default: false.

--fst_compat_symbols[=BOOLEAN]
Require symbol tables to match when appropriate. Default: true.

--fst_default_cache_gc[=BOOLEAN]
Enable garbage collection of cache. Default: true.

--fst_default_cache_gc_limit=N
Set the cache byte size that triggers garbage collection. Default: 1048576.

--fst_error_fatal[=BOOLEAN]
If true, FST errors are fatal. Otherwise, returned objects are flagged as bad. For example, FSTs are returned with the kError property set to true, and FST weights set
so that Member() returns false. Default: true.

--fst_field_separator=STRING
Set the characters used as a separator between printed fields. Default: " ".

--fst_read_mode=STRING
Set the default file reading mode for mappable files, either "read" or "map". Default: "read".

--fst_type=TYPE
Set the output FST type. Default: vector.

--fst_verify_properties[=BOOLEAN]
Verify fst properties queried by TestProperties. Default: false.

--fst_weight_parentheses=STRING
Set the characters enclosing the first weight of a printed composite weight (e.g., pair weight, tuple weight, and derived classes) to ensure proper I/O of nested composite
weights. Must have size 0 (none) or 2 (open and close parenthesis). Default: no parentheses.

--fst_weight_separator=CHARACTER
Set the character separator between printed composite weights. Default: comma.

--generate_keys=N
Generate N digit numeric keys. Default: use file basenames.

--help[=BOOLEAN]
Show usage information. Default: false.

--helpshort[=BOOLEAN]
Show brief usage information. Default: false.

--initial-symbols[=BOOLEAN]
If --keep-symbols is true, store the symbol table for only the first FST in the archive. Default: true.

--keep-symbols[=BOOLEAN]
Store the symbol table in the FAR file. Default: false.

--key_prefix=STRING
Set the prefix to prepend to keys. Default: no prefix.

--key_suffix=STRING
Set the suffix to append to keys. Default: no suffix.

--save_relabel_ipairs=FILENAME
Save input relabel pairs to file. Default: do not save.

--save_relabel_opairs=FILENAME
Save output relabel pairs to file. Default: do not save.

--symbols=FILENAME
Read the label symbol table from a file. Default: do not read a table.

--tmpdir=PATHNAME
Set the temporary directory to use. Default: /tmp.

--token_type=TYPE
Set the token type, one of symbol, byte, or utf8. Default: symbol.

--unknown-symbol=STRING
String to substitute for an unknown symbol.

--v[=N]
Set the verbosity level. Default: 0.

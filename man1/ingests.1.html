NAME
ingests - Build an auxiliary file for multicat

SYNOPSIS
ingests -p <PCR PID> [-m <payload size>]>] <input ts>

DESCRIPTION
Ingests is a companion application designed to manipulate TS files. It reads the PCR values of the file, and builds the auxiliary file that is necessary for multicat. The combiâ
nation of ingests and multicat makes a simple and efficient TS file streamer.

OPTIONS
-h Show summary of options

-m <payloadsize>
Size of the payload

-p <PCR PID>
PCR PID to use

aggregartp(1). desaggregartp(1). multicat(1), offsets(1).

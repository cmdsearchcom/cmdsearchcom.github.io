NAME

gnome-specimen - Simple font preview and compare application for Gnome.

SYNOPSIS

gnome-specimen [OPTION...]

DESCRIPTION

gnome-specimen is a simple tool to view and compare fonts installed on your system. It shows list of all fonts available in the left pane and list of font previews on the right pane. You can also configure preview text and font size. You can also have configurable foreground and background colors used in the preview pane.

OPTIONS

-?, --help
Show this help message and exit.
--usage
Display brief usage message and exit.
--version
Show version of program and exit.

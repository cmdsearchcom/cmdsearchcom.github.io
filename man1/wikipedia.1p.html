NAME

wikipedia - lookup an entry in the wikipedia

SYNOPSIS

    % wikipedia [-l lang] perl

DESCRIPTION

wikipedia is a command line tool for looking up a topic in the wikipedia and printing it to STDOUT. Useful if you spend a lot of time in a shell and don't want to fire up a web browser to see what something is.

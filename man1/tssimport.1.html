NAME
tssimport - manual page for tssimport v713

DESCRIPTION
Import

Runs TPM2_Import

-hp parent handle

[-pwdp password for parent (default empty)] [-ik encryption key in file name] -ipu object public area file name -id duplicate file name -iss symmetric seed file name file
name [-salg symmetric algorithm (default none)]

-opr private area file name

-se[0-2] session handle / attributes (default PWAP)

01 continue 20 command decrypt 40 response encrypt

SEE ALSO
The full documentation for tssimport is maintained as a Texinfo manual. If the info and tssimport programs are properly installed at your site, the command

info tssimport

should give you access to the complete manual.

NAME
desktop2menu - create a menu file skeleton from a desktop file

SYNOPSIS
desktop2menu --help|--version

desktop2menu desktop file [package name]

DESCRIPTION
desktop2menu generates a skeleton menu file from the supplied freedesktop.org desktop file.

The package name to be used in the menu file may be passed as an additional argument. If it is not supplied then desktop2menu will attempt to derive the package name from the
data in the desktop file.

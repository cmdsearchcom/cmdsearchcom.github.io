NAME

fnfx-client - A client for customize fnfx hot-keys
 
 

DESCRIPTION


If you only want to use the internal commands (mute, volume control, changing LCD bright-ness...), there is no need to run the client fnfx. On the other side, if you really want to map the Fn-x combinations/hot-keys to launch programs, you will need to configure and start both - the server and client.
Fnfx does not need any extra parameters, you need only to copy the example file to ~/.fnfxrc and customize it.
 
This manual page documents briefly the fnfx-client commands.
 
This manual page was written for the Debian distribution because the original program does not have a manual page. See fnfx-client README for more information.

OPTIONS

-h
Show summary of options.
-d
Enable debug output.
-n
Do not send into daemon mode.

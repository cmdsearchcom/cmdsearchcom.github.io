NAME
hsqldb-databasemanager - GUI database management tool

SYNOPSIS
hsqldb-databasemanager [options]

DESCRIPTION
hsqldb-databasemanager is a GUI tool to manage databases via their JDBC interface.

OPTIONS
-driver classname
JDBC driver class to use

-url url
JDBC URL to connect to

-user name
user name used for connection

-password password
password for this user

-dir path
default directory

-script file
reads from script file

-noexit
do not call system.exit()

-? show a brief help

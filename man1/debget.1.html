NAME

debget - Fetch a .deb for a package in APT's database

SYNOPSIS

debget package [package ...]

DESCRIPTION

debget fetches a .deb for one or more packages from an Debian mirror. It uses apt-get(8) to find out the URL so it will download the same version from the same server as apt-get(8) would do. The .deb will be downloaded with curl(1) and stored in your current working directory.

KNOWN BUGS

debget will not work for packages that are available in apt-get(8)'s cache. If you need that functionality please use the download option in aptitude(8).

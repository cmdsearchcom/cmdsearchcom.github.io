NAME

wxcas - wx c aMule statistics

SYNOPSIS

wxcas

DESCRIPTION

wxcas is a program for displaying the contents of your online signature file in a nice wx Window on your Desktop. For this to work, you must enable the "Online Signature" option in aMule's preferences.
 
Based on Pedro de Oliveira's cas(1). wxcas was written by ThePolish <thepolish@vipmail.ru>
 
This app doesn't take any arguments.

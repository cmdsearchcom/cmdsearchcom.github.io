NAME
sort-pictures - sort pictures according to exif date

SYNOPSIS
sort-pictures

DESCRIPTION
Sort-pictures uses the exif tags in jpeg pictures recovered by recoverjpeg in directories. It creates hard links to organize sorted pictures.

Running sort-pictures will scan the current directory for files named after the template image?????*.jpg (image followed by at least five characters followed by .jpg). A new

NAME
addcr - inserts a CR ( 15) before each LF ( 12)

SYNOPSIS
addcr

DESCRIPTION
addcr inserts a CR ( 15) before each LF ( 12), i.e., at the end of each line of input. It does not insert a CR at the end of a partial final line.

addcr reads input from stdin and writes to stdout.

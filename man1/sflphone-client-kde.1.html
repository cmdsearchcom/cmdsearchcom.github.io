NAME

sflphone-client-kde - SIP and IAX2 compatible voice over IP softphone client for KDE.

SYNOPSIS

sflphone-client-kde

DESCRIPTION

SFLphone is meant to be a robust enterprise-class desktop phone. It provides functions like call transfer, call hold, multiple lines, multiple accounts support. SFLphone audio layer is built upon a native ALSA interface and a native PulseAudio interface. sflphone-client-kde is a KDE client for SFLphone; it communicates with the core side through D-Bus.

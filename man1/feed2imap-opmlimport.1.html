NAME
feed2imap-opmlimport - Convert an OPML subscription list to a feed2imap config file

SYNOPSIS
feed2imap-opmlimport

DESCRIPTION
feed2imap-opmlimport reads an OPML subscription list on standard input and outputs a feed2imap configuration file on standard output. The resulting configuration file will
require some tweaking.

BUGS
Should probably accept parameters to be able to change default values.

Homepage : http://home.gna.org/feed2imap/

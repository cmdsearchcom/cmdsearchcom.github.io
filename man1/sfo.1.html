NAME

sfo - Simple Fuzzer Oracle (process monitoring)

SYNOPSIS

sfo PROGRAM [ARGS]...

DESCRIPTION

sfo is a simple process monitor with some reporting capability which will indicate that the program under test has failed, along with some basic logging to record the failure type.

USAGE

sfo will run an application specified on its command line with optional arguments, and will report the application's subsequent behavior if it crashes.

NOTES

sfo is not currently complete. It is provided as a preview.

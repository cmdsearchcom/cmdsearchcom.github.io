NAME
MBK_VDD - define the high level power name pattern

SYNOPSIS
c-shell running
setenv MBK_VDD powername

DESCRIPTION
MBK_VDD sets the pattern to be matched in a name to indicate a power supply for the tools based upon mbk. Its default value is vdd. Therefore all names of the form ‘*vdd*’
indicates a power supply.

EXAMPLE
setenv MBK_VDD vcc

SEE ALSO
mbk(3), genlib(1), isvdd(3), isvss(3), MBK_VSS(1).

NAME
idjc-auto - Make a profile be automatic

SYNOPSIS
idjc auto [-h] profile_name

DESCRIPTION
When a profile is made automatic the Profile Manager window will not appear at startup and instead the automatic profile will load. Only one profile can be automatic at any one
time.

Options
-h, --help
Show a help message.

Requirements
profile_name
The name of a profile to make automatic.

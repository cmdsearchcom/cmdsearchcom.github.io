NAME
stemwords - Snowball word stemming utility

DESCRIPTION
stemwords [-l language] [-i input file] [-o output file] [-c character encoding] [-p[2]] [-h]

The input file consists of a list of words to be stemmed, one per line. Words should be in lower case, but (for English) A-Z letters are mapped to their a-z equivalents anyway.
If omitted, stdin is used.

If -c is given, the argument is the character encoding of the input and output files. If it is omitted, the UTF-8 encoding is used.

If -p is given the output file consists of each word of the input file followed by "->" followed by its stemmed equivalent. If -p2 is given the output file is a two column layâ
out containing the input words in the first column and the stemmed equivalents in the second column. Otherwise, the output file consists of the stemmed words, one per line.

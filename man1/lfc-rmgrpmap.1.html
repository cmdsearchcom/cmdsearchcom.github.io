NAME
lfc-rmgrpmap - suppress group entry corresponding to a given virtual gid or group name

SYNOPSIS
lfc-rmgrpmap --gid gid --group groupname

DESCRIPTION
lfc-rmgrpmap suppresses the group entry corresponding to a given virtual gid or group name. If both are specified, they must point at the same entry.

This command requires ADMIN privilege.

OPTIONS
--gid gid
specifies the Virtual Group Id.

--group groupname
specifies the new group name. It must be at most 255 characters long.

EXIT STATUS
This program returns 0 if the operation was successful or >0 if the operation failed.

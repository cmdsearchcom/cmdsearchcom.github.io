NAME
bundle-binstubs - Install the binstubs of the listed gems

SYNOPSIS
bundle binstubs GEM_NAME [--force] [--path PATH] [--standalone]

DESCRIPTION
This command generates binstubs for executables in GEM_NAME. Binstubs are put into bin, or the --path directory if one has been set. Calling binstubs with [GEM [GEM]] will create
binstubs for all given gems.

OPTIONS
--force
Overwrite existing binstubs if they exist.

--path The location to install the specified binstubs to. This defaults to bin.

--standalone
Makes binstubs that can work without depending on Rubygems or Bundler at runtime.

BUNDLE INSTALL --BINSTUBS
To create binstubs for all the gems in the bundle you can use the --binstubs flag in bundle install(1) bundle-install.1.html.

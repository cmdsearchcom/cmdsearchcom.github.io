NAME
update_routes - update the database of BBS and callsigns

SYNOPSIS
update_routes <BBS_callsign> [<BBS_callsign> ...]

DESCRIPTION
This program scans all messages downloaded from the specified BBS and updates the home BBS information (builds the /var/ax25/mail_routes database).

FILES
/var/ax25/mail_routes

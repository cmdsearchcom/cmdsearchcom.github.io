NAME
winswitch_applet - Window Switch applet

SYNOPSIS
winswitch_applet [--debug-mode]

DESCRIPTION
winswitch_applet starts the Window Switch applet.

OPTIONS
--debug-mode
Enable debug mode

FILES
.winswitch/client/client.conf
The applet configuration file.

DIAGNOSTICS
A number of debugging messages will be printed on screen. More if debug mode is enabled.

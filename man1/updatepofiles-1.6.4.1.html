NAME

updatepofiles - The Lazarus resource builder.
 

SYNOPSIS

updatepofiles filename1.po [filename2.po ... filenameN.po]
 

DESCRIPTION

updatepofiles Update internationalization files.
 

OPTIONS

No option. If called without any argument it gives a short help message.

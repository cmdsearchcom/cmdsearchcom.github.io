NAME
cronometer - nutrient intake and exercise tracker

SYNOPSIS
cronometer

DESCRIPTION
cronometer is a nutrient intake and exercise tracker with information on over 8000 foods. Please see the extensive documentation bundled with cronometer through its help menu.

OPTIONS
cronometer has no command line options.

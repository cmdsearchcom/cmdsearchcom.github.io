NAME
karlyriceditor â graphical karaoke lyrics editor

SYNOPSIS
karlyriceditor

DESCRIPTION
This manual page documents briefly the karlyriceditor command. It was written for the Debian distribution because the original program does not have a manual page.

karlyriceditor is a graphical karaoke lyrics editor. It helps you edit and synchronize lyrics with karaoke songs. It supports importing and exporting them from and to various
standard formats as well as rendering of karaoke videos.

The command karlyriceditor doesn’t accept any options and starts up the graphical user interface.

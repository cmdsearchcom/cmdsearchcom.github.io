NAME
oc adm config - Change configuration files for the client

SYNOPSIS
oc adm config [OPTIONS]

DESCRIPTION
Manage the client config files

The client stores configuration in the current user’s home directory (under the .kube directory as config). When you login the first time, a new config file is created, and subâ
sequent project changes with the ’project’ command will set the current context. These subcommands allow you to manage the config directly.

EXAMPLE
# Change the config context to use
oc adm config use-context my-context

# Set the value of a config preference
oc adm config set preferences.some true

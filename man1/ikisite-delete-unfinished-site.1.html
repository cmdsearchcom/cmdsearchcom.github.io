NAME
ikisite-delete-unfinished-site - ikiwiki-hosting helper

SYNOPSIS
ikisite-delete-unfinished-site username|--all [--dry-run]

DESCRIPTION
This helper script is run by by cron daily, and deletes sites that someone started to create, but never finished setting up.

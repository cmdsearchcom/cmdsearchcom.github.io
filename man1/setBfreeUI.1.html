NAME
setBfree - DSP tonewheel organ Graphical User Interface

SYNOPSIS
setBfreeUI

DESCRIPTION
setBfree - DSP tonewheel organ Graphical User Interface

setBfree is a MIDI-controlled, software synthesizer designed to imitate the sound and properties of the electromechanical organs and sound modification devices that brought
world-wide fame to the names and products of Laurens Hammond and Don Leslie.

Thes GUI takes no commandline arguments. see setBfree(1) for the commandline option and complete documentation.

CONFIGURATOIN FILES
setBfreeUI reads $XDG_CONFIG_HOME/setBfree/default.cfg and $XDG_CONFIG_HOME/setBfree/default.pgm on startup. see setBfree(1) for details.

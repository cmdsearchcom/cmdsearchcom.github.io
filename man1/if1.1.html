NAME
if1 - Single IF (Image File) Transformation

SYNOPSIS
if1 iffile outfile [OPTIONS]

DESCRIPTION
if1 transforms a single IF file.

OPTIONS
if1 accepts the following options:

--invert Inverts (negates) the image.

--log Natural logrithm of image.

--exp Natural exponential of image.

--debug Set debug mode.

--verbose Set verbose mode.

--version Print version.

--help Print terse help summary.

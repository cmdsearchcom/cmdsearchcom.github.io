NAME
gsec -- deprecated Firebird user management utility

SYNOPSIS
gsec [option...] command [parameter...]

DESCRIPTION
gsec used to be the utility to manage Firebird users.

Since Firebird 3.0, user management is provided via SQL. Please see sql.extensions/README.user_management in the documentation directory of your installation.

SEE ALSO
Additional documentation about Firebird can be found on the Firebird Project website, <http://www.firebirdsql.org>.

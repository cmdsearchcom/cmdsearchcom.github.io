NAME

ppit - pit for perl

SYNOPSIS

    $ ppit [get|set|switch] <value>
    $ ppit help
 Options:
        get <name>        => Get values of <name>.
        set <name>        => Config values of name with $EDITOR.
        switch <profile>  => Switch profile to <profile>.
        help              => show this manual

DESCRIPTION

pit command for Perl

OPTIONS
-a, --action
Read-only operations for Linode account information: show.

--api-key
API key to use when communicating with the Linode API. The API key can also be set using an environment variable (LINODE_API_KEY). Alternatively, you can specify the API
key in a .linodecli file in the working user’s home directory, using the format ‘api-key foobar‘.

-u, --username:
Optional. Allows users to specify the username, if using with multiple accounts and configuration files.

-j, --json:
Optional. JSON output.

-h, --help
Displays help documentation.

ACTIONS
SHOW

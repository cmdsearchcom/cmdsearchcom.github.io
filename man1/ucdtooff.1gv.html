NAME

ucdtooff - filter to convert an AVS 'ucd' (Unstructured Cell Data) format file into an OFF file.

SYNOPSIS

ucdtooff

DESCRIPTION

Ucdtooff reads from standard input and outputs the equivalent OFF stream to standard out.

anytoucd(1)

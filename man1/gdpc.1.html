NAME
gdpc - program for visualising molecular dynamic simulations

DESCRIPTION
This manual page documents briefly the gdpc command.

gdpc is a program, written in C and utilizing GTK, for visualising molecular dynamic simulations. It is a very versatile program and could easily be used for other purposes. gdpc
reads xyz input and custom formats and can write out pictures of the frames.

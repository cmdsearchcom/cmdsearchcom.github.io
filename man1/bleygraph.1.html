NAME

bleygraph - stats plotter for bley

SYNOPSIS

bleygraph [ options]

OPTIONS

--version
show program's version number and exit
-h, --help
show this help message and exit
-d DESTDIR, --destdir=DESTDIR
write to DESTDIR
-c CONFFILE, --config=CONFFILE
load configuration from CONFFILE
-q, --quiet
be quiet (no output)

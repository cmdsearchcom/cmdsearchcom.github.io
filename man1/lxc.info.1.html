NAME

lxc info - Show container or server information

SYNOPSIS

lxc info [<remote>:][<container>] [--show-log]

DESCRIPTION

Show container or server information.
lxc info [<remote>:]<container> [--show-log]
For container information.
lxc info [<remote>:]
For LXD server information.

OPTIONS

--debug
(= false)
Enable debug mode
--force-local
(= false)
Force using the local unix socket
--no-alias
(= false)
Ignore aliases when determining what command to run
--show-log
(= false)
Show the container's last 100 log lines?
--verbose
(= false)
Enable verbose mode

NAME

botch-convert-arch - convert the architecture of a Packages file

SYNOPSIS

botch-convert-arch [options] fromarch toarch inPackages outPackages

DESCRIPTION

convert architecture of a Packages file from fromarch to toarch

OPTIONS

-h, --help
show help
-v, --verbose
be verbose

NAME
maptransfer-server - allows easy up- and downloading of Maps for a game server.

SYNOPSIS
maptransfer [<config file>]

DESCRIPTION
Uploading Maps to a Game server can be a tedious process if you want to get the post-upload processing right. MapTransfer tries to automate this process as much as possible.

This program is the MapTransfer server.

OPTIONS
You can give the path to the configuration file as the first command line argument. If omitted, server.cfg is assumed.

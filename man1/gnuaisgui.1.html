NAME
gnuaisgui - GUI for gnuais

SYNOPSIS
gnuaisgui

DESCRIPTION
This manual page documents briefly the gnuaisgui program.

gnuais is a tool for demodulating and decoding AIS messages using the line/mic input of the sound card. This GUI plots the vessels on a map from OpenStreetMap. gnuais MUST be
started before gnuaisgui is started.

OPTIONS
There are no command line options.

NAME
why3-cpulimit - run a command with CPU and virtual memory limits

SYNOPSIS
why3-cpulimit TIME MEMORY -h|-s command args...

DESCRIPTION
Run a command with CPU and virtual memory limits. One of -h or -s must be specified. TIME specifies the maximum runtime in seconds. MEMORY specifies the maximum virtual memory
use in megabytes. In both cases, a limit of zero means that no limit should be set; i.e., the existing system limit is in effect.

OPTIONS
-h Hide the CPU time.

-s Show the CPU time.

NAME

gqrx - Software Defined Radio graphical user interface application.

DESCRIPTION

AM, FM and SSB receiver powered by GNU Radio and Qt
Waterfall spectrum display and user interface elements from Moe Wheatley's Cutesdr program, combined with a gnuradio signal processing engine and hardware support for the FUNcube Dongle receiver hardware.

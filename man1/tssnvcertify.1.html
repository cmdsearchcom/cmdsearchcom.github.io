NAME

tssnvcertify - manual page for tssnvcertify v713

DESCRIPTION

nvcertify
Runs TPM2_NV_Certify
-ha NV index handle
[-pwdn password for NV index (default empty)] -hk certifying key handle [-pwdk password for key (default empty)] [-halg [sha1, sha256, sha384] (default sha256)] [-salg signature algorithm [rsa, ecc] (default rsa)] -sz data size [-off offset (default 0)] [-os signature file name (default none)] [-oa attestation output file name]
-se[0-2] session handle / attributes (default PWAP)
01 continue

SEE ALSO

The full documentation for tssnvcertify is maintained as a Texinfo manual. If the info and tssnvcertify programs are properly installed at your site, the command
info tssnvcertify
should give you access to the complete manual.

NAME
zfr - apply Frobenius automorphism to a matrix

SYNOPSIS
zfr [OPTIONS] <Matrix> <Result>

DESCRIPTION
This program reads a matrix, applies the Frobenius automorphism x|->x^p, where p is the characteristic of the field, to each entry and writes out the result.

OPTIONS
-Q Quiet, no messages.

-V Verbose, more messages.

-T <MaxTime>
Set CPU time limit

INPUT FILES
Matrix The matrix.

OUTPUT FILES
Result The transformed matrix.

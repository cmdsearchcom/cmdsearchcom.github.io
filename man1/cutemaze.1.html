NAME
cutemaze - A single player word finding game based on Boggle

DESCRIPTION
Tanglet is a single player word finding game based on Boggle. The object of the game is to list as many words as you can before the time runs out. There are several timer modes
that determine how much time you start with, and if you get extra time when you find a word.

CONFIGURATION
Basic configuration file is located in ~/.GottCode/Tanglet.conf

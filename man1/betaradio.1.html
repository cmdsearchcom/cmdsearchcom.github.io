NAME
betaradio - An easy way to listen to internet radio of Taiwan.

SYNOPSIS
betaradio

DESCRIPTION
With this client you can listen to internet radio of Taiwan without opening a browser. It displays a icon on system tray that you can click and select the channel you want to
listen to.

NAME
ttembed - remove embedding restrictions from TrueType files

SYNOPSIS
ttembed [-n] fontfile

DESCRIPTION
ttembed sets the fsType field of the OS/2 table to zero and recalculates the OS/2 table’s checksum.

OPTIONS
-n Dry-run mode, which just displays the fsType field and filename of the given fonts if it is non-zero (i.e. if ttembed would change it.

NAME

wsdlpull-schema - C++ Web Services Schema Parser

SYNOPSIS

#include <schemaparser/...>

DESCRIPTION

WSDLPull includes a schema parser. That schema parser can be used in isolation to parse XSD schemas. The schema parser can validate an instance document of the XSD schema just parsed.
The library covers the following areas,
WSDL (Web Services Description Language) XSD Schema

NAME
checktargetsortedness - check BAM file for sortedness

SYNOPSIS
checkTargetSortedness infile.bam

OPTIONS
’infile.bam’ is the name of the BAM file to check for sortedness. The program will output a short message describing the sortedness status of the file.

NAME

ucc - UseCode Compiler for Exult
 

SYNOPSIS

ucc [FILE]
 

DESCRIPTION

Games supported by Exult ( Ultima 7, for example) consist in large part of Usecode. exult(1) basically implements a virtual machine the machine language of which is Usecode.
 
ucc is a compiler that targets this machine language.

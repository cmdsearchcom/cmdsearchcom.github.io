NAME
smascii - smascii

SYNOPSIS
smascii smaky_file converted_file [--unix]

DESCRIPTION
Tool to convert Smaky text file to Extended ASCII (ISO-8859-1).

smaky_file
the smaky text file

converted_file
the file converted

--unix the Carriage Return (Old Mac) will be converted to Line Feed (unix)

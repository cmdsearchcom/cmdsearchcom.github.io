NAME
imclean - mail/news garbage cleanup

SYNOPSIS
imclean [OPTIONS] [FOLDER] [MSGS...]

DESCRIPTION
The imclean command delete all mail/news messages from +trash folder.

If you do not want to delete all messages, specify message numbers.

This command is provided by IM (Internet Message).

OPTIONS
-s, --src=FOLDER
Set a folder to be cleaned up. "--src=+xxx" is equivalent to "+xxx". Default value is "+trash".

-S, --sshserver=SERVER
SSH port relay server.

-n, --noharm={on,off}
Do not delete files, show what will be performed.

-q, --quiet={on,off}
Do not show any messages.

-v, --verbose={on,off}
Print verbose messages when running.

--debug=DEBUG_OPTION
Print debug messages when running.

-h, --help
Display help message and exit.

--version
Output version information and exit.

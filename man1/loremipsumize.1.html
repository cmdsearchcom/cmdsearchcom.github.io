NAME
loremipsumize - Lorem-ipsumize a LyX document

SYNOPSIS
loremipsumize [infile] [outfile]

DESCRIPTION
This script is a helper tool intended to let you replace the actual content of your Lyx file to meaningless blind text (lorem ipsum) so you can send it in along with your bugreâ
ports about eLyXer. It will replace all alphanumeric texts longer than two words with "lorem ipsum" and leave the symbols. It is not 100% bulletproof so please review your docuâ
ments before sending for any confidential information left.

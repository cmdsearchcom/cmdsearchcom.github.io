NAME
tsspcrread - manual page for tsspcrread v713

DESCRIPTION
pcrread

Runs TPM2_PCR_Read

-ha pcr handle

-halg [sha1, sha256, sha384] (default sha256)

[-of data file]

SEE ALSO
The full documentation for tsspcrread is maintained as a Texinfo manual. If the info and tsspcrread programs are properly installed at your site, the command

info tsspcrread

should give you access to the complete manual.

NAME
gpk-dbus-service - GNOME PackageKit Session Service

SYNOPSIS
gpk-dbus-service [ --verbose ]

DESCRIPTION
This manual page documents briefly the gpk-dbus-service command.

gpk-dbus-service provides the PackageKit D-Bus session API in GNOME.

NAME

afex - client for anonymous FEX

SYNOPSIS

[args]

DESCRIPTION

afex is a client for anonymous FEX.
This manual page was written for the Debian distribution because the original program does not have a manual page.

USAGE

afex [ID] files...
file input
afex ID [-]
file output
 ... | afex [ID]
pipe input
afex ID | ...
pipe output
afex -ID
delete

NAME
gtg_new_task - Adds a task to the Getting Things GNOME! organizer

SYNOPSIS
gtg_new_task [-h | --help] [-i | --interactive]

DESCRIPTION
gtg_new_task creates a new task in the Getting Things GNOME! organizer for the GNOME desktop via the DBUS message bus. Getting Things GNOME! must be running for the command to
work.

Options
[-h | --help]
Shows a brief usage help.

[-i | --interactive]
Accepts a task description via stdin.

FILES
None

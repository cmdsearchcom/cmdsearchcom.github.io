NAME
trash-restore - Restore for Command line trash utility.

SYNOPSIS
trash-restore

DESCRIPTION
Use for restore a trashed file or directory, in the original path. This command is a part of trash-cli package that provides a command line interface trashcan utility compliant
with the FreeDesktop.org Trash Specification. It remembers the name, original path, deletion date, and permissions of each trashed file.

EXAMPLES
$ trash-restore
0 2007-08-30 12:36:00 /home/andrea/foo
1 2007-08-30 12:39:41 /home/andrea/bar
2 2007-08-30 12:39:41 /home/andrea/bar2
3 2007-08-30 12:39:41 /home/andrea/foo2
4 2007-08-30 12:39:41 /home/andrea/foo
What file to restore [0..4]: 4
$ ls foo
foo

NAME
botan-config-1.10 - Botan Configuration Query program

SYNOPSIS
botan-config-1.10 [--prefix[=DIR]] [--version] [--libs] [--cflags]

DESCRIPTION
base-config is used to access the configuration information of Botan.

HOMEPAGE
More information about Botan Project can be found at <http://botan.randombit.net/>.

UL(1) General Commands Manual UL(1)

NAME

UL - Molecular Simulator

SYNOPSIS

UL

DESCRIPTION

This manual page documents briefly the UL command.
UL is a new extendible molecular simulation program that also includes data management and analysis capabilities.

AdunCore(1), AdunServer(1), GNUstep(7), defaults(1), BioCocoa(1).

CONGRUITY(1) User Commands CONGRUITY(1)

NAME

congruity - A simple GUI to program Logitech(R) Harmony(R) remotes

SYNOPSIS

congruity FILE

DESCRIPTION

congruity is intended to process files provided by the Logitech(R) Harmony(R) configuration website, currently members.harmonyremote.com.
You should configure your web browser to execute congruity whenever a file of type EZHex, EZUp, or EZTut is downloaded.

AUTHOR

Written by Stephen Warren.

REPORTING BUGS

Report bugs to <concordance-devel@lists.sourceforge.net>.
congruity utilitizes libconcord. Until congruity grows its own sourceforge project, it piggy-backs off the concordance/libconcord mailing list.

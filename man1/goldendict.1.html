NAME
goldendict - a feature-rich dictionary lookup program

SYNOPSIS
goldendict

DESCRIPTION
Goldendict is a cross-platform and international dictionary written in QT.

Features
Â· Use of WebKit for an accurate articles’ representation, complete with all formatting, colors, images and links.

Â· Support of multiple dictionary file formats, namely:

Â· Babylon .BGL files, complete with images and resources

Â· StarDict .ifo/.dict./.idx/.syn dictionaries

Â· Dictd .index/.dict(.dz) dictionary files

Â· ABBYY Lingvo .dsl source files, together with abbreviations. The files can be optionally compressed with dictzip. Dictionary res-urces can be packed together into a .zip
file.

Â· ABBYY Lingvo .lsa/.dat audio archives. Those can be indexed separately, -r be referred to from .dsl files.

Â· Support for Wikipedia, Wiktionary, or any other MediaWiki-based sites to perform lookups in.

Â· Ability to use arbitrary websites as dictionaries via templated Url patterns.

Â· Hunspell-based morphology system, used for word stemming and spelling suggestions.

Â· Ability to index arbitrary directories with audio files for pronunciation lookups.

Â· Full Unicode case, diacritics, punctuation and whitespace folding. This means the ability to type in words without any accents, correct case, punctuation or spaces (e.g.
typing ’Grussen’ would yield ’gruessen’ in German dictionaries).

Â· Scan popup functionality. A small window pops up with the translation of a word chosen from another application.

Â· Support for global hotkeys. You can spawn the program window at any point, or directly translate a word from the clipboard.

Â· Tabbed browsing in a modern Qt 4 interface.

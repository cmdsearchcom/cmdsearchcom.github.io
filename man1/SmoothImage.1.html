SMOOTHIMAGE(1) User Commands SMOOTHIMAGE(1)

NAME

SmoothImage - part of ANTS registration suite

DESCRIPTION

Usage: SmoothImage ImageDimension image.ext smoothingsigma outimage.ext {sigma-is-in-spacing-coordinates-0/1} {medianfilter-0/1}
if median, then sigma means radius of filtering A separate sigma can be specified for each dimension, e.g., 1.5x1x2

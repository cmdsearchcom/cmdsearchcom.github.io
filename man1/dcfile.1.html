DCFILE(1) DICOM PS3 - Describe DICOM file content DCFILE(1)

NAME

dcfile - ACR/NEMA DICOM PS3 ... DICOM PS3 - Describe DICOM file content

SYNOPSIS

dcfile
 
See the file ../dicom3tools/gen.so.
 
[ -b|brief ] [ -filename ]
 
See the file ../dicom3tools/optin.so.
 

DESCRIPTION

dcfile reads the named dicom or acr-nema input file and describes the encoding method of the file, ie. what transfer syntax is in use, and whether or not a Part 10 style meta-information header is present.

OPTIONS

The description goes to standard error.
The basic input switches are described in dcintro(1). Options specific to this program are:
-b|brief
Display a single line brief summary.
-filename
Show the name of the file supplied in the arguments; a hyphen will be reported if no filename was supplied.

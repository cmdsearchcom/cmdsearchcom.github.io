NAME
tunes2pod - convert an iTunesDB into GNUtunesDB’s XML

SYNOPSIS
tunes2pod [OPTION]...

DESCRIPTION
Convert an iTunesDB into XML (aka GNUtunesDB.xml)

--help display this help and exit

--version
output version information and exit

-m, --mount=DIRECTORY
iPod mountpoint. Default is $IPOD_MOUNTPOINT

--force
Disable ’sync’ checking. tunes2pod may do stupid things :)

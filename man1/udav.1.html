NAME

udav - program for data visualization based on MathGL library

SYNOPSIS

udav [scriptfile]

DESCRIPTION

udav reads MGL scripts from scriptfile to produce plots of specified functions or data. The program will create a GUI window which show the script result and allow to edit the script itself.

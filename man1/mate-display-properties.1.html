NAME
mate-display-properties - customize MATE display behavor

SYNOPSIS
mate-display-properties [OPTIONS...]

DESCRIPTION
This executable is part of the package ’mate-control-center’: The MATE Control Center.

The MATE display preference tool enables you to customize the screen resolution and refresh rate for your desktop.

OPTIONS
This program accepts the standard MATE and GTK options.

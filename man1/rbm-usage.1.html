NAME

rbm-usage - Open manual page for command

SYNOPSIS

rbm usage <command>
rbm --help <command>
rbm <command> --help

DESCRIPTION

Open manual page for the selected command.

EXAMPLES

$ rbm usage rpm
 => man rbm-rpm
$ rbm deb --help
 => man rbm-deb
$ rbm --help tar
 => man rbm-tar

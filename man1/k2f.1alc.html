NAME

k2f, FSM translator ALLIANCE format from/to Berkeley format
 
 
See the file man1/alc_origin.1alc.
 

SYNOPSIS

k2f In_format Out_format Input_name [ Output_name ]
 

DESCRIPTION

k2f is a FSM translator from ALLIANCE format (".fsm") to Berkeley format ( ".kiss2").

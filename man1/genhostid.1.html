NAME
genhostid - generate and set a hostid for the current host

SYNOPSIS
genhostid

DESCRIPTION
genhostid generates a random hostid and stores it in /etc/hostid, if /etc/hostid does not already exist.

SEE ALSO
hostid(1), gethostid(2), sethostid(2)

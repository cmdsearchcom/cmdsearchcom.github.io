NAME
keyboardcast - keyboard multiplexer

SYNOPSIS
keyboardcast

DESCRIPTION
This manual page documents briefly the keyboardcast command.

keyboardcast is a program that can send keystrokes to multiple windows simultaneously. It can also start gnome-terminal processes, for instance to log in to several machines
simultaneously via ssh

OPTIONS
keyboardcast takes no options

JHOVE(1) General Commands Manual JHOVE(1)

NAME

JHOVE - identify, validate, and characterize digital objects.

SYNOPSIS

jhove [options] files...
 
jhoveview

DESCRIPTION

This manual page documents briefly the jhove and jhoveview commands.
jhoveview has a graphical user interface and no significant command line options. jhove has a command line interface, see full documentation in online at http://hul.harvard.edu/jhove/using.html#invocation.

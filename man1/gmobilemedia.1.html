NAME

gmobilemedia - program to manipulate filesystem on mobile phone

SYNOPSIS

gmobilemedia

DESCRIPTION

This manual page documents briefly the gmobilemedia command.
gmobilemedia is a program that allows you to browse filesystem in your cell phone.

OPTIONS

This program does not accept any command line parameters.

NAME
fvstereodecoder - FireVision stereo image decoder

SYNOPSIS
fvstereodecoder directory

DESCRIPTION
This program reads in all FireVision raw files of the specified directory. It assumes Bumblebee stereo camera images in these files and converts them to YUV422 planar color
images. These are then written side by side into a JPEG file.

EXAMPLES
fvstereodecoder imagedir
Decode all raw images in imagedir.

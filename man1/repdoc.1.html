NAME
repdoc - strip doc-strings from C source.

SYNOPSIS
repdoc [ doc-file [src-files...] ]

DESCRIPTION
repdoc strip doc-strings from C source.

OPTIONS
doc-file
generate a doc-file from c source files.

The programs are documented fully by John Harper available via the Info system.

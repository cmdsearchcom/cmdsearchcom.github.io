NAME
sxvol-rename - rename an SX volume

SYNOPSIS
sxvol rename [OPTIONS] sx://[profile@]cluster/OLDNAME NEWNAME

DESCRIPTION
Change the name of an existing volume. Only cluster administrators can perform this operation.

OPTIONS
-h, --help
Print help and exit

--full-help
Print help, including hidden options, and exit

-V, --version
Print version and exit

-D, --debug
Enable debug messages

-c, --config-dir=PATH
Path to the SX configuration directory (default: ~/.sx)

-f, --filter-dir=PATH
Path to the SX filter directory (default: /usr/lib/sxclient)

EXAMPLES
To rename the volume ’foo’ to ’bar’ run:
sxvol rename sx://admin@cluster/foo bar

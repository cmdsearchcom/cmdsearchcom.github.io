NAME
xfce4-about - About and Credits dialog for Xfce

DESCRIPTION
Usage:
xfce4-about [OPTION...]

Help Options:
-h, --help
Show help options

--help-all
Show all help options

--help-gtk
Show GTK+ Options

Application Options:
-V, --version
Version information

--display=DISPLAY
X display to use

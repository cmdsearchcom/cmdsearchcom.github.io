NAME

pytrademgen - Python wrapper around the C++ Simulated Travel Demand Generation Library

SYNOPSIS

pytrademgen [--prefix] [-v|--version] [-h|--help] [-b|--builtin] [-i|--input <path-to-input>] [-l|--log <path-to-output-log-file>]

DESCRIPTION

pytrademgen is a small Python wrapper around the TraDemGen library.
pytrademgen accepts the following options:
-h, --help
 

Produce that message and show usage.
-l, --long
 

Produce long formatted output (rather than the shorter one).
See the output of the `pytrademgen --help' command for default options.

SEE ALSO

trademgen(1), trademgen-config(1), trademgen-library(3)

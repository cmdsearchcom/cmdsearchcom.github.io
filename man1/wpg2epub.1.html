NAME
wpg2epub - manual page for wpg2epub 0.9.5

SYNOPSIS
wpg2epub [OPTIONS] INPUT OUTPUT

DESCRIPTION
‘wpg2epub’ converts WordPerfect Graphics images to ePub.

OPTIONS
--help show this help message

--version
print version and exit

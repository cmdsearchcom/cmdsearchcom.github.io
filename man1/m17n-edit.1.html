NAME
m17n-edit - edit multilingual text

SYNOPSIS
m17n-edit [ XT-OPTION ...] [ OPTION ... ] FILE

DESCRIPTION
Display FILE on a window and allow users to edit it.

XT-OPTIONs are standard Xt arguments (e.g. -fn, -fg).

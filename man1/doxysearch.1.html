NAME

doxysearch.cgi - search engine used for searching in doxygen documentation.

SYNOPSIS

doxyindexer.cgi

DESCRIPTION

CGI binary that is used by doxygen generated HTML output to search for words. The tool uses the search index called doxysearch.db produced by doxyindexer.

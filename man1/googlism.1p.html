NAME
googlism - Search Googlism.com

SYNOPSIS
googlism query [ type ]

DESCRIPTION
A utility to search http://googlism.com/. Type "who" is assumed if it is not specified.

SEE ALSO
WWW::Search, WWW::Search::Googlism

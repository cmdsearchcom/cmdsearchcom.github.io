NAME

perldepgentest - Check that rpmbuild generates dependencies correctly

SYNOPSIS

perldepgentest [DIRECTORY]

DESCRIPTION

This is a regression test suite for Perl RPM dependency generator.
The only optional argument overrides directory with testing data.
See <http://permalink.gmane.org/gmane.linux.redhat.fedora.devel/187548> for discussion about rpmbuild reentrancy.

NAME

lxc version - Print the version number of this client tool

SYNOPSIS

lxc version

DESCRIPTION

Print the version number of this client tool.

OPTIONS

--debug
(= false)
Enable debug mode
--force-local
(= false)
Force using the local unix socket
--no-alias
(= false)
Ignore aliases when determining what command to run
--verbose
(= false)
Enable verbose mode

mtx-metapost(1) MetaPost to PDF processor mtx-metapost(1)

NAME
mtx-metapost

SYNOPSIS
mtxrun --script metapost [ OPTIONS ... ] [ FILENAMES ]

DESCRIPTION
MetaPost to PDF processor

OPTIONS
--rawmp
raw metapost run

--metafun
use metafun instead of plain

--latex
force --tex=latex

--texexec
force texexec usage (mkii)

--split
split single result file into pages

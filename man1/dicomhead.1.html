NAME
dicomhead - prints a dump of the header data in DICOM images

SYNOPSIS
dicomhead <DICOM file>

DESCRIPTION
dicomhead prints a dump of the information in the DICOM header of the specified image file. In addidtion to similar tools, dicomhead also extracts some information from the
Siemens shadow set (number of groups, number of channels and sagital, axial and transverse position).

NAME

leafpad - GTK+ based simple text editor

DESCRIPTION

leafpad is a simple GTK+ based text editor. The user interface is similar to Windows(tm) notepad. It aims to be lighter than GEdit and KWrite and to be as useful as them.
WARNING: The encoding feature is derived from built-in Glib's iconv feature (g_convert), which you can check with command "iconv -l". Leafpad can use only the listed encoding. Any other encoding cause serious data loss.

SYNOPSIS

  leafpad [options] [filename]

OPTIONS

--codeset=CODESET
Set codeset to open file.
--display=DISPLAY
X display to use.
--help
Dispaly the command line options.
--screen=SCREEN
X screen to use.
--sync
Make X calls synchronous.
--tab-width=WIDTH
Set tab width.
--version
Display version number.

ENVIRONMENT

None.

FILES

None.

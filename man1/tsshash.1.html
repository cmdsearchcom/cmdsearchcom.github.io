NAME
tsshash - manual page for tsshash v713

DESCRIPTION
hash

Runs TPM2_Hash

-hi hierarchy (e, o, p, n) (default null)

[-halg [sha1, sha256, sha384] (default sha256)] -if input file to be hashed [-oh hash file name] [-tk ticket file name]

SEE ALSO
The full documentation for tsshash is maintained as a Texinfo manual. If the info and tsshash programs are properly installed at your site, the command

info tsshash

should give you access to the complete manual.

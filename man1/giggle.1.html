GIGGLE(1) General Commands Manual GIGGLE(1)

NAME

giggle - Gtk+ interface to git trees

SYNOPSIS

giggle

DESCRIPTION

giggle is a graphical frontend for the git directory tracker (think of gitk on GTK+).

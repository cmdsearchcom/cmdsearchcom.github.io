NAME

latexdraw - vector drawing program for LaTeX using PSTricks
 

SYNOPSIS

latexdraw file
 

DESCRIPTION

latexdraw is a vector drawing program for LaTeX using PSTricks

OPTIONS

At present, latexdraw accepts no command line options other than (optionally) the name of a file to open.
 

FILES

/usr/share/latexdraw/templates/
system-wide template repository
$HOME/.latexdraw/templates/
user's template repository
$HOME/.latexdraw/.preferences.xml
user settings
$HOME/.latexdraw/.cacheShared
cached jpeg images of system templates
$HOME/.latexdraw/.cache
cached jpeg images of user's templates

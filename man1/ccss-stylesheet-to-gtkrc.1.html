NAME

ccss-stylesheet-to-gtkrc — program to dump css files to gtkrc format

SYNOPSIS

ccss-stylesheet-to-gtkrc [file]

DESCRIPTION

This manual page documents briefly the ccss-stylesheet-to-gtkrc command.
This manual page was written for the Debian distribution because the original program does not have a manual page. Instead, it has documentation in the GNU Info format; see below.
ccss-stylesheet-to-gtkrc is a program that parse a css file and then convert it to gtkrc style files.

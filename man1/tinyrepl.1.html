NAME

tinyrepl - Tiny REPL

SYNOPSIS

    $ tinyrepl
    re.pl$ "s" x 5
    "sssss"
    re.pl$ exit
    $ tinyrepl --plugin HintPersistence

DESCRIPTION

tinyrepl is a minimal pure-Perl REPL. It is just a small wrapper around Eval::WithLexicals.

OPTIONS

"--plugin="
Loads a plugin into the REPL. See "with_plugins" in Eval::WithLexicals.

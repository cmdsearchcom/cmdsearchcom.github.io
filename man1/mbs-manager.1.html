NAME

mbs-manager - manual page for mbs-manager 1.3.24

DESCRIPTION

usage: mbs-manager [-?]
{shell,runssl,generatelocalhostcert,db,runserver,gendevfedmsgcert,upgradedb,build_module_locally,cleardb} ...

positional arguments:

{shell,runssl,generatelocalhostcert,db,runserver,gendevfedmsgcert,upgradedb,build_module_locally,cleardb}
shell
Runs a Python shell inside Flask application context.
runssl
Runs the Flask app with the HTTPS settings configured in config.py
generatelocalhostcert
Creates a public/private key pair for message signing and the frontend
db
Perform database migrations
runserver
Runs the Flask development server i.e. app.run()
gendevfedmsgcert
Creates a CA, a certificate signed by that CA, and generates a CRL.
upgradedb
Upgrades the database schema to the latest revision
build_module_locally
Performs local module build using Mock
cleardb
Clears the database

optional arguments:

-?, --help
show this help message and exit

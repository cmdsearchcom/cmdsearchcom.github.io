NAME
tssload - manual page for tssload v713

DESCRIPTION
load

Runs TPM2_Load

-hp parent handle

-pwdp password for parent key (default empty)

-ipu public key file name

-ipr private key file name

-na name file name

-se[0-2] session handle / attributes (default PWAP)

01 continue

SEE ALSO
The full documentation for tssload is maintained as a Texinfo manual. If the info and tssload programs are properly installed at your site, the command

info tssload

should give you access to the complete manual.

NAME

fvwm-crystal.mplayer-wrapper - mplayer wrapper for fvwm-crystal

SYNOPSIS

fvwm-crystal.mplayer-wrapper [ directory] [directory] [...]

DESCRIPTION

fvwm-crystal.mplayer-wrapper is an helper for fvwm-crystal to control mplayer as a music player. It helps to manage playlists through a named pipe.
This manual page was written for the Debian distribution because the original program does not have a manual page.

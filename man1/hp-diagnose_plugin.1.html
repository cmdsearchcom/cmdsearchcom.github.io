NAME

hp-diagnose_plugin - Diagnose Plugin Utility

DESCRIPTION

Diagnose HP Plugin. Installs plugins if absent

SYNOPSIS

hp-diagnose_plugin [OPTIONS]

OPTIONS

Set the logging level:
-l<level> or --logging=<level> <level>: none, info*, error, warn, debug (*default)
Run in debug mode:
-g (same as option: -ldebug)
This help information:
-h or --help

SEE ALSO

hp-plugin hp-setup hp-firmware

NAME

hmt-slam-gui - Robotic Hybrid Metric Topological (HMT) mappping GUI

SYNOPSIS

hmt-slam-gui

DESCRIPTION

hmt-slam-gui is a GUI application which allows loading, editing and building complex HMT maps for robotics application.

BUGS

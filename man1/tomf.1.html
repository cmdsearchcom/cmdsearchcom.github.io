NAME

tomf - copies files to a multifile

SYNOPSIS

tomf [-h] [-b blocksize] filename1 ... outname

DESCRIPTION

-h
create MultiFile as HDF5 instead of regular file.
-b
Blocksize in bytes; default 1048576.

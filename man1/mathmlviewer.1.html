NAME
mathmlviewer - a viewer for MathML documents

SYNOPSIS
mathmlviewer [OPTION]... FILE...

DESCRIPTION
-v, --version
Output version information

-d, --debug
Debug mode

-h, --help
This small usage guide

--verbose[=0-3]
Display messages

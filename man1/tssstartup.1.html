TSSSTARTUP(1) User Commands TSSSTARTUP(1)

NAME
tssstartup - manual page for tssstartup v713

DESCRIPTION
startup

Runs TPM_Startup

[-c startup clear (default)] [-s startup state] [-st run TPM2_SelfTest] [-sto run only TPM2_SelfTest (no startup)]

SEE ALSO
The full documentation for tssstartup is maintained as a Texinfo manual. If the info and tssstartup programs are properly installed at your site, the command

info tssstartup

should give you access to the complete manual.

NAME
jiv - Image display utility

SYNOPSIS
jiv [options] [file...]

DESCRIPTION
The jiv command displays a JPEG-2000 image on an X display. Use the arrow keys for scrolling and < and > for zooming. Please use the --help command line switch and the JasPer
Software Reference Manual for more information.

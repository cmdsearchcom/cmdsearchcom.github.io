NAME
mdb2rec - mdb to rec converter

SYNOPSIS
mdb2rec [OPTIONS]... MDB_FILE [TABLE]

DESCRIPTION
Convert an mdb file into a rec file.

-s, --system-tables
include system tables.

-e, --keep-empty-fields
don’t prune empty fields in the rec output.

-l, --list-tables
dump a list of the table names contained in the mdb file.

--help print a help message and exit.

--version
show version and exit.

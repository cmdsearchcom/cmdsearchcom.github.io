NAME
lbmount â mount ltspfs based filesystems to a user owned subdir in /media

SYNOPSIS
lbmount

DESCRIPTION
This manual page documents briefly the lbmount command.

lbmount is a suid wrapper that creates mountpoints in /media and bind mounts ltspfs mounted devices to there lbmount

OPTIONS
Usage: lbmount [--umount] mediadir

ltspfs (1). ltspfsmounter (1).

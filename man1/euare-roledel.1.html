NAME
euare-roledel - Delete a role

SYNOPSIS
euare-roledel [-c] [-p] [--as-account ACCOUNT] [-U URL]

[--region USER@REGION] [-I KEY_ID] [-S KEY]
[--security-token TOKEN] [--debug] [--debugger] [--version] [-h] ROLE

DESCRIPTION
Delete a role

positional arguments:
ROLE name of the role to delete (required)

optional arguments:
-c, --recursive
remove all IAM resources associated with the role first

-p, --pretend
list the resources that would be deleted instead of actually deleting them. Implies -c.

--as-account ACCOUNT
[Eucalyptus cloud admin only] run this command as the administrator of another account

-U URL, --url URL
identity service endpoint URL

--region USER@REGION
region and/or user names to search when looking up config file data

-I KEY_ID, --access-key-id KEY_ID

-S KEY, --secret-key KEY

--security-token TOKEN

--debug
show debugging output

--debugger
launch interactive debugger on error

--version
show the program’s version and exit

-h, --help
show this help message and exit

NAME
genmod - make submodule

SYNOPSIS
genmod [OPTIONS] <Name> <Number>

DESCRIPTION
This program makes a submodule, i.e., it takes the mountains contained in that submodule and spins them up. The result is written to Name.sNumber. The program assumes that the
submodule structure has been calculated with mksub(1).

If the -m option is used, Number is treated as the number of a mountain. This mountain is generated and written to Name.mNumber.

OPTIONS
-Q Quiet, no messages.

-V Verbose, more messages.

-T <MaxTime>
Set CPU time limit

-m Make mountain; works after mkinc(1).

INPUT FILES
Name.sub
Submodule information, generated by mksub(1).

Name.v Mountains.

OUTPUT FILES
Name.sNumber
The result (without -m).

Name.mNumber
The result (with -m).

SEE ALSO
mkinc(1), mksub(1)

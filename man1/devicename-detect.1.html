NAME
devicename-detect - Pick a default name for a device

SYNOPSIS
devicename-detect [-h|--help|-V|--version]

DESCRIPTION
devicename-detect outputs a default name for the device based on autodetectable information from various sources (dmidecode, devicetype-detect, cpuinfo).

OPTIONS
-h, --help
display a usage message and exit

-V, --version
output version information and exit

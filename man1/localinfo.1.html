localinfo [-imdn46gslh] [domainname]

Description
The localinfo utility will obtain the local network addresses and print them.

Options
The localinfo utility accepts following command line options:

-i include interface name.

-m map IPv4 addresses.

-d require a reverse DNS entry.

-n use numeric host names.

-4 IPv4 only.

-6 IPv6 only (but including mapped IPv4 addresses).

-g Global addresses.

-s Site-level addresses.

-l Link-level addresses.

-h Host-internal addresses.

Examples
You want to find out local IPv6 addresses:

$ localinfo -6

You want to find out to link-local addresses

$ localinfo -l -n

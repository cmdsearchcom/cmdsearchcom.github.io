NAME
yuvsum - YUV conversion program from the GATOS package.

SYNOPSIS
yuvsum < gatos.yuv

DESCRIPTION
The program yuvsum averages all images in gatos.yuv (must all be same size) into one ppm format image.

SEE ALSO
gatos, gatos-conf, atitv, atisplit, atitogif, atitojpg, atitoppm, xatitv.

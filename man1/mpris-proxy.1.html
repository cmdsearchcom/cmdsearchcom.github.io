mpris-proxy - Bluetooth mpris-proxy

DESCRIPTION
Usage:
mpris-proxy [OPTION...]

Help Options:
-h, --help
Show help options

Application Options:
-v, --version
Show version information and exit

-e, --export
Export remote players

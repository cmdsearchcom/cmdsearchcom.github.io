NAME
docker-pause - Pause all processes within one or more containers

SYNOPSIS
docker pause CONTAINER [CONTAINER...]

DESCRIPTION
The docker pause command suspends all processes in the specified containers. On Linux, this uses the cgroups freezer. Traditionally, when suspending a process the SIGSTOP signal
is used, which is observable by the process being suspended. With the cgroups freezer the process is unaware, and unable to capture, that it is being suspended, and subsequently
resumed. On Windows, only Hyper-V containers can be paused.

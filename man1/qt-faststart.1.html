NAME

qt-faststart - utility for Quicktime files

SYNOPSIS

qt-faststart
 

DESCRIPTION

qt-faststart is a utility that rearranges a Quicktime file such that the moov atom is in front of the data, thus facilitating network streaming.

OPTIONS

Options processed by the executable:
<infile.mov>
The source Quicktime file.
<outfile.mov>
The destination Quicktime file.

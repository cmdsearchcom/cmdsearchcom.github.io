NAME

kdb-list - List keys in the key database

SYNOPSIS

kdb list

DESCRIPTION

This command will list all the available Elektra plugins that were compiled together with the source.

OPTIONS

-H, --help
Show the man page.
-V, --version
Print version info.
-v, --verbose
Explain what is happening.
-0, --null
Use binary 0 termination

EXAMPLES

To list all available plugins: kdb list

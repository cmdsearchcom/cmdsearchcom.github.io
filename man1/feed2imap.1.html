NAME
feed2imap - clever RSS/ATOM feed aggregator

SYNOPSIS
feed2imap [OPTIONS]

DESCRIPTION
feed2imap is an RSS/Atom feed aggregator. After Downloading feeds (over HTTP or HTTPS), it uploads them to a specified folder of an IMAP mail server. The user can then access the
feeds using Mutt, Evolution, Mozilla Thunderbird or even a webmail.

-V, --version
Show version information.

-v, --verbose
Run in verbose mode.

-c, --rebuild-cache
Rebuilds the cache. Fetches all items and mark them as already seen. Useful if you lose your .feed2imap.cache file.

-f, --config file
Use another config file (~/.feed2imaprc is the default).

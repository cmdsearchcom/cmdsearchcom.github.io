SHOWFOND(1) General Commands Manual SHOWFOND(1)

NAME

showfond - display information about Macintosh font resources

SYNOPSIS

showfond macfile...

DESCRIPTION

The program showfond dumps information about Macintosh font resources (FOND, NFNT, sfnt) found in the file(s) specified on the command line.

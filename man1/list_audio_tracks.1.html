LIST_AUDIO_TRACKS(1) list_audio_tracks LIST_AUDIO_TRACKS(1)

NAME

list_audio_tracks is an alias for icedax.

SYNOPSIS

list_audio-tracks

DESCRIPTION

list_audio_tracks equals to the invocation of icedax, so all the arguments for the latter can be used.

SEE ALSO

icedax(1)

AUTHOR

icedax was written by Heiko Eissfeldt and others.

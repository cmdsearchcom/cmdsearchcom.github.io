NAME
jsonld-java-tools - JSON for Linking Data Java tools.

DESCRIPTION
jsonld-java-tools

The following options are understood:

--base
base URI. (default: )

--context
The context.

--format

RDFFormat: The output file format to use. Valid values are: binaryrdf, turtle, n3, rdfjson, rdfxml, trig, nquads, jsonld, trix, ntriples default: N-Quads
(mimeTypes=application/n-quads, text/x-nquads, text/nquads; ext=nq)

--outputForm
The way to output the (default: expanded) results from fromRDF. Valid values are: compacted, expanded, flattened

--help
Show this help string

OPTIONS REQUIRED
The following options (required) are understood:

--inputFile
File: The input file.

--process

The processing to perform. Valid values are: expand, compact, frame, normalize, flatten, fromrdf, tordf.

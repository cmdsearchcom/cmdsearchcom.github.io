NAME

kwiki - The Kwiki Command Line Tool

USAGE

    > kwiki -new cgi-bin/my-kwiki
    Kwiki software installed! Point your browser at this location.

DESCRIPTION

Kwiki is a simple extendable wiki framework, written in Perl.
See Kwiki::Command for more information on using the command line tool.

NOTE

If you are on Mac OS X, use the command "perldoc Kwiki.pm" to get the Kwiki module documentation.

NAME
claws-mail-newmail-plugin - Message header summary to a log file.

DESCRIPTION
This manual page documents briefly the claws-mail-newmail-plugin.

This manual page was written for the Debian distribution because the original program does not have a manual page.

claws-mail-newmail-plugin is a plugin (loadable module) for the Claws Mail mailer.

This plugin writes a summary of new mail headers to a log file. Writing is done after sorting.

USAGE
Before using a plugin you must instruct Claws Mail to load it on startup.

For this you must go âConfigurationâ menu on main window toolbar, open âPlugins...â dialog, click on the âLoad plugin...â button and select the plugin file, named newmail.so,
and press the âOpenâ button.

FILES
~/Mail/NewLog
Default filename where header summary is written.

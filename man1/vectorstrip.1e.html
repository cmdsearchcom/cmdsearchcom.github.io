NAME

vectorstrip - Removes vectors from the ends of nucleotide sequence(s)

SYNOPSIS

vectorstrip -sequence seqall -readfile toggle -vectorsfile  infile -mismatch  integer -besthits boolean -alinker string -blinker string [ -allsequences  boolean] -outfile  outfile -outseq seqoutall
vectorstrip -help

DESCRIPTION

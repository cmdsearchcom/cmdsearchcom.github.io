?[1mNAME?[0m
bearing - returns the azimuth bearing and great circle path distance
between two specified points
 
?[1mSYNOPSIS?[0m
?[1mbearing ?[22mfile
 
?[1mDESCRIPTION?[0m
?[1mbearing ?[22mreads a pair of .qth files specified on the command line, and
returns the azimuth bearing and great circle path distance between the
two points specified. A -metric switch is available so that distances
can be provided in kilometers rather than statute miles. SPLAT! pro-
vides similar distance and bearing information between two specific
site locations. The bearing utility, however, provides the information
quickly and easily over great distances without having to run SPLAT!
 
 
?[1mEXAMPLES?[0m
bearing ai4qr.qth 940-repeater.qth

LINGOT(1) General Commands Manual LINGOT(1)

NAME

lingot - accurate and easy to use musical instrument tuner

SYNOPSIS

lingot

DESCRIPTION

lingot is a musical instrument tuner. It's accurate, easy to use, and highly configurable. Originally conceived to tune electric guitars, it can now be used to tune any instrument.
 

USAGE

lingot does not have any command line argument and is fully configured through its GUI.

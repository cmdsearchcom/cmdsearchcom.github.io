AVERAGE_GENOME_SIZE(1) User Commands AVERAGE_GENOME_SIZE(1)

NAME

average_genome_size - Calculate the average genome size in Grinder libraries

DESCRIPTION

Usage:

average_genome_size <db_fasta> <rank_file> average_genome_size --help average_genome_size --man average_genome_size --usage average_genome_size --version

Required arguments:

<db_fasta>
FASTA file containing the full-genomes used to produce the Grinder library.
<rank_file>
Grinder rank file that describes the library composition.

grinder(7), grinder(1), average_genome_size(1) and change_paired_read_orientation(1).

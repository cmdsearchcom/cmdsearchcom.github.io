NAME

termcast - broadcast your terminal sessions for remote viewing

VERSION

version 0.13

SYNOPSIS

  termcast [options] [command]

DESCRIPTION

Runs "command" (or a shell, if the command isn't given), while broadcasting it to a termcast server. See App::Termcast for options documentation.

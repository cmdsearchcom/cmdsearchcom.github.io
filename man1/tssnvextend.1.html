TSSNVEXTEND(1) User Commands TSSNVEXTEND(1)

NAME

tssnvextend - manual page for tssnvextend v713

DESCRIPTION

nvextend
Runs TPM2_NV_Extend
-ha NV index handle
-pwdn password for NV index (default empty)
-ic data string
-if data file
-se[0-2] session handle / attributes (default PWAP)
01 continue

SEE ALSO

The full documentation for tssnvextend is maintained as a Texinfo manual. If the info and tssnvextend programs are properly installed at your site, the command
info tssnvextend
should give you access to the complete manual.

NAME

ebook2odt - manual page for ebook2odt 0.9.5

SYNOPSIS

ebook2odt [ OPTIONS] INPUT [OUTPUT]

DESCRIPTION

`ebook2odt' converts e-books to ODF. If OUTPUT is omitted, the result is printed as Flat ODF to standard output.

OPTIONS

--help
show this help message
--version
print version and exit
--stdout
print the result as flat XML to standard output

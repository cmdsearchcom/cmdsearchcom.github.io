NAME

ldaptor-ldap2dnszones - LDAPtor DNS zone file exporter
 

SYNOPSIS

ldaptor-ldap2dnszones [ OPTION..] DOMAIN OUTPUTFILE [FILTER]

OPTIONS

--base=
LDAP base dn
--service-location=
Service location, in the form BASEDN:HOST[:PORT]
--reverse=
Write out reverse zone, in the form ADDRESS/NETMASK:FILE
 

DESCRIPTION

ldaptor-ldap2dnszones exports DNS zone data to a file.

NAME

tsspolicymaker - manual page for tsspolicymaker v713

DESCRIPTION

policymaker
[-halg hash algorithm (default sha256) -if input policy statements in hex ascii [-of] output file - policy hash in binary [-pr] stdout - policy hash in hex ascii

SEE ALSO

The full documentation for tsspolicymaker is maintained as a Texinfo manual. If the info and tsspolicymaker programs are properly installed at your site, the command
info tsspolicymaker
should give you access to the complete manual.

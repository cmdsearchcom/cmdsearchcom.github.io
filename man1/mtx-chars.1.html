mtx-chars(1) MkII Character Table Generators mtx-chars(1)

NAME
mtx-chars

SYNOPSIS
mtxrun --script chars [ OPTIONS ... ] [ FILENAMES ]

DESCRIPTION
MkII Character Table Generators

OPTIONS
--xtx generate xetx-*.tex (used by xetex)

--pdf generate pdfr-def.tex (used by pdftex)

--entities
generate entities table

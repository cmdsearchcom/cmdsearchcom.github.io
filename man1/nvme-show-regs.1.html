NAME
nvme-show-regs - Reads and shows the defined NVMe controller registers.

SYNOPSIS
nvme show-regs <device>

DESCRIPTION
For the NVMe device given, sends an identify namespace command and provides the result and returned structure.

The <device> parameter is mandatory and must be the nvme admin character device (ex: /dev/nvme0). The program uses knowledge of the sysfs layout to map the device to the pci
resource stored there and mmaps the memory to get access to the registers.

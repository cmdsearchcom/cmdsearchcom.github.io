NAME
MBK_VSS - define the ground power name pattern

SYNOPSYS
c-shell running
setenv MBK_VSS groundname

DESCRIPTION
MBK_VSS sets the pattern to be matched in a name to indicate a ground node for the tools based upon Its default value is vss. Therefore all names of the form ‘*vss*’ in indiâ
cates a ground node.

EXAMPLE
setenv MBK_VSS gnd

NAME
geiser-racket - REPL server for geiser and racket

SYNOPSIS
geiser-racket [-p port] [--port port]

DESCRIPTION
geiser-racket starts a instance of racket listening on the designated port, or a random port if none is specified. Note that this listens on all TCP addresses, which may be a
security concern for some users.

OPTIONS
-p port
--port port
Listen on the specified port.

SEE ALSO
For more information see the info documentation for geiser.

NAME
cipux_rpcd - CipUX XML-RPC server daemon

VERSION
version 3.4.0.0

SYNOPSIS
cipux_rpcd

DESCRIPTION
This is the CipUX XML-RPC daemon for the CipUX::Task API. The daemon provides CipUX commands to RPC clients.

OPTIONS
-c

Same as --cfg

--cfg

Specifies the configuration.

-D

The same as --debug

debug

h

The same as --help

help

n

The same as --nofork

nofork

Do not for the server.

V

The same as --version.

version

Prints the version.

USAGE
cipux_rpcd [OPTIONS]

EXAMPLE
cipux_rpcd

cipux_rpcd --debug

REQUIRED ARGUMENTS
None.

DIAGNOSTICS
None.

CONFIGURATION
None.

DEPENDENCIES
CipUX::RPC::Server::Daemon
version

INCOMPATIBILITIES
Not known.

BUGS AND LIMITATIONS
Supports https only via stunnel4.

SEE ALSO
See the CipUX webpage and the manual at <http://www.cipux.org> See the mailing list <http://sympa.cipworx.org/wws/info/cipux-devel>

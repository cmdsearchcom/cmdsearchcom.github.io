BLITZTEST(1) General Commands Manual BLITZTEST(1)

NAME
blitztest - program to test blitz imaging filter library

SYNOPSIS
blitztest imagefile

DESCRIPTION
This manual page documents briefly the blitztest command. Blitztest shows a window to demonstrate what the capabilities of the blitz library is.

TSSHIERARCHYCONTROL(1) User Commands TSSHIERARCHYCONTROL(1)

NAME

tsshierarchycontrol - manual page for tsshierarchycontrol v713

DESCRIPTION

hierarchyControl
Runs TPM2_HierarchyControl
-hi authhandle hierarchy e, o, p
-he enable hierarchy e, o, p, n
-pwda authorization password (default empty)
-state (0 to disable, 1 to enable (default enable)
-se[0-2] session handle / attributes (default PWAP)
01 continue 20 command decrypt 40 response encrypt

SEE ALSO

The full documentation for tsshierarchycontrol is maintained as a Texinfo manual. If the info and tsshierarchycontrol programs are properly installed at your site, the command
info tsshierarchycontrol
should give you access to the complete manual.

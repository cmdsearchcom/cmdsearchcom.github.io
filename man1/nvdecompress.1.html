NAME
nvdecompress - Decompresses the given DDS file.

SYNOPSIS
nvdecompress [OPTION] INFILE [OUTFILE]

DESCRIPTION
nvdecompress is part of the NVIDIA texture toolset. The tool decompresses the given infile into an image file.

AUTHOR
nvdecompress is part of the NVIDIA texture tools

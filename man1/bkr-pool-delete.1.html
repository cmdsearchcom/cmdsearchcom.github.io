NAME

bkr-pool-delete - Delete a system pool

SYNOPSIS

bkr pool-delete <poolname> ..
 

DESCRIPTION

Deletes a Beaker system pool.
 
(Note: this command requires Python 2.6 or later)
 
New in version 20.
 
 
Common bkr options are described in the Options section of bkr(1).

EXIT STATUS

Non-zero on error, otherwise zero.

EXAMPLES

Delete an existing system pool called "beakerdevs":
 
bkr pool-delete beakerdevs


SEE ALSO

bkr(1), bkr pool-create, bkr pool-modify

NAME
hugo-convert-toTOML - Convert front matter to TOML

SYNOPSIS
hugo convert toTOML [OPTIONS]

DESCRIPTION
toTOML converts all front matter in the content directory to use TOML for the front matter.

OPTIONS INHERITED FROM PARENT COMMANDS
--config=""
config file (default is path/config.yaml|json|toml)

--log[=false]
Enable Logging

--logFile=""
Log File path (if set, logging enabled automatically)

-o, --output=""
filesystem path to write files to

--quiet[=false]
build in quiet mode

-s, --source=""
filesystem path to read files relative from

--unsafe[=false]
enable less safe operations, please backup first

-v, --verbose[=false]
verbose output

--verboseLog[=false]
verbose logging

NAME
PTcrop - Crop TIFF images

SYNOPSIS
PTcrop [options] tiff_file(s)

DESCRIPTION
PTcrop is a command-line tool that crops a TIFF image to its bounding rectangle, preserving its original dimensions.

OPTIONS
-p prefix
Prefix for output files (defaults to cropped%4d).

-f Force processing: Overwrite output files if they exists (use with care).

-x Delete source files (use with care).

-q Quiet run

-h Output help summary.

SEE ALSO
PTuncrop(1)

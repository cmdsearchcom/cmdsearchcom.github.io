NAME
fakechanges - reverse engineer a Debian .changes file.

SYNOPSIS
fakechanges [OPTION] files...

DESCRIPTION
fakechanges can reverse engineer the .changes file from the .debs, .dsc,
.tar.gz, .diff.gz, etc from a package build.

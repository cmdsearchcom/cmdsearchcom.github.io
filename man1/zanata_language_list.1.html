NAME
zanata_language_list - List supported languages of a Zanata server

SYNOPSIS
zanata_language_list -h | --help

zanata_language_list [options] <ZanataURL>

DESCRIPTION
This program gets the currently supported list from a Zanata server specified by ZanataURL

ARGUMENTS
ZanataUrl
The URL of zanata server, such as ’http://translate.zanata.org/’

OPTIONS
-h, --help:
Print brief help message and exits.

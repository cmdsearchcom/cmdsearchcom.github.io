NAME

_______TaskjugglerUI ____ - a KDE graphical front-end for taskjuggler

SYNOPSIS


TaskjugglerUI
[  Qt-options  ] [     KDE-options    ] [    URL     ]

DESCRIPTION

TaskjugglerUI is a graphical user interface for taskjuggler.
It's completely an integrated project management environment.

OPTIONS

--help
Show help about options
--help-qt
Show Qt specific options
--help-kde
Show KDE specific options
--help-all
Show all options
--author
Show author information
-v --version
Show version information
--license
Show license information
URL
TaskJuggler Project (TJP) file to open

NAME
menueditor - program for creating menu profiles.

SYNOPSIS
menueditor

DESCRIPTION
This manual page explains the menueditor program. This program can be used to create menu profiles for users.

menueditor create archive containing menu files and .directory file for added submenus.

SEE ALSO
profilemanager(1), desktop-profiles(7)

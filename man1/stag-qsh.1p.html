NAME
stag-qsh - DBStag query shell

SYNOPSIS
stag-qsh
help

DESCRIPTION
This is an interactive database query shell. For a full description, execute this script and type ’help’

ARGUMENTS
-d DBNAME

This is either a DBI locator or the logical name of a database in the DBSTAG_DBIMAP_FILE config file

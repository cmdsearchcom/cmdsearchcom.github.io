NAME

argonaut-server - Dispatching action received from FusionDirectory to the clients

SYNOPSIS

argonaut-server

DESCRIPTION

argonaut-server - argonaut-server dispatch actions received from FusionDirectory and send it to the clients. It is modular and can load various modules at run time.

BUGS

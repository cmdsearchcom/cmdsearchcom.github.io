NAME
npm-start - Start a package

SYNOPSIS
npm start [-- <args>]

DESCRIPTION
This runs an arbitrary command specified in the package’s "start" property of its "scripts" object. If no "start" property is specified on the "scripts" object, it will run node
server.js.

As of npm@2.0.0 http://blog.npmjs.org/post/98131109725/npm-2-0-0, you can use custom arguments when executing scripts. Refer to npm help run-script for more details.

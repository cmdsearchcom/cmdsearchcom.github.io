NAME
tmxviewer - a simple tile map viewer

SYNOPSIS
tmxviewer [OPTIONS] [FILES...]

DESCRIPTION
This application can be used to quickly view a map that was created by the Tiled Map Editor.

OPTIONS
-h --help
Displays the help

-v --version
Displays the version

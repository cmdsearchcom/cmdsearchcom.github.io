NAME
CCSM - compizconfig-settings-manager

SYNOPSIS
ccsm [options]

OPTIONS
-h, --help
show this help message and exit

-p PLUGIN, --plugin=PLUGIN
Directly jump to the page of PLUGIN

-c CATEGORY, --category=CATEGORY
Directly jump to CATEGORY

-v, --version
Version

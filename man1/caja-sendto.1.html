NAME

caja-sendto - convenience application to send a file via email or instant messenger

SYNOPSIS

caja-sendto [OPTIONS | FILES...]

DESCRIPTION

Send FILE(s) via email or instant messenger.
 
A dialog window presents a choice of carrier application and recipient of the file(s). Recipent names can be selected from a list or autocompleted. The selected application is then opened with the file(s) and recipient ready for transfer.
 
The application is intented to integrate with caja and is written for the MATE graphical desktop.

OPTIONS

-?, --help
Show a help message listing all the options and their meanings.

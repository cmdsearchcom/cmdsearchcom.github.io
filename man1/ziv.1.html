NAME

ziv - invert matrix or permutation

SYNOPSIS

ziv [ OPTIONS] <File> [<Result>

DESCRIPTION

This program inverts a matrix or permutation.

OPTIONS

-Q
Quiet, no messages.
-V
Verbose, more messages.
-T <MaxTime>
Set CPU time limit

INPUT FILES

Matrix
A square matrix or permutation.

OUTPUT FILES

Result
Inverse matrix or permutation.

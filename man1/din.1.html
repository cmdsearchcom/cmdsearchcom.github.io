din â This program’s purpose is to be an audio synthesizer of a 3rd kind.

SYNOPSIS
din

DESCRIPTION
This manual page documents briefly the din and checkdotdin commands.

This manual page was written for the Debian distribution because the original program does not have a manual page. Instead, it has documentation in the GNU Info format;
see below.

din is a new audio synthesizer. It claims to be "a synth of a 3rd kind" that will revolutionize the world of audio synths. It has many features and is open source.

This program does not accept command-line arguments.

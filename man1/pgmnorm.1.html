NAME

pgmnorm - replaced by pnmnorm

DESCRIPTION

This program is part of Netpbm(1)
pgmnorm was replaced in Netpbm 9.25 (March 2002) by pnmnorm(1)
pnmnorm is backward compatible with pgmnorm, but it also handles PPM images.

NAME
pamtoavs - convert a Netpbm image to an AVS X image

SYNOPSIS
pamtoavs [netpbmfile]

DESCRIPTION
This program is part of Netpbm(1)

pamtoavs reads a Netpbm image as input and produces a Stardent <acronym xml:lang="en" title="Application Visualization System">AVS</acronym> X image as output. AVS X images are

gts2dxf(1) GNU Triangulated Surface utils gts2dxf(1)

NAME

gts2dxf - converts a GTS file to DXF format.
 

SYNOPSIS

gts2dxf < input.gts > output.dxf
 

DESCRIPTION

This manual page documents briefly the gts2dxf command.

NAME
tsssequenceupdate - manual page for tsssequenceupdate v713

DESCRIPTION
sequenceupdate

Runs TPM2_SequenceUpdate

-hs seqeunce handle

-pwds password for sequence (default empty)

-if input file to be HMACed

-se[0-2] session handle / attributes (default PWAP)

01 continue

SEE ALSO
The full documentation for tsssequenceupdate is maintained as a Texinfo manual. If the info and tsssequenceupdate programs are properly installed at your site, the command

info tsssequenceupdate

should give you access to the complete manual.

NAME
winswitch_ssh_session - Window Switch SSH Session Helper

SYNOPSIS
winswitch_ssh_session [--help]

winswitch_ssh_session clientID sessionID command [arguments ...]

DESCRIPTION
winswitch_ssh_session starts Window Switch SSH sessions for Microsoft Windows clients.

OPTIONS
--help Usage information

clientID
The ID of the user starting the session

sessionID
The ID of this session

command
The command to start

arguments
Optional arguments for the command

DESCRIPTION
Starts the given command and notifies the Window Switch server when it terminates.

This is not meant to be used directly. It is used by Microsoft Windows clients to ensure that tunnels are closed once the command terminates.

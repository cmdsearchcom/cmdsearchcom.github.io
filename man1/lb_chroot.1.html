NAME

lb chroot - Complete the chroot stage
 

SYNOPSIS

lb chroot [live-build options]
 

DESCRIPTION

lb chroot is a high-level command (porcelain) of live-build(7), the live systems tool suite.
lb chroot calls all necessary live-build programs in the correct order to complete the chroot stage.
 

OPTIONS

lb chroot has no specific options but understands all generic live-build options. See live-build(7) for a complete list of all generic live-build options.
 

FILES

none
 

SEE ALSO

live-build(7)
This program is a part of live-build.

NAME

lou_checkhyphens - Check the accuracy of hyphenation in liblouis Braille translation tables

SYNOPSIS

lou_checkhyphens [ OPTION]

DESCRIPTION

Check the accuracy of hyphenation in Braille translation for both translated and untranslated words.
-h, --help
display this help and exit
-v, --version
display version information and exit

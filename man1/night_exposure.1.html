NAME
night_exposure - make a single exposure on a camera

SYNOPSIS
night_exposure [-t time] [-s shutter on|off] [-b 1|2|3] [-o output]

DESCRIPTION
The options are as follows:

options:

-t
exposure time in seconds

-s
shutter state, on - open, off - close for dark frame

-r
select region by format: (x1,y1,x2,y2), default: full area

-b
binning, 1 - 1x1, 2 - 2x2, 3 - 3x3, default: 3

-c
select chip, I - imaging (default), T - tracking

-m
readout mode (see output of ’night_power info’)

-o
name of an output file, default to nightview.fits

-name object’s name

-obsname
observer’s name

-p
show progress indicator

-pn
show progress indicator (on new line)

-host address:port internet address server, default: local connect

Use Ctrl-C to interrupt the exposure or download in progress.

ENVIRONMENT
Setting of environment variable NIGHTVIEW_HOST is equivalent to -host option.

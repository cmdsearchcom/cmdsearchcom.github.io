RFDIR(1) Rfio User Commands RFDIR(1)

NAME

rfdir - Remote directory list

SYNOPSIS

rfdir directory
 
rfdir filename

DESCRIPTION

The remote directory list program provides an interface to the shift remote file I/O daemon (rfiod) for listing a remote directory or file. The filename or directory argument is either a remote file name of the form:
hostname:path
or a local file name (not containing the :/ character combination). The output from the rfdir command is a full directory listing similar to that of the ls -al command for listing local files or directories.

SEE ALSO

rcp(1), rfiod(1)

NOTES

rfdir does not support regular expressions (regexp(5)) in the directory or filename argument.
The date and time displayed by rfdir are for the last modification to the file.

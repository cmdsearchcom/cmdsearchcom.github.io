NAME
euscale-delete-scheduled-action - Delete a scheduled action

SYNOPSIS
euscale-delete-scheduled-action -g ASGROUP [-U URL]

[--region USER@REGION] [-I KEY_ID]
[-S KEY] [--security-token TOKEN] [--debug] [--debugger] [--version] [-h] ACTION

DESCRIPTION
Delete a scheduled action

positional arguments:
ACTION name of the scheduled action to delete (required)

optional arguments:
-g ASGROUP, --auto-scaling-group ASGROUP
name of the auto-scaling group the scheduled action is associated with (required)

-U URL, --url URL
auto-scaling service endpoint URL

--region USER@REGION
region and/or user names to search when looking up config file data

-I KEY_ID, --access-key-id KEY_ID

-S KEY, --secret-key KEY

--security-token TOKEN

--debug
show debugging output

--debugger
launch interactive debugger on error

--version
show the program’s version and exit

-h, --help
show this help message and exit

NAME

dccleancrw - Canon (CRW) RAW photo file recovery utility
 

SYNOPSIS

dccleancrw crw_0001.crw crw_0002.crw ...
 
 

DESCRIPTION

Because they are parsed from the end, Canon CRW files become unreadable if garbage data is appended to them, as often happens when files are recovered from damaged media. This program truncates CRW files to the correct size.

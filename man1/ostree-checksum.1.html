NAME
ostree-checksum - Checksum a file or directory

SYNOPSIS
ostree checksum {PATH}

DESCRIPTION
Generates a checksum for a given file or directory.

EXAMPLE
$ ostree checksum file1

67e382b11d213a402a5313e61cbc69dfd5ab93cb07fbb8b71c2e84f79fa5d7dc

NAME
libfm-pref-apps - sets preferred applications for programs based on libfm

SYNOPSIS
libfm-pref-apps

DESCRIPTION
libfm-pref-apps is an application that changes preferred applications - web browser and mail client. Those setting are appliable for programs based on libfm but should work also
for other XDG standard compliant programs.

FILES
~/.local/share/applications/mimeapps.list

pcmanfm(1)

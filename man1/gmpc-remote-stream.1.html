NAME
gmpc-remote-stream - a remote control for GMPC.

SYNOPSIS
gmpc-remote-stream [OPTION]

DESCRIPTION
If gmpc is compiled with IPC support gmpc-remote can send commands to the running instance of gmpc. gmpc-remote-stream is a wrapper around gmpc-remote that will pass it the
extra option --stream.

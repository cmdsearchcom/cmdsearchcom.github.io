NAME

cashe - Tool to access CAShe storage

SYNOPSIS

cashe {COMMAND} [OPTIONS...]

DESCRIPTION

View/alter objects within the a CAShe store.

COMMANDS

list
List the entries in the CAShe store. Just displays a prefix, the checksum type, the checksum of the object and the size of the object.
 
The prefix is either " " meaning the object is linked to at least one other file, so can't be removed; "*" meaning the object is newer than the configuration age; "!" meaning the object is older than the configuration age.
 
--path or -p to use another path to a CAShe, default is too use the system CAShe in /var/cashe/CAShe.
 
--sort-by to sort the entires listed by filename, atime, ctime, mtime, nlink or size.
info
List most of the information about the entries in the CAShe store.
summary
Show summary information about all the entries in the CAShe store. There are three types of data: free (has no links and is new enough to be in the high/newer part of storage), old (has no links and is old) and used (has one or more links, so can't be deleted).
check
List the entries in the CAShe store, but also check that the objects are valid. A True/False is displayed instead of the size, but if the user has permission then the object is automaticaly removed if the check fails.
config
Print the configuration information for the specified CAShe store, which is used for the cleanup operation.
load
Load an object from the CAShe store to a specified filename.
 
--copy-only don't try to link() the file.
 
--link try to link() the file.
save
Save a specified filename to an object in the CAShe store.
 
--copy-only don't try to link() the file.
 
--link try to link() the file.
save-fast
Save a specified filename to an object in the CAShe store, given the specified checksum data (without any internal checking).
 
--copy-only don't try to link() the file.
 
--link try to link() the file.
unlink
Remove an object from the CAShe store.
cleanup
Cleanup bjects from the CAShe store, given the specified (or default) configuration.
ls-extra
List extra files in the CAShe store.
rm-extra
Remove extra files from the CAShe store.
list-files
Works like the "list" command, but just displays the filename for the objects and the sort must be by timestamp.
recent
Works like the "list-files" command, but only displays the last 10 objects.
rsync-from
Rsync files to another CAShe store from the CAShe store at a specified destination. This works over ssh, and if you specify just a host ending with ":" then the path for the CAShe you are using is used as the path. This is like: rsync source CAShe
rsync-to, rsync2
Rsync files from the CAShe store to another CAShe store at a specified destination. This works over ssh, and if you specify just a host ending with ":" then the path for the CAShe you are using is used as the path. This is like: rsync CAShe destination
help
Show the command line help for the cashe command.

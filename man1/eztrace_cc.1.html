NAME
eztrace_cc - Compile an OpenMP application for eztrace

SYNOPSIS
eztrace_cc COMPILATION_COMMAND

DESCRIPTION
Compile an OpenMP application for eztrace.

EXAMPLES
$ eztrace_cc gcc -o openmp_program openmp_program.c -fopenm

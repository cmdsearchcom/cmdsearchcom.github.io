NAME
mkdtskel - Perl code skeleton generator to process XML files with XML::DT

SYNOPSIS
mkdtskel [-lines=20000] <xmlfile>

mkdtskel -dtd <dtdfile>

mkdtskel -html <htmlfile>

DESCRIPTION
Use this command to prepare a skeleton file with basic code needed to process your XML file with XML::DT; The command checks the element names and for each one, the attributes.
This information is described on the generated file to remember the programmer.

SEE ALSO
XML::DT(1), mkdtdskel(1), mkxmltype(1) and perl(1)

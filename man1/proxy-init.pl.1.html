NAME
proxy-init.pl

SYNOPSIS
proxy-init [ -cert /path/to/cert.pem ] [ -key /path/to/cert’s/key.pem ] [ -out /path/to/save/proxy ] [ -vomsAC /path/to/VOMS/AC ] .
[ -lifetime N (hours, default 12 hours) ] [ -pl N ] [ -(old|new|rfc|limited)]

DESCRIPTION
Creates a 512 bit proxy certificate optionally including a VOMS attribute certificate.

NAME

ztr - transpose matrices

SYNOPSIS

ztr [ OPTIONS] <Mat> <Result>

DESCRIPTION

This program transposes a matrix.

OPTIONS

-Q
Quiet, no messages.
-V
Verbose, more messages.
-T <MaxTime>
Set CPU time limit

INPUT FILES

Mat
The matrix to transpose.

OUTPUT FILES

Result
The transposed matrix.

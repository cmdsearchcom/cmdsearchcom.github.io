NAME
igv - Integrative Genomics Viewer

SYNOPSIS
igv
or
java -jar /usr/share/java/igv.jar

DESCRIPTION
The Integrative Genomics Viewer (IGV) is a high-performance visualization tool for interactive exploration of large, integrated datasets. It supports a wide variety of data types
including sequence alignments, microarrays, and genomic annotations.

To cite your use of IGV, please reference
http://www.broadinstitute.org/igv

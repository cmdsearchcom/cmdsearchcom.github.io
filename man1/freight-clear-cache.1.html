NAME
freight-clear-cache - clears existing package repositories

SYNOPSIS
freight clear-cache [-c conf] [-v] [-h] [manager/distro][...]

DESCRIPTION
freight-clear-cache clears the cache generated during freight-cache runs so that it can be regenerated on the next freight-cache invocation.

OPTIONS
-c conf, --conf=conf
Use an alternate configuration file.

-v, --verbose
Verbose mode.

-h, --help
Show a help message.

FILES
/etc/freight.conf, ~/.freight.conf, etc/freight.conf, .freight.conf
The default configuration files. See freight(5).

ENVIRONMENT VARIABLES
FREIGHT_CONF
The pathname of an alternate configuration file. See freight(5).

THEME SONG
The New Pornographers - "All the Old Showstoppers"

AUTHOR
Richard Crowley r@rcrowley.org

SEE ALSO
Part of freight(1).

freight-cache(1) can be used to build the repository again.

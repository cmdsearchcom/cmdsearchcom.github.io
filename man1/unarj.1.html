NAME
unarj - Unarchive ARJ files

SYNOPSIS
unarj [options] files...

DESCRIPTION
The unarj command is a simple wrapper to ease transition from the non-free unarj.

OPTIONS
This program has the same command line options as arj(1). It also supports to be called without any options as a synonim for l. For the full command line list please see arj(1)
man page.

SEE ALSO
arj(1), rearj(1).

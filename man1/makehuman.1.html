NAME
makehuman - application for the modelling of 3-Dimensional characters

SYNOPSIS
makehuman

DESCRIPTION
MakeHuman is an open source (so it’s completely free), innovative and professional software for the modelling of 3-Dimensional humanoid characters. MakeHuman

Features that make this software unique include a new, highly intuitive GUI and a high quality mesh, optimized to work in subdivision surface mode (for example, Zbrush). Using
MakeHuman, a photorealistic character can be modeled in less than 2 minutes.

After installing the application, you can run run it from the Applications menu.

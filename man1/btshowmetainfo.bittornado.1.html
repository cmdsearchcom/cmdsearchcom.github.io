NAME

btshowmetainfo — display the meta-information in bittorrent files

SYNOPSIS

btshowmetainfo file [file ...]

DESCRIPTION

This manual page documents briefly the btshowmetainfo command.
This manual page was written for the Debian distribution because the original program does not have a manual page.
btshowmetainfo is a program which will display the meta-information stored in a bittorrent file.

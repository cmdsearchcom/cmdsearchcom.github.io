NAME
monitor - receive logging information from LPD

SYNOPSIS
monitor [ -u ] [ -t ] [ port ]

DESCRIPTION
The monitor program is a template for a printer status monitoring program. It will open the specified TCP and/or UDP port, and then wait for accounting or other information to
be sent. It prints this information on it standard output.

OPTIONS
-u wait for connections on the UDP port.

-t wait for connections on the TCP port.

port Use the specified port number.

EXIT STATUS
The following exit values are returned:

zero (0) Successful completion.

non-zero (!=0) An error occurred.

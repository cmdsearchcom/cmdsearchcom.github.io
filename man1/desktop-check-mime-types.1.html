NAME
desktop-check-mime-types - compare MIME types in installed .desktop files

SYNOPSIS
desktop-check-mime-types

DESCRIPTION
desktop-check-mime-types compares MIME types in installed .desktop files with the ones installed in debian/*.mime files, reporting which ones are not listed into .mime files.

This tool is part of gnome-pkg-tools package.

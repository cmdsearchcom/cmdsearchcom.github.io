NAME

xsdck - rnv plugin for XSD datatypes
 

SYNOPSIS

rnv -d xsdck [RNV_OPTIONS] grammar.rnc [document.xml ...]
 

DESCRIPTION

Both RNV and RVP can use pluggable datatypes. This module provides support for XSD datatypes.

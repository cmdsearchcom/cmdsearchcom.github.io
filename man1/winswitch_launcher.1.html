NAME

winswitch_launcher - Window Switch session launcher

SYNOPSIS

winswitch_launcher <session file>

DESCRIPTION

winswitch_launcher starts a Winswitch session using the specified session file.

OPTIONS

--help
Display basic usage information.

DIAGNOSTICS

A number of debugging messages will be printed on screen.

NAME
oprof_start - A GUI interface for OProfile

SYNOPSIS
oprof_start [ options ]

DESCRIPTION
The oprof_start application provides a GUI interface to control the operation of OProfile. Running oprof_start requires root privileges and does not support operf due to
oprof_start using the older opcontrol script.

ENVIRONMENT
No special environment variables are recognised by oprof_start.

FILES
/root/.oprofile/daemonrc
Configuration file for opcontrol

/var/lib/oprofile/samples/
The location of the generated sample files.

VERSION
This man page is current for oprofile-0.9.9.

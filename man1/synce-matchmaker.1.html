NAME

synce-matchmaker - Configure/view a device's partnership status
 

SYNOPSIS

synce-matchmaker [-d LEVEL] [status|create|clear INDEX]
 

DESCRIPTION

synce-matchmaker is used to control a (pre Windows Mobile 5) device's partnership status. It can show the current status, create a new partnership or clear an existing partnership.
 

OPTIONS

status
Show the partnership status for the device. (Default)
create
Create a partnership with the device.
clear INDEX
Clear a partnership on the device (INDEX is 1 or 2).

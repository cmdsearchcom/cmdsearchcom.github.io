NAME

pdal-config - query PDAL build options

SYNOPSIS


pdal-config
[ --cflags] [--cxxflags] [--defines] [--includes] [--libs] [--plugin-dir] [--version] [--python-version]

DESCRIPTION

pdal-config allows querying the options used to build the PDAL library.

OPTIONS

--cflags
Compiler flags
--cxxlibs
C++ Compiler flags
--defines
Compiler defines
--includes
Compiler includes
--libs
Libraries
--plugin-dir
Plugin directory
--version
Library version
--python-version
Python version

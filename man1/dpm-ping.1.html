DPM-PING(1) DPM Administrator Commands DPM-PING(1)

NAME

dpm-ping - check if Disk Pool Manager is alive and print its version number

SYNOPSIS

dpm-ping [ -h server ]

DESCRIPTION

dpm-ping checks if the Disk Pool Manager is alive and prints its version number.

OPTIONS

-h server
specify the Disk Pool Manager hostname. This can also be set thru the DPM_HOST environment variable.

EXIT STATUS

This program returns 0 if the operation was successful or >0 if the operation failed.

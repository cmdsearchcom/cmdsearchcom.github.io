NAME

vpbscan - Scan the PCI bus for Voicetronix hardware

SYNOPSIS

vpbscan

DESCRIPTION

The vpbscan command will detect any installed Voicetronix hardware and report its configuration to stdout.
Unlike vpbconf it can do this even if none of the hardware drivers are loaded yet. It is used by VpbConfigurator to determine the initial config to use.

NAME
lxpolkit - PolicyKit agent for LXDE

SYNOPSIS
lxpolkit

DESCRIPTION
lxpolkit is a PolicyKit agent which interacts with PolicyKit daemon to provide extended privileges to applications.

OPTIONS
lxpolkit does not support any options.

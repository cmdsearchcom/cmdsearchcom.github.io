NAME
asn1Decoding - ASN.1 DER decoder

SYNOPSIS
asn1Decoding [OPTION] DEFINITIONS ENCODED ASN1TYPE

DESCRIPTION
Decodes DER data in ENCODED file, for the ASN1TYPE element described in ASN.1 DEFINITIONS file, and print decoded structures.

-h, --help
display this help and exit

-v, --version
output version information and exit

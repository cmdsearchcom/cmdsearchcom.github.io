NAME
mouse-dpi-tool - mouse resolution estimation tool

SYNOPSIS
mouse-dpi-tool <evdev device>

DESCRIPTION
mouse-dpi-tool reads mouse movement events and calculates the sampling frequency and distance covered in device units. Combined with a measurement of the actual distance physiâ
cally covered, this allows the mouse’s resolution to be estimated.

OPTIONS
mouse-dpi-tool accepts no options.

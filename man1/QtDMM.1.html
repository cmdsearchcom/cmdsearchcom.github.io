NAME

QtDMM - GUI for Digital Multimeters

SYNOPSIS

qtdmm

DESCRIPTION

QtDMM presents a graphical user interface to digital multimeters. It caters for the loggin of data, its export and the triggering of external applications. The recorder features manual start, scheduled start (at a given time) and triggered automatic start when given thresholds are reached. Additionally you can start an external application when given thresholds are reached.

GRR(1) General Commands Manual GRR(1)

NAME

Grr - an RSS reader for GNUstep

SYNOPSIS

Grr

DESCRIPTION

This manual page documents briefly the Grr command. This manual page was written for the Debian distribution because the original program does not have a manual page.
Grr is a program that you can manage RSS feeds with. It is also known as RSS Reader or Günther's RSS Reader.

SEE ALSO

GNUstep(7).

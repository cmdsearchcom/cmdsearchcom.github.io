NAME
fasta_nucleotide_changer - Convets FASTA sequences from/to RNA/DNA

DESCRIPTION
usage: fasta_nucleotide_changer [-h] [-z] [-v] [-i INFILE] [-o OUTFILE] [-r] [-d] Part of FASTX Toolkit 0.0.14 by A. Gordon (assafgordon@gmail.com)

[-h] = This helpful help screen.

[-z] = Compress output with GZIP.

[-v] = Verbose mode. Prints a short summary.

with [-o], summary is printed to STDOUT. Otherwise, summary is printed to STDERR.

[-i INFILE]
= FASTA/Q input file. default is STDIN.

[-o OUTFILE] = FASTA/Q output file. default is STDOUT. [-r] = DNA-to-RNA mode - change T’s into U’s. [-d] = RNA-to-DNA mode - change U’s into T’s.

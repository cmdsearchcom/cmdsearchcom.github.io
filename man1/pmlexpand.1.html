NAME
pmlexpand â expand and remove defaults in a PhotoML XML file

SYNOPSIS
pmlexpand [-h] [-a | -b] [[infile] outfile]

DESCRIPTION
pmlexpand expands defaults into the relevant elements and removes the defaults elements. Input is written to stdout if no output file is specified, and read from stdin if no
input file is specified.

OPTIONS
pmlexpand accepts the following options:

-h Display usage information.

-a Replace defaults elements with a comment indicating the position from which they were removed.

-b Comment out the defaults elements.

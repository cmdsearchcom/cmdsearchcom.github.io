TSSNVWRITE(1) User Commands TSSNVWRITE(1)

NAME

tssnvwrite - manual page for tssnvwrite v713

DESCRIPTION

nvwrite
Runs TPM2_NV_Write
[-hia hierarchy authorization (o, p)(default index authorization)] -ha NV index handle -pwdn password for NV index (default empty) -ic data string -if data file -id data values, pinPass and pinLimit
if none is specified, a 0 byte write occurs -id is normally used for pin pass or pin fail indexes
-off offset (default 0)
-se[0-2] session handle / attributes (default PWAP)
01 continue

SEE ALSO

The full documentation for tssnvwrite is maintained as a Texinfo manual. If the info and tssnvwrite programs are properly installed at your site, the command
info tssnvwrite
should give you access to the complete manual.

NAME

lsm — Monitor connectivity status

SYNOPSIS

lsm [OPTIONS...]

DESCRIPTION

LSM is a link connectivity monitor tool.
LSM can be used to monitor a Linux router/firewall connectivity. If it has multiple connections it can change routing when an up/down event happens by utilizing external script.

OPTIONS

-h | --help
Show summary of options.
-v --version
Show version of program.
-c --config FILE
Specify an alternative config file instead of the default one.
-p --pidfile FILE
Specify an alternative pid file instead of the default one.
-f --no-fork
Make instance independent of others one already running.

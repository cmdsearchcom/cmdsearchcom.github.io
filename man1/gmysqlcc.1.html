NAME

gmysqlcc - a GTK+ client for mysql databases.

DESCRIPTION

gmysqlcc will help you to use your MySQL servers, do requests on them, manage their configuration (users, process, etc.), dump datas and structure and more..
 
If you need more information about gmysqlcc visit the project page: http://gmysqlcc.thepozer.org/

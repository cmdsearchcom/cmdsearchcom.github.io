NAME

ubuntu-server-tip - Display one short tip about the Ubuntu Server
 

DESCRIPTION

ubuntu-server-tip displays one random tip about the Ubuntu Server, using a fortune(1) database.
 
To enable/disable printing this in the /etc/motd at login, edit /etc/default/ubuntu-server-tips.
 
http://launchpad.net/ubuntu-server-tips

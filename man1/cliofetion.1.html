NAME

cliofetion - open source command line fetion protocol client

SYNOPSIS

cliofetion -f SOURCE -p PASSWORD -t DEST -d MESSAGE
 
cliofetion -f SOURCE -p PASSWORD -m -d MESSAGE
 

DESCRIPTION

cliofetion is a fetion client for Linux based on GTK+2.0, using Fetion Protocol Version 4.
-f
specify which number you will use to send a message
-t
specify which number you will send a message to
-d
message you will send
-m
send a message to myself
-p
accound password

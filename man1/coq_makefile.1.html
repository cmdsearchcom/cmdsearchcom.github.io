NAME
coq_makefile - The Coq Proof Assistant makefile generator

SYNOPSIS
coq_makefile [ arguments ]

DESCRIPTION
coq_makefile is a makefile generator for Coq proof developments.

OPTIONS
-h Will give you a description of the whole list of options of coq_makefile.

coqtop(1), coqtc(1), coqdep(1).

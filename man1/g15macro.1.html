NAME
g15macro - A simple Macro recording/playback application for G15Daemon

DESCRIPTION
This package provides a way to record, playback and display keyboard macro using the M and G keys found on some Logitech keyboards.

OPTIONS
g15macro understands the following options:

-h show usage information.

-d dump current configuration to stdout.

-g print debugging information.

-k configure multimedia and extra keys then exit

-v show program version.

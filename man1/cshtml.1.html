NAME

cshtml - manual page for cshtml 1.3.2

SYNOPSIS

cshtml [ options] proj.js, where options are:

DESCRIPTION

--defect-url-template arg
e.g. http://localhost/index.php?proj=%d&defect=%d
--diff-base arg
use the given list of defects as diff base
--plain-text-url arg
generate a link to plain-text version
--scan-props-placement arg (=bottom) placement of the table with scan properties: top, bottom, none
-q [ --quiet ]
do not report any parsing errors
--help
produce help message
--version
print version

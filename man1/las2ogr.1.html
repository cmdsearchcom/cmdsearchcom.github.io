NAME
las2ogr - convert LAS file to data source using OGR

SYNOPSIS
las2ogr [-h | -i <infile> -o <outfile> -t <format> | -formats]

DESCRIPTION
las2ogr converts LAS file to data source using OGR drivers available from GDAL/OGR package. This utility is useful to translate cloud of points collected by LiDAR sensors to
vectors for easier data exchange and processing.

OPTIONS
-h print usage information

-i >infile>
input ASPRS LAS file

-o >outfile>
output file

-f >format>
OGR format for output file

-formats
list supported OGR formats

EXAMPLE
Shows how to convert LAS file to ESRI Shapefile with POINT geometry type:

$ las2ogr -i mydata.las -o points.shp -f "ESRI Shapefile"

NAME

jalv.qt - Run an LV2 plugin as a JACK application (Qt version).
 

SYNOPSIS

jalv.qt PLUGIN_URI
 

DESCRIPTION

This is a version of Jalv with a GUI implemented in Qt. It is mainly for developer testing purposes, for a production ready program use jalv.gtk.

jalv(1), jalv.gtkmm(1), jalv.qt(2), lv2ls(1), jackd(1)

NAME
kgrab - screen shot utility for KDE 4

SYNOPSIS
kgrab [Qt-options] [KDE-options] ...

DESCRIPTION
kgrab is a screen capturing utility used in KDE 4.

OPTIONS
All KDE and Qt programs accept a some common command-line options. kgrab has no application-specific options.

--‘help
Show help about options

--‘help-qt
Show Qt specific options

--‘help-kde
Show KDE specific options

--‘help-all
Show all options

--‘author
Show author information

--‘version
Show version information

--‘license
Show license information

-- Indicates end of options

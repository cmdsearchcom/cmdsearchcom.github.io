NAME
code-of-conduct-signing-assistant - Sign the Ubuntu Code of Conduct easily

SYNOPSIS
code-of-conduct-signing-assistant -p [profile]

DESCRIPTION
Code of Conduct Signing Assistant is an open source program that helps you sign the Ubuntu Code of Conduct easily!

No more worries about PGP keys or decrypting mail: just follow the steps the Code of Conduct Signing Assistant guides you through and in the end you have signed the Ubuntu Code
of Conduct!

code-of-conduct-signing-assistant has no commandline options.

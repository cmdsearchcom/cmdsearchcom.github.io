NAME
atomic-help - Get help for container images

SYNOPSIS
atomic help [-h|--help] IMAGE

atomic help provides documentation for the specified container IMAGE.

The documentation is extracted from the container image. atomic searches for the documentation in following locations:

item File placed inside root of the container named either help.1 or README.md. item Label named help which should contain executable command to display the documentation.

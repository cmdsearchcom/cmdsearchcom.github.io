testComponent - Example of a component for DIET using the LogCentral

NAME
testComponent - Example of a component that connects to the LogCentral and publish some messages. It is used as an example for the DIET program.

SYNOPSYS
testComponent

DESCRIPTION
testComponent connects to the LogCentral, publishes some messages and then disconnects.

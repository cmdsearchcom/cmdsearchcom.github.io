NAME

bibledit-xiphos - Bibledit Xiphos helper

DESCRIPTION

Bibledit-Xiphos interfaces between Bibledit and Xiphos. It is started by typing "bibledit-xiphos" on the commandline. For more information see the graphical user interface.

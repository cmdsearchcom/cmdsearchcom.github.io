NAME

guilt-export - Export a patch series (to be used by quilt)

SYNOPSIS

guilt export [<target_dir>]

DESCRIPTION

Export a guilt series to be used by quilt.

OPTIONS

<target_dir>
Name of the directory to export the patch series to. (defaults to patches).

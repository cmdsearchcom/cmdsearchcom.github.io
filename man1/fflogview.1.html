NAME

fflogview - Show log messages

SYNOPSIS

fflogview [-h] [hostname[:port]]

DESCRIPTION

This program connects to a Fawkes instance over the network and shows all log messages created by that instance.

OPTIONS

-h
Show help instructions.
[hostname[:port]]
Optionally hostname and port to connect to. If not given defaults to localhost:1910.

EXAMPLES

fflogview myrobot:1910
Show log messages of Fawkes instance running on host myrobot and listening on port 1910.

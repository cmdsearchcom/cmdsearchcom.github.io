NAME

dotpath - Draw a non-overlapping wordmatch dotplot of two sequences

SYNOPSIS

dotpath -asequence sequence -bsequence  sequence -wordsize  integer [-overlaps  boolean] [-boxit boolean] -graph graph
dotpath -help

DESCRIPTION

dotpath is a command line program from EMBOSS (“the European Molecular Biology Open Software Suite”). It is part of the "Alignment:Dot plots" command group(s).

OPTIONS

Input section

-asequence sequence
-bsequence sequence

Required section

-wordsize integer
Default value: 4

Output section

-overlaps boolean
Displays the overlapping matches (in red) as well as the minimal set of non-overlapping matches Default value: N
-boxit boolean
Default value: Y
-graph graph

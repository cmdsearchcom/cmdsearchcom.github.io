NAME
mailnag-config - Mailnag setup utility

SYNOPSIS
mailnag-config

DESCRIPTION
mailnag-config is a utility that makes it easy to setup Mailnag initially. This only needs to be run once; after running it and closing the configuration window, Mailnag will be
started automatically.

mailnag(1)

NAME
gmk_sym - create rectangular symbols for gschem from a text file

SYNOPSIS
gmk_sym [ -d ] inputfile.txt > outputfile.sym

gmk_sym -h|-?

DESCRIPTION
gmk_sym is a program to create rectangular symbols for gschem from a file composed of comma separated lines.

OPTIONS
-d Turn on debug output

-h, -? Display usage information

FILE FORMAT

Â· 4th value: box’s hoz size, in pins spacings

Â· 6th value: uref prefix, like U or J

Â· 7th value: Footprint

Â· 8th value: Total number of pins on device (including hidden)

Â· All other valid lines describes the symbol’s pins:

Â· 1st value: pin name

Â· 2nd value: pin number

Â· 3rd value: pin shape, choice of: line, clock, dot&line

Â· 4th value: side of box to attach the pin,choice of: R, L, T, B

Â· 5th value: location of pin on side of box, in pin spacings

Â· 6th value: (optional) pin type attribute: in, out, io, oc, oe, pas, tp, tri, clk, pwr

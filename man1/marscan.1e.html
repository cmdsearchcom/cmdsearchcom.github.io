NAME
marscan - Finds matrix/scaffold recognition (MRS) signatures in DNA sequences

SYNOPSIS
marscan -sequence seqall -outfile report

marscan -help

DESCRIPTION
marscan is a command line program from EMBOSS (âthe European Molecular Biology Open Software Suiteâ). It is part of the "Nucleic:Gene finding,Nucleic:Functional
sites,Nucleic:Transcription" command group(s).

OPTIONS
Input section
-sequence seqall

Output section
-outfile report
File for output of MAR/SAR recognition signature (MRS) regions. This contains details of the MRS in normal GFF format. The MRS consists of two recognition sites, one of 8 bp
and one of 16 bp on either sense strand of the genomic DNA, within 200 bp of each other.

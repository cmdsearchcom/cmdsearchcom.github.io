NAME
gitunpack - GNU Interactive Tools - Unified archive unpacking

SYNTAX
gitunpack dir files...

DESCRIPTION
gitunpack is a shell script that accepts a directory and a set of archives as its command line parameters, and then attempts to unpack those archives in the given directory,
selecting the utility used to unpack the archives based on the archive extensions.

NAME

babiloo - dictionary viewer with multi-languages support

SYNOPSIS

babiloo [-h] [--help] [-g] [--gui] [-d] [--debug] [-T] [--test] [--version]

DESCRIPTION

Babiloo is a Free Open Source tool written in Python for view dictionaries files like Sdictionary, StarDict, etc

OPTIONS

-h --help
Show Babiloo command-line options.
 
-g, --gui
Run application in GUI mode. This is the default.
 
-d, --debug
Print debug messages to stout and logfile.
 
-T, --test
Used by developers for testing.
 
--version
Show program's version number and exit.

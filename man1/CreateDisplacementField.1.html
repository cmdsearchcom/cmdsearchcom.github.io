CREATEDISPLACEMENTFIELD(1) User Commands CREATEDISPLACEMENTFIELD(1)

NAME
CreateDisplacementField - part of ANTS registration suite

SYNOPSIS
CreateDisplacementField ImageDimension EnforceZeroBoundaryFlag{0/1} ComponentImage1 [ ComponentImage2 [...ComponentImageN] ] OutputImage

DESCRIPTION
Create an itkImage of itkVector pixels (NOT an itkVectorImage), using each scalar input component image for each vector component. An itkImage of itkVectors is the standard type
for displacement fields in ITK. All component images (up to 8) are assumed to have the same size, offset, origin, and spacing. The ’EnforceZeroBoundaryFlag’ option will create
zero-valued vectors along the borders when enabled (pass 1), and is recommended for better displacement field behavior.

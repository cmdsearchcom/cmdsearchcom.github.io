NAME

Glipper - Clipboard manager for GNOME

SYNOPSIS

glipper
 

DESCRIPTION

Glipper is a clipboard manager for the GNOME Desktop. Glipper sits in the panel and collects everything that is copied to the clipboard. Glipper maintains a history both of the `copy' clipboard, usually filled with `Ctrl+C' and of the `select' clipboard, filled when text is selected.

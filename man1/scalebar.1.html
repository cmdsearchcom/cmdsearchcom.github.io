NAME
scalebar - create a scalebar image from a mapfile

SYNOPSIS
scalebar [mapfile outputimage | -v]

DESCRIPTION
scalebar creates a scalebar image from a mapfile. Output is either PNG or GIF depending on what version of the GD library used.

OPTIONS
mapfile
Path to the mapfile.

outputimage
Path to the output image file.

-v Display mapserver version and build options.

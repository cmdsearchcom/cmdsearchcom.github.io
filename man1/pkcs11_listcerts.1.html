NAME
pkcs11_listcerts - SmartCard PKCS#11 certificates listing

SYNTAX
pkcs11_listcerts [debug]

DESCRIPTION
pkcs11_listcerts display all the certificates.

OPTIONS
debug Enable debugging output.

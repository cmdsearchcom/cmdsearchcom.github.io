NAME

afl-gotcpu - CPU gizmo for American Fuzzy Lop (afl)

SYNOPSIS

afl-gotcpu

DESCRIPTION

This tool provides a fairly accurate measurement of CPU preemption rate. It is meant to complement the quick-and-dirty load average widget shown in the afl-fuzz UI.
See /usr/share/doc/afl-doc/docs/parallel_fuzzing.txt.gz for more information.

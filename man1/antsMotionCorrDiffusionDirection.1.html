NAME
antsMotionCorrDiffusionDirection - part of ANTS registration suite

DESCRIPTION
COMMAND:
antsMotionCorrDiffusionDirection

antsMotionCorrDiffusionDirection

OPTIONS:
-s, --scheme dwi.scheme

Camino scheme file specificy acquisition parameters.

-b, --bvec dwi.bvec

bvec image specifying diffusion directions.

-p, --physical ref.nii.gz

image in dwi space

-m, --moco MOCOparams.csv

motion correction parameters from antsMotionCorr.

-o, --output corrected.scheme

Specify the output file for corrected directions.

-h

Print the help menu (short version). <VALUES>: 0

--help

Print the help menu. <VALUES>: 1, 0

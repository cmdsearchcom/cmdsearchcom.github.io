NAME

kcov-merge - Merge code coverage data from multiple kcov output directories

SYNOPSIS

kcov-merge [options] outdir dir1 [dir2...]

DESCRIPTION

This manual page documents briefly the kcov-merge command. kcov-merge is a utility to merge the code coverage data gathered from multiple kcov output directories.

OPTIONS

-h, --help
Display help message and exit
-l, --limits=low,high
Setup limits for low/high coverage (default: 25,75).

EXAMPLES

Generate aggregate coverage from ./kcov1 and ./kcov2 directories and generate HTML output in ./kcov-aggregate.
kcov-merge ./kcov-aggregate ./kcov1 ./kcov2

HTML OUTPUT

The HTML output shows a summary of the executed and non-executed lines aggregated from each of the individual kcov output directories. It also shows coverage data for each file/binary included in any of the individual kcov output directories.

COBERTURA OUTPUT

Kcov-merge also outputs data in the Cobertura XML format, which allows integrating kcov-merge output in Jenkins (see http://cobertura.sf.net and http://jenkins-ci.org).
The Cobertura output is placed in a file named out-path/cobertura.xml.

NAME
wvPS - convert msword documents to PS

SYNOPSIS
wvPS in_word_doc out_ps_doc

DESCRIPTION
wvPS converts word documents into PS (Postscript) format. You can then use the PS version to print, view using gv, ggv,ghostview,etc..

REQUIREMENTS
wvDVI dvips

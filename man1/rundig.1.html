NAME
rundig - sample script to create a search database for ht://Dig

SYNOPSIS
rundig [options]

DESCRIPTION
rundig is a sample script to create a search database for the ht://Dig search engine.

OPTIONS
-v Verbose mode. This increases the verbosity of the program.

-a Use alternate work files.

-s Print statistics about the dig after completion

-c configfile
Use alternate config file.

detailed description of ht://Dig and its commands.

ATIG(1) Another Twitter IRC Gateway ATIG(1)

NAME

atig - Another Twitter IRC Gateway

DESCRIPTION

usage: atig [opts]

OPTIONS

-p, --port [PORT=16668]
port number to listen
-h, --host [HOST=localhost]
host name or IP address to listen
-l, --log LOG
log file
--debug
Enable debug mode
--memprof
Enable memory profiler
-c, --conf [file=~/.atig/config] atig configuration file default is '~/.atig/config'

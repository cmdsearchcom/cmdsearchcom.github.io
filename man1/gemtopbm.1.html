NAME

gemtopbm - replaced by gemtopnm

DESCRIPTION

This program is part of Netpbm(1)
gemtopbm was replaced in Netpbm 9.1 (May 2000) by gemtopnm(1)
gemtopnm is backward compatible with gemtopbm, but works on color images as well.

NAME
lit2epub - simple LIT to EPUB conversion

SYNOPSIS
lit2epub lit-document [epub-document]

DESCRIPTION
lit2epub is a command line tool for converting a LIT document to an EPUB document. It uses clit and zip for the conversion.

lit2epub needs to be run with at least one arguments, the LIT document to be converted. The second argument, epub-document, is the resulting EPUB document. If it is not speciâ
fied, the output EPUB is named after the input LIT document, with the .lit extension changed in .epub, and placed in the same directory as the LIT document.

DRPYTHON(1) General Commands Manual DRPYTHON(1)

NAME

DrPython - simple and customizable editor for the Python language

SYNOPSIS

drpython [ files ... ]

DESCRIPTION

This manual page documents briefly the drpython commands. This manual page was written for the Debian distribution because the original program does not have a manual page. For more information, refer to DrPython help menu.
drpython is a clean and simple yet powerful and highly customizable editor/environment for developing programs written in Python.

OPTIONS

This program has no options.

SEE ALSO

python(1), idle(1).

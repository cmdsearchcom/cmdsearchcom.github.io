NAME
StarDict - A Cross-Platform and international dictionary written in Gtk2

SYNOPSIS
stardict

or select StarDict from the Utility submenu of the Applications menu.

DESCRIPTION
StarDict is a Cross-Platform and international dictionary written in Gtk2. It has powerful features such as "Glob-style pattern matching," "Scan selection word," "Fuzzy query,"
etc.

For full documentation, see the StarDict GNOME Help Documentation.

OPTIONS
-h do not show splash screen

NAME

backup-driver - entry point for cron or inittab to start backups

SYNOPSIS

backup-driver [-test]
 

DESCRIPTION

`backup-driver' is the script to actually run to do a backup. It assumes that it is being run from cron or inittab unless passed the `test' option

OPTIONS

-test
Do not attempt to use backup-bringup to change runlevel. Use this argument when running backup-driver from the command line.

FILES

/etc/chiark-backup/settings.pl
Configuration file for the whole of chiark-backup

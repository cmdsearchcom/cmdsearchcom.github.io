NAME

bvd - Boogie Verification Debugger

SYNOPSIS

bvd

DESCRIPTION

bvd is an interactive verification condition debugger for for Microsoft Research's Boogie programming language.

OPTIONS

bvd is a graphical program and accepts no options on the command line.

NAME
claws-mail-spamassassin - Filtering messages through SpamAssassin.

DESCRIPTION
This manual page documents briefly the claws-mail-spamassassin plugin.

This manual page was written for the Debian distribution because the original program does not have a manual page.

claws-mail-spamassassin is a plugin (loadable module) for the Claws Mail mailer.

This plugin allows filtering incoming mails usin SpamAssassin to detect spam messages.

The plugin also provides the ability to teach spamd to recognize spam and ham, using the external command sa-learn in local modes or spamd in TCP mode (this one requires
SpamAssassin >= 3.1.x).

A toolbar button for marking messages as spam or ham can be added to the main window or the message window (see âConfigurationâ menu, âPreferencesâ dialog, âCustomize toolbarsâ
option).

Messages classified as spam are moved to the configured folder or the defaul trash if no folder was configured.

USAGE
Before using a plugin you must instruct Claws Mail to load it on startup.

For this you must go âConfigurationâ menu on main window toolbar, open âPlugins...â dialog, click on the âLoad plugin...â button and select the plugin file, named
spamassassin.so, and press the âOpenâ button.

FILES
/usr/lib/claws-mail/plugins/spamassassin.so
The loadable module for claws-mail-spamassassin.

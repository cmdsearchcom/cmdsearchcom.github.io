NAME
lxsettings-daemon â lightweight XSettings daemon for LXDE

SYNOPSIS
lxsettings-daemon -f file

This manual page documents briefly the lxsettings-daemon command.

DESCRIPTION
lxsettings-daemon is a program that provides XSettings daemon. The daemon handles the session settings for applications.

lxsettings-daemon is a part of the LXDE project.

OPTIONS
-f file, --file=file path to the configuration file

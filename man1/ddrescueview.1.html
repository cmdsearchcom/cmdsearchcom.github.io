NAME
ddrescueview - graphical viewer for GNU ddrescue log files

SYNOPSIS
ddrescueview

DESCRIPTION
This small tool allows the user to graphically examine ddrescue’s log files in a user friendly GUI application. The Main window displays a block grid with each block’s color repâ
resenting the block types it contains. Many people know this type of view from defragmentation programs.

OTHER INFO

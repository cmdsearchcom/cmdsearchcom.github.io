NAME
jsb-stop - The JSONBOT stop program

SYNOPSIS
jsb-stop stops a running JSONBOT

DESCRIPTION
If you need to stop a running JSONBOT (shell) you can do that with this program. It uses a pid file found in the <datadir>/run directory.

USAGE
Usage: jsb-stop

Options:
--version show program’s version number and exit
-h, --help show this help message and exit
-d DATADIR, --datadir=DATADIR
datadir to use

DOCUMENTATION
See http://jsonbot.org for more documentation or see http://jsonbot.googlecode.com

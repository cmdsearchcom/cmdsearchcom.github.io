NAME
tsssignapp - manual page for tsssignapp v713

DESCRIPTION
signapp

Runs TPM2_Sign application, including creating and loading a primary storage key and primary key

[-hi hierarchy e, o, p, n (default null)] [-pwdp password for primary key (default empty)] [-pwdk password for signing key (default empty)] [-halg [sha1, sha256, sha384]
(default sha256)] -if input message to hash and sign

SEE ALSO
The full documentation for tsssignapp is maintained as a Texinfo manual. If the info and tsssignapp programs are properly installed at your site, the command

info tsssignapp

should give you access to the complete manual.

NAME
classmate-screen-switch - program to switch to a virtual screen on a classmatePC

SYNOPSIS
classmate-screen-switch

DESCRIPTION
This manual page documents briefly the classmate-screen-switch command.

classmate-screen-switch adds an applet to the classmatePC notification area that enables you to switch to a bigger virtual screen

OPTIONS
This program follows the usual GNU command line syntax but has no options. This manpage is only here to fulfill the debian policy. The app is started by a .desktop file from
/etc/xdg/autostart.

NAME
jana-ecal-time-2 - program to test if DST+Zone auto-adjust works

SYNOPSIS
jana-ecal-time-2

DESCRIPTION
This manual page documents briefly the jana-ecal-time-2 command.

This manual page was written for the Debian distribution because the original program does not have a manual page.

jana-ecal-time-2 is a program that creates a time object for 2:00 1/1/2007, GMT/BST (+1) and changes the timezone to FJT (+12). If all goes well, the time should be adjusted
forward by 11 hours.

SEE ALSO
jana-ecal-event(1), jana-ecal-store-view(1), jana-ecal-time(1)

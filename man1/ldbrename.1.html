NAME
ldbrename - Edit LDB databases using your favorite editor

SYNOPSIS
ldbrename [-h] [-o options] {olddn} {newdn}

DESCRIPTION
ldbrename is a utility that allows you to rename trees in an LDB database based by DN. This utility takes two arguments: the original DN name of the top element and the DN to
change it to.

OPTIONS
-h
Show list of available options.

-H <ldb-url>
LDB URL to connect to. See ldb(3) for details.

-o options
Extra ldb options, such as modules.

ENVIRONMENT
LDB_URL
LDB URL to connect to (can be overrided by using the -H command-line option.)

VERSION
This man page is correct for version 1.1 of LDB.

ldb(3), ldbmodify, ldbdel, ldif(5)

NAME

ladr4-apps - undocumented LADR4 applications

DESCRIPTION

Some programs in the ladr4-apps package currently have no manual pages. You can obtain documentation on some of these applications via the prover9 manual, which is available on Debian systems via the package prover9-doc, at /usr/share/doc/prover9-doc/manual/index.html. Alternatively invoking the application with the -help option may produce documentation. Patches to add manual pages are welcome, and may be sent to the Debian package maintainer, whose details are listed below.

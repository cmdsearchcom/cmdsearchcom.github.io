NAME
mlink - calculates lod scores and risk with two of more loci

SYNOPSIS
mlink [options] ipedfile.dat pedfile.dat

DESCRIPTION
This manual page documents briefly the mlink command. In fact this is a very raw intend to provide a manpage because the Debian GNU/Linux distribution requires each program to
have a manpage. Any enhancement would be greatly apreciated.

mlink is part of the fastlink software package.

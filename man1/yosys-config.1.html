NAME
yosys-config - Yosys Open SYnthesis Suite Config

SYNOPSIS
yosys-config [ -exec ] [ --prefix pf ] args

DESCRIPTION
This manual page documents briefly the yosys-config command.

yosys-config is a shell script which can be used to query compiler options and other information needed for building loadable modules for yosys. It is similar in behaviour to
pkg-config.

OPTIONS
A summary of options is included below.

--cxx

--cxxflag

--ldflags

--ldlibs

--bindir

--datdir

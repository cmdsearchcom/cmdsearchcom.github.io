NAME

atitoppm - YUV conversion program from the GATOS package.

SYNOPSIS

atitoppm < gatos.yuv

DESCRIPTION

The program atitoppm convert the ".yuv" files created by gatos to a series of " .ppm" files.

gatos, gatos-conf, atitv, atisplit, atitogif, atitojpg, xatitv, yuvsum, scanpci.

NAME
hdf24to8 - convert an 24-bit image in an HDF file to a 8-bit image in another HDF file

SYNOPSIS
hdf24to8 input_hdf_file output_hdf_file

DESCRIPTION
hdf24to8 converts a 24-bit image in an HDF file into an 8-bit image in an other HDF file.

SEE ALSO
hdf(5), hdf8to24(1)

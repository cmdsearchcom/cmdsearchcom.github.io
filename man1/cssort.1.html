NAME
cssort - manual page for cssort 1.3.2

SYNOPSIS
cssort [options] [file1.err [...]], where options are:

DESCRIPTION
--key arg (=path)
checker, path

-q [ --quiet ]
do not report any parsing errors

--color
use colorized console output (default if connected to a terminal)

--no-color
do not use colorized console output

--help produce help message

--version
print version

NAME
ocaml-xgettext - program to extract translatable strings from OCaml source file.

SYNOPSIS
ocaml-xgettext [camlp4 arguments] [filename]

DESCRIPTION
This manual page documents briefly the ocaml-xgettext

ocaml-xgettext is a camlp4 top level. This top level is compiled with the printer module pr_gettext.cmo. It outputs an OCaml marshalled data structure that can only be understood
by ocaml-gettext. The purpose of this program is to be a back end for OCaml source code string extraction. You should not use it directly.

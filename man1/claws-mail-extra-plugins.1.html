NAME
claws-mail-extra-plugins - Claws Mail extra plugins meta-package.

DESCRIPTION
claws-mail-extra-plugins is a meta package for installing all extra plugins for the Claws Mail mailer.

See below for the corresponding manpages of each installed plugin.

HISTORY
The extra plugins is a collection of plugins which was initially released as a separate tarball download containing all the contributed plugins from the Claws Mail website.

To allow faster releases, early in 2013 all the extra plugins were incorporated to the main CVS tree, and are now part of the main (and only) release tarball.

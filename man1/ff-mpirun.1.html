NAME
ff-mpirun - a tool for running jobs in parallel within FreeFem++

SYNOPSIS
ff-mpirun

DESCRIPTION
This manual page documents briefly the scope of the ff-mpirun command.

ff-mpirun is a script that facilitates the process of running jobs in parallel within FreeFem++. It is part of the FreeFem++ suite which is used for the solution of Partial Difâ
ferential Equations (PDE’s) via the Finite Element Method (FEM).

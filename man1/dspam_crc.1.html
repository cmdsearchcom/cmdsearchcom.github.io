NAME
dspam_crc - calculate the DSPAM CRC/hash value for a text token

SYNOPSIS
dspam_crc [token]

DESCRIPTION
dspam_crc is used to calculate the DSPAM CRC/hash value for a text token.

OPTIONS
[token]
Data that will be CRC-ized/hashed.

EXIT VALUE
0 Operation was successful.
other Operation resulted in an error.

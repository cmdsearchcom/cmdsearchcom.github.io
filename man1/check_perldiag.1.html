NAME
check_perldiag - check a localized version of perldiag for consistency

SYNOPSIS
From the command line:

check_perldiag POD2::FR::perldiag

DESCRIPTION
This script compares a translated version of perldiag with the default perldiag installed with perl. It compares each error message in the two files and tells you if they do not
match. A warning is issued if the two files contain a different number of error messages defined.

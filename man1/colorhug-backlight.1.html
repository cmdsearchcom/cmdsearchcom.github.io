NAME
colorhug-backlight - Backlight utility for the ColorHug colorimeter

SYNOPSIS
colorhug-backlight [ --verbose ] [ --repair ]

DESCRIPTION
This manual page documents briefly the colorhug-backlight command.

colorhug-backlight uses the ColorHug device to alter the backlight level to compensate for the changing ambient light level.

OPTIONS
This program follows the usual GNU command line syntax, with long options starting with two dashes (‘-’). A summary of options is included below.

--help
Show summary of options.

--verbose
Show extra debugging information.

NAME
tsspcrextend - manual page for tsspcrextend v713

DESCRIPTION
pcrextend

Runs TPM2_PCR_Extend

-ha pcr handle

[-halg [sha1, sha256, sha384] (default sha256)]

-halg may be specified more than once

-ic data string, 0 pad appended to halg length

-if data file, 0 pad appended to halg length

SEE ALSO
The full documentation for tsspcrextend is maintained as a Texinfo manual. If the info and tsspcrextend programs are properly installed at your site, the command

info tsspcrextend

should give you access to the complete manual.

NAME

maki - IRC daemon
 

SYNOPSIS

maki [OPTION...]
 

DESCRIPTION

maki provides a central IRC daemon clients can connect to.
 

OPTIONS

-?, --help
Show help options.
-d, --daemon
Run as daemon.
--dbus-server
Start DBus server.
-v, --verbose
Output debug messages.
--version
 

EXAMPLES

$ maki --daemon --verbose

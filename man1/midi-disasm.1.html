NAME
midi-disasm - disassemble midi files

SYNOPSIS
midi-disasm [ -d ] [ -s ] [ -q ] [ file ]

DESCRIPTION
midi-disasm disassembles a standard midi file into readable/editable text. midi-disasm prefers the file suffix ".mid" -- it will supply it if a file by the name given on the
command line cannot be found. Its output is to stdout.

There are the following options for midi-disasm:

-d displays times as deltas from the time of the last midi event. A keyword "DELTA" is placed in the header line in the disassembly.

-s displays times in seconds and factions instead of midi ticks; the format is "<seconds>.<centiseconds>.<16nths of a centisecond>".

-q suppresses note-on and note-off output lines.

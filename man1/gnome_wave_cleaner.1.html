NAME
gwc - GNOME Wave Cleaner, Audio file denoiser

SYNOPSIS
gwc filename

gnome_wave_cleaner filename

DESCRIPTION
This manual page documents briefly the gnome_wave_cleaner , the GNOME wave cleaner application.

This manual page was written for the Debian distribution because the original program does not have a manual page. Instead, it has documentation in the HTML format; see below.

gwc is an audio file denoiser. It can be used to remove unwanted clicks and background noise from your recordings. It has a GNOME (gtk) user interface and all operations are perâ
formed through that.

OPTIONS
gwc follows the usual GNU command line syntax, with long options starting with two dashes (‘-’).

Options for gwc are the common GNOME options. For further information on these see the output of the --help option.

--help Show summary of options.

--version
Show version of program.

--usage
Show usage summary.

SEE ALSO
The help menu entry in gwc, and the files in /usr/share/gnome/help/gnome_wave_cleaner contain further documentation.

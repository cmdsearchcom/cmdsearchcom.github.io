NAME
cipuxpasswd - XML-RPC Client to change own password.

VERSION
version 3.4.0.3

USAGE
cipuxpasswd

OPTIONS
None

DESCRIPTION
Implements a XML-RPC client to change once own password using the CipUX XML-RPC server.

REQUIRED ARGUMENTS
None.

DIAGNOSTICS
None.

EXIT STATUS
0 on success.

CONFIGURATION
None. See CipUX::Passwd.

DEPENDENCIES
CipUX::Passwd
version

INCOMPATIBILITIES
None reported.

BUGS AND LIMITATIONS
No bugs have been reported.

SEE ALSO
See the CipUX webpage and the manual at <http://www.cipux.org>

See the mailing list <http://sympa.cipworx.org/wws/info/cipux-devel>

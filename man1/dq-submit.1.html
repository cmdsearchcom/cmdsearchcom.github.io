NAME
dq-submit - submit tasks to an IPC::DirQueue queue

SYNOPSIS
dq-submit --dir qdirectory file ...

DESCRIPTION
dq-submit will submit tasks to an "IPC::DirQueue" directory. Multiple task files can be specified on the command-line.

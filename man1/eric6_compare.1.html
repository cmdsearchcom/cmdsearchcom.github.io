NAME
eric6_compare - Simple graphical compare tool

SYNOPSIS
eric6_compare [options] files...

DESCRIPTION
This manual page documents briefly the eric6_compare program. This manual page was written for the Debian distribution because the original program does not have a manual page.
Instead, it has documentation provided in HTML format.

OPTIONS
These programs follow the usual GNU command line syntax, with long options starting with two dashes (‘-’). A summary of options is included below.

-h, --help
Show summary of options.

--version
Show version of program.

--nokde
Don’t use KDE widgets

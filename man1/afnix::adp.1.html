NAME
adp - afnix documentation processor

SYNOPSIS
axi [i] afnix-adp.axl [m] [][] [options]

OPTIONS
[h]
prints the help message

[v]
prints the program version

[x]
enable xhtml output mode

[o] file
set the output file name

DESCRIPTION
adpis the documentation processor. The adpoperates with an uri and produces a high quality documentation suitable for various distribution media.

VERSION
The current version is the 2.6.3release.

AUTHOR
The adpclient has be written by (amaury@afnix.org) Amaury Darsch.

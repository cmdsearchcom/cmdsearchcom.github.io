NAME
pmlcreate â construct PhotoML files from a single input file in a simple text format

SYNOPSIS
pmlcreate [-h] [-e] [-x] inputfile

DESCRIPTION
pmlcreate constructs PhotoML files from a single input file in a simple text format. The input file summarises basic information such as camera body, film, processing, image creâ
ation date, and exposure details. The output represents an initial version suitable for further editing.

OPTIONS
pmlcreate accepts the following options:

-h Display usage information.

-e Display an example input file.

-x Output intermediate XML representation of input file data.

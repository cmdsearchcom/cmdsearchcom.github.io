NAME
lrzcat - Uncompress LRZ files to STDOUT

SYNOPSIS
lrzcat [options] FILE [... FILE]

DESCRIPTION
lrzcat is identical to "lrzip -d -o -" used to decompress files to STDOUT.

OPTIONS
See lrzip(1).

ENVIRONMENT
None.

FILES
None.

NAME
immstool - debugging tool for imms

DESCRIPTION
immstool is mostly a debugging tool used by imms developer. However it has some functionality that advanced user would like to use. For details please see upstream homepage:
http://imms.luminal.org/.

SEE ALSO
analyzer(1), immsd(1)

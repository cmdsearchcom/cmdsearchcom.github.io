NAME

bkr-whoami - Show your Beaker username

SYNOPSIS

bkr whoami [options]

DESCRIPTION

Prints to stdout a dict with username and proxied_by_username if proxy permissions are granted to this user.

OPTIONS

Common bkr options are described in the Options section of bkr(1).

EXIT STATUS

Non-zero on error, otherwise zero.

NAME
idjc-noauto - No automatic profile

SYNOPSIS
idjc noauto [-h]

DESCRIPTION
Removes the automatic profile property after which when starting idjc if no profile is chosen on the command line the Profile Manager will appear in order that one be selected
manually.

Options
-h, --help
Show a help message.

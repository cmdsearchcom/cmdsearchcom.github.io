FREEBIRTH(1) General Commands Manual FREEBIRTH(1)

NAME

freebirth — Bass synthesizer/sample player/sequencer

SYNOPSIS

freebirth

DESCRIPTION

This manual page documents briefly the freebirth program.
This manual page was written for the Debian GNU/Linux distribution because the original program does not ship with any documentation. There are some instructions available on the freebirth website (link to URL http://www.bitmechanic.com/projects/freebirth/doc.html) .

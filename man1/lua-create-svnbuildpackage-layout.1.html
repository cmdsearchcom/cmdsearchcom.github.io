NAME

lua-create-svnbuildpackage-layout - program to create an svn-buildpackage compliat repository for a lua package.

SYNOPSIS

lua-create-svnbuildpackage-layout PKG

DESCRIPTION

lua-create-svnbuildpackage-layout is a program that creates a standard filesystem layout for svn-buildpackage and adds some templates files for dh-lua.

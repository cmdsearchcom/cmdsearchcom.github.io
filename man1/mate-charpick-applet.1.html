NAME
mate-charpick-applet - Character Picker Applet for the MATE panel.

SYNOPSIS
mate-charpick-applet [ --activate-goad-server [=GOAD_ID] ]

DESCRIPTION
mate-charpick-applet allows you to easily write many characters which are not available on standard keyboards such as accented characters, certain mathematical symbols and puncâ
tuation, and some other special symbols.

To add this applet to a Panel, right-click on the Panel and choose Applets->Utility->Character Picker.

OPTIONS
--activate-goad-server
(Internal use only) GOAD server ID to activate.

GOAD_ID is usually the applet name [mate-charpick-applet]

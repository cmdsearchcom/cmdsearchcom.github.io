NAME
lspgpot - extracts the ownertrust values from PGP keyrings and list them in GnuPG ownertrust format.

SYNOPSIS
lspgpot

DESCRIPTION
lspgpot extracts the ownertrust values from PGP keyrings and list them in GnuPG ownertrust format.

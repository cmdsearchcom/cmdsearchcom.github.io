PMD2RAW(1) User Commands PMD2RAW(1)

NAME

pmd2raw - manual page for pmd2raw 0.0.3

SYNOPSIS

pmd2raw [ OPTION] INPUT

DESCRIPTION

`pmd2raw' is used to test libpagemaker.

OPTIONS

--callgraph
display the call graph nesting level
--help
show this help message
--version
show version information and exit

NAME

fep - FEP server running sub-shell

SYNOPSIS

fep [options] [command]
 

DESCRIPTION

fep is a shell wrapper with FEP support. It can be controlled by clients (such as fepcli) running inside the sub-shell, through the libfep library. fep executes the SHELL environment variable. You can specify another command as its argument.

OPTIONS

The fep command follows the usual GNU command line syntax, with long options starting with two dashes (`-'). A summary of options is included below.
-h, --help
Show summary of options.
-l, --log-file=FILE
Specify a log file.

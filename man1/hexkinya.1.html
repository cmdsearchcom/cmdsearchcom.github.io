NAME
hexkinya - Create the Private Use Area Kinya syllables

SYNOPSIS
hexkinya > plane15.hex

DESCRIPTION
hexkinya contains pre-defined initial, middle, and final alphabet glyphs to form syllables. The output is the Kinya Syllables Private Use Area block of the ConScript Unicode
Registry (CSUR). The output range is U+0F0000 through U+0FDE6F, inclusive.

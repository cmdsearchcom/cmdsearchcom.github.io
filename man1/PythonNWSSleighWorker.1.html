NAME
PythonNWSSleighWorker - Executes the Python NWS Sleigh Worker

SYNOPSIS
PythonNWSSleighWorker

DESCRIPTION
This script is executed on the worker machines, possibly via ssh, to start the Python NWS sleigh workers. It is executed when constructing a Sleigh object, and is intended for
internal use.

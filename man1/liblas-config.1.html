NAME
liblas-config - query libLAS build options

SYNOPSIS
liblas-config [--libs] [--cflags] [--cxxflags] [--defines] [--includes] [--version]

DESCRIPTION
liblas-config allows querying the options used to build the libLAS library.

OPTIONS
--libs Libraries

--cflags
Compiler flags

--cxxlibs
C++ Compiler flags

--defines
Compiler defines

--includes
Compiler includes

--version
library version

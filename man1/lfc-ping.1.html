NAME
lfc-ping - check if name server is alive and print its version number

SYNOPSIS
lfc-ping [ -h server ]

DESCRIPTION
lfc-ping checks if the name server is alive and prints its version number.

OPTIONS
-h server
specify the name server hostname. This can also be set thru the LFC_HOST environment variable.

EXIT STATUS
This program returns 0 if the operation was successful or >0 if the operation failed.

NAME
atitogif - YUV conversion program from the GATOS package.

SYNOPSIS
atitogif < gatos.yuv

DESCRIPTION
The program atitogif convert the ".yuv" files created by gatos to a series of ".gif" files.

NOTE
Unlike atitoppm, this application requires various utilities to be present (such as ppmtogif, etc...). If they are not, this program will fail.

gatos, gatos-conf, atitv, atisplit, atitojpg, atitoppm, xatitv, yuvsum, scanpci.

DESCRIPTION
eventssquencecomplete

[-ha pcr handle (default NULL)] -hs seqeunce handle [-pwds password for sequence (default empty)] [-if input file to be added (default no data)] [-of1 sha1 output digest
file] [-of2 sha256 output digest file] [-of3 sha384 output digest file]

-se[0-2] session handle / attributes (default PWAP)

01 continue

NAME
GAiksaurus - English-language thesaurus

SYNOPSIS
gaiksaurus [word]

DESCRIPTION
Aiksaurus is an English-language thesaurus library. This is a graphical interface to it.

-v, --version
Output version information and quit.

-h, --help
Output this help message and quit.

For more help, visit

http://www.aiksaurus.com/

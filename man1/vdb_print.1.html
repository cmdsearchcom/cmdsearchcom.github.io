NAME

vdb_print - prints information about OpenVDB grids

SYNOPSIS

vdb_print in.vdb [in.vdb ...] [options]

DESCRIPTION

Which: prints information about OpenVDB grids Options:
-l, -stats
long printout, including grid statistics
-m, -metadata
print per-file and per-grid metadata
-version
print version information

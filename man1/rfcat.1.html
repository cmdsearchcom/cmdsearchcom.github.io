NAME
rfcat - Remote file concatenation to standard output

SYNOPSIS
rfcat filename1 [filename2[...]]

DESCRIPTION
The remote file I/O concatenation program provides an interface to the LCG remote file I/O daemon (rfiod) for concatenating files from remote and/or local hosts to standard outâ
put. Each filename argument is either a remote file name of the form:

hostname:path

or a local file name (not containing the :/ character combination)

or an HSM filename /castor/...

With no filename, or when filename is -, read standard input. The streaming mode (v3) is used.

RETURN CODES
0 Ok.
1 Bad parameter.
2 System error.
16 Device or resource busy.
28 No space left on device.
196 Request killed.

cat(1), rfiod(1)

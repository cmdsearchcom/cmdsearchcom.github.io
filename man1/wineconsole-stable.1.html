NAME
wineconsole - The Wine console

SYNOPSIS
wineconsole [option] command

DESCRIPTION
wineconsole is the Wine console manager, used to run console commands and applications. It allows running the console either in the current terminal (curses) or in a newly made
window (user).

OPTIONS
--backend={user|curses}
If user is chosen, a new window will be created for the console. The curses option will make wineconsole try to setup the current terminal as a Wine console.

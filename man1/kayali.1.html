NAME
kayali - a qt and maxima-based computer algebra system

DESCRIPTION
This manual page explains the kayali program.

kayali is a Qt based Computer Algebra System (CAS) that can also be used as an advanced replacement for KDE KCalc. It is essentially a front end GUI for Maxima (and is easily
extended to other CAS back-ends) and Gnuplot

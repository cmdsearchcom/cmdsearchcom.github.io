NAME
pjtoppm - convert an HP PaintJet file to a PPM image

SYNOPSIS
pjtoppm

[paintjet]

DESCRIPTION
This program is part of Netpbm(1)

pjtoppm reads an HP PaintJet file as input and converts it into a PPM image. This was a quick hack to save some trees, and it only handles a small subset of the paintjet comâ
mands. In particular, it will only handle enough commands to convert most raster image files.

REFERENCES
HP PaintJet XL Color Graphics Printer User’s Guide

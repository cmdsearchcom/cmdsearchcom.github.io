NAME
PTtiffdump - Compare two TIFF images

SYNOPSIS
PTtiffdump [options] tiff_file1 tiff_file2

DESCRIPTION
PTtiffdump compares the contents of 2 different tiff files. If the byte is different, it gets output.

OPTIONS
-o Overwrite output file if it exists.

-q Quiet run

-h Output help summary.

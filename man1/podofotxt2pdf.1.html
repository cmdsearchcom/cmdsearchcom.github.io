NAME
podofotxt2pdf - convert plain text files into PDF files

SYNOPSIS
podofotxt2pdf [-fontname <name>] [-utf8] [inputfile] [outputfile]

DESCRIPTION
podofotxt2pdf is one of the command line tools from the PoDoFo library that provide several useful operations to work with PDF files. It can convert a plain text file into a PDF
file.

OPTIONS
-fontname <name>

Use the font <name>.

-utf8

specifies that the input text is UTF8 encoded instead of Windows ANSI encoding.

[inputfile]

Input PDF file.

[outputfile]

Output PDF file.

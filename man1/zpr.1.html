NAME

zpr - print permutations or matrices

SYNOPSIS

zpr [ OPTIONS] <Binfile> [<Textfile>]

DESCRIPTION

This program prints the contents of a MeatAxe data file in readable format. The text produced by zpr can be converted into binary format by the zcv(1) program.
If there is only one argument on the command line, zpr writes to stdout. A second argument, if present, is taken as the output file name.
To find out the contents of a MeatAxe file, use the -s option. To generate output readble by GAP, use -G. Both options can be combined.

OPTIONS

-Q
Quiet, no messages.
-V
Verbose, more messages.
-T <MaxTime>
Set CPU time limit
-G, --gap
Produce output in GAP format.
-s, --summary
Show headers only.

NAME
fgjs - FlightGear Joystick Utility

SYNOPSIS
fgjs

DESCRIPTION
fgjs is a joystick utility for the FlightGear Flight Simulator. It allows you to generate an appropriate configuration file for your joystick. For more information visit
<http://wiki.flightgear.org/Joystick>.

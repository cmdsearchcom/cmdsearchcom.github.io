NAME
bwrap-oci - Run an OCI (Open Container Initiative) container through bubblewrap. bwrap-oci converts the OCI configuration file, a JSON document which specifies how to run the
container, to a command line for bubblewrap.

SYNOPSIS
bwrap-oci [OPTION...] [COMMAND]

DESCRIPTION
bwrap-oci converts an open containers spec file to a command line for bubblewrap and run a container through it.

OPTIONS
General options:

--help
Print help and exit

--configuration
Specify the configuration file to use. If not specified ’configuration.json’ is used.

--dry-run
Print the command line used for bubblewrap but do not execute it.

NAME
magicsort - Categorize files by their file(1) magic

SYNOPSIS
magicsort directory

DESCRIPTION
Invokes the system’s file utility on all files in directory non-recursively. For each different string that file outputs it will create a sub-directory of that name and move the
file in there.

SEE ALSO
magicrescue(1), dupemap(1)

NAME

hugin_stitch_project - Stitch a hugin project

SYNOPSIS

hugin_stitch_project [options] [<project> <images> ...]

DESCRIPTION

A GUI tool to stitch a single hugin project.

OPTIONS

-h|--help
Show help message
-o|--output=<str>
Output prefix
-t|--threads=<num>
Number of threads
-d|--delete
Delete .pto file after stitching

NAME

bibledit-bibletime - Bibledit BibleTime helper

DESCRIPTION

Bibledit-BibleTime interfaces between Bibledit and BibleTime. It is started by typing "bibledit-bibletime" on the commandline. For more information see the graphical user interface.

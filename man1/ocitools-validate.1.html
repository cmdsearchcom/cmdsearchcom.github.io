NAME

ocitools-validate - Validate a OCI bundle
 
 

SYNOPSIS

ocitools validate [ --help] [ --path[=PATH]
 
 

DESCRIPTION

Validate an OCI bundle
 
 

OPTIONS

--help
Print usage statement
 
**--path="PATH"
Path to bundle

NAME

gpsketcher - a Glade-Perl code sketcher for Gtk2::GladeXML::Simple

DESCRIPTION

gpsketcher 0.4, a Glade-Perl code sketcher for Gtk2::GladeXML::Simple
usage: gpsketcher -g glade_file [options]
[options]:
-h
This message
-o output_file
file to write the output code to

Prints the program version

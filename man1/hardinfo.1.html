HARDINFO(1) General Commands Manual HARDINFO(1)

NAME

hardinfo - shows hardware information in a GTK+ window

SYNOPSIS

hardinfo

DESCRIPTION

hardinfo parses some files in /proc and displays their information in a user-friendly window.
 
This manual page was written for the Debian distribution because the original program does not have a manual page.

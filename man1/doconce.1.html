NAME
doconce - run a Doconce command

SYNOPSIS
doconce command [OPTIONS]

DESCRIPTION
The doconce script takes a command and runs a corresponding utility. Run the script without arguments to see a list of available commands.

BUGS
Send comments, questions, bug reports etc. to doconce@googlegroups.com.

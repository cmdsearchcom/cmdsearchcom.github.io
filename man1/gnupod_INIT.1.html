NAME
gnupod_INIT - prepares a ’virgin’ iPod for GNUpod

SYNOPSIS
gnupod_INIT [OPTION]...

DESCRIPTION
Prepares a ’virgin’ iPod for GNUpod

--help display this help and exit

--version
output version information and exit

-m, --mount=DIRECTORY
iPod mountpoint. Note, gnupod_INIT ignores $IPOD_MOUNTPOINT

-d, --disable-convert
Don’t try to convert an existing iTunesDB (Only for experts)

-f, --france
Create IPOD_MOUNTPOINT/iPod_Control/Device/Limit. This File tells iPods with Firmware 1.x to limit the Volume to 100dB (EU/France law...). Firmware 2.x ignores this File.
(But you can use the --volume switch of mktunes.pl)

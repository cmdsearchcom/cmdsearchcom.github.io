NAME

ardour5-copy-mixer, ardour5-export, ardour5-fix_bbtppq - session utilities for the ardour digital audio workstation

SYNOPSIS

ardour5-copy-mixer [ --help ] [ arguments ]
 
ardour5-export [ --help ] [ arguments ]
 
ardour5-fix_bbtppq [ --help ] [ arguments ]

DESCRIPTION

ardour5-copy-mixer copies mixer-settings from one session to another.
 
ardour5-export exports an ardour session from the commandline.
 
ardour5-fix_bbtppq converts an ardour session with 5.0-5.3 midi sources to be compatible with ardour5.4.

OPTIONS

--help
Display detailed usage information.

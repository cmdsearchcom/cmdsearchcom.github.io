NAME

xpra_browser - Shows the xpra sessions found via mDNS

SYNOPSIS

xpra_browser [ FILENAME]

DESCRIPTION

xpra_browser is a browser for xpra(1) sessions found via mDNS, it allows you to easily connect to those sessions.

NAME
mate-window-properties - configure MATE network proxy

SYNOPSIS
mate-window-properties [OPTIONS...]

DESCRIPTION
This executable is part of the package ’mate-control-center’: The MATE Control Center.

The MATE window preference tool enables you to customize window behaviour such as focus behaviour, titlebar actions, compositing, etc. for your desktop.

OPTIONS
This program accepts the standard MATE and GTK options.

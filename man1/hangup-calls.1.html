NAME
hangup-calls

SYNOPSIS
hangup-calls

DESCRIPTION
Hangup-calls is a hangup script that hangs up on all calls not in the whitelist.

If hupmode is set to 3 (announcement mode), the sctipt will play the default recording or one set in ncidd.conf.

FILES
/usr/share/ncid/extensions/hangup-calls
/usr/share/ncid/recording/*.rmd
/etc/ncid/ncidd.whitelist

ncidd.8, ncidd.whitelist.5, hangup-closed-skel.1, hangup-message-skel.1, hangup-skel.1

NAME
asex - client for anonymous SEX

SYNOPSIS
asex [args]

DESCRIPTION
asex is a client for anonymous SEX.

This manual page was written for the Debian distribution because the original program does not have a manual page.

USAGE
asex [ID] files...
file input

asex ID [-]
file output

... | asex [ID]
pipe input

asex ID | ...
pipe output

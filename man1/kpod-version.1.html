NAME

kpod-version - Display the KPOD Version Information
 

SYNOPSIS

kpod version [ --help|-h]
 

DESCRIPTION

Shows the the following information: Version, Go Version, Git Commit, Build Time, OS, and Architecture.
 
kpod [GLOBAL OPTIONS]
 
kpod [GLOBAL OPTIONS] version [OPTIONS]
 

GLOBAL OPTIONS

--help, -h
Print usage statement

kpod(1), crio(8), crio.conf(5)

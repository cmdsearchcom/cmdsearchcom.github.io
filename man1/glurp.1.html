NAME

glurp - GTK+ frontend for controlling Music Player Daemon (MPD)

SYNOPSIS

glurp [-v|--version] [-h|--help] [--config= file]

DESCRIPTION

glurp is a GTK+ client for MPD with a simple single-window interface that features a playlist, a database list, a list of streams and also other controls.

USAGE

Using glurp should be intuitive since it is based around a simple GUI. Simply run glurp and choose the config tab to configure and connect it to a running MPD process.

OPTIONS

-v, --version
prints version string.
-h, --help
prints short usage information.
--config=file
use the given file as the config file instead of the default ( ~/.glurp).

SEE ALSO

mpd(1), mpc(1).

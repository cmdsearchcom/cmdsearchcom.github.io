NAME
vbox - interact with recorded messages

SYNOPSIS
vbox [-f|--force] [-o|--mono] [-s|--noledstatus] [-c|--playcmd player] [-m|--hostname server] [-p|--port port] [-r |--reload SECS] [-v|--version] [-h|--help]

DESCRIPTION
This is the program with which you can listen to recorded messages.

autovbox(1), vboxtoau(1), vboxmode(1), vboxcnvt(1), vbox(5)

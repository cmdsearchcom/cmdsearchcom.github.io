NAME
mtx-unzip

SYNOPSIS
mtxrun --script unzip [ OPTIONS ... ] [ FILENAMES ]

DESCRIPTION
Simple Unzipper

OPTIONS
--list list files in archive

--junk flatten unzipped directory structure

--extract
extract files

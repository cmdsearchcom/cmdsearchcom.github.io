NAME
panoinfo - List details about the libpano13 library

SYNOPSIS
panoinfo

DESCRIPTION
panoinfo is a simple utility built alongside the libpano13 library that outputs a list of details about the currently installed version of this library. It is particularly useful
for debugging.

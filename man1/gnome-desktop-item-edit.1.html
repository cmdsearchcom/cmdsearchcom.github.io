NAME

gnome-desktop-item-edit — tool to edit .desktop file

SYNOPSIS

gnome-desktop-item-edit [--create-new ] [--help ]

DESCRIPTION

gnome-desktop-item-edit allows you to edit .desktop file.

OPTIONS

--create-new
Create new file in the given directory.
--help
Display help

gnome-options(7)

NAME
euare-roleupdateassumepolicy - Update a role’s trust policy, the policy that allows entities to assume a role

SYNOPSIS
euare-roleupdateassumepolicy (-f FILE | -s SERVICE) [-o]

[--as-account ACCOUNT] [-U URL]
[--region USER@REGION] [-I KEY_ID] [-S KEY] [--security-token TOKEN] [--debug] [--debugger] [--version] [-h] ROLE

DESCRIPTION
Update a role’s trust policy, the policy that allows entities to assume a role

positional arguments:
ROLE role to update (required)

optional arguments:
-f FILE
file containing the policy for the new role

-s SERVICE, --service SERVICE
service to allow access to the role (e.g. ec2.amazonaws.com)

-o also print the role’s new policy

--as-account ACCOUNT
[Eucalyptus cloud admin only] run this command as the administrator of another account

-U URL, --url URL
identity service endpoint URL

--region USER@REGION
region and/or user names to search when looking up config file data

-I KEY_ID, --access-key-id KEY_ID

-S KEY, --secret-key KEY

--security-token TOKEN

--debug
show debugging output

--debugger
launch interactive debugger on error

--version
show the program’s version and exit

-h, --help
show this help message and exit

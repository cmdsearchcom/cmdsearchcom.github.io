NAME

gastman - graphical console for asterisk

SYNOPSIS


gastman

DESCRIPTION

This manual page documents briefly the gastman command.
The gastman is a GK2 based graphical console for Asterisk. Three views are offered : a graphical view (where you can manipulate calls), the list view (calls), queue view and an asterisk console.

NAME
estload - stresser for web applications

SYNOPSIS
estload [-t num] [-l num] [-i num] [-p] [-q] [file|url]

DESCRIPTION
This manual page documents briefly the estload commands.

estload is a program that stress web applications on url.

OPTIONS
A summary of options is included below.

-t number of threads.

-l number of loops.

-i interval time in msec.

-p print out the response.

-q quiet.

estcmd(1), estmaster(1), estconfig(1).

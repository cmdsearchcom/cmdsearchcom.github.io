NAME
podofotxtextract - Extract all text from a PDF file

SYNOPSIS
podofotxtextract [inputfile]

DESCRIPTION
podofotxtextract is one of the command line tools from the PoDoFo library that provide several useful operations to work with PDF files. It can extract text from a PDF file.

OPTIONS
[inputfile]

Input PDF file.

NAME

ldaptor-fetchschema - command line LDAP schema fetcher
 

SYNOPSIS

ldaptor-fetchschema [ options]

OPTIONS

--base=
LDAP base dn
--service-location=
Service location, in the form BASEDN:HOST[:PORT]
 

DESCRIPTION

ldaptor-fetchschema takes a base distinguished name (DN) and fetches the LDAP schema definition from the LDAP server specified by --service-location.
 

SEE ALSO

ldaptor-find-server(1), ldaptor-search(1)

NAME

lizardfs-dirinfo - show directories stats

SYNOPSIS

lizardfs dirinfo [-n|-h|-H] OBJECT...

DESCRIPTION

dirinfo is extended, LizardFS-specific equivalent of du -s command. It prints summary for each specified object (single file or directory tree).

OPTIONS

-n, -h, -H
These options are described in lizardfs(1).

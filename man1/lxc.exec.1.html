NAME

lxc exec - Execute commands in containers

SYNOPSIS

lxc exec [<remote>:]<container> [-t] [ -T] [-n] [--mode=auto|interactive|non-interactive] [ --env KEY=VALUE...] [ --] <command line>

DESCRIPTION

Execute commands in containers.
Mode defaults to non-interactive, interactive mode is selected if both stdin AND stdout are terminals (stderr is ignored).

OPTIONS

-T
(= false) Disable pseudo-terminal allocation
--debug
(= false)
Enable debug mode
--env
(= [])
Environment variable to set (e.g. HOME=/home/foo)
--force-local
(= false)
Force using the local unix socket
--mode (= "auto")
Override the terminal mode (auto, interactive or non-interactive)
-n
(= false) Disable stdin (reads from /dev/null)
--no-alias
(= false)
Ignore aliases when determining what command to run
-t
(= false) Force pseudo-terminal allocation
--verbose
(= false)
Enable verbose mode

NAME
lxc monitor - Monitor a local or remote LXD server

SYNOPSIS
lxc monitor [<remote>:] [--type=TYPE...]

DESCRIPTION
Monitor a local or remote LXD server.

By default the monitor will listen to all message types.

Message types to listen for can be specified with --type.

OPTIONS
--debug
(= false)

Enable debug mode

--force-local
(= false)

Force using the local unix socket

--no-alias
(= false)

Ignore aliases when determining what command to run

--type (= [])

Event type to listen for

--verbose
(= false)

Enable verbose mode

EXAMPLES
lxc monitor --type=logging

Only show log message.

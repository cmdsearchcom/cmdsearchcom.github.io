NAME
trash-put - Command line trash utility.

SYNOPSIS
trash-put [arguments] ...

DESCRIPTION
Trash-cli package provides a command line interface trashcan utility compliant with the FreeDesktop.org Trash Specification. It remembers the name, original path, deletion date,
and permissions of each trashed file.

ARGUMENTS
Names of files or directories to move in the trashcan.

EXAMPLES
$ trash-put foo # trashes foo

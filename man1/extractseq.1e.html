NAME
extractseq - Extract regions from a sequence

SYNOPSIS
extractseq -sequence sequence -regions range [-separate boolean] -outseq seqoutall

extractseq -help

DESCRIPTION
extractseq is a command line program from EMBOSS (âthe European Molecular Biology Open Software Suiteâ). It is part of the "Edit" command group(s).

OPTIONS
Input section
-sequence sequence

Required section
-regions range
Regions to extract. A set of regions is specified by a set of pairs of positions. The positions are integers. They are separated by any non-digit, non-alpha character.
Examples of region specifications are: 24-45, 56-78 1:45, 67=99;765..888 1,5,8,10,23,45,57,99 Default value: $(sequence.begin)-$(sequence.end)

Additional section
-separate boolean
If this is set true then each specified region is written out as a separate sequence. The name of the sequence is created from the name of the original sequence with the
start and end positions of the range appended with underscore characters between them, eg: XYZ region 2 to 34 is written as: XYZ_2_34 Default value: N

Output section
-outseq seqoutall

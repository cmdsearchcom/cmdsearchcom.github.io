NAME
zcv - convert text representation of a matrix to binary format

SYNOPSIS
zcv [OPTIONS] <TextFile> <DataFile>

DESCRIPTION
This program converts a text file into binary format. If the input file name is "-", input is read from stdin.

Text File Format
The text file is interpreted line by line. Empty lines are ignored completely. If the line contains one or more "#" characters, which may occur at any position, the first "#"
and all remaining characters on this line are ignored.

Each object, for example a matrix or a permutation, consists of a one line header followed by the data. Here are some examples of possible header formats:

matrix field=16 rows=10 cols=10
permutation degree=10026
polynomial field=2 degree=23

The header may be given in a different format, for example:

MeatAxeFileInfo := "matrix field=5 rows=100 cols=100";

Other header formats are supported for compatibility with older versions of the MeatAxe.

After an old type 1 header, the format of the data follwing the header is fixed. There must be at most 80 characters per line, lines must be filled as much as possible, and each
row of the matrix starts with a new input line. There are no blanks allowed to separate numbers. In all other modes, the data part consists of a sequence of integers in free
format, separated by any combination of blanks, tabs, or newlines.

OPTIONS
-Q Quiet, no messages.

-V Verbose, more messages.

-T <MaxTime>
Set CPU time limit

NAME

pmlupgrade ‐ upgrade PhotoML XML files to latest DTD version

SYNOPSIS

pmlupgrade [ -h] [-n] xmlfile [xmlfile] ...

DESCRIPTION

pmlupgrade is a shell script that upgrades PhotoML XML files to the latest DTD version.

OPTIONS

pmlupgrade accepts the following options:
-h
Display usage information.
-n
Preserve numeric character references instead of converting them into the referenced character.

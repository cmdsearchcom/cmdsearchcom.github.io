YUM-CHANGELOG(1) User Manuals YUM-CHANGELOG(1)

NAME

yum-changelog

SYNOPSIS

The yum-changelog package

DESCRIPTION

yum-changelog(1) is a Yum plugin for viewing package changelogs before/after updating. yum will invoke yum-changelog(1) plugin if the --changelog option or the changelog command is used with yum.

OPTIONS

--changelog
Show changelog delta of updated packages

COMMANDS

changelog
Show changelog data of packages listed (same format as the list command). The first argument is required and is either "all" for all the changelog entries, a date for the changelog entries since a specified point in time or a number for a given number of changelog entries.

FILES

/etc/yum/pluginconf.d/changelog.conf
The system wide configuration file. See yum-changelog.conf(5) for more information.

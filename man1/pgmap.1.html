NAME
pgmap - utility for getting information from kernel’s pagemap interface

SYNOPSIS
pgmap [-ndpFPs]

DESCRIPTION
Pgmap utility gets informations from /proc/kpagecount, /proc/kpageflags and /proc/[pid]/pagemap. Using libpagemap library for that.

OPTIONS
-h for help

-n simulate non-root = only RES and SWAP

-d without headers

-p prints numbers in pages (instead of default kB)

-F prints info from kpageflags file

-P pid prints only specified pid

-s ID sorting by [uss|pss|shr|res|swap|pid][+-]

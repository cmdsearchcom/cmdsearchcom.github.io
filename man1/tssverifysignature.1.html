TSSVERIFYSIGNATURE(1) User Commands TSSVERIFYSIGNATURE(1)

NAME

tssverifysignature - manual page for tssverifysignature v713

DESCRIPTION

verifySignature
Runs TPM2_Verifysignature
-hk key handle
[-halg [sha1, sha256, sha384] (default sha256)] [Asymmetric Key Algorithm]
[-rsa (default)] [-ecc curve (P256)]
-if input message file name
-ih input hash file name
-is signature file name
-raw signature in raw format (default TPMT_SIGNATURE)
[-tk ticket file name]

SEE ALSO

The full documentation for tssverifysignature is maintained as a Texinfo manual. If the info and tssverifysignature programs are properly installed at your site, the command
info tssverifysignature
should give you access to the complete manual.

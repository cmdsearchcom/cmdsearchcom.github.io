NAME
pmd2odg - manual page for pmd2odg 0.9.5

SYNOPSIS
pmd2odg [OPTIONS] INPUT [OUTPUT]

DESCRIPTION
‘pmd2odg’ converts Adobe PageMaker documents to ODF. If OUTPUT is omitted, the result is printed as Flat ODF to standard output.

OPTIONS
--help show this help message

--version
print version and exit

--stdout
print the result as flat XML to standard output

REPORTING BUGS
Report bugs to <https://sourceforge.net/p/libwpd/tickets/>.

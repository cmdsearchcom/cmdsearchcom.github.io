NAME

bdd - Mutli Reduced Ordered Binary Decision Diagrams
 
See the file man1/alc_origin.1.
 

DESCRIPTION

bdd is a library that enables to represent a boolean expression as a Multi Reduced Ordered Binary Decision Diagrams.

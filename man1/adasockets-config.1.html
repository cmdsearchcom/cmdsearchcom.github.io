NAME

adasockets-config - script to get information about the installed AdaSockets library

SYNOPSIS

adasockets-config [--version] [--libs] [--cflags]

DESCRIPTION

adasockets-config will echo a line on standard output with appropriate flags for gnatmake .
 

SEE ALSO

gnatmake(1), pkg-config(1)

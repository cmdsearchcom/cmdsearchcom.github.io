NAME
ibtoopdiff - Fabric Topology Matcher

SYNOPSYS
ibtopodiff [-v][-h] -t <topo file> -d <subnet file> -s <start node name>
-p <start port num> -g <start port guid>

DESCRIPTION
This utility performs matching between a given topology file (see man ibdm-topology-file) and a subnet.lst file (this file provides a dump of all links in the discovered fabric
and generated by ibdiagnet or OpenSM).

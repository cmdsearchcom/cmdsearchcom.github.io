NAME
dpigs - Show which installed packages occupy the most space

SYNOPSIS
dpigs [options]

DESCRIPTION
dpigs sorts the installed packages by size and outputs the largest ones. Per default dpigs displays the largest 10 packages. You can change this value by using the -n option (see
"OPTIONS"). The information is taken from the dpkg status file with grep-status(1).

OPTIONS
-n, --lines=N
Display the N largest packages on the system (default 10).

-s, --status=FILE
Use FILE instead of the default dpkg status file (which is /var/lib/dpkg/status currently).

-S, --source
Display the largest source packages of binary packages installed on the system.

-H, --human-readable
Display package sizes in human-readable format (like ls -lh or du -h)

-h, --help
Display some usage information and exit.

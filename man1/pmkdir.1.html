NAME

pmkdir - make directory
 

SYNOPSIS

pmkdir [-d LEVEL] [-p DEVNAME] [-h] DIRECTORY
 

DESCRIPTION

pmkdir creates a directory on a device connected through SynCE.
 
Forward slashes ('/') on the command line are converted to backward slashes ('\').
 

OPTIONS

-d LEVEL
Set debug log level:
0 - No logging (default)
1 - Errors only
2 - Errors and warnings
3 - Everything
 
-p DEVNAME
Use the device with the given name, instead of the default.
 
-h
Display help message.
 
FILE
Full path name to the directory that should be created.

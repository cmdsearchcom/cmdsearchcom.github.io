NAME

coop-computing-tools - share resources in a heterogeneous, and unreliable computing environment

DESCRIPTION

The Cooperative Computing Tools (cctools) are a collection of programs designed to assist users with the difficulties of building and managing complex, fault-prone distributed systems. These components are:
Parrot - A personal virtual file system.
Chirp - A distributed file and storage system.
Makeflow - A workflow engine similar to Make.
Work Queue - A flexible master-worker library.
All-Pairs - A computational abstraction.
Wavefront - A computational abstraction.
Watchdog - A reliable process manager.
ftsh - A fault tolerant shell language.
FTP-Lite - A light weight FTP client library.

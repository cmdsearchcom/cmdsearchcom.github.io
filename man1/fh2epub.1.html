NAME

fh2epub - manual page for fh2epub 0.9.5

SYNOPSIS

fh2epub [ OPTIONS] INPUT OUTPUT

DESCRIPTION

`fh2epub' converts Adobe FreeHand documents to ePub.

OPTIONS

--help
show this help message
--version
print version and exit

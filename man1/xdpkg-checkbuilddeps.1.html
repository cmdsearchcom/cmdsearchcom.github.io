NAME

xdpkg-checkbuilddeps - check whether a package's build dependencies are fulfilled

SYNOPSIS

xdpkg-checkbuilddeps [-B] [-p package ]

DESCRIPTION

xdpkg-checkbuilddeps reads the debian/xcontrol (or debian/control, if the former does not exist), and tests whether the packages mentioned in the Build-Depends, Build-Depends-Tools and Build-Depends-Indep are currently fulfilled.

OPTIONS

The -B option specifies that the Build-Depends-Indep field should be ignored, as only architecture-dependent packages will be built.
The -p option allows you to query the per-package build-dependencies for a specific package. If no -p options are given, all non-optional non-bootstrap packages are selected.

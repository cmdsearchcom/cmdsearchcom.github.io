NAME
lxd-benchmark - The container lightervisor - benchmark

SYNOPSIS
lxd-benchmark spawn [--count=COUNT] [--image=IMAGE] [--privileged=BOOL] [--parallel=COUNT]

DESCRIPTION
debian/tmp/usr/bin/lxd-benchmark delete [--parallel=COUNT]

--count
(= 100)

Number of containers to create

--freeze
(= false)

Freeze the container right after start

--image (= "ubuntu:")

Image to use for the test

--parallel
(= -1)

Number of threads to use

--privileged
(= false)

Use privileged containers

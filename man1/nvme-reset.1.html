NAME
nvme-reset - Reset the nvme controller.

SYNOPSIS
nvme reset <device>

DESCRIPTION
Requests NVMe controller reset. The <device> param is mandatory and must be an NVMe character device (ex: /dev/nvme0).

OPTIONS
None

NAME
wmiv - quick image viewer using wrlib

SYNOPSIS
wmiv [image(s)|directory]

OPTIONS
--help print help text

--version
print version

KEYS
[+] zoom in

[-] zoom out

[Esc] actual size

[D] launch diaporama mode

[L] rotate image on the left

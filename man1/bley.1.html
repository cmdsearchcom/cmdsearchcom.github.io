NAME
bley - intelligent greylisting daemon for Postfix

SYNOPSIS
bley [options]

OPTIONS
--version
show program’s version number and exit

-h, --help
show this help message and exit

-p PID_FILE, --pidfile=PID_FILE
use PID_FILE for storing the PID

-c CONFFILE, --config=CONFFILE
load configuration from CONFFILE

-v, --verbose
use verbose output

-d, --debug
don’t daemonize the process and log to stdout -f, --foreground don’t daemonize the process

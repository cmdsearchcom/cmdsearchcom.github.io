NAME

tsseventextend - manual page for tsseventextend v713

SYNOPSIS

eventextend -if <measurement file> [-v]

DESCRIPTION

Extends a measurement file (binary) into TPM PCRs
Where the arguments are...
-if <input file> is the file containing the data to be extended

SEE ALSO

The full documentation for tsseventextend is maintained as a Texinfo manual. If the info and tsseventextend programs are properly installed at your site, the command
info tsseventextend
should give you access to the complete manual.

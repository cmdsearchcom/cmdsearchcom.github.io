NAME
cqsearch - create codequery database

SYNOPSIS
cqsearch [-s <sqdbfile> [-p <n>] [-t <term>] -[e|f] ] [-d] [-v] [-h]

OPTIONS
-s
CodeQuery sqlite3 db file path

-p <n>
parameter is a number denoted by n

Â· 1: Symbol (default)

Â· 2: Function or macro definition

Â· 3: Class or struct

Â· 4: Files including this file

Â· 5: Full file path

Â· 6: Functions calling this function

Â· 7: Functions called by this function

Â· 8: Calls of this function or macro

Â· 9: Members and methods of this class

Â· 10: Class which owns this member or method

Â· 11: Children of this class (inheritance)

Â· 12: Parent of this class (inheritance)

Â· 13: Functions or macros inside this file

-t
search term without spaces if Exact Match is switched off, wild card searches are possible. Use * and ?

-e
Exact Match switched ON Case-sensitive

-f
Exact Match switched OFF (fuzzy search) Case-insensitive with wild card search (default)

-d
debug

-v
version

-h
help

The combinations possible are -s -t -e, -s -t -f. The additional optional arguments are -d.

NAME
glue-validator - Validates information against the GLUE information model.

SYNOPSIS
Usage: /usr/bin/glue-validator -t <test class> [OPTIONS]

DESCRIPTION
The glue-validator reads information from either an LDIF file or an LDAP server. The information is validated against the GLUE information model.

OPTIONS
-t --test
The test class [glue1|glue2].

Server Mode: Obtains LDIF from an OpenLDAP server.

-h --host
Hostname of the LDAP server.

-p --port
Port for the LDAP server.

-b --bind
The bind point for the LDAP server.

File Mode: Obtains LDIF directly from a file.

-f --file
An LDIF file

Options:

-d --debug
Debug level 0-3, default 0

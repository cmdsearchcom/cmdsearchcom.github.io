NAME
foomatic-extract-text - Generate .po files from Foomatic UI strings

SYNOPSIS
foomatic-extract-text

DESCRIPTION
The foomatic-extract-text command generates .po files from the UI strings in the currently installed Foomatic database.

It is not intended to be called directly by the user.

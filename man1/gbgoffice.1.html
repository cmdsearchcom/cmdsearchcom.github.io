GBGOFFICE(1) General Commands Manual GBGOFFICE(1)

NAME

gbgoffice - GTK BG Office assistant

SYNOPSIS

gbgoffice [OPTIONS]...

DESCRIPTION

gbgoffice - GTK BG Office assistant

OPTIONS

-d, --dump-data-dir
Dumps direcotry where shared files are instaled.
-l, --language=[bg/en]
Sets language to Bulgarian or English.
-h, --help
Display help message.
-v, --version
Display the version of the program.

FILES

~/.gbgoffice
Per user configuration file.

ENVIRONMENT

LANG
Specify language which will be used (Bulgarian or English)

BUGS

Trayicon may not work with all window managers.

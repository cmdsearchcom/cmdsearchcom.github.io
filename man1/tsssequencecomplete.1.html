TSSSEQUENCECOMPLETE(1) User Commands TSSSEQUENCECOMPLETE(1)

NAME

tsssequencecomplete - manual page for tsssequencecomplete v713

DESCRIPTION

sequencecomplete
Runs TPM2_SequenceComplete
-hs seqeunce handle
-pwds password for sequence (default empty)
-if input file to be added (default no data)
-of result file name (default none)
[-tk ticket file name]
-se[0-2] session handle / attributes (default PWAP)
01 continue

SEE ALSO

The full documentation for tsssequencecomplete is maintained as a Texinfo manual. If the info and tsssequencecomplete programs are properly installed at your site, the command
info tsssequencecomplete
should give you access to the complete manual.

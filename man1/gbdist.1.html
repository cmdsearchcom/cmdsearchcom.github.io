NAME

gbdist - Produce cumulative distribution from data

SYNOPSIS

gbdist [ options]

DESCRIPTION

Compute the empirical probability distribution function as F(x_k) = k/(N+1) where x_k is the k-th largest observation, that is k-1 observations are smaller then x_k and n-k are larger, and N is the sample size.

OPTIONS

-r
print right cumulated distribution function, i.e. 1-F(x)
-i
print multiple points for identical observations
-t
print the distribution for each column of input
-F
specify the input fields separators (default " \t")
-v
verbose mode
-h
this help

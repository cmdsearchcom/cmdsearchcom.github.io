NAME

livemix — live mixer table

SYNOPSIS

livemix [ options ]

DESCRIPTION

livemix is a mixer table designed for live performances based on Jack and LADSPA.

OPTIONS

file
Optional file containing a mixer table configuration.
--verbose
To add some console messages.

EXAMPLE

livemix /usr/share/livemix/sample/default.lm

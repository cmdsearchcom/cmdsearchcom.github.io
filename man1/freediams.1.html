NAME

FreeDiams - pharmaceutical drugs prescriptor

SYNOPSIS

freediams

DESCRIPTION

FreeDiams prescriber is the result of FreeMedForms prescriber plugins built into a standalone application. FreeDiams is developed by a international community and is mainly intended for health professionals. It can be used alone to prescribe and / or test drug interactions within a prescription. It can be linked to any application thanks to its command line parameters.
FreeDiams can use multiple drugs database. By default, only the FDA (USA) database is installed. You can download and install new database directly from the application.
FreeDiams is fully integrated in the FreeMedForms open source EMR and can be used as external prescriber by any EMR using its interoperability interface.
 

OPTIONS

-v / --version
Displays the current version of FreeDiams.
--exchange-in=</path/to/xml/exchange-in.xml>
defines the "in" exchange file.
More parameters are documented at
http://www.freemedforms.com/en/manuals/freediams/command_line

NAME
tssnvwritelock - manual page for tssnvwritelock v713

DESCRIPTION
nvwritelock

Runs TPM2_NV_WriteLock

[-hia hierarchy authorization (o, p)(default index authorization)] -ha NV index handle -pwdn password for NV index (default empty)

-se[0-2] session handle / attributes (default PWAP)

01 continue

SEE ALSO
The full documentation for tssnvwritelock is maintained as a Texinfo manual. If the info and tssnvwritelock programs are properly installed at your site, the command

info tssnvwritelock

should give you access to the complete manual.

NAME
Sweep - a sound wave editor

SYNOPSIS
sweep-audio-editor [-h] [--help] [-v] [--version] [--display display] [--ignore-failed-lock] [filename]...

DESCRIPTION
Sweep is a sound wave editor.

OPTIONS
Sweep accepts the following options:

-h, --help
Display a list of all commandline options.

-v, --version
Output the version info.

--ignore-failed-lock
Continue when attempt to lock the preferences file fails. For use when the users home directory is on an NFS file system. (possibly unsafe)

--display display
Use the designated X display.

ENVIRONMENT
DISPLAY to get the default host and display number.

XENVIRONMENT
to get the name of a resource file that overrides the global resources stored in the RESOURCE_MANAGER property.

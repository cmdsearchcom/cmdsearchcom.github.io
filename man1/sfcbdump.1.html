SFCBDUMP(1) General Commands Manual SFCBDUMP(1)

NAME

sfcbdump - sfcBroker repository dumper

SYNOPSIS

sfcbdump [ -hsapqm] schemafile

OPTIONS

Allowed options:
-s, --stringbuffer-details
-a, --arraybuffer-details
-p, --property-details
-q, --qualifier-details
-m, --method-details
-h, --help

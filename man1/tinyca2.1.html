TINYCA2(1) General Commands Manual TINYCA2(1)

NAME
tinyca2 - Very simple Certification Authority

SYNOPSIS
tinyca2

DESCRIPTION
This manual page documents briefly the tinyca2 command. This manual page was written for the Debian distribution because the original program does not have a manual page and was
modified for Foobar Linux.

tinyca2 is a very simple Certification Authority. It features a graphical interface to create Certificate Requests, sign them with the key of the Certificate Authority and
export the signed certificate and the corresponding keys in various formats.

On first time, run with --ManageCA to create information for a new Certificate Authority.

OPTIONS
--version
show program version/exit

--ManageCA
enable CA management options

FILES
/etc/tinyca2/openssl.cnf
Openssl configuration template for tinyca

~/.TinyCA/
Storage place for the different CAs

SEE ALSO
openssl(1).

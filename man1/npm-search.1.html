NAME
npm-search - Search for packages

SYNOPSIS
npm search [-l|--long] [search terms ...]

aliases: s, se, find

DESCRIPTION
Search the registry for packages matching the search terms.

If a term starts with /, then it’s interpreted as a regular expression. A trailing / will be ignored in this case. (Note that many regular expression characters must be escaped
or quoted in most shells.)

NAME
apoo - An Assembly course aid

SYNOPSIS
apoo

DESCRIPTION
This manual page documents briefly the apoo command.

This manual page was written for the Debian GNU/Linux distribution because the original program does not have a manual page. Instead, it has documentation in
/usr/share/doc/apoo.

apoo

A virtual machine (a CPU) created for teaching purposes. All the programs are written in Python and are very easy to extend to incorporate new assembly-like pseudo-intuctions. A
TK-based interface is provided, to help debugging or just examining the execution of programs. Another program permits a tutor to create exercises and write conditions to grade
the students solutions.

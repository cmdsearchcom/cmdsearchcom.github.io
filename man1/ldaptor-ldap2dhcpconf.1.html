NAME
ldaptor-ldap2dhcpconf - LDAPtor dhcpd config file exporter

SYNOPSIS
ldaptor-ldap2dhcpconf [options]

OPTIONS
--base=
LDAP base dn

--dns-domain=
DNS domain name [default: example.com]

--service-location=
Service location, in the form BASEDN:HOST[:PORT]

DESCRIPTION
ldaptor-ldap2dhcpconf exports DHCP configuration from an LDAP server to a format usable by dhcpd.

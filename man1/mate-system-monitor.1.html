NAME
mate-system-monitor - view and control processes

SYNOPSIS
mate-system-monitor [OPTIONS]

DESCRIPTION
The mate-system-monitor allows you to view and control the processes running on your system. You can access detailed memory maps, send signals, and terminate the processes.

In addition, the mate-system-monitor provides an overall view of the resource usage on your system, including memory and CPU allocation, as well as network usage. It also allows
you to view file system information such as Device, Type, Mountpoints, and Disk Usage.

The System tab will display basic information about your system like Hostname, Kernel, MATE Version, Installed Memory, and Processor Information.

OPTIONS
This program accepts all the standard MATE and GTK+ options, which follow the usual GNU command line syntax, with long options starting with two dashes (’-’). In addition to
those standard MATE options, the mate-system-monitor accepts the following options:

-s, --show-system-tab
Show the System tab.

-?, -h, --help
Print standard command line options.

NAME
run_auxiliary_logger - manual page for run_auxiliary_logger 07.33

SYNOPSIS
run_auxiliary_logger Device_name remote_logfile_name time_seconds

DESCRIPTION
Device_name:
VRPN name of data source to contact

example:
Logger0@localhost

remote_logfile_name: The name of the file to log remotely time_seconds: How long to log before closing and exiting

NAME
lrs-rat2float - Convert rational coefficients to floating point

SYNOPSIS
lrs-rat2float [FILE]

DESCRIPTION
Lrs-rat2float reads a polyhedron file, or stdin if no file is given, and converts its rational coefficients to floating point approximations.

The resulting polyhedron is printed to stdout.

FILE FORMAT
The input file should be in the format documented at http://cgm.cs.mcgill.ca/~avis/C/lrslib/USERGUIDE.html#File%20Formats.

EXIT STATUS
The exit status is one if an error is encountered, or zero otherwise.

NAME

opahfirev
 
 
(Linux) Scans the system and reports hardware and firmware information about all the HFIs in the system.

Syntax

opahfirev

Options

no option
Returns information about all of the HFIs in the system.
 
--help
Produces full help text.
 

Example

# opahfirev
 
######################
 
server.intel.com#8217#160; - HFI 02:00.0
 
Board:#8217#160;ChipABI 3.0, WFR_ID 0x1, ChipRev 7.0 patch 0x00000003, SW Compat 3
 
SN:#8217#160; 0x0057501a
 
Bus:#8217#160; PCIe,8000MHz,x16
 
GUID:#8217#160; 0011:7501:0157:501a

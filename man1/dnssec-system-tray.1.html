NAME
dnssec-system-tray - System Tray for monitoring log files for DNSSEC errors

DESCRIPTION
A simply system-tray application that monitors log files (eg libval, or bind/named logfiles) for DNSSEC error messages that should be displayed to the user.

Further information on this tool can be found on the DNSSEC-Tools wiki page, of which this application is a component:

http://www.dnssec-tools.org/wiki/index.php/DNSSEC-System-Tray

SEE ALSO
For more information on the DNSSEC-Tools project, see the project’s web page:

http://www.dnssec-tools.org/

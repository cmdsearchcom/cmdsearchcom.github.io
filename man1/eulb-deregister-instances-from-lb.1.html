NAME
eulb-deregister-instances-from-lb - Remove one or more instances from a load balancer

SYNOPSIS
eulb-deregister-instances-from-lb --instances INSTANCE1,INSTANCE2,...

[--show-empty-fields] [-U URL]
[--region USER@REGION] [-I KEY_ID] [-S KEY] [--security-token TOKEN] [--debug] [--debugger] [--version] [-h] ELB

DESCRIPTION
Remove one or more instances from a load balancer

positional arguments:
ELB name of the load balancer to modify (required)

optional arguments:
--instances INSTANCE1,INSTANCE2,...
IDs of the instances to remove from the load balancer (required)

--show-empty-fields
show empty values as "(nil)"

-U URL, --url URL
load balancing service endpoint URL

--region USER@REGION
region and/or user names to search when looking up config file data

-I KEY_ID, --access-key-id KEY_ID

-S KEY, --secret-key KEY

--security-token TOKEN

--debug
show debugging output

--debugger
launch interactive debugger on error

--version
show the program’s version and exit

-h, --help
show this help message and exit

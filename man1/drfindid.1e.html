NAME
drfindid - Find public databases by identifier

SYNOPSIS
drfindid -query string -sensitive boolean -subclasses boolean -drfindid outresource

drfindid -help

OPTIONS
Input section
-query string

Additional section
Advanced section
-sensitive boolean
By default, the query keywords are matched against the EDAM term names (and synonyms) only. This option also matches the keywords against the EDAM term definitions and will
therefore (typically) report more matches. Default value: N

-subclasses boolean
Extend the query matches to include all terms which are specialisations (EDAM sub-classes) of the matched type. Default value: Y

Output section
-drfindid outresource

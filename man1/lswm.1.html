LSWM(1) General Commands Manual LSWM(1)

NAME
lswm - discover new wiimotes

SYNOPSIS
lswm [options]

DESCRIPTION
This manual page documents briefly the lswm command.

OPTIONS
-h Show summary of options.

-a List all bluetooth devices (not just wiimotes)

-l Long format (device details)

-q Quiet mode

NAME
KellySlater - part of ANTS registration suite

SYNOPSIS
KellySlater ImageDimension Segmentation.nii.gz WMProb.nii.gz GMProb.nii.gz Out.nii {GradStep-1-2D,2-3D} {#Its-~50} {ThickPriorValue-6} {Bool-use-curvature-prior} {smoothing}
{BoolUseEuclidean?}

DESCRIPTION
this is a kind of binary image registration thing with diffeomorphisms Segmentation.nii.gz -- should contain the value 3 where WM exists and the value 2 where GM exists

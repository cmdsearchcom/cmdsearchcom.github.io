NAME

kadu - instant messenger for Gadu-Gadu protocol

SYNOPSIS

kadu

DESCRIPTION

Gadu-Gadu is very popular instant messenger for MS Windows in Poland. kadu is Gadu-Gadu's client for Unix-like systems. The most complete documentations of kadu and related modules is at the project's home page at:
 
http://www.kadu.net
 

ENVIRONMENT

On startup kadu chacks if $CONFIG_DIR variable is set. If not, kadu will use default directory (which is $HOME/.kadu) to store it's configuration files. However, one may wish to run another instance of kadu, for different Gadu-Gadu account, on the same system account. In such case, one needs to set $CONFIG_DIR so it points to the directory that is supposed to hold configuration files for the other account.

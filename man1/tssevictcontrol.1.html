NAME
tssevictcontrol - manual page for tssevictcontrol v713

DESCRIPTION
evictcontrol

Runs TPM2_EvictControl

-hi authhandle hierarchy o, p

-ho object handle

-hp persistent handle

-pwda authorization password (default empty)

-se[0-2] session handle (default PWAP)

01 continue 20 command decrypt 40 response encrypt

SEE ALSO
The full documentation for tssevictcontrol is maintained as a Texinfo manual. If the info and tssevictcontrol programs are properly installed at your site, the command

info tssevictcontrol

should give you access to the complete manual.

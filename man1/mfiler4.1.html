MFILER4(1) General Commands Manual MFILER4(1)

NAME

mfiler4 a console based file manager
 

SYNOPSIS

[-c command] [--version] [ filer initial directory or script file ]
 

DESCRIPTION

a two pane console based file manager.
 
The following options are available -c " string"
 
Commands described by string are executed and exit immediately. -s -x When refreshing screeen, use "erase" function. This may hold down filicker in screen.
 

USAGE

run mfiler4 and hit ? key

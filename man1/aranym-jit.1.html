NAME
ARAnyM - Atari Running on Any Machine - JIT enabled

SYNOPSIS
aranym [options]

DESCRIPTION
This is version of aranym built with support for JIT (Just In Time) CPU compiler that makes ARAnyM about 10 times faster. For more information about ARAnyM, please, have a look
at aranym(1).

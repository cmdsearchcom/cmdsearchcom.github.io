NAME

git-annex-vcycle - switch view to next layout

SYNOPSIS

git annex vcycle

DESCRIPTION

When a view involves nested subdirectories, this cycles the order.
For example, when the view is by year/author/tag, vcycle will switch it to author/tag/year.

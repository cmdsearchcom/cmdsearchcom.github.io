NAME

eficas - eficas command

SYNOPSIS

eficas
 

DESCRIPTION

This manual page documents briefly the eficas command.
 
eficas is an application to help the creation of Code Aster command files. Thanks to this graphical user interface, a user can intuitively create a Aster command file without knowing the syntax.

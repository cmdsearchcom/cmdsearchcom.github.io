NAME

symnew - symmetrized tensor product

SYNOPSIS

symnew [ OPTIONS] <Mode> <Inp> <Out>

DESCRIPTION

Compute the symmetrized tensor product, where <Mode> is the symmetrization mode, one of e2, e3, e4, s2, or m3. The input matrix is in <Inp> and the output matrix is written to <Out>.

OPTIONS

-Q
Quiet, no messages.
-V
Verbose, more messages.
-T <MaxTime>
Set CPU time limit
-G
Produce output in GAP format. This option implies -Q.

randomip - generate random IP addresses

SYNOPSIS
randomip [OPTIONS] [NUM] [BYTE1] [BYTE2] [BYTE3]

DESCRIPTION
randomip is a simple tool to generate specified number of random IP addresses. By default it generates 10,000 of them. One can specify how many IP addresses to generate followed
by the first, second and third byte of the IP address.

OPTIONS
-h --help
print this help

-v --version
print version information

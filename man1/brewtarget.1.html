NAME

brewtarget - Helps you brew beer.

DESCRIPTION

Brewtarget is a calculator for brewing beer. It is a Qt-based program which allows you to create recipes from a database of ingredients. It calculates all the important parameters, helps you with mash temperatures, and just makes the process of recipe formulation much easier. It has no command line options.

NAME

wks2raw - debug the conversion library

SYNOPSIS

wks2raw [ OPTION] <Works Spreadsheet Document>

OPTIONS

--callgraph:
Display the call graph nesting level
-h, --help:
Shows this help message
-v, --version:
Output wks2raw version

NAME

jsb-sed - The JSONBOT sed program

SYNOPSIS

jsb-sed is used to sed all files in a directory with provided sedstring.

DESCRIPTION

This program allows for sedding of whole directories, used when refactoring jsonbot code. Hope its usefull for other purposes as well.

USAGE

Usage: jsb-sed <dir> <sedstring>

DOCUMENTATION

See http://jsonbot.org for more documentation or see http://jsonbot.googlecode.com

KOJID(1) User Commands KOJID(1)

NAME

kojid - Koji build daemon

SYNOPSIS

kojid [ options]

OPTIONS

-h, --help
show this help message and exit
-c FILE, --config=FILE
use alternate configuration file
--user=USER
specify user
--password=PASSWORD
specify password
-f, --fg
run in foreground
--force-lock
force lock for exclusive session
-v, --verbose
show verbose output
-d, --debug
show debug output
--debug-task
enable debug output for tasks
--debug-xmlrpc
show xmlrpc debug output
--skip-main
don't actually run main
--maxjobs=MAXJOBS
Specify maxjobs
--minspace=MINSPACE
Specify minspace
--sleeptime=SLEEPTIME
Specify the polling interval
--admin-emails=ADMIN_EMAILS
Address(es) to send error notices to
--topdir=TOPDIR
Specify topdir
--topurl=TOPURL
Specify topurl
--workdir=WORKDIR
Specify workdir
--pluginpath=PLUGINPATH
Specify plugin search path
--plugin=PLUGIN
Load specified plugin
--mockdir=MOCKDIR
Specify mockdir
--mockuser=MOCKUSER
User to run mock as
-s SERVER, --server=SERVER
url of XMLRPC server

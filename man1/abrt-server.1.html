NAME
abrt-server - Unix socket for ABRT.

SYNOPSIS
abrt-server [-u UID] [-spv[v]...]

DESCRIPTION
abrt-server is executed by abrtd daemon to handle socket connections. Every application in system is able to invoke creation of a new problem directory by following the
communication protocol (described below in section PROTOCOL).

OPTIONS
-u UID
Use UID as client uid

-s
Log to system log.

-p
Add program names to log.

-v
Log more detailed debugging information.

NAME

pft show - Show the built site

SYNOPSYS

pft show [-v]

DESCRIPTION

Show the built site

OPTIONS

--browser <command>
The browser can be specified by name (e.g. "firefox") or as a shell command, where %s is replaced with the file name (e.g. "firefox [options] %s").
 
This flag overrides the "system.editor" setting in the main configuration file. If neither the flag nor the setting are specified, the $BROWSER environment variable will be honored.
--help | -h
Show this help.

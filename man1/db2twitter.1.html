NAME
db2twitter - simple Twitter bot using a database to build and send tweets

SYNOPSIS
db2twitter db2twitter.ini

DESCRIPTION
db2twitter fetches data from a database using the SQLAlchemy library, filling a user-defined template to build a tweet and send it. db2twitter keeps track of already sent tweets
and only sends a new tweet if it was not sent before. db2twitter information are stored in a sqlite database.

EXAMPLES
Launch db2twitter

$ db2twitter db2twitter.ini

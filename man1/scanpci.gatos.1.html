NAME
scanpci - PCI bus utility from the GATOS package.

SYNOPSIS
scanpci [-v12]

DESCRIPTION
The program scanpci will scan for and print details of devices on the PCI bus.

OPTIONS
-1 Try PCI configuration type 1.

-2 Try PCI configuration type 2 (deprecated).

-v Verbose mode.

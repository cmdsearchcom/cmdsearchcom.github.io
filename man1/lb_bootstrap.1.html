NAME
lb bootstrap - Complete the bootstrap stage

SYNOPSIS
lb bootstrap [live-build options]

DESCRIPTION
lb bootstrap is a high-level command (porcelain) of live-build(7), the live systems tool suite.

lb bootstrap calls all necessary live-build programs in the correct order to complete the bootstrap stage.

OPTIONS
lb bootstrap has no specific options but understands all generic live-build options. See live-build(7) for a complete list of all generic live-build options.

FILES
none

SEE ALSO
live-build(7)

This program is a part of live-build.

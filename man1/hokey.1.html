NAME

hokey - hopenpgp-tools

SYNOPSIS

hokey COMMAND

DESCRIPTION

hokey (hopenpgp-tools) 0.19.4 Copyright © 2012-2016 Clint Adams
hOpenPGP Key utility

Available options:

-h,--help
Show this help text
-V,--version
Show version information

Available commands:

canonicalize
arrange key components in a canonical ordering
fetch
fetch key(s) from keyserver
lint
check key(s) for 'best practices'

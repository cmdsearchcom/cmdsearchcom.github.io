NAME

pitivi - Non-linear video editor

SYNOPSIS

pitivi [PROJECT_FILE]

DESCRIPTION

Pitivi is a free, intuitive and featureful movie editor for the Linux desktop.
pitivi starts the video editor, optionally loading PROJECT_FILE.

OPTIONS

-h, --help
Show help message and exit

NAME

dwp - The DWARF packaging utility

SYNOPSIS

dwp [ options] [file...]

DESCRIPTION

-h, --help
Print this help message
-e EXE, --exec EXE
Get list of dwo files from EXE (defaults output to EXE.dwp)
-o FILE, --output FILE
Set output dwp file name
-v, --verbose
Verbose output
--verify-only
Verify output file against exec file
-V, --version
Print version number

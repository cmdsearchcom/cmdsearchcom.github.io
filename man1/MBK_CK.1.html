NAME
MBK_CK - define the clock name pattern

SYNOPSYS
c-shell running
setenv MBK_CK cke

DESCRIPTION
MBK_CK sets the pattern to be matched in a name to indicate a clock for the tools based upon mbk. Its default value is ck. Therefore all names of the form ‘*ck*’ indicates a
clock.

EXAMPLE
setenv MBK_CK vcc

mbk(3), genlib(1), isvdd(3), isvss(3), isck(3), MBK_VSS(1), MBK_VDD(1).

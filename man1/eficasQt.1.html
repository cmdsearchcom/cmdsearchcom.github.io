NAME
eficasQt - eficasQt command

SYNOPSIS
eficas

DESCRIPTION
This manual page documents briefly the eficasQt command.

eficasQt is a Qt application to help the creation of Code Aster command files. Thanks to this graphical user interface, a user can intuitively create a Aster command file without
knowing the syntax.

NAME
c2sim - codec2 testing framework

DESCRIPTION
Codec2 test and evaluation program

USAGE
c2sim 3200|2400|1600|1400}1300|1200 InputFile OutputRawFile [OPTIONS]

OPTIONS
--help display help and exit

--rate Set rate

--gain Set gain

--lpc Set LPC model

Codec2 simulation. Combines encoder and decoder and allows switching in and out various algorithms and quantisation steps. Used for algorithm development.

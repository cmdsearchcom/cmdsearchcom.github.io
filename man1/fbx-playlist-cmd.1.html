NAME

fbx-playlist-cmd - create playlists for the FreeBox

SYNOPSIS

fbx-playlist-cmd files...

DESCRIPTION

This manual page documents briefly the fbx-playlist-cmd command.
fbx-playlist-cmd is a program that create on its standard output a playlist that can be given to the vlc-fbx program to be play by the FreeBox. Standard extension for vlc playlist files is ".m3u" .

SEE ALSO

fbx-playlist(1), vlc-fbx(1).
 
The format of the playlist is documented fully in the /usr/share/doc/freeplayer/options-playlist.txt file.

NAME
ambix-deinterleave - AMBIsonics eXchange utility

SYNOPSIS
ambix-deinterleave [options] infile

DESCRIPTION
Split an ambix file into several mono files

OPTIONS
-h, --help
print this help

-v, --version
print version info

-p, --prefix
output prefix

This splits an ambix file into several mono files, naming them according to type: If infile is ’FieldRecording.caf’, this will extract audio data into ’FieldRecordâ
ing-ambi000.caf’, ’FieldRecording-ambi001.caf’, ..., ’FieldRecording-extra000.caf’, ...

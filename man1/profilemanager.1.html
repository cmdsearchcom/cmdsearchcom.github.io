NAME
profilemanager - program for applying menu profiles to groups.

SYNOPSIS
profilemanager

DESCRIPTION
This manual page explains the profilemanager program. This program can be used to apply menueditor created profiles to groups of users.

profilemanager apply menu profile to user using desktop-profiles.

FILES
Those directory contains applicable profiles:
/usr/share/edubuntu-menueditor/menus/
/etc/edubuntu-menueditor/

SEE ALSO
menueditor(1), destop-profiles(7)

NAME
qstail - an utility printing the end of a log file starting at the specified pattern.

SYNOPSIS
qstail -i <path> -p <pattern>

DESCRIPTION
qstail shows the end of a log file beginning with the line containing the specified pattern. This may be used to show all lines which has been written after a certain event
(e.g., server restart) or time stamp.

OPTIONS
-i <path>
Input file to read the data from.

-p <pattern>
Search pattern (literal string).

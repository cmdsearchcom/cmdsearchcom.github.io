NAME
amideco - Decompress flashfiles equipped with an AMI BIOS

SYNOPSIS
amideco filename [OPTIONS]

DESCRIPTION
amideco is a program which can decompress BIOS images which contain an AMI BIOS.

OPTIONS
-l List BIOS structure.

-x Extract BIOS modules.

-d Create directory.

-h Show help text and exit.

NAME
system-config-httpd - HTTPD configuration utility

SYNTAX
system-config-httpd

DESCRIPTION
This is a new graphical tool to configure the common web server Apache in an easy and simple way. It supports the basic configuration options and even allows to set up multiple
virtual hosts.

OPTIONS
None

FILES
/usr/bin/system-config-httpd
/usr/share/system-config-httpd/*
/etc/alchemist/switchboard/system-config-httpd.switchboard.adl
/etc/alchemist/namespace/system-config-httpd/local.adl
/etc/alchemist/namespace/system-config-httpd/rpm.adl
/var/cache/alchemist/system-config-httpd/*

EXAMPLES
To run this program simply type:

system-config-httpd

SEE ALSO
httpd(8)

BUGS
Not all possible checks for correct values are done yet.

The current /etc/httpd/conf/httpd.conf file is NOT parsed, so changing the configuration with system-config-httpd will overwrite any manual changes to the httpd.conf file or the
changes done with system-config-httpd are discarded. In case of overwriting the manually editied file a backup file will be generated.

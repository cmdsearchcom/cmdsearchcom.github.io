NAME
mysqldumpslow - Summarize slow query log files

SYNOPSIS

DESCRIPTION
The MySQL slow query log contains information about queries that take a long time to execute (see Section 5.2.5, âThe Slow Query Logâ). mysqldumpslow parses MySQL slow query log
files and prints a summary of their contents.

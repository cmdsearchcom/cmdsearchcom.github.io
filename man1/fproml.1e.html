NAME

fproml - Protein phylogeny by maximum likelihood

SYNOPSIS

DESCRIPTION

fproml is a command line program from EMBOSS (“the European Molecular Biology Open Software Suite”). It is part of the "Phylogeny:Molecular sequence" command group(s).

NAME
ltsp-agent â Agent for LTSP-Cluster.

SYNOPSIS
ltsp-agent

DESCRIPTION
ltsp-agent is a daemon running a set of plugins (provided by other packages) and used as a base for the new ltsp-cluster infrastructure.

<configuration file> The configuration is usually stored in /etc/ltsp/agent.conf The ssl certificate is usually stored in /etc/ltsp/agent.pem

AVAILABILITY
ltsp-agent is part of the ltsp-cluster-agent package and the latest versions are available in source form from https://launchpad.net/ltsp-cluster.

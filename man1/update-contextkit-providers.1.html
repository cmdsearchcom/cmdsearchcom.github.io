NAME
update-contextkit-providers - updates the cached cache.cdb context properties registry database.

SYNOPSIS
update-contextkit-providers [directory]

DESCRIPTION
update-contextkit-providers reads the context properties registry (in xml format) and produces an updated cached database - cache.cdb. The database is used by libcontextsubscrâ
bier for quick introspection of the registry.

OPTIONS
directory the location containing the xml registry. The database will be written there. If no directory is specified, update-contextkit-providers will use the CONâ
TEXT_PROVIDERS environment variable. If that is not preset as well, a compiled-in registry prefix path will be used (usually /usr/share/contextkit/providers).

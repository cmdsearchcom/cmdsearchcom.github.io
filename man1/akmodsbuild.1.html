NAME
akmodsbuild - manual page for akmodsbuild 0.5.6

SYNOPSIS
akmodsbuild [OPTIONS] <SRPMS>

DESCRIPTION
Rebuilds kmod SRPM(s)

OPTIONS
-k, --kernels -- build for kernel-versions (output from ’uname -r’)

-l, --logfile <file> -- save rpmduild output to <file>

-o, --outputdir <dir> -- save rpms and logs here (current directory)

-t, --target -- target-arch (output from ’uname -m’)

-v, --verbose -- increase verboseness

-q, --quiet -- be more quiet

-h, --help -- show usage

-V, --version -- show version

NAME

gnome-video-arcade - a MAME front-end

SYNOPSIS

gnome-video-arcade [ OPTION...]

DESCRIPTION

gnome-video-arcade is a clean and simple MAME front-end for any freedesktop.org compliant desktop environment.
The program is targeted towards non-technical arcade enthusiasts who just want to easily play classic arcade games on their desktop. Though it may be suitable for use in custom arcade cabinets, gnome-video-arcade is not designed for that purpose.

OPTIONS

-?, --help
Show help options
-b, --build-database
Build the games database
-i, --inspect=NAME
Inpect an emulator setting
-v, --version
Show the application version
-w, --which-emulator
Show which emulator will be used
--display
X display to use

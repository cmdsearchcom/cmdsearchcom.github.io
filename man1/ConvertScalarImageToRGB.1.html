NAME

ConvertScalarImageToRGB - part of ANTS registration suite

SYNOPSIS

ConvertScalarImageToRGB imageDimension inputImage outputImage mask colormap [ customColormapFile] [minimumInput] [maximumInput] [ minimumRGBOutput=0] [maximumRGBOutput=255] <vtkLookupTable>

DESCRIPTION

Possible colormaps: grey, red, green, blue, copper, jet, hsv, spring, summer, autumn, winter, hot, cool, overunder, custom

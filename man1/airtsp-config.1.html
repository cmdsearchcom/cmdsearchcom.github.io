NAME

airtsp-config - Configuration Script for the C++ Simulated Airline Schedule Manager Library

SYNOPSIS

airtsp-config [--prefix] [--version] [--libs] [--cflags]

DESCRIPTION

airtsp-config is a tool that is used by configure to determine the compiler and linker flags that should be used to compile and link programs that use AirTSP. It is also used internally by the .m4 macros, that are included with AirTSP, for GNU autoconf.

OPTIONS

airtsp-config accepts the following options:
--help
 

Display this help and exit
--prefix
 

Show the AirTSP installation prefix.
--version
 

Print the currently installed version of AirTSP on the standard output.
--libs
 

Print the linker flags that are necessary to link with the AirTSP library.
--cflags
 

Print the compiler flags that are necessary to compile with the AirTSP library.
--sampledir
 

Show where the (StdAir CSV) sample files are located.
--mandir
 

Show where the manual pages are located.
--infodir
 

Show where the information pages are located.
--docdir
 

Show the documentation base directory.
--htmldocpath
 

Show where the HTML documentation is located.
--refmanpath
 

Show where the PDF documentation is located.

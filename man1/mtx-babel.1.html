mtx-babel(1) Babel Input To UTF Conversion mtx-babel(1)

NAME

mtx-babel

SYNOPSIS

mtxrun --script babel [ OPTIONS ... ] [ FILENAMES ]

DESCRIPTION

Babel Input To UTF Conversion

OPTIONS

--language=string
conversion language (e.g. greek)
--structure=string
obey given structure (e.g. 'document', default: 'context')
--convert
convert babel codes into utf

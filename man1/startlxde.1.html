NAME

startlxde - X session script to prepare environment and run lxsession.
 

SYNOPSIS

startlxde
 
 

DESCRIPTION

startlxde is used to start the desktop session of LXDE. By default it loads openbox, lxpanel, pcmanfm, dbus-launch, and lxsession.

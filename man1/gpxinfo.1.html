NAME
gpxinfo â command line utility to extract basic statistics from a GPX file

SYNOPSIS
gpxinfo file ...

DESCRIPTION
gpxinfo extracts basic statistics from a GPX file. It prints meta info about the file and calculates data about tracks and routes in the file, among which are times, distances, and
uphill and downhill information.

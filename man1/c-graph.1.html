NAME
c-graph - manual page for c-graph

SYNOPSIS
c-graph [OPTION]...

DESCRIPTION
-d, --dedicate print the dedication and exit

-h, --help print these help options and exit -n, --no-splash invoke GNU C-Graph with no splash screen

-v, --version print this version of GNU C-Graph and exit

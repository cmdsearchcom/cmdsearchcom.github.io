NAME
rbm-pkg - Build a package

SYNOPSIS
rbm pkg <project> [options]

DESCRIPTION
Build a package for the selected project. Depending on configuration or selected distribution, this can be an rpm, deb or other type of package.

CONFIGURATION OPTIONS
pkg_type
This is the name of the option that will be used by the pkg command as the script to build the package. This can be rpm or deb for rpm and debian packages. This option is
usually set in distribution specific configuration as it depends on the distribution being used.

See rbm_config(7) for all other options.

COMMAND LINE OPTIONS
See rbm_cli(7).

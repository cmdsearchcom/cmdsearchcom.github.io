NAME
stem - Stem Documentation

Stem is a Python controller library for Tor. With it you can use Tor’s control protocol to script against the Tor process, or build things such as arm. Stem’s latest version is
1.5.4 (released January 3rd, 2017). .SH AUTHOR Damian Johnson (atagar@torproject.org)

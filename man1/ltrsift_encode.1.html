NAME

ltrsift_encode - encode sequences into a LTRsift-compatible format

SYNOPSIS

ltrsift_encode <sequence file> <indexname>

DESCRIPTION

This command creates an index with name <indexname> from the FASTA/EMBL/GenBank file <sequence file>.

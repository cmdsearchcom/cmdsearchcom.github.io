NAME
djbdns - djbdns DNS tools

DESCRIPTION
djbdns is a collection of DNS tools, originally written by Dr D J Bernstein. This release of djbdns includes the following ones

+ dnscache

+ dnsfilter

+ dnsip

+ dnsipq

+ dnsmx

+ dnsname

+ dnsq

+ dnsqr

+ dnstrace

+ dnstracesort

+ dnstxt

+ tinydns

+ tinydns-data

+ tinydns-edit

+ tinydns-get

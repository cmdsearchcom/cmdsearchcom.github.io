NAME

capify — Stub out the files 'config/deploy.rb' and 'Capifile' on rails application.

SYNOPSIS

capify project_location

DESCRIPTION

capify Stub out the files 'config/deploy.rb' and 'Capifile' on rails application.
To use the capify script just call it, and pass the location of your project’s root directory. If you are already in that directory, you can simply type:
$ capify .

NAME
InVesalius - 3D medical imaging reconstruction software

SYNOPSIS
invesalius-3.0 [-d] [-i <dicom_folder>]

invesalius-3.0 [-d] [<invesalius_project_file>]

DESCRIPTION
InVesalius generates 3D medical imaging reconstructions based on a sequence of 2D DICOM files acquired with CT or MRI equipments. InVesalius is internationalized (currently
available in English, Portuguese, French, German, Spanish, Greek and Chinese) and provides several tools:

Â· DICOM support including: (a) ACR-NEMA version 1 and 2; (b) DICOM version 3.0 (including various encodings of JPEG - lossless and lossy -, RLE)

Â· Image manipulation features (zoom, pan, rotation, brightness/contrast, etc.)

Â· Segmentation based on 2D slices

Â· Pre-defined threshold ranges according to tissue of interest

Â· Edition tools (similar to Paint Brush) based on 2D slices

Â· 3D surface creation

Â· 3D surface connectivity tools

Â· 3D surface exportation (including: binary STL, OBJ, VRML, Inventor)

Â· High-quality volume rendering projection

Â· Pre-defined volume rendering presets

Â· Volume rendering crop plane

Â· Picture exportation (including: BMP, TIFF, JPG, PostScript, POV-Ray)

OPTIONS
-h display a short help text

-d, --debug
debug mode. Causes InVesalius to print debugging messages.

-i <dicom folder>
InVesalius opens a <dicom folder>. In the presence of more than one series in the given folder, InVesalius opens the larger one.

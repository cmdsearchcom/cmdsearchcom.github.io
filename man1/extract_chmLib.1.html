NAME

extract_chmLib - Extracts a chm file into a directory.

SYNOPSIS

extract_chmLib [ FILE]... [DIR]...

DESCRIPTION

extract_chmLib extracts the contents of the specified chm file into the specified directory.

USAGE

extract_chmLib <chmfile> <dir>

OPTIONS

extract_chmLib has no options.

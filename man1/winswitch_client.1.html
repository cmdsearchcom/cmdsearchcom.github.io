NAME
winswitch_client - Window Switch applet

SYNOPSIS
winswitch_client [--debug-mode]

DESCRIPTION
winswitch_client starts a GUI-less Window Switch client.

OPTIONS
--debug-mode
Enable debug mode

FILES
.winswitch/client/client.conf
The client configuration file.

DIAGNOSTICS
A number of debugging messages will be printed on screen. More if debug mode is enabled.

OVERVIEW
Generally, users will want to use winswitch_applet instead as it provides a GUI and the means of starting and controlling sessions. winswitch_client on the other hand does not
include any user interface, it can only receive commands from the servers it will connect to, or from local socket commands (ie: open file remotely).

NAME

ochusha - a GUI BBS browser for 2ch.net

SYNOPSIS

ochusha
 

DESCRIPTION

This manual page documents briefly the ochusha.
 
ochusha is a program that gives a GUI based on GTK+ for accessing BBS like 2ch.net.
 
Since it tends to be extremely user friendly, there's neither command line options that can be specified by users nor configuration files that should be prepared by users.

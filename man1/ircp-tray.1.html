NAME
ircp-tray - IrDA and OBEX wireless file transfer

ircp-tray is a wireless file transfer program for Linux. It stays inside your system tray, listening for incoming OBEX file transfer request, as well as sending file out to
remote devices via IrDA.

OPTIONS
A summary of options is included below.

-h Show summary of options.

-v Show version of program.

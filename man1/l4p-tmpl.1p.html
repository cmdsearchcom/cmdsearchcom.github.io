NAME
l4p-tmpl - Print out a Log4perl template configuration

SYNOPSIS
l4p-tmpl >l4p.conf

DESCRIPTION
l4p-tmpl prints out the text of a template Log4perl configuration for starting a new Log4perl configuration file.

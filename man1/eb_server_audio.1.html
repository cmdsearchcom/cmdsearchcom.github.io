EB_SERVER_AUDIO(1) User Commands EB_SERVER_AUDIO(1)

NAME
eb_server_audio - gather entropy on systems with a spare sound-card

DESCRIPTION
eb_server_audio v1.0, (C) 2009-2012 by folkert@vanheusden.com

-i host eb-host to connect to

-d dev audio-device, default hw:1

-o file file to write entropy data to

-l file log to file ’file’

-s log to syslog

-n do not fork

-P file file to write the PID to

-X file file to read the password from

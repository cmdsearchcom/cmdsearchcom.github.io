NAME
bkr-harness-test - Generate Beaker job to test harness installation

SYNOPSIS
bkr harness-test [workflow options] [options]

DESCRIPTION
Generates a Beaker job to test that the harness can be installed correctly on all available combinations of distro family, variant, and arch.

This is intended to catch misconfigurations and missing/incomplete harness repos, not find bugs in the harness.

OPTIONS
Common workflow options are described in the Workflow options section of bkr(1).

Common bkr options are described in the Options section of bkr(1).

EXIT STATUS
Non-zero on error, otherwise zero.

EXAMPLES
Test the harness:

bkr harness-test

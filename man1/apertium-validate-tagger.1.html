NAME
apertium-validate-tagger - This application is part of ( apertium )

This tool is part of the apertium open-source machine translation toolbox: http://www.apertium.org.

SYNOPSIS
apertium-validate-tagger <input_file>

DESCRIPTION
apertium-validate-tagger is a script that checks the validity of a set of rules which enforce the state to state transition probabilities used by the part-of-speech tagger. The
script uses xmllint to validate against a DTD.

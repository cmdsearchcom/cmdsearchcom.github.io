NAME

idxmerge - merge fweb-produced index files.

SYNOPSIS

idxmerge -efR file1.idx...
 

DESCRIPTION

idxmerge merges index files produced by the invocation of fweave with option -XI.
 

MANUAL

Further documentation can be obtained from texinfo, available online through emacs' info browser (menu item fweb ), or in HTML format.

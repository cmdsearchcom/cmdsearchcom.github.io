NAME

rajce-get - Download images from rajce.net.

VERSION

version 1.152450

SYNOPSIS

rajce-get [OPTIONS]

DESCRIPTION

This program will download images from rajce.net. Images are saved in directory username/album
Behind proxy server try:
export http_proxy=http://login:password@proxyserver:port

OPTIONS AND ARGUMENTS

         -u, --url http://username.rajce.idnes.cz/album
                        Address of requested album
         -v, --verbose
                        Verbose mode
         -s, --sorted
                        Sort output file names in the same order as they
                        appear in album
         -V, --version
                        Show version and exit
         -h, --help
                        Show help

EXAMPLES

rajce-get --sorted --verbose --url http://username.rajce.idnes.cz/album

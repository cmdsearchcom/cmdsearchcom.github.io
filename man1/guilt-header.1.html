NAME
guilt-header - Print a patch header

SYNOPSIS
guilt header [-eE] [<patchname>]

DESCRIPTION
Prints either the topmost patchâs header or the header of a specified patch.

-e
Open the header in an editor, instead of printing it.

-E
Open the raw patch in an editor, instead of printing it.

<patchname>
Name of the patch. If a patch with exactly this name exists, use it. Otherwise, treat the name as a regexp; if the regexp matches a single patch, use it. Otherwise, list all
matching patch names to stderr and fail.

NAME
dvdainfo - prints information about a DVD-Audio disc

SYNOPSIS
dvdainfo [OPTIONS]

DESCRIPTION
dvdainfo takes a mounted DVD-Audio disc and writes information about the it to standard output. This information includes the length, start sector, end sector, codec, sample
rate, channel count and bits-per-sample of each track in each title.

OPTIONS
-h, --help
show a list of options and exit

--version
display version and exit

-c, --cdrom=CDROM
the CD-ROM device of the mounted disc

-A, --audio-ts=DIR
the path to the AUDIO_TS directory in the mounted disc

NAME
wfindent - Indents and optionally converts Fortran program sources

SYNOPSIS
wfindent [OPTION]... [FILE]...

See ’man 1 findent’ for the options.

DESCRIPTION
Wfindent is a wrapper for findent(1), calling findent for each of the files. The files are overwritten by the indented version. A sanity check is made before oerwriting a input
file.

Example:
wfindent -i4 *.f90

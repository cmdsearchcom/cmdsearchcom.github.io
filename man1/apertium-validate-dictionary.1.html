NAME
apertium-validate-dictionary - This application is part of ( apertium )

This tool is part of the apertium open-source machine translation toolbox: http://www.apertium.org.

SYNOPSIS
apertium-validate-dictionary <input_file>

DESCRIPTION
apertium-validate-dictionary is a script that validates a dictionary file against the apertium DTD file for dictionaries using the xmllint utility.

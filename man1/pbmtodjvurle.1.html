NAME
pbmtodjvurle - convert a PBM image to DjVu Bitonal RLE format

SYNOPSIS
pbmtodjvurle

[pbmfile [rlefile]]

DESCRIPTION
This program is part of Netpbm(1)

pbmtodjvurle reads a PBM image as input and produces DjVu Bitonal RLE format as output.

DRC(1) General Commands Manual DRC(1)

NAME
drc - a program for acoustic compensation of audio systems

SYNOPSIS
drc [options] DRCFile
DRCFile is name of DRC configuration file

DESCRIPTION
This manual page documents briefly the drc program. This manual page was written for the Debian distribution because the original program does not have a manual page.

drc is program used to generate correction filters for acoustic compensation of HiFi and audio systems in general, including listening room compensation.
Program generates just the FIR correction filters, which can be used with a real time or offline convolver to provide real time or offline correction.

drc doesn’t provide convolution features, and provides only some simplified, although really accurate, measuring tools.

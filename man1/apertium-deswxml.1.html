NAME
apertium-deswxml - This application is part of ( apertium )

This tool is part of the apertium open-source machine translation toolbox: http://www.apertium.org.

SYNOPSIS
apertium-deswxml [ -h ] [ -i ] [ -n ] [ <input file> [ <output file> ] ]

DESCRIPTION
apertium-deswxml is an WXML format processor. Data should be passed through this processor before being piped to lt-proc. The program takes input in the form of an WXML document
and produces output suitable for processing with lt-proc. WXML tags and other format information are enclosed in brackets so that lt-proc treats them as whitespace between
words.

OPTIONS
-h, --help
Display this help. -i Makes the addition of trailing sentence terminator (".") unconditional, often leading to duplicates. -n Suppresses the addition of a trailing senâ
tence terminator.

EXAMPLE
You could write the following to show how the word "gener" is analysed:

echo "<b>gener</b>" | apertium-deswxml | lt-proc ca-es.automorf.bin

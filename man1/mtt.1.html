NAME

mtt - teletext browser for X11 and console.

SYNOPSIS

mtt [ options ]

DESCRIPTION

mtt is a browser for teletext pages. If the DISPLAY environment variable is set mtt will come up with the X11 GUI (motif), otherwise it will use the tty it was started from.

OPTIONS

mtt understands the usual toolkit options (-geometry + friends). Additional options are:
-help
print help text.
-debug
enable debug messages.
-device <dev>
vbi device (default is /dev/vbi0).
-tty
force using the console mode.

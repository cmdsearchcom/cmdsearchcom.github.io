genome music bmr NAME
genome music bmr - Calculate gene coverages and background mutation rates.

VERSION
This document describes genome music bmr version 0.04 (2016-01-01 at 23:10:18)

SYNOPSIS
genome music bmr ...

DESCRIPTION
These tools are part of the MuSiC suite.

SUB-COMMANDS
GENERAL
calc-bmr
Calculates mutation rates given per-gene coverage (from "music bmr calc-covg"), and a mutation list

calc-covg
Uses calcRoiCovg.c to count covered bases per-gene for each given tumor-normal pair of BAMs.

calc-covg-helper
Uses calcRoiCovg.c to count covered bases per-gene for a tumor-normal pair of BAMs.

calc-wig-covg
Count covered bases per-gene for each given wiggle track format file.

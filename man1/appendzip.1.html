NAME

Usage: - manual page for Usage: appendzip exe-file zipfile

SYNOPSIS

appendzip exe-file zipfile

DESCRIPTION

This tool appends a zipfile at the end of any other file (most often a .exe under MS-Windows). The openEmbeddedZipFile() function can then be used to read the file.

NAME
glam2mask - masks a GLAM2 motif in sequences

SYNOPSIS
glam2mask [options] my_motif.glam2 my_seqs.fa

DESCRIPTION
glam2mask masks a glam2 motif out of sequences, so that weaker motifs can be found. Masking replaces residues aligned to key positions with the symbol Â´xÂ´. By alternately
applying glam2 and glam2mask several times, it is possible to find the strongest, second-strongest, third-strongest, etc. motifs in a set of sequences.

OPTIONS (DEFAULT SETTINGS)
-o
Output file (stdout).

-x
Mask character (x).

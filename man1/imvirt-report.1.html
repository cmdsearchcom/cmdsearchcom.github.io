NAME

imvirt-report - system reporting to improve detection

SYNOPSIS

imvirt-report

DESCRIPTION

This manual page documents briefly the imvirt-report script. imvirt-report collects various system information and sends them to the upstream authors to provide a better detection support of virtualizing technologies.

USAGE

imvirt-report does not need any arguments.

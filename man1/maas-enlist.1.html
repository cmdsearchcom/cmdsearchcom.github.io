NAME
maas-enlist -- MaaS Enlistment tool, for enlisting with a MaaS server over a network

SYNOPSIS
maas-enlist --serverurl=[protocol://]127.0.0.1 --hostname=test-node --mac-address=AA:BB:CC:DD:EE:FF,AA:BB:CC:DD:EE:F1

DESCRIPTION
: maas-enlist <parameters>

Parameters:

-s, --serverurl
-- resolvable IP or hostname of the MaaS server. May optionally prefix http:// or https:// to explicitly set the protocol. If unspecified, the protocol is http://.

-n, --hostname
-- hostname to assign the node

-i, --interface
-- network interface to register

-a, --arch
-- architecture to register the node with

--interface is optional; if not specified, all network interfaces will be registered

all other parameters are required

WARNING
When using http:// as the protocol, all traffic sent unencrypted. It is strongly recommended that http:// only be used in a trusted network segment.

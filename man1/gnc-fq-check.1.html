NAME

gnc-fq-check - check for the presence of Finance::Quote
From gnc-fq-helper

SYNOPSIS

gnc-fq-check

DESCRIPTION

Input: <none>
Output (on standard output, one output form per input line):
A list of quote sources supported by Finance::Quote, or the single term missing-lib if finance quote could not be executed.
Exit status
0 - success non-zero - failure

WHY3REALIZE(1) User Commands WHY3REALIZE(1)

NAME

why3realize - produce a realization of a Why3 theory

SYNOPSIS

why3realize [ OPTIONS] <THEORY>

DESCRIPTION

Produce a realization of a Why3 theory.

OPTIONS

-L, --library <dir>
Add dir to the library search path.
-help, --help
Show a list of options.

NAME

kpod-load - load an image from docker archive
 

SYNOPSIS

kpod load NAME[:TAG|@DIGEST] [ --help|-h]
 

DESCRIPTION

kpod load copies an image from docker-archive stored on the local machine. kpod load reads from stdin by default or a file if the input flag is set. The quiet flag suppresses the output when set.
 
kpod [GLOBAL OPTIONS]
 
kpod load [GLOBAL OPTIONS]
 
kpod load [OPTIONS] NAME[:TAG|@DIGEST] [GLOBAL OPTIONS]
 

OPTIONS

--input, -i Read from archive file, default is STDIN
 
--quiet, -q Suppress the output
 

GLOBAL OPTIONS

--help, -h
Print usage statement

NAME
lorem - Generate random Latin looking text using Text::Lorem

SYNOPSIS
Generate 3 paragraphs of Latin looking text:

$ lorem -p 3

Generate 5 Latin looking words:

$ lorem -w 5

Generate a Latin looking sentence:

$ lorem -s 1

DESCRIPTION
lorem is a simple command-line wrapper around the "Text::Lorem" module. It provides the same three basic methods: Generate "words", generate "sentences", and generate
"paragraphs".

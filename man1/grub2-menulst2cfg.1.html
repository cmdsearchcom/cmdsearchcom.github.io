NAME

grub-menulst2cfg — Convert a configuration file from GRUB 0.xx to GRUB 2.xx format.
 

SYNOPSIS

grub-menulst2cfg [INFILE [OUTFILE]]
 

DESCRIPTION

grub-menulst2cfg converts a configuration file from GRUB 0.xx to the current format.

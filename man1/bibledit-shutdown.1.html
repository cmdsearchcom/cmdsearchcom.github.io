NAME
bibledit-shutdown - internal command for bibledit-gtk, a Bible editor

DESCRIPTION
Bibledit is a Bible editor. It is started by typing "bibledit-gtk" on the commandline. Binary "bibledit-shutdown" is used by Bibledit internally.

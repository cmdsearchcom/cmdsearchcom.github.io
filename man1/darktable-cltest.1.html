NAME
darktable-cltest - check if there is a usable OpenCL environment for darktable to use

SYNOPSIS
darktable-cltest

DESCRIPTION
darktable is a digital photography workflow application for Linux, Mac OS X and several other Unices. It’s described further in darktable(1).

darktable-cltest checks if there is a usable OpenCL environment on your system that darktable can use. It emits some debug output that is equivalent to calling darktable -d
opencl and then terminates.

SEE ALSO
darktable(1)

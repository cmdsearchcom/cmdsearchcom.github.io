NAME
pomdump - dump the POM parse tree for a POD document

SYNOPSIS
pomdump MyFile.pm

DESCRIPTION
This script uses Pod::POM to parse a Pod document and then invokes the dump method on the top level node, resulting in a visualization of the structure of the POD document (the
parse tree).

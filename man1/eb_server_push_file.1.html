NAME
eb_server_push_file - push entropy data from file to eb-host

DESCRIPTION
eb_server_push_file v1.0, (C) 2009-2012 by folkert@vanheusden.com

-i host eb-host to connect to

-f file file to write to

-l file log to file ’file’

-s log to syslog

-n do not fork

-P file file to write the PID to

-X file file to read the password from

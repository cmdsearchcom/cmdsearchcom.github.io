NAME
keylaunch - A utility for binding commands to a hot key.

SYNTAX
keylaunch [no options] <no arguments>

DESCRIPTION
KeyLaunch is a utility for binding commands to a hot key. See the example rc file for information

FILES
$HOME/.keylaunchrc

EXAMPLES
To use keylaunch add it to your .xinitrc

Keylaunch reads a users .keylaunchrc file

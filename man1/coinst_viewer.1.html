NAME

Coinst_viewer - view co-installability kernel generated by coinst for X
 

SYNOPSIS

coinst_viewer file.dot
 

DESCRIPTION

coinst_viewer reads the graph of a co-installability produced with coinst and then layouted with dot, and displays it.

NAME

tsspcrallocate - manual page for tsspcrallocate v713

DESCRIPTION

pcrallocate
Runs TPM2_PCR_Allocate
Allocates SHA-1 and/or SHA-256 banks for a full set of PCR 0-23. Not all TPMs support two banks
[-pwdp hierarchy password (default empty)] -sha1 allocate a SHA-1 bank -sha256 allocate a SHA-256 bank

SEE ALSO

The full documentation for tsspcrallocate is maintained as a Texinfo manual. If the info and tsspcrallocate programs are properly installed at your site, the command
info tsspcrallocate
should give you access to the complete manual.

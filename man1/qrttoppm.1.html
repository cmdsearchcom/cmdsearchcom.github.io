NAME
qrttoppm - convert output from the QRT ray tracer to a PPM image

SYNOPSIS
qrttoppm

[qrtfile]

DESCRIPTION
This program is part of Netpbm(1)

qrttoppm reads a QRT file as input and produces a PPM image as output.

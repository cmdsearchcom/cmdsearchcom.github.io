NAME
ebnupgrade - smooths transition from old ndtpd

SYNOPSIS
ebnupgrade [option...]

OPTIONS
-c FILE --configuration-file FILE
input old configuration from FILE (default: /etc/ebnetd/ebnetd.conf)

-h --help
display this help, then exit

-o FILE --output-file FILE
output new configuration to FILE (default: /etc/ebnetd/ebnetd.conf.new)

-q --quiet --silence
suppress report messages

-t --test
only check for input file

-v --version
display version number, then exit

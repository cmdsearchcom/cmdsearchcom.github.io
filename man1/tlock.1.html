NAME

tlock - terminal lock

SYNOPSIS

tlock [OPTIONS]
 

DESCRIPTION

tlock is a console based application intended to lock the terminal until the correct password is supplied by the user. Terminal is locked with the user's login password.
 

OPTIONS

The following options are supported by tlock
-h, --help
Prints the short usage information
-v, --version
Prints the tlock version information

NAME
efax-gtk-faxfilter - Wrapper around efax-gtk-socket-client

SYNOPSIS
efax-gtk-faxfilter
efax-gtk-faxfilter is a script which drives efax-gtk-socket-client - it is the way the user identifies the hostname and the port number for efax-gtk-socket-client.

efax(1), efax-gtk(1), efax-gtk-socket-client(1)

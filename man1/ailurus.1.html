NAME

Ailurus - a simple application installer and GNOME tweaker

SYNOPSIS

ailurus [ OPTIONS ]

DESCRIPTION

Ailurus is a simple application installer and GNOME tweaker which makes Linux easier to use. It supports Fedora, Ubuntu, ArchLinux and Mint.

BUGS

Please report all bugs to the Ailurus bug tracker: http://code.google.com/p/ailurus/issues/list

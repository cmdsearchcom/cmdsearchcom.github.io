NAME

podofoimg2pdf - Convert images to PDF files

SYNOPSIS

podofoimg2pdf [output.pdf] [-useimgsize] [image1 image2 image3 ...]

DESCRIPTION

podofoimg2pdf is one of the command line tools from the PoDoFo library that provide several useful operations to work with PDF files. This tool will combine any number of images into a single PDF. This is useful for creating a document from scanned images. Large pages will be scaled to fit the page and images smaller than the defined page size will be centered.
Supported image formats:
JPEG
 
PNG
 
TIFF

OPTIONS

-useimgsize
Use the image size as page size instead of A4

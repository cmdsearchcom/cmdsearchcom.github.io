NAME
mprof-decoder - Console decoder for the logging profiler output files

SYNOPSIS
mprof-decoder file

DESCRIPTION
mprof-decoder Console decoder for the logging profiler output files

Decodes the contents of a logging profiler output file and prints the data on standard output.

Options
None

ENVIRONMENT VARIABLES
None

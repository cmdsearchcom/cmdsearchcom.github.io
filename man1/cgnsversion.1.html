NAME

cgnsversion - returns the CGNS version of the file

DESCRIPTION

usage : cgnsversion [options] Version CGNSfile [CGNSoutfile] options:
-v : verbose output
-r : remove links
-k : keep all nodes

options:

-v : verbose output
-r : remove links

NAME

git-annex-indirect - switch repository to indirect mode

SYNOPSIS

git annex indirect

DESCRIPTION

Switches a repository back from direct mode to the default, indirect mode.
Note that the direct mode/indirect mode distinction is removed in v6 git-annex repositories.

NAME
init-ltsp â init wrapper to configure LTSP

DESCRIPTION
init-ltsp is run very early in the boot process of an LTSP system as the init process, configuring the system for LTSP and then handing off booting of the system to the real
init(1). The bulk of the logic is implemented in shell fragments in /usr/share/ltsp/init-ltsp.d/. Variables configured typically include values set from lts.conf(5), or from the
kernel commandline.

EXAMPLE
A typical invocation of init-ltsp from the kernel commandline might look like:

init=/sbin/init-ltsp quiet root=/dev/nbd0

SEE ALSO
init(1), lts.conf(5)

AVAILABILITY
init-ltsp is part of ltsp package and the latest versions are available in source form from https://launchpad.net/products/ltsp.

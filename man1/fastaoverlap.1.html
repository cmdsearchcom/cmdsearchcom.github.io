DESCRIPTION
These programs are not described by the upstream author, however, some information is available via the --help and -h arguments for all these programs. Whoever is using any of
these programs in routine, is requested to provide a seed of a respective man page to the upstream author.

OPTIONS
-h, --help Show summary of options.

-v, --version
Show version of program.

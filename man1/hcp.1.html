NAME

mpicc, mpiCC, hcc, hcp - Compile LAM/MPI C/C++ programs.

SYNOPSIS

hcc [-showme] ...
hcp [-showme] ...

OPTIONS

-showme
Does not invoke the underlying C/C++ compiler. Instead, it shows the command line that would be executed to compile the C/C++ program.
See cc(1) and CC(1) (or whatever your underlying C/C++ compilers are) for all other options.

DESCRIPTION

The names hcc and hcp have been deprecated in favor of mpicc and mpiCC. Please see their corresponding man pages.

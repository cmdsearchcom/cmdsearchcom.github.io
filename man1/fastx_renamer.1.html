NAME
fastx_renamer - FASTA/Q Renamer

DESCRIPTION
usage: fastx_renamer [-n TYPE] [-h] [-z] [-v] [-i INFILE] [-o OUTFILE] Part of FASTX Toolkit 0.0.14 by A. Gordon (assafgordon@gmail.com)

[-n TYPE]
= rename type:

SEQ - use the nucleotides sequence as the name. COUNT - use simply counter as the name.

[-h] = This helpful help screen.

[-z] = Compress output with GZIP.

[-i INFILE]
= FASTA/Q input file. default is STDIN.

[-o OUTFILE] = FASTA/Q output file. default is STDOUT.

LIBNFO-READER(1) General Commands Manual LIBNFO-READER(1)

NAME

libnfo-reader - an utility to read NFO files which uses libnfo

SYNOPSIS

libnfo-reader <file>

DESCRIPTION

This manual page documents briefly the libnfo-reader command.
libnfo-reader parses an NFO file and outputs its contents to the standard output.

OPTIONS

none

TITLE

iinfo -- print information about images

SYNOPSIS

Usage:   iinfo [options] filename...

DESCRIPTION

This program is part of the OpenImageIO (http://www.openimageio.org) tool suite. Detailed documentation is avaliable in pdf format with the OpenImageIO distribution.

OPTIONS

--help
Print help message
-v
Verbose output
-m %s
Metadata names to print (default: all)
-f
Prefix each line with the filename
-s
Sum the image sizes
-a
Print info about all subimages
--hash
Print SHA-1 hash of pixel values
--stats
Print image pixel statistics (data window)

NAME
compose-create-legacy-composeinfo - create legacy .composeinfo file

SYNOPSIS
compose-create-legacy-composeinfo [OPTIONS...] COMPOSE

DESCRIPTION
compose-create-legacy-composeinfo reads composeinfo.json and generates metadata in the legacy INI-style format.

OPTIONS
-h, --help
Print help and exit.

-p, --path=DIR
Set directory in which to create the files.

By default the compose/ subdirectory will be used. This is where most consumers expect the file to be.

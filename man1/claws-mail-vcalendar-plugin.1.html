NAME
claws-mail-vcalendar-plugin - vCalendar plugin for Claws Mail

DESCRIPTION
This manual page documents briefly the claws-mail-vcalendar-plugin.

This manual page was written for the Debian distribution because the original program does not have a manual page.

claws-mail-vcalendar-plugin is a plugin (loadable module) for the Claws Mail mailer.

Allows displaying vCalendar messages in a similar way other well-known mailers do. Only the meeting subset of the vCalendar format is currently supported.

USAGE
Before using a plugin you must instruct Claws Mail to load it on startup.

For this you must go âConfigurationâ menu on main window toolbar, open âPlugins...â dialog, click on the âLoad plugin...â button and select the plugin file, named vcalendar.so,
and press the âOpenâ button.

Once loaded, a new mailbox named âvCalendarâ will appear on the folder list window. In this folder a âMeetingsâ inbox is shown, which is where your meetings can be created.
Contextual menu of this folder is also useful for changing view mode or adding webCal subscriptions.

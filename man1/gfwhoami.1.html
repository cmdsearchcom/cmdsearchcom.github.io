NAME
gfwhoami - print user name on the Gfarm filesystem

SYNOPSIS
gfwhoami [options]

DESCRIPTION
gfwhoami prints a global user name on the Gfarm filesystem.

OPTIONS
-v
Prints the Subject DN of the user for GSI authentication.

-h
Displays a list of command options.

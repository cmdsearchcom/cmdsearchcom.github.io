NAME
ikiwiki-hosting-web-daily - compact and maintain all sites

SYNOPSIS
ikiwiki-hosting-web-daily

DESCRIPTION
This script is run daily by cron, as a low-priority job. It does various cleanup and bookkeeping tasks, including rotating sites logs, and updating their calendars.

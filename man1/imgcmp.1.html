NAME

imgcmp - Image comparison utility
 

SYNOPSIS

imgcmp [options]
 

DESCRIPTION

The imgcmp command compares two images of the same geometry with respect to a given metric. Please use the --help command line switch and the JasPer Software Reference Manual for more information.

Data model
The basic atom of storage in Tarantool is called tuple. A tuple can have arbitrary number of fields. The first field or fields in the tuple are always the identifying unique key.
Tuples form spaces. It is possible to search tuples in spaces using the primary or secondary keys. It is possible to change, as well as add or remove fields.

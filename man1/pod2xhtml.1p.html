NAME
pod2xhtml - convert .pod files to .xhtml files

SYNOPSIS
pod2xhtml [--help] [--infile INFILE] [--outfile OUTFILE] [OPTIONS]

DESCRIPTION
Converts files from pod format (see perlpod) to XHTML format.

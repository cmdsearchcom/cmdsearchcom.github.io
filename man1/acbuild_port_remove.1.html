NAME

acbuild-port-remove - Remove a port
 
 

SYNOPSIS

acbuild port remove [OPTIONS]
 
 

DESCRIPTION

Updates the ports in the ACI's manifest to include a port with the given name and value
 
 

OPTIONS INHERITED FROM PARENT COMMANDS

--debug[=false]
Print out debug information to stderr
 
--modify=""
Path to an ACI to modify (ignores build context)
 
--no-history[=false]
Don't add annotations with the command that was run
 
--work-path="."
Path to place working files in
 
 

EXAMPLE

 
acbuild port remove https

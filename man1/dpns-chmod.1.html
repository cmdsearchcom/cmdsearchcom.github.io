NAME
dpns-chmod - change access mode of a DPNS directory/file in the name server

SYNOPSIS
dpns-chmod absolute_mode path...

DESCRIPTION
dpns-chmod sets the access mode of DPNS directory/file(s) in the name server to the octal value in absolute_mode. Symbolic mode changes are not supported yet.

The effective user ID of the process must match the owner of the file or the caller must have ADMIN privilege in the Cupv database.

path specifies the DPNS pathname. If path does not start with /, it is prefixed by the content of the DPNS_HOME environment variable.

EXIT STATUS
This program returns 0 if the operation was successful or >0 if the operation failed.

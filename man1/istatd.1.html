NAME

istatd — serving statistics to your istat iphone application

SYNOPSIS

istatd [-c config] [-a address] [-p port] [-d]
 

DESCRIPTION

istatd is a server daemon for serving statistics to your istat iphone application. istatd collects data such as cpu, mem, network and disk usage and keeps the history. once connecting from the iphone and entering the lock code this data will be sent to the iphone and shown in fancy graphs.
 

OPTIONS

-v
Print version number
-d
Run in the process in background
-c config
Use a custom config file search path
-a address
Listen for network connections on this address
-p port
Listen for network connections on this port
-u user
User to run daemon as
-g group
Group to run daemon as
--pid path
Custom pid file location
--clear-sessions
Clear all saved authorized sessions.

FILES

/etc/istat.conf
Configuration for network settings, lock code, running user, devices to monitor and more.

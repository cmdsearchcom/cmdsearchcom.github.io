NAME
yokadid - commandline todo system

SYNOPSIS
yokadid [options]...

DESCRIPTION
yokadid is a Yokadi daemon that remind you due tasks. If you want to be automatically reminded of due tasks, you can use the Yokadi daemon. The Yokadi daemon can be launched via
desktop autostart services.

In most desktop environments, you just need to create a symbolic link to yokadid (or a shell script that calls it) in $HOME/.config/autostart/.

ln -s ‘which yokadid‘ $HOME/.config/autostart/

OPTIONS
These programs follow the usual GNU command line syntax, with long options starting with two dashes (‘-’). A summary of options is included below.

-d, FILE --db=FILE
TODO database.

-k, --kill
Kill Yokadi Daemon (you can specify database with -db if you run multiple Yokadid).

-f, --foreground
Don’t fork background. Useful for debug.

-h, --help
Show summary of options and exit.

TSSCREATEEK(1) User Commands TSSCREATEEK(1)

NAME

tsscreateek - manual page for tsscreateek v713

DESCRIPTION

createek
Parses and prints the various EK NV indexes specified by the IWG Creates a primary key based on the EK NV indexes
-te print EK Template -no print EK nonce -ce print EK certificate -cp CreatePrimary using the EK template and EK nonce
[-noflush Do not flush the primary key after validation
[-root filename validate EK certificates against the root)]
filename contexts a list of PEM certificate filenames, one per line the list may contain up to 100 certificates
-alg (rsa or ec)

SEE ALSO

The full documentation for tsscreateek is maintained as a Texinfo manual. If the info and tsscreateek programs are properly installed at your site, the command
info tsscreateek
should give you access to the complete manual.

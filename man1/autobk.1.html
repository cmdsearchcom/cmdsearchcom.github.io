AUTOBK(1) IFEFFIT Suite AUTOBK(1)

NAME

autobk - automatic XAFS background removal

SYNOPSIS

autobk

DESCRIPTION

autobk is a command line program which belongs to the IFEFFIT XAFS data analysis suite and is generally invoked from one of the interactive user interfaces, ifeffit or gifeffit.
Program documentation in PDF format may be found in /usr/share/doc/ifeffit/pdf/autobk.pdf.

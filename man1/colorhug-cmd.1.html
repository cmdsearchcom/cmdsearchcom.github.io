NAME
colorhug-cmd - A command line utlity for the ColorHug colorimeter

SYNOPSIS
colorhug-cmd [ --verbose ]

DESCRIPTION
This manual page documents briefly the colorhug-cmd command.

colorhug-cmd allows you to do low level commands on your Hughski ColorHug colorimeter.

OPTIONS
This program follows the usual GNU command line syntax, with long options starting with two dashes (‘-’). A summary of options is included below.

--help
Show summary of options.

--verbose
Show extra debugging information.

ENVIRONMENT
Setting COLORHUG_OUTPUT=plain will not print any human readable text which makes the output easy to parse.

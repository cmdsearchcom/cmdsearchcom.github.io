NAME
btsethttpseeds â sets http-seeds information in torrents

SYNOPSIS
btsethttpseeds seedURLlist file [file ...]

DESCRIPTION
This manual page documents briefly the btsethttpseeds command.

This manual page was written for the Debian distribution because the original program does not have a manual page.

btsethttpseeds is a program which will change the http seed URLs of an existing torrent file. The already existing torrent specified by the file argument will be modified to use
the new http seed URLs given by the seedURLlist argument. These alternate URLs will be used to seed the file if there are no other seeds available.

The seedURLlist can be ’0’, or formatted as follows:

URL[|URL ...]

if the seedURLlist is ’0’, any http seeds currently in file will be stripped.

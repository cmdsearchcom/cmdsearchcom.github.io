NAME
sysconftoolcheck - check a sysconftool configuration file

SYNOPSIS
sysconftoolcheck {filename}

DESCRIPTION
sysconftoolcheck reads filename and checks if it is a valid sysconftool configuration file. sysconftoolcheck is a useful tool to use in order to spot often occuring typos.
sysconftoolcheck can’t really know whether or not a sysconftool configuration file will work as expected, but it can be used to flag some obvious errors and omissions.
sysconftoolcheck abilities will increase, over time.

NAME

dawgdic-find - find words in a DAWG-based dictionary

USAGE

dawgdic-find [options] dictionary-file [lexicon-file]

DESCRIPTION

dawgdic-find queries DAWG dictionary dictionary-file for words read from lexicon-file (or from standard input).

OPTIONS

-h
Display help and exit.
-g
Load dictionary with guide.
-r
Load dictionary with ranked guide.

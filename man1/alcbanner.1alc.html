NAME

alcbanner - Display a standardized banner for Alliance tools
 
 
See the file man1/alc_origin.1alc.
 
 
 

SYNOPSIS

alcbanner tool toolversion purpose copyrightdates
 

DESCRIPTION

alcbanner display on stdout a standardized banner for the Alliance tools. This is a compiled version of the alliancebanner(2) function.
 
 

EXAMPLE

alcbanner GenPat 3.1 "Procedural GENeration of test PATterns" 1991

DIAGNOSTICS

The output is centered for a 80 columns screen, so it can be used on most display easily, and not found at execution time.
 
alliancebanner: Error: Resulting size bigger than 80 columns not allowed
The tool argument must be such that its resulting size isn't wider that 80 culumns. This means, since the character set is proportional, that the longest name to be output is about 8 characters long.
alliancebanner: Error: Character out of [0-9A-Za-z] range
The tool parameter has a non allowed character in it. For simplicity purposes, only numbers and letters are accepted.

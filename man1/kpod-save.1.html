kpod-save 1 July 2017 kpod NAME
kpod-save - Save an image to docker-archive or oci

SYNOPSIS
kpod save NAME[:TAG] [--help|-h]

DESCRIPTION
kpod save saves an image to either docker-archive on the loacl machine. kpod save writes to STDOUT by default and can be redirected to a file using the output flag. The quiet
flag suppresses the output when set.

kpod [GLOBAL OPTIONS]

kpod save [GLOBAL OPTIONS]

kpod save [OPTIONS] NAME[:TAG] [GLOBAL OPTIONS]

OPTIONS
--output, -o Write to a file, default is STDOUT

--quiet, -q Suppress the output

GLOBAL OPTIONS
--help, -h
Print usage statement

EXAMPLES
# kpod save --quiet -o alpine.tar alpine:2.6

# kpod save > alpine-all.tar alpine

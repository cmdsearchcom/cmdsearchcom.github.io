JMAPVIEWER(1) User Commands JMAPVIEWER(1)

NAME

jmapviewer - JMapViewer is a java component which allows one to easily integrate an OSM map view into your Java application, including demo program 'jmapviewer'.

SYNOPSIS

jmapviewer

DESCRIPTION

jmapviewer is a demo program for the JMapViewer library.

OPTIONS

There are no options for 'jmapviewer'.

ENVIRONMENT VARIABLES

There are no environment variables for 'jmapviewer'.

SEE ALSO

See https://wiki.openstreetmap.org/wiki/JMapViewer for more information.

NAME

mkdarts - convert sorted dictionary data to DoubleArray File

SYNOPSIS

mkdarts DictionaryFile DoubleArrayFile
 

DESCRIPTION

This manual page documents briefly the mkdarts command.
mkdarts is a program that convert already sorted dictionary data to DoubleArray File.

darts(1).

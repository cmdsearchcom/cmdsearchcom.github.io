NAME
jalv.gtkmm - Run an LV2 plugin as a JACK application (Gtkmm version).

SYNOPSIS
jalv.gtkmm PLUGIN_URI

DESCRIPTION
This is a version of Jalv with a GUI implemented in Gtkmm. It is mainly for developer testing purposes, for a production ready program use jalv.gtk.

SEE ALSO
jalv(1), jalv.gtk(1), jalv.qt(2), lv2ls(1), jackd(1)

NAME

startlazarus - starts lazarus program. fpdoc.
 

SYNOPSIS

lazarus
 

Description

startlazarus is used to start the lazarus program. It automatically checks if the user has built a custom lazarus and starts it. It is also used to restart the IDE, when the user clicks restart in the IDE.
 

Usage

You can invoke startlazarus without any arguments.

NAME

iptux - IP Messanger client
 

SYNOPSIS

iptux
 

DESCRIPTION

iptux is an IP Messenger client for Linux. For more information on IP Messanger, check http://www.ipmsg.org/index.html.en .
 
iptux support:

- auto detect other clients in the intranet.

- send message to other clients.

- send file to other clients.
 
Homepage: http://code.google.com/p/iptux
 

FILES

~/.config/iptux: users' local settings directory

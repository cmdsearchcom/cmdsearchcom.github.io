NAME
camomilecharmap - a charmap database translators for camomile.

SYNOPSIS
camomilecharmap [-d { dir }] [-help | --help] [file]

DESCRIPTION
This manual page documents briefly the camomilecharmap command.

This manual page was written for the Debian GNU/Linux distribution because the original program does not have a manual page.

camomilecharmap is made to convert standard charmap file into .mar file. This convert a standard charmap file into the internal database format of camomile.

-d dir
Output directory.

-help, --help
Display help message.

file
Charmap file.

FEDREPO-REQ-ADMIN(1) User Commands FEDREPO-REQ-ADMIN(1)

NAME

fedrepo-req-admin - manual page for fedrepo-req-admin 1.5.0

SYNOPSIS

fedrepo-req-admin [ OPTIONS] ACTION [TICKET_ID]

DESCRIPTION

Process src.fedoraproject.org repository requests.
Possible actions include "list", "process", "processall".
Examples:
fedrepo-req-admin list
fedrepo-req-admin process 1
fedrepo-req-admin processall
fedrepo-req-admin process 1 --force

OPTIONS

-f, --force
Skips Bugzilla ticket validation
-h, --help
Show this message and exit.

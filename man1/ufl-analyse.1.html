NAME
ufl-analyse - Analyse a .ufl file to find errors.

SYNOPSIS
ufl-analyse [OPTIONS] input.ufl

DESCRIPTION
Analyse a .ufl file to find errors. Optionally write information about the forms for further inspection.

OPTIONS
-h, --help
Display help text and exit.

-q QUIET, --quiet=QUIET
Do not print form information to screen.

-w WRITE, --write=WRITE
Write form information to file.

EXAMPLES
ufl-analyse --quiet=0 --write=1 mass.ufl

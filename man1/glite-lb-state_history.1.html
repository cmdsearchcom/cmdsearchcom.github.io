NAME
glite-lb-state_history - utility to show job state changes stored in Logging and Bookkeeping

SYNOPSIS
glite-lb-state_history [-f FIELDS] JOBID

DESCRIPTION
glite-lb-state_history shows the job state changes from the events stored in Logging and Bookkeeping.

OPTIONS
-f FIELDS
Job status fields to show in the history listing.

Basic information always shown:
event name
jobid
job status

Fields shown by default:
network_server
jdl:VirtualOrganisation
destination
done_code
reason

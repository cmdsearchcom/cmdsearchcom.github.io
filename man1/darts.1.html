DARTS(1) General Commands Manual DARTS(1)

NAME

darts - program to do common prefix search interactively

SYNOPSIS

darts DoubleArrayFile
 

DESCRIPTION

This manual page documents briefly the darts command.
darts is a program for interactive common prefix search.

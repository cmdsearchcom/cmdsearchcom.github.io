ARDUINO(1) General Commands Manual ARDUINO(1)

NAME
arduino - an IDE and uploader for Arduino development boards.

SYNOPSIS
arduino

DESCRIPTION
This program takes no options and will load the IDE in an X11 window.

http://www.arduino.cc/

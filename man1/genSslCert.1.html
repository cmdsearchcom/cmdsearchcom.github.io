NAME

genSslCert - Generate SSL certificates for Small-Footprint CIM Broker (sfcb)

SYNOPSIS

genSslCert

DESCRIPTION

Script to generate the server.pem and client.pem SSL certificate files to enable the Small-Footprint CIM Broker to handle https requests from CIM clients.

FILES

/usr/share/sfcb/genSslCert.sh
The genSslCert script.
/etc/sfcb/server.pem
The server certificate file.
/etc/sfcb/client.pem
The client certificate file.

PREREQUISITES

Requires OpenSSL package to be installed.

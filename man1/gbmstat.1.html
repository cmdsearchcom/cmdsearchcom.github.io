NAME
gbmstat - Computing statistics in a moving windows

SYNOPSIS
gbmstat [options]

DESCRIPTION
Compute moving average. Data are read from standard input.

OPTIONS
-s number of steps to average (default 10)

-t compute average for each column of input

-D ignore NAN entries in computation

-O set the output, comma separated list of mean, adev, std, var, skew, kurt, min ,max and nonan (default mean). nonan works only with option -D

-F specify the input fields separators (default " ")

-h this help

NAME
ltpanel - shows a tasklist in X until killed.

SYNOPSIS
ltpanel [-w=width] [-h=height] [-n=nrows] [-y=ycoord]

DESCRIPTION
Displays a tasklist.

-h display this help and exit

-v output version information and exit

NAME
bkr-workflow-simple - Simple workflow to generate Beaker jobs

SYNOPSIS
bkr workflow-simple [workflow options] [options]

DESCRIPTION
Generates a Beaker job using the (not-so-simple) options available for all workflows.

OPTIONS
Common workflow options are described in the Workflow options section of bkr(1).

Common bkr options are described in the Options section of bkr(1).

EXIT STATUS
Non-zero on error, otherwise zero.

EXAMPLES
Schedule a job on RHEL6 Server containing all tests for the apache and tomcat6 packages on a random x86_64 machine:

bkr workflow-simple --arch x86_64 --random --family RedHatEnterpriseLinux6 --variant Server --package apache --package tomcat6

NAME

carto - Carto map stylesheet compiler

SYNOPSIS

carto [OPTION] <source MML file>

DESCRIPTION

Carto is a stylesheet renderer for Mapnik. It's an evolution of the Cascadenik idea and language, with an emphasis on speed and flexibility.

OPTIONS

-v --version
Parse JSON map manifest
-b --benchmark
Outputs total compile time
-n --nosymlink
Use absolute paths instead of symlinking files

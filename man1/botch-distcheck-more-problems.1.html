NAME
botch-distcheck-more-problems - a tool like dose-distcheck but return more than the first reason

SYNOPSIS
botch-distcheck-more-problems [-h] [-v] Packages Sources

DESCRIPTION
This is a wrapper around dose-distcheck. It accepts the exact same commandline arguments and it outputs the same yaml format. The only difference is, that instead of only giving
on reason for why a binary package cannot be installed, it gives multiple reasons.

OPTIONS
--verbose
Be verbose.

--help
Print help message.

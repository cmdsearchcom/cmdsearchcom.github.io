NAME
lirc-make-devinput - create a devinput.lircd.conf for the local site.

SYNOPSIS
lirc-make-devinput [input.h]

ARGUMENTS
input.h
Header file parsed to create the configuration file.

DESCRIPTION
Reads the system header files and creates a devinput.lircd.conf which matches the local system (i. e., the local kernel). It normally requires that the kernel-headers package is
installed.

FILES
/usr/include/linux/input.h
Default system header file

SEE ALSO
The documentation for lirc is maintained as html pages. They are located under html/ in the documentation directory.

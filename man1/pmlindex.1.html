NAME
pmlindex â constructs an HTML index of PhotoML files

SYNOPSIS
pmlindex [-h] [-y] [-s size] (([-e] [-m]) | [-d]) [-p dstâpath] xmlfile [xmlfile] ...

DESCRIPTION
pmlindex is a shell script that builds an indexed set of HTML descriptions of the PhotoML files provided as command line arguments. If an HTML descriptions exists, it is only
reconstructed if it has an older time stamp than the corresponding PhotoML file.

OPTIONS
pmlindex accepts the following options:

-h Display usage information.

-y Group table entries by year.

-s size Set the primary font size in points.

-e Include a column providing exposure details when these are present in the PhotoML data.

-m Include location map when coordinates are specified in the PhotoML data.

-d Generate detailed output rather than the default summary output.

-p dstâpath
Specify the destination path for the HTML files.

NAME
a2ps-lpr-wrapper â lp/lpr wrapper script for GNU a2ps on Debian

SYNOPSIS
a2ps-lpr-wrapper [-d printer] [files]

DESCRIPTION
This manual page documents briefly the a2ps-lpr-wrapper command.

a2ps-lpr-wrapper uses either lp or lpr to send files to printer. It determines which of the two programs is currently installed and adds the appropriate parameters.

OPTIONS
-d printer
Destination system printer.

files List of files to be printed.

SEE ALSO
a2ps (1).

The programs are documented fully by the a2ps documentation available via the Info system.

NAME
DBusViewer â Show connections to the Bus and introspect on them.

DESCRIPTION
This manual page documents briefly the DBusViewer command.

DBusViewer shows connections to the Bus in a window. It allows you to introspect on them and save the corresponding Java interface definitions.

NAME

wmgenmenu - Writes a menu structure usable as ~/GNUstep/Defaults/WMRootMenu to stdout

SYNOPSIS

wmgenmenu [options]

DESCRIPTION

wmgenmenu writes a menu structure usable as ~/GNUstep/Defaults/WMRootMenu to stdout.
 
The output format is a PropList.

OPTIONS

--help
print a help message with the list of options
--version
print the version of Window Maker from which the program comes

BUGS

The program is using a built-in structure and list of applications, which means it cannot discover new programs, but it filters the programs that cannot be found in the PATH.

NAME

foomatic-nonumericalids - Convert numerical printer IDS to clear-text IDs

SYNOPSIS

foomatic-nonumericalids
 

DESCRIPTION

This script changes all old cryptic numerical printer IDs from the old PostGreSQL time to clear-text printer IDs in the installed Foomatic database.
 
It is very unlikely you need to do this.

NAME
claws-mail-address-keeper - Keep unknown outgoing addresses in address book

DESCRIPTION
This manual page documents briefly the claws-mail-address-keeper.

This manual page was written for the Debian distribution because the original program does not have a manual page.

claws-mail-address-keeper is a plugin (loadable module) for the Claws Mail mailer.

It allows saving typed outgoing addresses to address book automatically in case they are unknown, i.e., not already present in the address book.

USAGE
Before using a plugin you must instruct Claws Mail to load it on startup.

For this you must go âConfigurationâ menu on main window toolbar, open âPlugins...â dialog, click on the âLoad plugin...â button and select the plugin file, named
address_keeper.so, and press the âOpenâ button.

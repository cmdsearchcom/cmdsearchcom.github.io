NAME
idzebra-abs2dom - Converts .abs files to DOM XML configuration files

SYNOPSIS
idzebra-abs2dom [file]

DESCRIPTION
idzebra-abs2dom converts grs filter .abs files to DOM XML filter index XSLT files. The melm and xelm directives are converted to XSLT rules. Conversion of elm directives are not
supported.

zebraidx(1)

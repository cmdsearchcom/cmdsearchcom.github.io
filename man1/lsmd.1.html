LSMD(1) libStorageMgmt plug-in daemon LSMD(1)

NAME
Daemon - lsmd

DESCRIPTION
libStorageMgmt plug-in daemon. Plug-ins execute in their own process space for fault isolation and to accommodate different plug-in licensing requirements. Runs as an unpriviâ
leged user.

OPTIONS
--plugindir = The directory where the plugins are located

--socketdir = The directory for IPC sockets

-v = Verbose logging

-d = New style daemon (systemd) non-forking

BUGS

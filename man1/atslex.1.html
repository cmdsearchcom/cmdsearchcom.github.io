NAME

atslex - Lexer generator for ATS

SYNOPSIS

atslex  < file.lats > file_lats.dats

DESCRIPTION

atslex takes a lexer specification via stdin and outputs a dynamic ATS source file to stdout

SEE ALSO

atscc(1), atsopt(1).
 
These programs are documented fully by The ATS/Anairiats user's manual available via the web at http://www.ats-lang.org/

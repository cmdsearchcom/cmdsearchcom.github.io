NAME
choreonoid - an integrated robotics GUI environment

SYNOPSIS
choreonoid [OPTIONS]

DESCRIPTION
Choreonoid is an integrated robotics GUI environment, which allows users to add their own functions on the basis of various basic robotics functions.

OPTIONS
--project arg
load a project file

--start-playback
start playback automatically

--model arg
load a 3d model file

--hrpmodel arg
load an OpenHRP model file

-h, --help
show help message

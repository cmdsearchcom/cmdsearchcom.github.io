NAME
installation-report-generator - A small tool to collect installation data for support purposes

SYNOPSIS
installation-report-generator

DESCRIPTION
Installation Report Generator is a small program to help users in gathering informations about system software and hardware configuration, in order to prepare a page for the
Italian wiki’s Notebook Section. It is designed to be simple and easy to use. The generated reports can be proposed to the wiki’s Editors in order to be published.

For more detailed informations about Italian wiki’s notebook section, visit: http://wiki.ubuntu-it.org/Hardware/Notebook.

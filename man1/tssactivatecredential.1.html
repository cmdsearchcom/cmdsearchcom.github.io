NAME

tssactivatecredential - manual page for tssactivatecredential v713

DESCRIPTION

activatecredential
Runs TPM2_ActivateCredential
-ha activation handle of object associated with the certificate
-hk handle of loaded decryption key
-icred input credential file name
-is secret file name
[-pwda password for activation key (default empty)] [-pwdk password for decryption key (default empty)]
[-ocred output credential file name (default do not save)]
-se[0-2] session handle (default PWAP)
01 continue 20 command decrypt 40 response encrypt

SEE ALSO

The full documentation for tssactivatecredential is maintained as a Texinfo manual. If the info and tssactivatecredential programs are properly installed at your site, the command
info tssactivatecredential
should give you access to the complete manual.

NAME
rbm-showconf - Show rbm configuration

SYNOPSIS
rbm showconf [project] [config name] [options]

DESCRIPTION
Without argument, dumps all configuration. With one argument, shows the configuration for the selected project. With two or more arguments, show the projectâs config for a
specific value.

COMMAND LINE OPTIONS
See rbm_cli(7).

EXAMPLES
Show all configuration :

$ rbm showconf

Show configuration for rbm project :

$ rbm showconf rbm

Show git_url value for the rbm project :

$ rbm showconf rbm git_url

SEE ALSO
rbm(1), rbm_config(7)

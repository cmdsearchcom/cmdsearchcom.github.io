SUPERKB(1) User Commands SUPERKB(1)

NAME
superkb - A hotkey-based application launcher with on-screen keyboard.

DESCRIPTION
usage: superkb [options]

OPTIONS
-0 Quit when Superkb is ready (for timing and debugging).

-d level
Show debug messages up to the specified verbosity level.

-h Shows this help.

SEE ALSO
The full documentation for superkb and its configuration is maintained at the project website:

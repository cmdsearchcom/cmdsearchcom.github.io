NAME

lrzuntar - Extract whole tarball from .tar.lrz files

SYNOPSIS

  lrzuntar [options] DIRECTORY.tar.lrz

DESCRIPTION

lrzuntar is identical to "lrztar -d" used to decompress lrzip compressed tarballs.

OPTIONS

See lrzip(1)

ENVIRONMENT

None.

FILES

None.

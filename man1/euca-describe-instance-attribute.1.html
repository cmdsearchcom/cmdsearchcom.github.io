NAME

euca2ools - Show one of an instance's attributes.

DESCRIPTION

Show one of an instance's attributes.
Note that exactly one attribute may be shown at a time.
euca-describe-instance-attribute [-a, --access-key <access_key> ][-s, --secretkey <secret_key> ][--config <config_path>
][--debug][--debugger][-h, --help][--region <region_name> ][-U, --url <url> ][--version ][--euca-auth][-t, --instancetype][--kernel][--ramdisk][--user-data ][--disable-api-termination][--instanceinitiated-shutdown-behavior][--root-devicename][-b, --block-device-mapping][--sourcedest-check][-g, --group-id][-p, --product-code ][--ebs-optimized] instance_id
REQUIRED PARAMETERS
instance_id
ID of the instance to describe
OPTIONAL PARAMETERS
-a,--access-key
User's Access Key ID.
-s,--secret-key
User's Secret Key.
--config
Read credentials and cloud settings from the specified config file (defaults to $HOME/.eucarc or /etc/euca2ools/eucarc).
--debug
Turn on debugging output.
--debugger
Enable interactive debugger on error
-h,--help
Display this help message.
--region
region to direct requests to
-U,--url
URL of the Cloud to connect to.
--version
Display the version of this tool.
--euca-auth
Use NC authentication mode
-t,--instance-type
show the instance's type
--kernel
show the instance's kernel ID
--ramdisk
show the instance's ramdisk ID
--user-data
show any user-data supplied to the instance
--disable-api-termination
whether the instance can be terminated using euca-terminate-instances
--instance-initiated-shutdown-behaviorwhether the instance will stop or terminate
when shut down
--root-device-name
name of the instance's root device volume
-b,--block-device-mapping
block device mappings
--source-dest-check
whether a VPC instance has source/destination checking of its network traffic enabled
-g,--group-id
the security groups the instance belongs to
-p,--product-code
product codes associated with the instance
--ebs-optimized
whether the instance is optimized for EBS I/O

EBNDAILY(1) User Commands EBNDAILY(1)

NAME

ebndaily - get stats for ebnetd and send it to admins

SYNOPSIS

ebndaily [ option...] mail-address...

OPTIONS

-a INTEGER --ages INTEGER
keep INTEGER ages of old syslog files (default: 7)
-c TYPE --compressor TYPE
how to compress old syslog files; compress, gzip, bzip2 or none (default: none)
-h --help
display this help, then exit
-l FILE --log-file FILE
specify a syslog file (default: /var/log/ebnetd.log)
-n --no-mail
do not send report mail
-v --version
display version number, then exit
-1 --fast
compress faster when using gzip or bzip2
-9 --best
compress better when using gzip or bzip2

Arguemnts:

mail-address
receipient of a report from ebndaily.

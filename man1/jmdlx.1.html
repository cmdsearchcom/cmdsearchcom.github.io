NAME
jmdlx - wxWidgets jugglemaster

SYNOPSIS
jmdlx [options] [siteswap]

DESCRIPTION
This is the wxWidgets interface for jugglemaster. If no siteswap was given on the commandline ‘3’ will be used. If you do not know what a siteswap is, please have a look at
http://www.juggling.org/help/siteswap/faq.html.

OPTIONS
-h, --help
Show summary of options.

-p, --pattern <name>
Use a named pattern from the pattern library.

-s, --style <name>
Use given style. Valid values are Normal, Reverse, Shower, Mills Mess, Center, Windmill and Random.

NAME

dom-apply-patches - Applies all patches found in debian/patches/series in a new local branch patch-queue

SYNOPSIS

dom-apply-patches

DESCRIPTION

•dom-apply-patches creates a local branch patch-queue from upstream branch
 
•Applies every patch found in master:debian/patches/series

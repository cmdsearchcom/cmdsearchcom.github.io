NAME
tssdictionaryattacklockreset - manual page for tssdictionaryattacklockreset v713

DESCRIPTION
dictionaryattacklockreset

Runs TPM2_DictionaryAttackLockReset

[-pwd lockout auth password (default empty)]

-se[0-2] session handle / attributes (default PWAP)

-se1 session handle / attributes (default PWAP)

01 continue

SEE ALSO
The full documentation for tssdictionaryattacklockreset is maintained as a Texinfo manual. If the info and tssdictionaryattacklockreset programs are properly installed at your
site, the command

info tssdictionaryattacklockreset

should give you access to the complete manual.

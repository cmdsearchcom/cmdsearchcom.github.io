NAME

mailutils-config - deprecated mailutils build configuration tool

DESCRIPTION

This is a deprecated interface to GNU Mailutils configuration facilities. It will be removed in future versions. Please, consider using these alternatives instead:
Traditional usage
| Use instead
------------------------------+-------------------
mailutils-config --compile
| mailutils cflags
mailutils-config --link
| mailutils ldflags
mailutils-config --info
| mailutils info
mailutils-config --query
| mailutils query
mailutils-config --query=FILE | mailutils query -f FILE
For more information, try `mailutils --help'.

SEE ALSO

The complete GNU mailutils documentation is maintained as a Texinfo manual. If the mailutils-doc package is installed, the command

NAME
chewing-editor - chewing userphrase editor

SYNOPSIS
chewing-editor [-v|--version] [-h|--help] [-d|--debug]

DESCRIPTION
chewing-editor is a cross platform chewing user phrase editor written in Qt5. It provides a user friendly GUI to manage user phrase. With it, user can customize user phrase to
enhance chewing input performance.

-d, --debug

Enable debug message.

-v, --version

Print program version.

-h, --help

Print help message.

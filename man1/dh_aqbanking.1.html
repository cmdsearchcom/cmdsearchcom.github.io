NAME
dh_aqbanking - substitutes the correct AqBanking dependencies

SYNOPSIS
dh_aqbanking

DESCRIPTION
dh_aqbanking is a debhelper program that is responsible for generating the ${aqbanking:Depends} substitutions and adding them to substvars files.

SEE ALSO
debhelper(7)

This program is a part of libaqbanking35-dev.

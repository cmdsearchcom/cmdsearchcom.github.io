NAME
s2tc_decompress - Decompresses the texture into an image

SYNOPSIS
s2tc_decompress

DESCRIPTION
s2tc_decompress allows decompression of S2TC format textures into TGA images.

OPTIONS
-i INFILE.dds

-o OUTFILE.tga

-l libtxc_dxtn.so Path to an implementation of libtxc-dxtn

AUTHOR
s2tc_decompress is part of the S2TC toolset

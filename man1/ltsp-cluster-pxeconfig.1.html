NAME
ltsp-cluster-pxeconfig â Symlink generator for LTSP-Cluster and pxelinux.

SYNOPSIS
ltsp-cluster-pxeconfig

DESCRIPTION
ltsp-cluster-pxeconfig is a daemon scanning the LTSP-Cluster database and generating a symlink to the right pxe configuration file for each nodes.

<configuration file> The configuration is usually stored in /etc/default/ltsp-cluster-pxeconfig

AVAILABILITY
ltsp-cluster-pxeconfig is part of ltsp-cluster-pxeconfig package and the latest versions are available in source form from https://launchpad.net/ltsp-cluster.

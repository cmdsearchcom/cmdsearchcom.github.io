mtx-pdf(1) ConTeXt PDF Helpers mtx-pdf(1)

NAME

mtx-pdf

SYNOPSIS

mtxrun --script pdf [ OPTIONS ... ] [ FILENAMES ]

DESCRIPTION

ConTeXt PDF Helpers

OPTIONS

--info
show some info about the given file
--metadata
show metadata xml blob
--fonts
show used fonts (--detail)
--linearize
linearize given file

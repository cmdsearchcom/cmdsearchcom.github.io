NAME
lxtask - a lightweight GTK+ task manager.

SYNOPSIS
lxtask

DESCRIPTION
LXTask - lightweight and desktop-independent task manager derived from xfce4-taskmanager with all dependencies on xfce removed, new features, and some improvement of the user
interface.

OPTIONS
No options are supported by this program.

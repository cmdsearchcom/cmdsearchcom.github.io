NAME
desktopnova - change your desktop wallpaper automatically

DESCRIPTION
DesktopNova is a program, that can change your desktop wallpaper automatically in a given time interval and/or at the program’s start. At the moment the program can only be used
with a gnome system.

To start DesktopNova you can execute desktopnova in a terminal.

You can also start the program via the starter in the gnome menu.

* Applications > Accessories > DesktopNova

FILES
DesktopNova looks for the configuration at the following location:

~/.config/desktopnova/desktopnova.xml

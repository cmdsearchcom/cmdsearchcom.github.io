wxGUI Map Swipe DESCRIPTION
The Map Swipe is a wxGUI component which allows the user to interactively compare two raster maps of the same area by revealing different parts of the raster maps. It is useful
e.g. for comparing raster maps from different time periods. Map Swipe can be launched from the menu File -> Map Swipe.

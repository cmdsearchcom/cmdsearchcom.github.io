NAME

gajim-history-manager — a tool to manage gajim(1) logs

SYNOPSIS

gajim-history-manager -h

gajim-history-manager [-c directory]

DESCRIPTION

gajim-history-manager is a tool to manage (do some cleanup) log file of gajim(1). Use this program to delete or export logs. For more information on database logs see <https://trac.gajim.org/wiki/LogsDatabase>.

OPTIONS

-h --help
Show help options
-c --config-path directory
Where to look for logs file

FILES

$XDG_DATA_DIR/gajim/logs.db
The history database log file used when [-c] is not specified.

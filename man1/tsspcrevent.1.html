NAME

tsspcrevent - manual page for tsspcrevent v713

DESCRIPTION

pcrevent
Runs TPM2_PCR_Event
-ha pcr handle
-ic data string
-if data file
[-of1 sha1 output digest file] [-of2 sha256 output digest file] [-of3 sha384 output digest file]

SEE ALSO

The full documentation for tsspcrevent is maintained as a Texinfo manual. If the info and tsspcrevent programs are properly installed at your site, the command
info tsspcrevent
should give you access to the complete manual.

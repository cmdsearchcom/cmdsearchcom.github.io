NAME
dostime - manual page for dostime 2.1.1

SYNOPSIS
dostime [-opt] <time>

DESCRIPTION
Where -opt is one or more of:
--dec display the result in decimal

--dos convert a Unix time to a DOS time (default)

--help display this help screen

--hex display the result in hexadecimal

--unix convert a DOS time to a Unix time

--version
print the library version and exit

EXAMPLES
to convert a Unix time to a DOS time:

dostime 956794294

to convert a DOS time to a Unix time:

dostime --unix 681216369

Output: <time entered on command line> <converted time> <date (YYYY/MM/DD)> <time (HH:MM:SS)>

Note: <converted time> shows -1 if an error occurs

NAME
gsymfix - automatically fix common issues with gEDA symbols

SYNOPSIS
gsymfix [ options ] symbol_file

DESCRIPTION
gsymfix reads a gEDA symbol file and does the following:

Â· prints a warning if there is no pinnumber attribute on a pin;

Â· Renumbers pinseq attributes on all pins, starting with 1;

Â· Adds a pinseq attribute to a pin if none exists;

Â· Adds a pinlabel=n_a attribute to a pin if none exists;

Â· Adds a pintype=io attribute to a pin if none exists;

Â· Adds a numslots=0 attribute to the symbol if none exists;

Â· Adds a device=XXX attribute to the symbol if none exists;

Â· Adds a footprint=XXX attribute to the symbol if none exists.

The idea here is to fix common cut-and-paste issues and fix up the symbol so that it will pass gsymcheck(1) with no errors or warnings.

OPTIONS
gsymfix accepts the following options:

--help Displays this help message.

--verbose
Enables verbose output.

--vverbose
Enables *very* verbose output.

--version
Shows the version of this program.

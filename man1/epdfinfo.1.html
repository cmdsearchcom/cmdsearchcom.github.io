NAME
epdfinfo - server for rendering and manipulating pdf.

SYNOPSIS
epdfinfo

DESCRIPTION
This program is intended to be run from the pdf-tools Emacs package. It works on a command/response basis, you could look at the pdf-info.el code to see how to interact with it.

NAME
wmlsdasm - WMLScript disassembler

SYNOPSIS
wmlsdasm [-cfnh] file...

DESCRIPTION
wmlsdasm un-compilers WMLScript bytecode files into a source version. It is useful for debugging the WMLScript compiler.

SEE ALSO
wmlsc(1), kannel(8).

NAME
po2xliff - Convert Gettext PO localization files to XLIFF localization files.

SYNOPSIS
po2xliff [--version] [-h|--help] [--manpage] [--progress PROGRESS] [--errorlevel ERRORLEVEL] [-i|--input] INPUT [-x|--exclude EXCLUDE] [-o|--output] OUTPUT [-t|--template TEMâ
PLATE] [-S|--timestamp]

DESCRIPTION
See: http://docs.translatehouse.org/projects/translate-toolkit/en/latest/commands/xliff2po.html for examples and usage instructions.

OPTIONS
--version
show program’s version number and exit

-h/--help
show this help message and exit

--manpage
output a manpage based on the help

--progress
show progress as: dots, none, bar, names, verbose

--errorlevel
show errorlevel as: none, message, exception, traceback

-i/--input
read from INPUT in po, pot formats

-x/--exclude
exclude names matching EXCLUDE from input paths

-o/--output
write to OUTPUT in xlf format

-t/--template
read from TEMPLATE in xlf format

-S/--timestamp
skip conversion if the output file has newer timestamp

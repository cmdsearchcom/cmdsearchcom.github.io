NAME

kgb-split-config - helper script for splitting out project configurations

SYNOPSIS

kgb-split-config

DESCRIPTION

kgb-split-config extracts project configurations from /etc/kgb-bot/kgb.conf and dumps them as separate files into /etc/kgb-bot/kgb.conf.d/.

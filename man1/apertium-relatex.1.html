NAME

apertium-relatex - This application is part of ( apertium )
This tool is part of the apertium open-source machine translation toolbox: http://www.apertium.org.

SYNOPSIS

apertium-relatex [ <input file> [ <output file> ] ]

DESCRIPTION

apertium-relatex This filter preprocess apertium generator output and removes superblanks marks to do subsequent processing in apertium pipeline.

OPTIONS

-h, --help
Display this help.

SEE ALSO

apertium-destxt(1), apertium-prelatex(1), apertium-postlatex(1), apertium-desatex(1), apertium-postlatex-raw(1), lt-proc(1), apertium(1).

BUGS

Complicated constructions in LaTeX (i.e. custom defined tags) are not (yet) supported.

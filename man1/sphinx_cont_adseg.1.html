NAME

sphinx_cont_adseg - Read audio from the audio device and segment it

SYNOPSIS

sphinx_cont_adseg sampling-rate silence-threshold

DESCRIPTION

This program reads audio from the system audio device and segments it into individual non-silence regions. The argument sampling-rate is the sampling rate to read audio at (e.g. 16000), while the argument silence-threshold is the number of seconds to wait before deciding that a non-silence region is done (e.g. 1.0).

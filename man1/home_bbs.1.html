NAME

home_bbs - find home BBS or force a home BBS for the callsign

SYNOPSIS

home_bbs <user_callsign> [<bbs_callsign>]

DESCRIPTION

When only callsign is specified, home_bbs scans the BBS database and tries to find the home BBS for the callsign. When also the address is specified, home_bbs changes the database to force the specified BBS as the home BBS for the callsign.

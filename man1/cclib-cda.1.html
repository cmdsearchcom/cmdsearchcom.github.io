NAME
cclib-cda - command-line interface for charge decomposition analysis with cclib

SYNOPSIS
cclib-cda [files]

DESCRIPTION
The command cclib-cda a simple command-line interface to cclib that invokes the charge decomposition analysis (CDA) for a given log file.

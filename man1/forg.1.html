NAME
forg - graphical gopher:// client

SYNOPSIS
forg [gopher-url]

DESCRIPTION
forg is a graphical gopher client written in Python. It includes bookmarking, browser-style navigation, launching specific file types with user-specified programs, and Gopher+
compliance.

OPTIONS
gopher-url
Specifies the initial URL to open when forg starts. If this is not specified, forg will load the home page as set in ~/.forg/forgrc or will load the built in default of
gopher://gopher.floodgap.com:70/. forg requires that an absolute URI be used; thus the gopher:// protocol part is required. The port number nor any path information needs
to be specified, however.

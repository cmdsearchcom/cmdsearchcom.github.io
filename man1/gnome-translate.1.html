NAME
gnome-translate - GNOME interface to libtranslate.

SYNOPSIS
gnome-translate

DESCRIPTION
GNOME interface to libtranslate.

GNOME Translate is a GNOME interface to libtranslate. It can translate a text or web page between several natural languages, and it can automatically detect the source language
as you type.

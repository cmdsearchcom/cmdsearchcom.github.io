NAME

beauti - Bayesian Evolutionary Analysis Utility

SYNOPSIS

beauti [ nexus_file template_file beast_file]

DESCRIPTION

BEAUti is a graphical user-interface (GUI) application for generating beast-mcmc(1) XML files.

BUGS

The BEAUti bug tracking system can be found at http://code.google.com/p/beast-mcmc/issues/list.

SEE ALSO

beast-mcmc(1).

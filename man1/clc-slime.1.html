NAME

clc-clbuild - invoke lisp with clbuild active

SYNOPSIS

clc-slime
 

DESCRIPTION

This is a thin wrapper round clbuild for use with common lisp controller. In effect it will just run
 
clbuild slime
 
this is not working at the moment due to a patch we need to have in clbuild

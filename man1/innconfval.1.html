NAME
innconfval - get an InterNetNews configuration parameter

SYNOPSIS
innconfval [ -f ] [ parameter... ]

DESCRIPTION
Innconfval prints the values of the parameters specified on the command line. Values are retrieved from the inn.conf(5) file and are described there.

OPTIONS
-f Values are retrieved by using the GetFileConfigValue if the ‘‘-f’’ flag is used. Otherwise the GetConfigValue routine is used. Both are described in libinn(3).

NAME
pmd2svg - manual page for pmd2svg 0.0.3

SYNOPSIS
pmd2svg [OPTION] INPUT

DESCRIPTION
‘pmd2svg’ converts Adobe PageMaker documents to SVG.

OPTIONS
--help show this help message

--version
show version information and exit

NAME
xnightview - interactive control center for camera.

SYNOPSIS
xnightview [-] [-i fits]

DESCRIPTION
Without options create a frame. A right-click opens a control panel for a CCD. An option will display an image. The options are as follows:

-i fits load FITS image

- get names of FITS images by reading of std. input

-h print help

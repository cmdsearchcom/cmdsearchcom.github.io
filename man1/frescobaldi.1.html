NAME
frescobaldi - LilyPond Music Editor

SYNOPSIS
frescobaldi [options] files

DESCRIPTION
LilyPond Music Editor

OPTIONS
Arguments:
files LilyPond files to open, may also be textedit URLs

Options:
-v, --version
Show version information

-h, --help
Show help about options and exit

-e ENC, --encoding=ENC
Encoding to use

-l NUM, --line=NUM
Line number to go to, starting at 1

-c NUM, --column=NUM
Column to go to, starting at 0

--start <session>
Session to start (‘none’ for empty session)

-n, --new
Always start a new instance

SEE ALSO
Frescobaldi features a user manual accessible via the User Guide entry in the Help menu.

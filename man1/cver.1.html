NAME
Cver - Verilog Simulator

SYNOPSIS
cver [options][verilogfiles...]

DESCRIPTION
Cver is a full 1995 IEEE P1364 standard Verilog simulator. This is "GPL Cver", the GPL version.

SEE ALSO
The full documentation for Cver , please visit http://www.pragmatic-c.com/gpl-cver

KVPNC(1) General Commands Manual KVPNC(1)

NAME

kvpnc - KDE frontend for various vpn clients

SYNOPSIS

kvpnc [ Qt-options] [KDE-options] [options]

DESCRIPTION

Currently supported protocols:
 
 
* Cisco (vpnc (free), vpnclient (propritary))
 
* IPSec (FreeS/WAN, Openswan, strongSwan, racoon)
 
* PPTP
 
* OpenVPN
 
* L2TP (l2tpd, xl2tpd, openl2tp) over IPSec (racoon, FreeS/WAN, Openswan, strongSwan)
 
* Vtun
 
* SSH

Generic options:

--help
Show help about options
--help-qt
Show Qt specific options
--help-kde
Show KDE specific options
--help-all
Show all options
--author
Show author information
-v, --version
Show version information
--license
Show license information
--
End of options

OPTIONS

--openvpnimport <file>
Import OpenVPN config file
--ciscoimport <file>
Import Cisco PCF file
KDE: 4.2.2 (KDE 4.2.2) KVpnc: 0.9.1

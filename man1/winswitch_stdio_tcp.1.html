NAME

winswitch_stdio_tcp - Window Switch TCP Socket Helper

DESCRIPTION

winswitch_stdio_tcp is used for connecting (and potentially starting) a winswitch_server instance over ssh.
 
It is not meant to be used directly but needs to be present on the PATH so that remote clients can find it easily.

NAME
acbuild - the application container build system

SYNOPSIS
acbuild [OPTIONS]

DESCRIPTION
the application container build system

OPTIONS
--debug[=false]
Print out debug information to stderr

--modify=""
Path to an ACI to modify (ignores build context)

--no-history[=false]
Don’t add annotations with the command that was run

--work-path="."
Path to place working files in

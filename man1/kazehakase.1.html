NAME
kazehakase - Gecko based web browser using GTK

SYNOPSIS
kazehakase [OPTION...] [URL...]

DESCRIPTION
kazehakase is a web browser using the Mozilla rendering and networking engines.

OPTIONS
-a ACTION, --action ACTION
Invoke the action in kazehakase.

-v, --version
Show the version.

-h, --help
Show this help.

EXAMPLES
kazehakase --action "KzWindowTabPopup/TabCopyTitle 1 3"
Copy the title of the 3rd tab in the 1st window.

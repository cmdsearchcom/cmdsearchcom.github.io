NAME

bkr-list-labcontrollers - List Beaker lab controllers

SYNOPSIS

bkr labcontroller-list [options]

DESCRIPTION

Prints to stdout a list of all the lab controllers attached to Beaker. Lab controllers which have been removed are not included in the list.

OPTIONS

Common bkr options are described in the Options section of bkr(1).

EXIT STATUS

Non-zero on error, otherwise zero.

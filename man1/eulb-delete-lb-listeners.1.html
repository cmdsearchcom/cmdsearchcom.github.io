NAME
eulb-delete-lb-listeners - Delete one or more listeners from a load balancer

SYNOPSIS
eulb-delete-lb-listeners -l PORT1,PORT2,... [-U URL]

[--region USER@REGION] [-I KEY_ID] [-S KEY]
[--security-token TOKEN] [--debug] [--debugger] [--version] [-h] ELB

DESCRIPTION
Delete one or more listeners from a load balancer

DESCRIPTION
If a listener named with -l/--lb-ports does not exist, this command still succeeds.

positional arguments:
ELB name of the load balancer to modify (required)

optional arguments:
-l PORT1,PORT2,..., --lb-ports PORT1,PORT2,...
port numbers of the listeners to remove (required)

-U URL, --url URL
load balancing service endpoint URL

--region USER@REGION
region and/or user names to search when looking up config file data

-I KEY_ID, --access-key-id KEY_ID

-S KEY, --secret-key KEY

--security-token TOKEN

--debug
show debugging output

--debugger
launch interactive debugger on error

--version
show the program’s version and exit

-h, --help
show this help message and exit

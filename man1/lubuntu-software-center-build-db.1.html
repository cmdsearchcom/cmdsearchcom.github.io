NAME

lubuntu-software-center-build-db - Utility to build the database used by lubuntu-software-center
 

DESCRIPTION

lubuntu-software-center-build-db is an utility to build and rebuild the database used by lubuntu-software-center. See lubuntu-software-center.1 for more details.

NAME
gprbuild, gprclean, gprinstall, gprconfig, gprslave - multi-language extensible build tool

DESCRIPTION
These programs compile, clean and install multi-language projects specified by a .gpr GNAT Project File.

The gprconfig tool creates configurations files for the other ones, interactively unless --batch is given.

The gprslave tool is used for distributed builds.

Each of them will list understood options on standard output with --help.

For examples and known problems, please look in the /usr/share/doc/gprbuild directory. Some hints specific to Debian may be found in /usr/share/doc/gprbuild/README.Debian. The

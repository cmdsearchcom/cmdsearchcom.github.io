NAME

SwapValuation - Example of using QuantLib

SYNOPSIS

SwapValuation

DESCRIPTION

SwapValuation is an example of using QuantLib.
 
It prices an Interest Rate Swap over a term structure and calculates its fair fixed rate and floating spread.

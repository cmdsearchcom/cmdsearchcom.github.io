ADDPASSWD(1) BSD General Commands Manual ADDPASSWD(1)

NAME
addpasswd â Password utility for the GNU Gatekeeper

SYNOPSIS
addpasswd config-file section user password

DESCRIPTION
Addpasswd is a tool to encrypt passwords for use in the GNU Gatekeeper config file. Please note that this is a rather weak encryption that is easy to break for a determined
attacker.

OPTIONS
config-file
Specify the configuration file to use.

section
Specify which section to write the password to.

user The user name.

password
The password to encode for this user.

FILES
/etc/gatekeeper.ini

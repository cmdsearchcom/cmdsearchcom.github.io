NAME

wavinfo - Display format information for wave files

SYNOPSIS

wavinfo file1.wav [file2.wav] [...]

DESCRIPTION

wavinfo displays information about wave files you pass to it on the command line. This includes information about the sampling rate and bit rate used.

SEE ALSO

wavbreaker(1), wavmerge(1).

NAME
anxenc â Encapsulate annotations and data into annodexed media

SYNOPSIS
anxenc [-o filename | --output filename ] [-h | --help ] [-t content-type | --type content-type ] filename ...

Description
anxenc encapsulates annotations and data into annodexed media.

Options
anxenc accepts the following options:

CMML generation options
-c, --cmml
Extract annotations as a valid CMML file (default)

--head Extract only the CMML head element

-g, --granulerate
Include information about media granulerates

Data track options
-m, --media
Extract media from annodexed media

Miscellaneous options
-o filename, --output filename
Write output to the specified filename instead of printing it to standard output.

-h, --help
Display usage information and exit.

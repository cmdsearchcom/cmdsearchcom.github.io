NAME
globash - a special shell for GLOBAL using GNU bash

SYNOPSIS
globash

DESCRIPTION
Globash is a special shell for GLOBAL using GNU bash. You can use a lot of functions to ease reading source code, like tag stack, tag mark and cookie. At first, you should make
tag files using gtags and invoke this command in the project. Please refer to the help (type Â´ghelp<RET>Â´) for detailed usage.

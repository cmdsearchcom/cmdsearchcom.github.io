TRACEPKTDUMP(1) User Commands TRACEPKTDUMP(1)

NAME

tracepktdump - output packets in human readable format

SYNOPSIS

tracepktdump [ -f exp | --filter=exp ] [ -c num | --count=num ] inputuri ...
 

DESCRPTION

tracepktdump verbosely outputs packets in a human readable form, suitable for diagnosing packets.
 
tracepktdump uses the libpacketdump library. There is an example tracepktdump program in the examples directory that shows how to use libpacketdump.
 
-f bpf-filter
--filter bpf-filter
Output only packets that match the bpf-filter expression. See tcpdump(1) for the syntax of the bpf-filter expression.
 
-c num
--count num
stop after displaying num packets.

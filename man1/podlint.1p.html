NAME
podlint - check POD for correctness using Pod::POM

SYNOPSIS
podlint MyFile.pm

DESCRIPTION
This script uses Pod::POM to parse a Pod document with full warnings enabled, effectively acting as a syntax and structure checker.

The -f option can be specified to have the parsed Pod Object Model printed to STDOUT with any markup errors fixed. Note there are some critical parse errors that can’t be
handled and fixed by the parser and in this case the script will terminate reporting the error.

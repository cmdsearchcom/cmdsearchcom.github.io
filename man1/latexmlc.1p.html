NAME
"latexmlc" - A omni-executable for LaTeXML, capable of
stand-alone socket-server and (soon) web service conversion.

SYNOPSYS
See the OPTIONS section in LaTeXML::Common::Config for usage information.
Also consult latexmlc --help

DESCRIPTION
latexmlc provides a client which automatically sets up a LaTeXML local server
if necessary (via latexmls).

If such server already exists, the client proceeds to communicate normally.

A stand-alone conversion (the default) can also be requested via --timeout=-1

SEE ALSO
latexmls, ltxmojo, LaTeXML::Common::Config

NAME
git-extras - Awesome GIT utilities

SYNOPSIS
git-extras [-v,--version] [-h,--help] [update]

OPTIONS
-v, --version

Show git-extras version number.

-h, --help

Show this help. This option can also be used for any of the extras commands.

update

Self update.

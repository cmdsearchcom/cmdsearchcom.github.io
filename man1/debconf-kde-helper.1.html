NAME

debconf-kde-helper - Provide KDE Debconf frontend

DESCRIPTION

This manual page documents the debconf-kde-helper command.
debconf-kde-helper provides a GUI frontend for the Debconf configuration system, based on the KDE platform libraries.

OPTIONS

Additional options:
-h, --help
show help about possible commands.
--socket-path PATH
Set path to Debconf socket file.

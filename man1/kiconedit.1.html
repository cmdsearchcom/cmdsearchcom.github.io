NAME
kiconedit - KDE Icon Editor

SYNOPSIS
kiconedit

DESCRIPTION
The KDE Icon Editor is a small graphics drawing program especially for creating icons using the standard KDE icon palette. It has most of the tools which are needed for easy creâ
ation of icons including: Line, Ellipse, Circle, Rectangle, Freehand, Eraser and Spraycan. More tools are planned but not fully implemented.

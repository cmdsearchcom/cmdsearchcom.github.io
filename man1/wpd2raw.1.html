WPD2RAW(1) User Commands WPD2RAW(1)

NAME

wpd2raw - debug the conversion library

SYNOPSIS

wpd2raw [ OPTION] <WordPerfect Document>

OPTIONS

--callgraph Display the call graph nesting level --help Shows this help message --password <password> Try to decrypt password protected document --version Output wpd2raw version

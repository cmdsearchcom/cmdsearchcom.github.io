NAME
is_aufs - program to check whether a filesystem is an aufs filesystem

SYNOPSIS
is_aufs MOUNTPOINT

DESCRIPTION
This manual page documents briefly the is_aufs command.

is_aufs is a program that checks whether a filesystem (mountpoint) is aufs.

OPTIONS
Pass the mountpoint as the parameter

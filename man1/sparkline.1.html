NAME

sparkline - A unicode sparkline generation tool

DESCRIPTION

usage: sparkline [-h] [--version] ...
Reads from command line args or stdin and prints a sparkline from the data. Requires at least 2 data points as input.

positional arguments:

data
Floating point data, any delimiter.

optional arguments:

-h, --help
show this help message and exit
--version, -v
Display the version number and exit.

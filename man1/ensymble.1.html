NAME
ensymble â developer utilities for Symbian OS

SYNOPSIS
ensymble [command] [options]

DESCRIPTION
This manual page documents briefly the ensymble command.

This manual page was written for the Debian distribution because the original program does not have a manual page.

ensymble is a program whose functions include generation of SIS (installation) packages, merging several SIS packages into one, (re-)signing existing SIS packages and modifying
extension DLL headers For full details, run ensymble with no arguments, or see /usr/share/doc/ensymble/README.

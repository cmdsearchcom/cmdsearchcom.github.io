EXTRACTREGIONFROMIMAGEBYMASK(1) User Commands EXTRACTREGIONFROMIMAGEBYMASK(1)

NAME

ExtractRegionFromImageByMask - part of ANTS registration suite

DESCRIPTION

Extract a sub-region from image using the bounding box from a label image, with optional padding radius. Usage : ExtractRegionFromImageByMask ImageDimension inputImage outputImage labelMaskImage [label=1] [padRadius=0]

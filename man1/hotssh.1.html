NAME

hotssh - a graphical interface to secure shell.

SYNOPSIS

hotssh

DESCRIPTION

HotSSH is an interface to Secure Shell, for GNOME and OpenSSH.
 
It intends to be a better experience than simply invoking "ssh" from an existing terminal window.

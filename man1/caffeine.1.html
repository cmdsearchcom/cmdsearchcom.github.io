NAME
caffeine - prevent desktop idleness in full-screen mode

SYNOPSIS
caffeine [--help|--version]

DESCRIPTION
Caffeine stops the desktop becoming idle when an application runs full-screen, and hence prevents screen savers or power-saving activating even if the machine is not being used.

OPTIONS
-h, --help
Show help and exit

-V, --version
Show program’s version number and exit

SEE ALSO
caffeine-indicator(1), caffeinate(1)

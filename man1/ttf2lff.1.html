NAME
ttf2lff - ttf to lff converter.

DESCRIPTION
ttf2lff is a font converter. It convert fonts in ttf format to lff format usseful to use with LibreCAD or another app that uses this format.

ttf2lff invocation parameters are printed to the console if ttf2lff is called without parameters.

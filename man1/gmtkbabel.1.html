NAME
gmtkbabel - Zenity GUI to mtkbabel

SYNOPSIS
gmtkbabel

DESCRIPTION
This manual page documents briefly the gmtkbabel command.

gmtkbabel consists of a set of shell scripts that, using zenity, provide a graphical interface to the command-line program mtkbabel.

NAME
hopsh 3.0.0 - send a command to a running Hop server.

SYNOPSIS
hopsh [ options ] [ input ]

DESCRIPTION
Hopsh is a tool that let users spawn hop from a command line interface.

OPTIONS
options Many options are available. Type ‘hopsh -help’ to print them.

The complete Hopsh documentation is available as a Web application. To access this documentation, starts the server and browse the url

http://localhost:8080/hop/doc

SEE ALSO
hop(1), hopreplay(1)

NAME

abw2epub - manual page for abw2epub 0.9.5

SYNOPSIS

abw2epub [ OPTIONS] INPUT OUTPUT

DESCRIPTION

`abw2epub' converts AbiWord documents to ePub.

OPTIONS

--help
show this help message
--version
print version and exit

NAME
FreeFem++-nw - executes FreeFem++ scripts

SYNOPSIS
FreeFem++-nw [ -v verbosity ] [ -ne ] [ -cd ] script.edp

DESCRIPTION
This manual page documents briefly the scope of the FreeFem++-nw command.

FreeFem++-nw is part of the FreeFem++ suite which is used for the solution of Partial Differential Equations (PDE’s) via the Finite Element Method (FEM). It can give only postâ
script plot output (batch version, no graphics windows via ffglut).

OPTIONS
A summary of options is included below.
-v verbosity : 0 -- 1000000 levels of FreeFem++ output
-ne : no edp script output
-cd : change dir to script dir

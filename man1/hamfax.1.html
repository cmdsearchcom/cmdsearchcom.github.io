NAME
hamfax - Qt based shortwave fax

SYNOPSIS
hamfax

DESCRIPTION
hamfax is a program for sending and receiving fax images. This is a mode used by hamradio operators. Hamfax uses a soundcard to send and receive images.

SEE ALSO
The hamfax documentation in html format in /usr/share/hamfax/hamfax.html, which can be viewed from within hamfax.

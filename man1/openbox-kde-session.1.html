NAME

openbox-kde-session — Run a KDE session with Openbox as the window manager
 

DESCRIPTION

openbox-kde-session runs a KDE session with Openbox as the window manager.
openbox-kde-session does not take any command line arguments.

SEE ALSO

openbox(1), openbox-gnome-session(1), openbox-session(1).

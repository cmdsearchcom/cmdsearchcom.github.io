NAME
gsnd - Run ghostscript (PostScript and PDF engine) without display

SYNOPSIS
gsnd [ options ] [ files ] ...

DESCRIPTION
This script simply invokes gs(1) with the -NODISPLAY flag, followed by any other arguments from the command-line.

gs(1)

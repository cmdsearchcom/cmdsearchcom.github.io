NAME

guilt-files - Print the list of files that the topmost patch changes

SYNOPSIS

guilt files [-v] [-a] [-l]

DESCRIPTION

Print the list of files that the topmost patch changes.

OPTIONS

-v
Verbose output
-a
List all files in all applied patches
-l
Add patch name to the output

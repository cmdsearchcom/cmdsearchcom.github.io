NAME
alt_key - generate keyboard accelerators

SYNOPSIS
diffpdf [file1] [file2]

DESCRIPTION
This manual page very briefly documents the alt_key command. (Click the GUI’s Help button or press F1 for more information.)

alt_key is a GUI application used to generate keyboard accelerators
Alt_Key is used to show where keyboard accelerators should go in menu option texts and dialog labels. The program instantly produces optimal results on the basis that the best
accelerator is the first character, the second best is the first character of a word, the third best is any character, the worst is no accelerator at all, and no accelerator
should be used more than once. It also allows accelerators to be specified by the user.

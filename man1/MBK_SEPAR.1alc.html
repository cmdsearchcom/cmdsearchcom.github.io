NAME

MBK_SEPAR - define the separator character for hierarchy

SYNOPSIS

c-shell running
setenv MBK_SEPAR character
 
See the file man1/alc_origin.1alc.
 

DESCRIPTION

MBK_SEPAR sets the character that is to be used while concatening names, during a flatten, for example.

EXAMPLE

	setenv MBK_SEPAR /

SEE ALSO

mbk(3), genlib(1), concatname(3), nameindex(3).

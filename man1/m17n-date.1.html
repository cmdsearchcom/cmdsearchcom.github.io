NAME

m17n-date - display date and time

SYNOPSIS

m17n-date [ OPTION ... ]

DESCRIPTION

Display the system date and time in many locales on a window.
The following OPTIONs are available.
•
--version
Print version number.
•
-h, --help
Print this message.

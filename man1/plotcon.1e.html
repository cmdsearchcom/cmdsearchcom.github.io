NAME
plotcon - Plot conservation of a sequence alignment

SYNOPSIS
plotcon -sequences seqset [-scorefile matrix] -winsize integer -graph xygraph

plotcon -help

DESCRIPTION
plotcon is a command line program from EMBOSS (âthe European Molecular Biology Open Software Suiteâ). It is part of the "Alignment:Multiple" command group(s).

OPTIONS
Input section
-sequences seqset
File containing a sequence alignment

-scorefile matrix
This is the scoring matrix file used when comparing sequences. By default it is the file ’EBLOSUM62’ (for proteins) or the file ’EDNAFULL’ (for nucleic sequences). These
files are found in the ’data’ directory of the EMBOSS installation.

Required section
-winsize integer
Number of columns to average alignment quality over. The larger this value is, the smoother the plot will be. Default value: 4

Output section
-graph xygraph

NAME

ldaptor-getfreenumber - get the first free uidNumber from an LDAP server
 

SYNOPSIS

ldaptor-getfreenumber [ options]
 

OPTIONS

--base=
LDAP base dn
--service-location=
Service location, in the form BASEDN:HOST[:PORT]
 

RESULT

ldaptor-getfreenumber prints the first free uidNumber in the scope of the specified base dn of the LDAP server specified by --service-location.
 

SEE ALSO

ldaptor-namingcontexts(1), ldaptor-search(1)

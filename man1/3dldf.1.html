NAME

3dldf - a system based on MetaPost for creating three-dimensional graphics

SYNOPSIS

3ldf [ OPTION]... 3DLDFNAME[.ldf]

DESCRIPTION

This is GNU 3DLDF Version 2.0.3. Copyright © 2013 The Free Software Foundation.
Valid options for GNU 3DLDF are:
--bison-trace
Prints information generated by Bison for tracing which parser rules are reduced and the state of the Bison stack
--multithread-input
Enables the use of multiple threads for input
--multithread-output
Enables the use of multiple threads for output
--multithread-include
Enables the use of multiple threads for inputfiles included in other input files
--quiet
Suppresses some output to standard outputand standard error when `3dldf' is running
--silent
Suppresses almost all output to standard outputand standard error when `3dldf' is running
--verbose
Causes status information to be printed to standard output when `3dldf' is running
--help
Prints this message and exits with return value 0
--version
Prints the version number of GNU 3DLDF to standard output and exits with return value 0

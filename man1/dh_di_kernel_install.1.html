NAME

dh_di_kernel_install - install kernel udeb files

SYNOPSIS

dh_di_kernel_install [debhelper options]

DESCRIPTION

dh_di_kernel_install is a debhelper program that runs "kernel-wedge install-files" and "kernel-wedge check" to install files into package build directories for kernel udebs.
It silently does nothing if the "kernel-versions" file does not exist.

SEE ALSO

debhelper(7), kernel-wedge(1)
This program is a part of dh-di.

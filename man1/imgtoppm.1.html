NAME
imgtoppm - convert an Img-whatnot file into a PPM image

SYNOPSIS
imgtoppm [imgfile]

DESCRIPTION
This program is part of Netpbm(1)

imgtoppmreads an Img-whatnot file as input and produces a PPM image as output. The Img-whatnot toolkit is available for FTP on venera.isi.edu, along with numerous images in this
format.

SEE ALSO
ppm(1)

AUTHOR

NAME
dh_golang - Generates Built-Using substvar

SYNOPSIS
dh_golang [debhelperÂ options]

DESCRIPTION
dh_golang is a debhelper program which adds the misc:Built-Using substvar based on the dependencies of the package being built. It uses go list to determine the packages imported
and dpkg-query to find the source package and version that provided that package.

NOTES
The best way to invoke dh_golang is by using dh --with=golang.

MBK_FILTER_SFX(1) MBK ENVIRONMENT VARIABLES MBK_FILTER_SFX(1)

NAME
MBK_FILTER_SFX - define the inputoutput filter suffixe.

DESCRIPTION
MBK_FILTER_SFX tells to Alliance the extention set by compression tools. This variable must be set in order to activate filters. Note the leading points of extention must be set
if necessary.

EXAMPLE
For compressed files with gzip :
setenv MBK_FILTER_SFX ".gz"

SEE ALSO
mbk(3), MBK_IN_FILTER(1), MBK_OUT_FILTER(1), mbkenv(3).

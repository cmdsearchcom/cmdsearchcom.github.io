NAME
system-switch-displaymanager - graphical and text mode interface for choosing graphical display manager

SYNOPSIS
system-switch-displaymanager [display manager name]

DESCRIPTION
system-switch-displaymanager provides a simple method of choosing between the various display manager available under Fedora, including GDM, KDM, XDM and WDM, LXDM, SDDM,
LIGHTDM. If X Windows is running, switchdesk will bring up a dialog box which allows the user to choose between the available display manager installed on the system. This diaâ
log follows the appearance of the user’s current desktop as closely as possible. If a display manager name (GDM, KDM, XDM, WDM) is supplied as an argument, or X is not running,
system-switch-displaymanager can be used from the command line.

RETURN CODES
system-switch-displaymanager returns 0 on success, 1 on error.

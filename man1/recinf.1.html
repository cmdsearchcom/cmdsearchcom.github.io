NAME

recinf - print information about a recfile

SYNOPSIS

recinf [ OPTION]... [FILE]...

DESCRIPTION

Print information about the types of records stored in the input.
-t, --type=RECORD_TYPE
print information on the records having the specified type.
-d, --descriptor
include the full record descriptors.
-n, --names-only
output just the names of the record files found in the input.
--help
print a help message and exit.
--version
show version and exit.

Special options:

-S, --print-sexps
print the data in sexps instead of rec format.

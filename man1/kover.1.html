NAME
kover - CD Cover printer

SYNOPSIS
kover [Qt-options] [KDE-options] [options] file

DESCRIPTION
Kover is an easy to use WYSIWYG CD cover printer with CDDB support.

Generic options:
--help Show help about options

--help-qt
Show Qt specific options

--help-kde
Show KDE specific options

--help-all
Show all options

--author
Show author information

-v, --version
Show version information

--license
Show license information

-- End of options

OPTIONS
--advise
Help me now!

--debug
Enable debug output.

Arguments:
file File to load

AVAILABILITY
kover can be found at http://lisas.de/kover/.

NAME

tssgetrandom - manual page for tssgetrandom v713

DESCRIPTION

getrandom
Runs TPM2_GetRandom
-by bytes requested
[-of output file, with -nz, appends nul terminator] [-nz get random number with no zero bytes (for authorization value)] -se[0-2] session handle / attributes
01 continue 20 command decrypt 40 response encrypt

SEE ALSO

The full documentation for tssgetrandom is maintained as a Texinfo manual. If the info and tssgetrandom programs are properly installed at your site, the command
info tssgetrandom
should give you access to the complete manual.

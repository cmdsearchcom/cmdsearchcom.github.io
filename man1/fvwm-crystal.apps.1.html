NAME
fvwm-crystal.apps - fvwm-crystal panels and menus generator

DESCRIPTION
fvwm-crystal.apps is the panels and menus generator of the fvwm-crystal desktop environment. This program is not meant for direct use but rather only called from the fvwm-crystal
system.

This manual page was written for the Debian distribution because the original program does not have a manual page.

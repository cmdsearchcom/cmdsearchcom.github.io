NAME
pmldigital â construct PhotoML files from EXIF data in digital images from a digital camera, or from data embedded in the output of a film scanner (only Nikon Coolscan scanners
are currently supported)

SYNOPSIS
pmldigital [-h] [-b] [-i] [-f] [-d] [-r] [-x] ([-s] | [-g groupâid]) [-t dateâtimeâoffset] imagefile [imagefile] ...

DESCRIPTION
pmldigital constructs PhotoML files from EXIF data in digital images from a digital camera, or from the private tags in TIFF files written by the Nikon Scan film scanner softâ
ware.

OPTIONS
pmldigital accepts the following options:

-h Display usage information.

-b Include file basename only in output.

-i Output intermediate XML representation of EXIF data.

-f Fast output excluding computation of image and file md5 hashes.

-d Construct digimage descriptions of scanned images instead of digital descriptions of digital camera images.

-r Skip JPEG images with same basename as a RAW image

-x For each specified image file, look for an XMP file with the same path and filename, but with a .xmp extension. If found, tags from this file override those in the main
image file.

-s Set group and frame (and image) ids from standard filenames.

-g groupâid
Specify group id for the set of image files.

-t dateâtimeâoffset
Apply an offset to date/time values in EXIF data. The offset string is in the format accepted by the ParseDateDelta function of the Date::Manip Perl module, for example
"+1y +2m +3d -2h +4mn +6s" for an offset adding 1 year, 2 months, and 3 days, subtracting 2 hours, and adding 4 minutes and 6 seconds.

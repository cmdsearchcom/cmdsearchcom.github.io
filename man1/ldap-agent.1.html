NAME
ldap-agent - SNMP agent for Directory Server

SYNOPSIS
ldap-agent [-D]<configuration-file>

DESCRIPTION
ldap-agent is an SNMP subagent for Directory Server

OPTIONS
-D Enable debugging

<configfile>
Configuration file for the ldap agent

SEE ALSO
snmpd(8)

USAGE
Sample usage:

ldap-agent /etc/dirsrv/config/ldap-agent.conf

AUTHOR
ldap-agent was written by the 389 Project.

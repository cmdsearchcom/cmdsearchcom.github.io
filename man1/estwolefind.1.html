NAME
estwolefind - listing WWWOFFLE cache for hyper estraier

SYNOPSIS
estwolefind dir

DESCRIPTION
This manual page documents briefly the estwolefind commands.

estwolefind is a program that lists up WWWOFFLE caches for input of estcmd. It outputs pathnames of WWWOFFLE cache files in the directory specified by first argument and correâ
sponding URLs in Tab separeted value formats.

EXAMPLES
The following command will register cache files of WWWOFFLE into HyperEstraier database.

estwolefind /var/cache/wwwoffle | estcmd gather -cl -fm -px @uri -sd -cm casket -

estcmd(1).

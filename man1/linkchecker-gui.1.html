LINKCHECKER-GUI(1) LinkChecker GUI client LINKCHECKER-GUI(1)

NAME

linkchecker-gui - GUI client to check links of websites and HTML documents

SYNOPSIS

linkchecker-gui [file-or-url-or-project]

DESCRIPTION

A graphical interface to check links of websites and HTML documents.

PROJECT FILES

A project file has a .lcp extension. The file stores all active preferences and the current checked URL. Project files can be loaded with the menu entry or with drag-and-drop on the GUI window.

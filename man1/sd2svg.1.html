NAME

sd2svg - manual page for sd2svg 0.0.4

SYNOPSIS

sd2svg [ OPTION] INPUT

DESCRIPTION

`sd2svg' converts StarOffice graphic documents and presentations to SVG.

OPTIONS

-h
show this help message
-o OUTPUT
write ouput to OUTPUT
-N
Output the number of sheets
-n NUM
choose the page to convert (1: means first page)
-v
show version information

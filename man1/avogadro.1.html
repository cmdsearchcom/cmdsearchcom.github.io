NAME
avogadro â Advanced molecular editor

SYNOPSIS
avogadro [--erase-config] file1 [file2 ...]

DESCRIPTION
Avogadro is an advanced molecular editor designed for cross-platform use in computational chemistry, molecular modeling, bioinformatics, materials science, and related areas. It
offers flexible rendering and a powerful plugin architecture.

OPTIONS
--erase-config
Overwrite settings with default values for the current user

EXAMPLES
avogadro file1 file2
Open chemical files file1 and file2

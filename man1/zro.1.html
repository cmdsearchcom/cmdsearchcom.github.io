NAME
zro - random orders of a group

SYNOPSIS
zro [OPTIONS] <Count> <Gen> ...

DESCRIPTION
This program calculates the order of Count random elements of the group generated by a set of matrices or permutations. This information can be helpful to find out which group
is generated by a given set of matrices or permutations.

OPTIONS
-Q Quiet, no messages.

-V Verbose, more messages.

-T <MaxTime>
Set CPU time limit

-G Produce output in GAP format. This option implies -Q.

-s Summary only.

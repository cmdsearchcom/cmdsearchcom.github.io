NAME
gnome-paint - simple, easy-to-use paint program for GNOME

SYNOPSIS
gnome-paint

DESCRIPTION
This manual page documents briefly the gnome-paint commands.

gnome-paint is a program inspired by MS Paint and designed for GNOME (and maybe other) desktop environment. It could be used to manipulate images in a very simple way.

NAME
anxinfo â Display information about annodexed media

SYNOPSIS
anxinfo [-v | --verbose ] [-h | --help ] filename ...

Description
anxinfo displays information about annodexed media. The default behavior is to list the id, serial number, granule rate and number of extra headers for each track in filename. If
the --verbose option is given, additional information including the timebase and duration of the file is displayed.

Options
anxinfo accepts the following options:

Miscellaneous options
-v, --verbose
Display verbose information about file

-h, --help
Display usage information and exit.

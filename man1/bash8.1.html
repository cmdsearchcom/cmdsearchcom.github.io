BASH8
A pep8 equivalent for bash scripts

This program attempts to be an automated style checker for bash scripts to fill the same part of code review that pep8 does in most OpenStack projects. It started from humble
beginnings in the DevStack project, and will continue to evolve over time.

INSTALLATION
At the command line:

$ pip install bash8

Or, if you have virtualenvwrapper installed:

$ mkvirtualenv bash8
$ pip install bash8

USAGE
To use bash8 in a project:

import bash8

NAME
Karbon - A Vector Graphics Drawing Application.

SYNOPSIS
karbon [Qt-options] [KDE-options] [Calligra-options] [file]

DESCRIPTION
Karbon is a vector drawing application with an user interface that is easy to use, highly customizable and extensible. That makes Karbon a great application for users starting to
explore the world of vector graphics as well as for artists wanting to create breathtaking vector art.

OPTIONS
Generic options:

--help Show help about options

--help-qt
Show Qt specific options

--help-kde
Show KDE specific options

--help-calligra
Show Calligra specific options

--help-all
Show all options

--author
Show author information

-v, --version
Show version information

--license
Show license information

Arguments:

file File to open

SEE ALSO
More detailed user documentation is available from help:/karbon (either enter this URL into Konqueror, or run khelpcenter help:/karbon).

OPENSTV-RUN-ELECTION(1) General Commands Manual OPENSTV-RUN-ELECTION(1)

NAME
OpenSTV - Single Transferable Vote and Instant Runoff Voting Software

SYNOPSIS
openstv-run-election [OPTION] METHOD BALLOT_FILE

DESCRIPTION
Runs an election for a given METHOD and BALLOT_FILE. Results are printed to the standard output. The following methods are available:

Approval
Borda
Bucklin
CambridgeSTV
Condorcet
Coombs
ERS97STV
FTSTV
GPCA2000STV
IRV
MeekNZSTV
MeekQXSTV
MeekSTV
MinneapolisSTV
NIrelandSTV
QPQ
RTSTV
SNTV
ScottishSTV
SuppVote
WarrenQXSTV
WarrenSTV

OPTIONS
-p PREC
Override default precision (in digits).

-r FORMAT
Report format: CsvReport, HtmlReport, MinimalReport, TextReport or YamlReport

-t TIEBREAK
Strong tie-break method: random (default), alpha or index

-w WEAKTIE
Weak tie-break method: method-default (default), strong, forward or backward

-s SEATS
Number of seats (for text-format ballot files).

-P Profile and send output to profile.out

-x NUM Specify repeat count (for profiling).

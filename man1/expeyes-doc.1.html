NAME
expeyes-doc - program to display the user manual of expeyes and expeyes-junior.

SYNOPSIS
expeyes-doc [lang]

expeyes-junior-doc [lang]

DESCRIPTION
expeyes-doc launches a viewer for the user manual of expeyes. If invoked as expeyes-junior-doc it launches a viewer for the manual of expeyes-junior.

The preferred language is detected automatically.

OPTIONS
lang
Enforces the language lang as preferred. For example, lang may be ’en’ or ’fr’.

ENVIRONMENT
LANGUAGE, LANG, LC_ALL
If defined, these variables are taken in account to find automatically the preferred language. These variables are taken in account only when the optional argument lang is
not given.

expeyes(1), expeyes-junior(1)

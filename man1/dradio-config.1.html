NAME
dradio-config - generate dradio configuration

SYNOPSIS
dradio-config [--rss]

DESCRIPTION
dradio-config is a script to help keep the dradio menu list configuration ~/.config/dradio/menu.xml up-to-date. It relies on curl, tidy, and xsltproc to screen scrape the DR
streams and podcasts pages, and outputs the resulting XML on stdout.

Invoke it as, e.g.

dradio-config > radio.xml

dradio-config --rss > podcast.xml

FILES
~/.config/dradio/menu.xml
The menu configuration file. See dradio(5) for further details.
http://www.dr.dk/netradio/wmp.asp
DR netradio direct links page.
http://www.dr.dk/Podcast
DR podcast direct links page.

NAME

shotwell - a photo organizer for the GNOME desktop
 

SYNOPSIS

shotwell [ OPTION...] [FILE]
 

DESCRIPTION

Shotwell is a photo organizer designed for the GNOME desktop. It allows you to import photos from your camera, view and edit them, and share them with others.

OPTIONS

-d, --datadir=DIRECTORY
Path to private data belonging to shotwell
--no-runtime-monitoring
Do not monitor library directory at runtime for changes.
--no-startup-progress
Do not display startup progress meter.
--display=DISPLAY
X display to use.
-V, --version
Output version information and exit.
-h, --help
Prints the command line options.

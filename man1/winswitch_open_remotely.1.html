NAME
winswitch_open_remotely - Window Switch helper to open files on a remote server

SYNOPSIS
winswitch_open_remotely filename

DESCRIPTION
winswitch_open_remotely is used by scripts and mime-type mappings to allow other software to open a file on a remote server.

It will contact the winswitch client (generally, the applet) and ask it to open the given file on the default remote server.

It is currently used by Thunar to provide an option in its ’Send To’ menu, but may also be used elsewhere.

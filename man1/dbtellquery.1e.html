NAME
dbtellquery - Show possible queries of a database

SYNOPSIS
dbtellquery -database string [-outfile outfile] [-html boolean]

dbtellquery -help

DESCRIPTION
dbtellquery is a command line program from EMBOSS (âthe European Molecular Biology Open Software Suiteâ). It is part of the "Information" command group(s).

OPTIONS
Input section
-database string
The unique identifier of the database. These are listed in the EMBOSS data file DRCAT.dat (field names in parenthesis) and can be the recommended or official unique
identifier (’ID’), the alternative identifier (’IDalt’), or the accession number (’Acc’) of the database. Any of these are acceptable. Official database identifiers and
accession numbers are taken from http://www.uniprot.org/docs/dbxref where available. You can use dbfind to find databases by keyword and return unique identifiers and other
information.

-outfile outfile
Default value: stdout

Html section
-html boolean
If you are sending the output to a file, this will format it for displaying as a table in a WWW document. Default value: N

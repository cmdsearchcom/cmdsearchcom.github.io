NAME

botch-fasofstats - extract feedback arc set from output of botch-print-stats

SYNOPSIS

botch-toolname [options] stats.json

DESCRIPTION

extract feedback arc set from stats.json

OPTIONS

-h, --help
Show help.
-v, --verbose
Be verbose.

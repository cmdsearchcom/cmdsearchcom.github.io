NAME
gogui-display - Go Text Protocol adapter showing the current board in a window.

SYNOPSIS
gogui-display [-config file] [-laf name] [-verbose] [program]

gogui-display -help

gogui-display -version

DESCRIPTION
Go Text Protocol adapter showing the current board in a window. If no program argument is given, the adapter generates answers to genmove commands by waiting for input on the
graphical board.

The program also tracks the standard error stream of the Go program for GoGui Live Graphics commands (see GoGui documentation).

OPTIONS
-config file
Read command line options and arguments from the given file.

-help
Display help and exit.

-laf name
Set Swing look and feel. See gogui(1) for allowed arguments.

-verbose
Print the GTP stream from the adapter to the Go program to standard error.

-version
Print version and exit.

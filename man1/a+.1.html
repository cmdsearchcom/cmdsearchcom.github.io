A+(1) General Commands Manual A+(1)

NAME

a+ - A+ interpreter

SYNOPSIS

a+ [files]...

DESCRIPTION

This manual page documents briefly the a+ command. This manual page was written for the Debian GNU/Linux distribution because the original program does not have a manual page. Instead, it has documentation in HTML format; install the package aplus-fsf-doc then open /usr/share/doc/aplus-fsf-doc/html/index.html.

AUTHOR

NAME

buxon - RDF sioc:Forum browser

SYNOPSIS

buxon [uri]

DESCRIPTION

buxon is an RDF sioc:Forum browser, created by the SWAML developers as well and written in PyGTK.
buxon expects as an input a URI pointing to an RDF file containing a representation of a SIOC forum. Typical inputs are outputs generated by SWAML.

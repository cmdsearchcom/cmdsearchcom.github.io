NAME

dynare++ - compute higher order approximations of DSGE models

DESCRIPTION

Dynare++ is used to compute higher order approximations of Dynamic Stochastic General Equilibrium models.
For a full documentation, including the list of options, please refer to the documentation in /usr/share/doc/dynare-doc/.

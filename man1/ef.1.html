NAME
ef - Utility to run a program under the Electric Fence memory debugger

SYNOPSIS
ef PROGRAM [ARGS...]

DESCRIPTION
The ef command is used to launch another program under the Electric Fence memory debugger. It sets up appropriate environment variables and then launches the specified program
with the specified arguments. See efence (3) manual page for more details about Electric Fence.

NAME

maki-remote - Remote IRC clients
 

SYNOPSIS

maki-remote [OPTION...] [USER@]HOST COMMAND ...
 

DESCRIPTION

maki-remote connects clients to a maki instance started with --dbus-server.
 

OPTIONS

--print-bus-address
Print bus address.
 

EXAMPLES

$ maki-remote user@host tekka
 
$ maki-remote --print-bus-address

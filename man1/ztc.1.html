ztc(1) User Commands ztc(1)

NAME
ztc - calculate matrix or permutation trace

SYNOPSIS
ztc [OPTIONS] <File>

DESCRIPTION
This program reads a matrix or permutation, calculates its trace, and outputs this to the user. If the input file is a matrix, it is read row by row and the diagonal entries are
added up. Note that the matrix need not be square. The result is printed in the form

Trace is N

For permutations, the trace is calculated as the number of fixed points.

OPTIONS
-Q Quiet, no messages.

-V Verbose, more messages.

-T <MaxTime>
Set CPU time limit

-G Produce output in GAP format. This option implies -Q.

INPUT FILES
File The matrix or permutation.

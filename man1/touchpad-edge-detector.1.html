NAME

touchpad-edge-detector - touchpad edge detector

SYNOPSIS

touchpad-edge-detector < evdev device>

DESCRIPTION

touchpad-edge-detector reads touchpad events and calculates the minimum and maximum values for the x and y coordinates.

OPTIONS

touchpad-edge-detector accepts no options.

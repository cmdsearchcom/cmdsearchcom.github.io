NAME

bkr-pool-systems - List systems in a pool

SYNOPSIS

bkr pool-systems [options] <pool-name>
 

DESCRIPTION

Lists systems in a pool.

OPTIONS

--format list, --format json
Display results in the given format. The default format is list, which lists one system per line. The format json displays systems as a JSON array.

 
Common bkr options are described in the Options section of bkr(1).

EXIT STATUS

Non-zero on error, otherwise zero.

EXAMPLES

List systems in the kernel-hw pool:
 
bkr pool-systems kernel-hw

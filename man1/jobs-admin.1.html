NAME
jobs-admin - graphical interface to jobservice

SYNOPSIS
jobs-admin

DESCRIPTION
jobs-admin does not accept any command-line options.

jobs-admin is a GTK+ utility that acts as a frontend to jobservice, which manages system jobs.

BUGS
Bugs can be reported at https://bugs.launchpad.net/jobsadmin.

NAME
bauble-admin - create bauble’s database and change some of its properties

SYNOPSIS
bauble-admin [ command ] [ options ] dbname

DESCRIPTION
The bauble-admin script makes it easier to create and database and change some of its properties

OPTIONS
-H,--host=hostname
the host to connect to

-u,--user=user
the user name to use when connecting to the server

-P,--port=port
the port on the server to connect to

-o,--owner=owner
the owner of the newly created database, if not set then use the user name

-p ask for a password

-d,--dbtype=dbtype
the database type

-v,--verbose
verbose output

-y’,--dryrun
don’t execute anything

BUGS
This script is considered ALPHA quality. You mileage may vary.

NAME
d.fontlist - Lists the available fonts.

KEYWORDS
display, settings

SYNOPSIS
d.fontlist
d.fontlist --help
d.fontlist [-lv] [--help] [--verbose] [--quiet] [--ui]

Flags:
-l
List fonts (default; provided for compatibility with d.font)

-v
List fonts verbosely

--help
Print usage summary

--verbose
Verbose module output

--quiet
Quiet module output

--ui
Force launching GUI dialog

DESCRIPTION
d.fontlist outputs a list of available fonts for use with GRASS display commands.

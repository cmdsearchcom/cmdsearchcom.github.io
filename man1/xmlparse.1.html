NAME
xmlparse - error-check and validate XML documents

SYNOPSIS
xmlparse
filename

DESCRIPTION
Parses the given file, validating it against its DTD if found. Prints messages about any errors, otherwise returns silently. (Value 0 is always returned in any case.) The
parse of the file is discarded.

HISTORY

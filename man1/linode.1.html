NAME
linode - command-line interface to the Linode platform

SYNOPSIS
linode [--object object] [object-options...]

This is a wrapper utility to the suite of Linode command line tools.

Objects
The following command line tools are available, and can be called directly or with linode object, e.g.: linode domain list. Each has its own help documentation accessible with
--help; for instance: linode -o domain --help.

If no object is specified, -o linode is implied.

NAME
lcp â Secure remote file copy

SYNOPSIS
lcp [-fvn] [[user@]host1:]file1 [[user@]host2:]file2

DESCRIPTION
lcp is a program that securely copies files between two hosts on a network. This is intended as a secure replacement for rcp(1). Note that both the source and the destination can
be on remote machines.

OPTIONS
Available options:

--help Show summary of options.

-f, --force
Overwrite existing files.

-v, --version
Display commands before they are executed.

-n, --dry-run
Don’t execute any commands. Implies -v.

ENVIRONMENT
LSFTP_RSH The program to use for the SSH tunnel. If the variable is not set, lcp defaults to using lsh.

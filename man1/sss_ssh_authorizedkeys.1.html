NAME
sss_ssh_authorizedkeys - get OpenSSH authorized keys

SYNOPSIS
sss_ssh_authorizedkeys [options] USER

DESCRIPTION
sss_ssh_authorizedkeys acquires SSH public keys for user USER and outputs them in OpenSSH authorized_keys format (see the âAUTHORIZED_KEYS FILE FORMATâ section of sshd(8) for
more information).

sshd(8) can be configured to use sss_ssh_authorizedkeys for public key user authentication if it is compiled with support for âAuthorizedKeysCommandâ option. Please refer to the
sshd_config(5) man page for more details about this option.

If âAuthorizedKeysCommandâ is supported, sshd(8) can be configured to use it by putting the following directives in sshd_config(5):

AuthorizedKeysCommand /usr/bin/sss_ssh_authorizedkeys
AuthorizedKeysCommandUser nobody

OPTIONS
-d,--domain DOMAIN
Search for user public keys in SSSD domain DOMAIN.

-?,--help
Display help message and exit.

EXIT STATUS
In case of success, an exit value of 0 is returned. Otherwise, 1 is returned.

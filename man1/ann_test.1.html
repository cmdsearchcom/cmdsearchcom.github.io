NAME
ann_test - A test and evaluation program for the ANN library

SYNOPSIS
ann_test < file .in

DESCRIPTION
The program ann_test provides a simple script language for setting up experiments involving ANN. It allows the user to generate data and query sets of various sizes, dimensions,
and distributions, to build kd- and bd-trees of various types, and then run queries and outputting various performance statistics.

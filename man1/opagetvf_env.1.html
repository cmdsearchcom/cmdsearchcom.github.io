NAME

opagetvf_env
 
 
Provides opagetvf_func and opagetvf2_func shell functions that query the parameters of a vFabric. Also exports values that indicate the PKEY, SL, MTU, and RATE associated with the vFabric. This script is designed to be sourced into bash scripts.
A usage example is provided in:
/usr/lib/opa-ff/src/mpi_apps/ofed.openmpi.params

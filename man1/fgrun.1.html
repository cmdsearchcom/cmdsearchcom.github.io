NAME
FGRun - FlightGear Flight Simulatior Graphical Launcher:

SYNOPSIS
fgrun [OPTIONS]

DESCRIPTION
FlightGear Launch Control (FGRun) is a graphical frontend for running the FlightGear Flight Simulator. It allows you to select a airport and aircraft and includes advanced
options like launching from an aircraft carrier, or changing the screen resoluton.

--fg-root=<DIR>

--fg-scenery=<DIR>

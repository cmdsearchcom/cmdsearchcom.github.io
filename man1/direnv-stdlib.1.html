NAME
direnv_stdlib - the ".envrc" stdlib

SYNOPSIS
direnv stdlib

DESCRIPTION
Outputs a bash script called the stdlib. The following commands are included in that script and loaded in the context of an ".envrc". In addition, it also loads the file in
" /.direnvrc" if it exists.

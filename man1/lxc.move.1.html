NAME
lxc move - Move containers within or in between LXD instances

SYNOPSIS
lxc move [<remote>:]<container>[/<snapshot>] [<remote>:][<container>[/<snapshot>]]

DESCRIPTION
Move containers within or in between LXD instances.

lxc move [<remote>:]<source container> [<remote>:][<destination container>]

Move a container between two hosts, renaming it if destination name differs.

lxc move <old name> <new name>

Rename a local container.

lxc move <container>/<old snapshot name> <container>/<new snapshot name>

Rename a snapshot.

OPTIONS
--debug
(= false)

Enable debug mode

--force-local
(= false)

Force using the local unix socket

--no-alias
(= false)

Ignore aliases when determining what command to run

--verbose
(= false)

Enable verbose mode

NAME
birdfont-export - generate TTF, EOT and SVG files from FFI-files

SYNOPSIS
birdfont-export [options] file

DESCRIPTION
birdfont-export generates fonts that can be installed and on your computer or deployed on your website.

See http://birdfont.org for use cases.

OPTIONS
-h, --help
Print command line options.

-o, --output [directory]
Write files to this directory

-s, --svg
Write SVG font

-t, --ttf
Write TTF and EOT fonts

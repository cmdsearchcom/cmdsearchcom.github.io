NAME
acdcheck

AUTHOR
Luke McCarthy <lukem@gene.pbi.nrc.ca>

SYNOPSIS
acdcheck [ --output FORMAT ] FILE [ FILE ... ]

DESCRIPTION
Reads one or more named input FILEs and attempts to parse them as if they were EMBOSS ACD (AJAX Command Definition) files.

OPTIONS
--output FORMAT
Upon successful parsing of each ACD file, print a representation of the parse tree to standard output in the specified format. FORMAT must be one of either ’XML’ or ’Perl’
(case-insensitive and without the quotes.)

NAME
guilt-help - open man page of a guilt command

SYNOPSIS
guilt help [<command> | <topic>]

DESCRIPTION
Open up the man page for a guilt command.

EXAMPLES
Open the guilt-status man page

$ guilt help status

Open the guilt man page

$ guilt help

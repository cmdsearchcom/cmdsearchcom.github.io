NAME

ListDBus — List the names on a DBus

SYNOPSIS

ListDBus [-y ] [--system ] [-s ] [--session ] [ -o ] [--owners ] [-u ] [--uids ] [-h ] [--help]

DESCRIPTION

This manual page documents briefly the ListDBus command.
ListDBus lists all the names on DBus.
 

OPTIONS

These programs follow the usual GNU command line syntax, with long options starting with two dashes (`-'). A summary of options is included below.
 
-s --session
List the Session Bus (This is the Default)
-y --system
List the System Bus (Default is Session)
-u --uids
Give the UIDs which own each name
 
-o --owners
Resolve any well-known names to their owners
 
-h --help
Show summary of options.

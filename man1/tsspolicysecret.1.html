NAME
tsspolicysecret - manual page for tsspolicysecret v713

DESCRIPTION
policysecret

Runs TPM2_PolicySecret

-ha authorizing entity handle

-hs session handle

-in nonceTPM file (default none)

-cp cpHash file (default none)

-pref policyRef file (default none)

-exp expiration (default none)

-pwde authorizing entity password (default empty)

[-tk ticket file name] [-to timeout file name]

-se[0-2] session handle / attributes (default PWAP)

01 continue 20 command decrypt 40 response encrypt

SEE ALSO
The full documentation for tsspolicysecret is maintained as a Texinfo manual. If the info and tsspolicysecret programs are properly installed at your site, the command

info tsspolicysecret

should give you access to the complete manual.

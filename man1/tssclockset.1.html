NAME
tssclockset - manual page for tssclockset v713

DESCRIPTION
clockset

Runs TPM2_ClockSet

-time new time

-hi hierarchy o, p (default platform)

-pwdp password for hierarchy (default empty)

-se[0-2] session handle / attributes (default PWAP)

01 continue

SEE ALSO
The full documentation for tssclockset is maintained as a Texinfo manual. If the info and tssclockset programs are properly installed at your site, the command

info tssclockset

should give you access to the complete manual.

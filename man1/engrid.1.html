NAME

enGrid - Mesh generation program.
 

SYNOPSIS

engrid
 
engrid [ -f FILE ]
 
engrid [ -h ]
 
engrid [ -appendlic FILE1 FILE2 ... ]
 
engrid [ -distbin ]
 
 

DESCRIPTION

enGrid is an open-source mesh generation program.
 

OPTIONS

-f FILE
start enGrid and open FILE
-h
Display usage instructions
-appendlic FILE1 FILE2 ...
Append license to files
-distbin
Create binary distribution

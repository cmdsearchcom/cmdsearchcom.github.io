NAME

botch-profile-build-fvs - remove droppable dependencies from graph

SYNOPSIS

botch-profile-build-fvs [options] graph.xml fvs.list

DESCRIPTION

Remove droppable dependencies from a given set of source packages. List is given on standard input if not supplied as the last positional argument.

OPTIONS

-h, --help
Show help.
--remove-weak=FILE
A filename containing a list of weak build dependencies
--remove-reduced=FILE
One or more filename containing a list of droppable build dependencies, separated by commas.
--verbose
Be verbose.

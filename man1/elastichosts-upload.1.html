NAME

elastichosts-upload - upload disk images to ElasticHosts

SYNOPSIS

elastichosts-upload [ OPTIONS ] FILENAME | BLOCK-DEVICE

OPTIONS

-c CHUNK
size of chunks (default: 5242880)
-d DRIVE-UUID
UUID of existing drive to image (default: creates new drive)
-n NAME
name for newly created drive (default: basename of FILENAME)
-o OFFSET
byte offset from which to resume upload (default: 0)
-z
input image is gzipped

NAME
mail-expire â program to extract outdated messages from mbox files

SYNOPSIS
mail-expire AGE FILE [FILES...]

DESCRIPTION
mail-expire is a small utility which only purpose is to help on keeping the size of multiple mailboxes as small as needed (by removing outdated messages).

The old messages are compressed with gzip and stored in the file with the name <MBOXNAME>.<MONTH>_<YEAR>.gz.

OPTIONS
-u If an existing archive file with the expected filename is found, another filename is to be choosen. For example: <MBOXNAME>(NUMBER).<MONTH>_<YEAR>.gz.

-t DIR Specify a different target directory for storing expired mailbox files. Default is the current directory.

--delete Drop the old messages. No backup will be made, be careful with this option.

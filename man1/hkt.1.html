NAME

hkt - hopenpgp-tools

SYNOPSIS

hkt COMMAND

DESCRIPTION

hkt (hopenpgp-tools) 0.19.4 Copyright © 2012-2016 Clint Adams
hOpenPGP Keyring Tool

Available options:

-h,--help
Show this help text
-V,--version
Show version information

Available commands:

export-pubkeys
export matching keys to stdout
findpaths
find short paths between keys
graph
graph certifications
list
list matching keys

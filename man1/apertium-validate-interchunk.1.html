NAME

apertium-validate-interchunk - This application is part of ( apertium )
This tool is part of the apertium open-source machine translation toolbox: http://www.apertium.org.

SYNOPSIS

apertium-validate-interchunk <input_file>

DESCRIPTION

This is an intermediate tool from Apertium level 2 engine. You should never have to use it independently.
It is a script that validates a set of structural interchunk rules against the apertium structural interchunk rules DTD using the xmllint utility.

OPTIONS

It has no options.

FILES

input_file A interchunk.xml file
modes.dtd The DTD used to validate the input file.

NAME

dnsipq - dns lookup tool
 

SYNOPSIS

dnsipq [ udn ]
 

DESCRIPTION

dnsipq feeds the name udn through qualification(5). It prints the fully qualified domain name and IP addresses on a single line. If the fully qualified domain name does not exist, dnsipq prints no addresses. You can list several udns; dnsipq prints each result on a separate line.
 
Normally dnsipq exits 0. If dnsipq encounters a temporary problem that prevents it from determining the list of IP addresses, it prints an error message and exits 111.

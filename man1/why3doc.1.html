NAME
why3doc - format Why3 theories in HTML

SYNOPSIS
why3doc [OPTIONS] [FILE ...]

DESCRIPTION
Format Why3 theories in HTML for human viewing.

OPTIONS
-o, --output <dir>
Write HTML files to dir.

-b Output the HTML body only.

-help, --help
Show a list of options.

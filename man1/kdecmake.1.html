NAME

kdecmake - cmake --help-custom-modules no longer supported
 

DESCRIPTION

CMake versions prior to 3.0 exposed their internal module help page generation functionality through the --help-custom-modules option. CMake versions 3.0 and above use other means to generate their module help pages so this functionality is no longer available to be exposed.

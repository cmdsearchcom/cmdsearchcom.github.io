NAME
fvwm-crystal.play-movies - mplayer wrapper for fvwm-crystal

SYNOPSIS
fvwm-crystal.play-movies [directory]

DESCRIPTION
fvwm-crystal.play-movies is an helper for fvwm-crystal to use mplayer to play videos contained in a given directory.

This manual page was written for the Debian distribution because the original program does not have a manual page.

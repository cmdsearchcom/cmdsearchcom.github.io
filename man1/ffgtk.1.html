NAME

FFGTK - Router Manager

SYNOPSIS

ffgtk
 

DESCRIPTION

FFGTK is a clean solution for controlling the FRITZ!Box or compatible router with Linux. It offers a rich feature list, including caller monitor, fax and softphone support.

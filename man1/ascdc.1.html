NAME
ascdc - AfterStep CD changer

SYNOPSIS
ascdc [-options ...]

DESCRIPTION
‘ascdc’ is an AfterStep CD changer.

-g Geometry. Standard X Location

-d Device. default value is /dev/hdd. IDE Device of the Changer

-c cdplayer. Location of program to launch for audio CDs.

-n Number to set now. If set, switches to slot 0 on launch.

-m Mount/Umount Directory. If desired umounts & mounts, eg /cdrom or /mnt

-w Withdrawn. Starts window in Withdrawn State (for WindowMaker)

NAME
s3gdump -- s3g decompiler

SYNOPSIS
s3gsump [-h] [FILE]

DESCRIPTION
Dump the contents of the .s3g specified in FILE to STDOUT in plain-text. If FILE is not provided, then the s3g is read from STDIN instead.

-?, -h
Shows help message

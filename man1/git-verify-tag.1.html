NAME

git-verify-tag - Check the GPG signature of tags

SYNOPSIS

git verify-tag <tag>...
 

DESCRIPTION

Validates the gpg signature created by git tag.

OPTIONS

-v, --verbose
Print the contents of the tag object before validating it.
<tag>...
SHA-1 identifiers of Git tag objects.

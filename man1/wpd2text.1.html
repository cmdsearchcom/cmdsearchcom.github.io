NAME
wpd2text - convert WordPerfect document into plain text

SYNOPSIS
wpd2text [OPTION] <WordPerfect Document>

OPTIONS
--info Display document metadata instead of the text --help Shows this help message --password <password> Try to decrypt password protected document
--version Output wpd2text version

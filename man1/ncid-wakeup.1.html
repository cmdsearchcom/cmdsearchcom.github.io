NCID-WAKEUP(1) General Commands Manual NCID-WAKEUP(1)

NAME

ncid-wakeup

SYNOPSIS

ncid --wakeup ncid-wakeup

DESCRIPTION

The ncid-wakeup output module is called to wakeup X-Windows for ncid in GUI mode or ncid using a output module. If another module is called, it is executed before that module. Since no information regarding the call is required, none is sent. It is called when the WakeUp option is set either on the command line or in the ncid.conf file.

REQUIREMENTS

This module only works with X-windows and Gnome.
 
Do not set the ncid "--program | -P" option.

FILES

/etc/ncid/ncid.conf
 
/usr/share/ncid/modules/ncid-wakeup

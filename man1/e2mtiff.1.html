NAME
e2mtiff - any to multi-page TIFF converter of the ExactImage toolkit

SYNOPSIS
bardecode [{-o | --output} output-file] file...

bardecode {-h | --help}

DESCRIPTION
ExactImage is a fast C++ image processing library. Unlike many other library frameworks it allows operation in several color spaces and bit depths natively, resulting in low
memory and computational requirements.

e2mtiff converts a set of images into a multi-page TIFF file.

OPTIONS
-o output-file, --output output-file
Save output image to the specified file.

-h, --help
Display help text and exit.

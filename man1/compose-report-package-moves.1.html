NAME

compose-report-package-moves - show changes between two composes

SYNOPSIS

compose-report-package-moves [ OPTIONS...] OLD_COMPOSE NEW_COMPOSE

DESCRIPTION

compose-report-package-moves detects and reports packages that have been moved between related variants. Specifically, it is targetted at identifying situations when package moves from Optional into the base variant or the other way around.
By default, the results will be stored in JSON format in current directory. There will be multiple files with different views of the data.

OPTIONS

-h, --help
Print help and exit.
-p, --path=DIR
Set directory in which to create the files.
-n, --name=NAME
Set custom identifier for the generated files. Defaults to compose ID of new compose.

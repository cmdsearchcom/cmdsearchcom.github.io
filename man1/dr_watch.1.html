NAME


dr_watch - execute a program periodically, showing output fullscreen

SYNOPSIS


dr_watch command

DESCRIPTION


dr_watch tests wether watch(1) is available, and runs it if found. On systems not shipping watch, it executes a crude reimplementation of this program and runs command repeatedly, displaying its output.

ENVIRONMENT


dr_watch honors integer DR_SLEEP: number of seconds to wait between execution.

KOJIRA(1) User Commands KOJIRA(1)

NAME
kojira - Koji repository administrator

SYNOPSIS
kojira [opts]

OPTIONS
-h, --help
show this help message and exit

-c FILE, --config=FILE
use alternate configuration file

--user=USER
specify user

--password=PASSWORD
specify password

--principal=PRINCIPAL
Kerberos principal

--keytab=KEYTAB
Kerberos keytab

-f, --fg
run in foreground

-d, --debug
show debug output

-q, --quiet
don’t show warnings

-v, --verbose
show verbose output

--with-src
include srpms in repos

--force-lock
force lock for exclusive session

--debug-xmlrpc
show xmlrpc debug output

--skip-main
don’t actually run main

--show-config
Show config and exit

--sleeptime=SLEEPTIME
Specify the polling interval

-s SERVER, --server=SERVER
URL of XMLRPC server

--topdir=TOPDIR
Specify topdir

--logfile=LOGFILE
Specify logfile

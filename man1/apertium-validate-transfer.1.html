NAME
apertium-validate-transfer - This application is part of ( apertium )

This tool is part of the apertium open-source machine translation toolbox: http://www.apertium.org.

SYNOPSIS
apertium-validate-transfer <input_file>

DESCRIPTION
apertium-validate-transfer is a script that validates a set of structural transfer rules against the apertium structural transfer rules DTD using the xmllint utility.

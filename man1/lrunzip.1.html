NAME

lrunzip - Uncompress LRZ files

SYNOPSIS

  lrunzip [options] FILE [... FILE]

DESCRIPTION

lrunzip is identical to "lrzip -d" used to decompress files.

OPTIONS

See lrzip(1).

ENVIRONMENT

None.

FILES

None.

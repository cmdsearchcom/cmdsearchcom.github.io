NAME

mtx-grep

SYNOPSIS

mtxrun --script grep [ OPTIONS ... ] [ FILENAMES ]

DESCRIPTION

Simple Grepper

OPTIONS

--pattern
search for pattern (optional)
--count
count matches only
--nocomment
skip lines that start with %% or #
--xml
pattern is lpath expression

NCID-SKEL(1) General Commands Manual NCID-SKEL(1)

NAME

ncid-skel

SYNOPSIS

ncid --no-gui --program ncid-skel

DESCRIPTION

The ncid-skel output module is used as a template for writing a shell script output module. All it does is send the Caller ID data to standard output. This makes it useful for troubleshooting.

CONFIGURATION

The ncid-skel output module is configured in ncid-skel.conf.
PageSkel
This sets the NCID message types ncid-skel will send. The different types are CID, OUT, HUP, BLK, WID, MSG, PID, and NOT.
 
Default: skel_types="CID OUT HUP BLK MSG PID NOT"
skel_raw
If set to 0, data will be shown as a single horizontal line. MSG and NOT are processed as with other clients. If set to 1, data will also be formatted vertically, one line per input received.
 
Default: skel_raw=0

REQUIREMENTS

run ncid-skel with ncid in a terminal window to view output:
 
ncid --no-gui --program ncid-skel

FILES

/etc/ncid/conf.d/ncid-skel.conf
 
/usr/share/ncid/modules/ncid-skel

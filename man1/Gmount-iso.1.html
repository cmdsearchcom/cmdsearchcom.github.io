NAME

Gmount - A tool to easily mount your CD images

SYNOPSIS

Gmount

DESCRIPTION

Gmount allows you to easily mount your cd images. This is a frontend to the 'mount -o loop -t iso9660 foo.iso /mountpoint' command. All you need is to enter the iso path and the mount point in the dedicated fields.

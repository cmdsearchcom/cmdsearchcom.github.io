NAME
sflphone-client-gnome - SIP and IAX2 compatible voice over IP softphone GTK+ client.

SYNOPSIS
sflphone-client-gnome

DESCRIPTION
SFLphone is meant to be a robust enterprise-class desktop phone. It provides functions like call transfer, call hold, multiple lines, multiple accounts support. SFLphone audio
layer is build upon a native ALSA interface and and a native PulseAudio interface. sflphone-client-gnome is a GTK+ client for SFLphone; it communicates with the core side
through DBus. SFLphone package comes with the core, sflphoned.

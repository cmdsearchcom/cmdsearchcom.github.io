NAME

mailutils - Mailutils multi-purpose tool

SYNOPSIS

mailutils [ OPTION...] COMMAND [CMDOPTS]

DESCRIPTION

GNU Mailutils multi-purpose tool.
Commands are:
mailutils 2047
decode/encode email message headers
mailutils acl
test access control lists
mailutils cflags
show compiler options
mailutils dbm
DBM management tool
mailutils filter
apply a chain of filters to the input
mailutils help
display a terse help summary
mailutils imap
IMAP4 client shell
mailutils info
show Mailutils configuration
mailutils ldflags
list libraries required to link
mailutils logger
log data using Mailutils log facility
mailutils pop
POP3 client shell
mailutils query
query configuration values
mailutils send
send a message
mailutils smtp
run a SMTP session
mailutils wicket
scan wickets for matching URLs
Try `mailutils COMMAND --help' to get help on a particular COMMAND.
OPTIONs are:
Global debugging settings
--debug-level=LEVEL
set Mailutils debugging level
--debug-line-info
show source info with debugging messages
Informational options
--show-config-options
show compilation options
-?, --help
give this help list
--usage
give a short usage message
-V, --version
print program version
Mandatory or optional arguments to long options are also mandatory or optional for any corresponding short options.

NAME
rt-attributes-viewer - show the content of an attribute

SYNOPSIS
# show the content of attribute 2
rt-attributes-viewer 2

DESCRIPTION
This script deserializes and print content of an attribute defined by <attribute id>. May be useful for developers and for troubleshooting problems.

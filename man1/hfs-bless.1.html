NAME

bless - make HFS+ filesystems bootable on EFI-based Apple hardware

SYNOPSIS

bless filename
 

DESCRIPTION

bless makes the file filename run as the bootloader on HFS+ filesystems on Apple hardware when using EFI. It does not support setting EFI boot variables - use efibootmgr for that.
 

SEE ALSO

mkfs.hfsplus(1), efibootmgr(1)

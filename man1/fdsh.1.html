NAME

fdsh - simple shell for fdclone

SYNOPSIS

fdsh [ -acefhikmnrstuxv ] [ args ]

DESCRIPTION

fdsh is a simple shell to be used as built-in shell of fdclone. Please refer fd(1); the manual page of fdclone for detail.

NAME
passenger-config - Show configuration settings for Phusion Passenger

SYNOPSIS
passenger-config --root | --version

DESCRIPTION
passenger-config shows the current configuration settings for Phusion Passenger. You need to select one of the following options.

OPTIONS
--root Shows the Phusion Passenger root directory

--version
Shows the current Phusion Passenger version number

passenger-memory-stats(1), passenger-stress-test(1), passenger-status(8)

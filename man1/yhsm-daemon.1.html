NAME
yhsm-daemon â Allow multiple users of a YubiHSM.

SYNOPSIS
yhsm-daemon [options] [device] [port]

DESCRIPTION
This is a daemon to allow multiple users of a YubiHSM without requiring permission to use the device. The daemon listens on a TCP port on localhost and allows multiple connecâ
tions to share a YubiHSM. Access the YubiHSM via the daemon by specifying a device string following the yhsm://<host>:<port> syntax:

hsm = YHSM(’yhsm://localhost:5348’)

Note that the daemon and clients need to share the same version number to be compatible.

OPTIONS
-h, --help
Shows this help message and exit

-D, --device
YubiHSM device name

-I, --interface
Network interface to bind to

-P, --port
TCP port to bind to

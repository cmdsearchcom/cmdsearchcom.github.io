NAME
webtidy - show errors in an (X)HTML document

SYNOPSIS
webtidy [filename or url]... (filename - reads STDIN)

DESCRIPTION
--context[=n]
Show the offending line (and n surrounding lines)

--noerrors
Ignore errors

--nowarnings
Ignore warnings

--help This message

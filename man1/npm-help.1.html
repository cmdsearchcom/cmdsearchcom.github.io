NAME
npm-help - Get help on npm

SYNOPSIS
npm help <term> [<terms..>]

DESCRIPTION
If supplied a topic, then show the appropriate documentation page.

If the topic does not exist, or if multiple terms are provided, then run the help-search command to find a match. Note that, if help-search finds a single subject, then it will
run help on that topic, so unique matches are equivalent to specifying a topic name.

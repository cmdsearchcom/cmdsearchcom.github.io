preparetips5

NAME
preparetips5
- extract text from tips file

SYNOPSIS
preparetips5

DESCRIPTION
preparetips5 is a script to extract the text from a tips file. It outputs the text so xgettext can add the tips to a PO file. PO files provide a human-readable string format used
for translations.

preparetips5 looks for data/tips as tips file.

NAME

verilator_profcfunc - Read gprof report created with --profile-cfuncs

SYNOPSIS

  verilator --profile-cfuncs ....
  gcc --ggdb -pg ....
  {run executable}
  gprof
  verilator_profcfuncs gprof.out

DESCRIPTION

Verilator_profcfunc reads a profile report created by gprof. The names of the functions are then transformed, assuming the user used verilator's --profile-cfuncs, and a report printed showing the percentage of time, etc, in each Verilog block.

ARGUMENTS

--help
Displays this message and program version and exits.

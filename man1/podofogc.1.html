NAME
podofogc - Garbage collection in a PDF file.

SYNOPSIS
podofogc [inputfile] [outpufile]

DESCRIPTION
podofogc is one of the command line tools from the PoDoFo library that provide several useful operations to work with PDF files. It can perform a garbage collection operation on
a PDF file. All objects that are not reachable from within the trailer are deleted.

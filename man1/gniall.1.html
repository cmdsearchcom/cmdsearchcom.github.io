NAME

gniall - A language learning program

SYNOPSIS

gniall [OPTIONS]...

DESCRIPTION

This manual page documents briefly gniall , a language learning program for GNOME.

OPTIONS

gniall accepts the standard GNOME application options. Try `gniall --help' to get more information about these options.

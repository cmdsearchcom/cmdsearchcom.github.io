NAME
btrfs-sxbackup-destroy - Destroy a backup job

SYNOPSIS
btrfs-sxbackup destroy [-h] [--purge] <subvolume> [<subvolume> ...]

DESCRIPTION
Destroy a btrfs-sxbackup job.

OPTIONS
<subvolume>
backup job source or destination subvolume. local path or SSH url

-h, --help
show help message and exit --purge removes all backup snapshots from source and destination

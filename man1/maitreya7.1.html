NAME
maitreya7 - program for the display of astrological charts, Vedic and Western.

SYNOPSIS
maitreya7
maitreya7 displays, stores, prints and converts astrological charts, both Vedic and Western.

DESCRIPTION
Maitreya is a free software (GPL) for Vedic and western astrology. Program runs under X11.

The program supports many features for the daily work of Vedic and Western astrologers. A large number of calculation options that make the program a stable basis for research
purposes.

Languages: English (default), French, German, Hungarian, Italian, Polish, Romanian, Russian, Spanish, Telugu.

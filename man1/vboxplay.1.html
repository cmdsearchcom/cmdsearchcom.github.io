NAME

vboxplay - play vbox sound files
 

SYNOPSIS

vboxplay soundfile [volume]
 

DESCRIPTION

vboxplay interfaces with sox to play the specified sound file. Note that currently the volume parameter is ignored.
 

SEE ALSO

sox(1)

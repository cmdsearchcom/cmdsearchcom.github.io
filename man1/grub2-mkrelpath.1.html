NAME

grub-mkrelpath — Generate a relative GRUB path given an OS path.
 

SYNOPSIS

grub-mkrelpath FILE
 

DESCRIPTION

grub-mkrelpath takes an OS filesystem path for FILE and returns a relative path suitable for use in a GRUB configuration file.

NAME

ffkbjoystick - Emulate joystick via keyboard and post to blackboard

SYNOPSIS

ffkbjoystick [-r host[:port]]

DESCRIPTION

This tool is used to emulate a joystick using the keyboard and post the data to a remote blackboard, similar to ffjoystick.

OPTIONS

-h
Show usage instructions.
-r host[:port]]
The host and port of the Fawkes instance to connect to.

EXAMPLES

ffkbjoystick -r myrobot
Open the default local joystick and write the data to the blackboard on myrobot.

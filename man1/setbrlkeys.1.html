NAME
setbrlkeys - setup a braille keymap

DESCRIPTION
usage: ./setbrlkeys [ -h | -l | -r]

-h print this help

-l bind left hand’s [aq]sdfvb to 6-dot braille keyboard

-r bind right hand’s bnjkl[m;] to 6-dot braille keyboard

NAME
ncid-yac - send Caller ID to YAC listeners

SYNOPSIS
ncid --no-gui --program ncid-yac

DESCRIPTION
The ncid-yac output module sends the Caller ID or message to YAC listeners.

CONFIGURATION
The ncid-yac output module is configured in ncid-yac.conf.

YACTYPES This sets the NCID message types ncid-yac will send. The different types are CID, OUT, HUP, BLK, WID, MSG, PID, and NOT.

Default: YACTYPES="CID OUT MSG PID NOT"

YACPORT The YAC network port

Default: YACPORT=10629

YACLIST A list of YAC clients, separated by spaces, inside quotes. List can be IP_Addresses or hostnames or both.

Default: YACLIST="localhost"

REQUIREMENTS
YAC listener

URL: http://www.sunflowerhead.com/software/yac/

FILES
/etc/ncid/conf.d/ncid-yac.conf
/usr/share/ncid/modules/ncid-yac

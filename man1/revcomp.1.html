NAME

revcomp - reverse complement a nucleic acid sequence

SYNOPSIS

revcomp [-options] <seqfile>

DESCRIPTION

This manual page documents briefly the revcomp command.
revcomp is a program that generates reverse complement of sequences.

OPTIONS

Available options:
-h
Help; print version and usage info.

NAME
whereami-1.0 - displays work location

SYNOPSIS
whereami [OPTIONS]

DESCRIPTION
whereami displays information about the machine(location) you are working on. Information like terminal name, present working directory, host name, and the host IP address. This
is extremely useful for those who tend to work remotely on several machines at the same time.

OPTIONS
-d --dir
display current working directory

-h --help
show this help

-i --hostip
display machines IP address

-n --hostname
display machine name

-t --ttyname
display terminal name

-v --version
display version information

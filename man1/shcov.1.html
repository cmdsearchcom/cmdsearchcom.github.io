NAME
shcov - collect execution coverage information for shell scripts

DESCRIPTION
shcov collects coverage data for bourne shell and bash scripts similar to gcov for C/C++ programs. It is used together with the shlcov tool to create HTML output of coverage
information.

SYNOPSIS
shcov [--output=where] [--shell=shell] script ...

--output= where
Place output in where instead of /tmp/shcov

--shell= what
Use shell command line what instead of bash -x

HOMEPAGE

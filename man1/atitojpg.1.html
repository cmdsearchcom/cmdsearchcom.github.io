NAME

atitojpg - YUV conversion program from the GATOS package.

SYNOPSIS

atitojpg < gatos.yuv

DESCRIPTION

The program atitojpg convert the ".yuv" files created by gatos to a series of " .jpg" files.

NOTE

Unlike atitoppm, this application requires various utilities to be present (such as ppmtojpg, etc...). If they are not, this program will fail.

gatos, gatos-conf, atitv, atisplit, atitogif, atitoppm, xatitv, yuvsum, scanpci.

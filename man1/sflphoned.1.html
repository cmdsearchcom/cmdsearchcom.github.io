NAME

sflphoned - SIP and IAX2 compatible voice over IP softphone core.

SYNOPSIS

sflphoned [OPTION]...

DESCRIPTION

SFLphone is meant to be a robust enterprise-class desktop phone. It provides functions like call transfer, call hold, multiple lines, multiple accounts support. SFLphone audio layer is build upon a native ALSA interface and and a native PulseAudio interface. sflphoned is the core of SFLphone; it communicates with the client side through DBus. You need to install a client to use the daemon, for instance GTK+ client, sflphone-client-gnome.

OPTIONS

-c, --console
Output the log to the console instead of syslog.
-d, --debug
Debug mode (more verbose output).
-h, --help
Print short list of command-line options.

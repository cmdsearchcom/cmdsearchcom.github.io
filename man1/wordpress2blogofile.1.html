NAME
wordpress2blogofile - Migrating existing Wordpress Blog to Blogofile

SYNOPSIS
wordpress2blogofile

DESCRIPTION
wordpress2blogofile is used to migrate an existing Wordpress Blog to Blogofile /_posts directory format.

For a complete User Guide please see the Blogofile online Documentation at http://www.blogofile.com/documentation/migrating_blogs.html

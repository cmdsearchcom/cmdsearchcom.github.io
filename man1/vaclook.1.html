NAME

vaclook - browses .vacation.db and shows addresses

SYNOPSIS

vaclook

DESCRIPTION

Vaclook browses the ~/.vacation.db file and shows the addresses that have been replied to, as well as the time/date of the most recent 'vacation' reply to each address.

EXAMPLE

sender@senders.domain: Sat Nov 20 00:39:51 1999

FILES

~/.vacation.db

vacation(1)

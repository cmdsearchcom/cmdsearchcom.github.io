NAME
identicurse - Identi.ca client with a curses-based UI

SYNOPSIS
identicurse

DESCRIPTION
This manual page documents briefly the identicurse command.

IdentiCurse is a Python written commandline client for Identi.ca and other status.net based microblogging sites. It offers tabs for different microblogging feeds (personal timeâ
line, mentions, direct messages and public timeline by default) and use IRC likeslash-prefixed commands.

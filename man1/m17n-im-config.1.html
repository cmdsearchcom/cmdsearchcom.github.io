NAME

m17n-im-config - an input method configuration utility for m17n-lib

SYNOPSIS

m17n-im-config

DESCRIPTION

m17n-im-config is a utility to maintain per-user configuration of input methods provided by the m17n library. m17n is an abbreviation of Multilingualization.

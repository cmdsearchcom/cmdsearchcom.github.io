NAME
lrs-float2rat - Convert floating point coefficients to rationals

SYNOPSIS
lrs-float2rat [FILE]

DESCRIPTION
Lrs-float2rat reads a polyhedron file, or stdin if no file is given, and converts its floating point coefficients to rational coefficients. Each floating point coefficient f is
converted to 10^k*f/10 for an appropriate value of k. The results are not reduced.

The resulting polyhedron is printed to stdout.

FILE FORMAT
The input file should be in the format documented at http://cgm.cs.mcgill.ca/~avis/C/lrslib/USERGUIDE.html#File%20Formats.

EXIT STATUS
The exit status is one if an error is encountered, or zero otherwise.

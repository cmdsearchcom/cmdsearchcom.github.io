NAME
tsspolicyticket - manual page for tsspolicyticket v713

DESCRIPTION
policyticket

Runs TPM2_PolicyTicket

-ha session handle

-to timeout file name

-cp cpHash file (default none)

-pref policyRef file (default none)

-na authName file (not hierarchy)

-hi hierarchy (authName is hierarchy)

-tk ticket file name

SEE ALSO
The full documentation for tsspolicyticket is maintained as a Texinfo manual. If the info and tsspolicyticket programs are properly installed at your site, the command

info tsspolicyticket

should give you access to the complete manual.

NAME

tuc - tensor uncondense

SYNOPSIS

tuc [ OPTIONS] <info> <cond> <uncond>

DESCRIPTION

This program is part of the tensor condensation package. It is used to uncondense one or more vectors, i.e., it calculates the embedding of (M⊗N)e_H into (M⊗N). The input vectors must be given with respect to the basis defined by precond(1), which is also used by tcond(1). The uncondensed vectors are calculated with respect to the semisimplicity basis produced by pwkond(1).

OPTIONS

-Q
Quiet, no messages.
-V
Verbose, more messages.
-T <MaxTime>
Set CPU time limit

INPUT FILES

info.tki
Tensor condense info file.
M.cfinfo
Constituent info file for the left factor. The name is taken from info.tki.
N.cfinfo
Constituent info file for the right factor. The name is taken from info.tki.
cond
Matrix with condensed vectors.

OUTPUT FILES

uncond
Matrix with uncondensed vectors.

SEE ALSO

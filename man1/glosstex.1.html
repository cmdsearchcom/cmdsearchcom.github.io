NAME
glosstex - prepare glossaries and lists of acronyms

SYNOPSIS
glosstex AUX GDF... [-v[0-5]]

DESCRIPTION
Generate a glossary using the glossary definitions in the .gdf file, GDF, for the LaTeX .aux file, AUX.

OPTIONS
-vVERBOSITY
explain what is being done; VERBOSITY is from 0 to 5

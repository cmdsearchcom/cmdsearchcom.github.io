NAME

gnotime-remote - The GnoTime remote control

SYNOPSYS

gnotime-remote <command>

DESCRIPTION

gnotime-remote allows the user to send commands to a running GnoTime directly from the command line. The currently available commands are:
start-timer
- starts the timer
stop-timer
- stops the timer
save-file
- saves project data to the data file
reload-file
- reloads project data from current data file

NAME

kchmviewer - Windows CHM viewer for KDE

SYNOPSIS

kchmviewer

DESCRIPTION

kchmviewer makes it possible to browse native Windows CHM files under KDE.
Complete documentation can be found in docbook format in /usr/share/doc/kde/HTML/<lang>/kchmviewer/ on this system.

NAME
afl-tmin - test case minimizer for American Fuzzy Lop (afl)

SYNOPSIS
afl-tmin [options] -- /path/to/fuzzed/app [params]

DESCRIPTION
afl-tmin is simple test case minimizer that takes an input file and tries to remove as much data as possible while keeping the binary in a crashing state or producing consistent
instrumentation output (the mode is auto-selected based on initially observed behavior).

OPTIONS
Run afl-tmin without any arguments to see a complete list of options.

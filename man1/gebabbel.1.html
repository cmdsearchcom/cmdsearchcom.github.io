NAME

gebabbel - GPS file conversion and transfer from/to GPS units

SYNOPSIS

Gebabbel is a graphical frontend for gpsbabel. Use the command gebabbel to start the program.

DESCRIPTION

Gebabbel is a Qt4 frontend for the famous gpsbabel, a tool to convert GPS file formats back and forth. Gebabbel wraps the complexity of gpsbabel in a (hopefully) easy to use graphical user interface.
In the main window you can rightclick on any of the three panes and select a source (left pane) and a destination (right pane) format. Optionally a filter can be specified (middle pane) to filter out unwanted data.
Select File/Run to start the conversion process.

CONNECTING GPS UNITS

If you have a GPS unit that is connected via a serial connection, use the device name as the filename. If your GPS unit is connected via USB, use "usb:" as the filename.

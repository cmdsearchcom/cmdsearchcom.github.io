NAME
zuk - uncondense vectors

SYNOPSIS
zuk [OPTIONS] <Vectors> <Orbits> <Result>

DESCRIPTION
This program reads a matrix which is assumed to be a condensed space of a permutation representation whose orbits are in the file Orbits. The vectors in Vectors are elongated so
as to lie in the original permutation space and written out to the file Result. Orbits must be an orbits file in the format defined by zmo. Here is an example:

2 0 4
Space = 1 3 2
2 0 2

Orbits = (1,2) (3,4,5,6) (7,8,9)

2 2 0 0 0 0 4 4 4
Result = 1 1 3 3 3 3 2 2 2
2 2 0 0 0 0 2 2 2

OPTIONS
-Q Quiet, no messages.

-V Verbose, more messages.

-T <MaxTime>
Set CPU time limit

INPUT FILES
Vectors
Vectors to uncondense. This is a matrix where the number of columns equals the number of orbits.

Orbits A file containing orbits tables produced by zmo.

OUTPUT FILES
Result Uncondensed vectors.

NAME
logique49 - evaluate logical functions

SYNOPSIS
logique49 [OPTIONS] COUNT [BIT]... [NAME EXPRESSION]...

DESCRIPTION
logique49 evaluates named EXPRESSION(s) for each set of COUNT BIT(s) binary values. BIT(s) and NAME(s) are expected valid C identifiers. EXPRESSION(s) may refer BIT(s) integral
variables and NAME(s) preceding expressions variables.

OPTIONS
-P, --print
Print the parsed expressions instead.

-s, --skip-header
Do not print the NAME(s) line header.

--help Print a usage message and exit with a status code indicating success.

--version
Print version information on standard output then exit.

SYNTAX
The allowed expression syntax is same as for express17(1).

EXAMPLES
logique49 2 a b c "a | b" d "a & b" e "c ^ d"

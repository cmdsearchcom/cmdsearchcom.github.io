NAME
flasm - A Flash (SWF) bytecode assembler/disassembler.

SYNOPSIS
flasm [options] filename

DESCRIPTION
flasm is a program for assembling and disassembling Flash (SWF) bytecode.

Please see flasm.html for more documentation.

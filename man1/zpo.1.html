NAME
zpo - power of a matrix or permutation

SYNOPSIS
zpo [OPTIONS] <In> <N> <Result>

DESCRIPTION
This program calculates the Nth power of a matrix or permutation. The exponent N may be negative. For compatibility with the old zsm program, the power may be specified in the
zsm format. The following example shows two equivalent ways to calculate the 69th power of a matrix:

zpo matrix 69 result
zpo matrix pwr69 result

OPTIONS
-Q Quiet, no messages.

-V Verbose, more messages.

-T <MaxTime>
Set CPU time limit

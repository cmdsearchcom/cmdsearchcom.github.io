NAME
nbdkit-example2-plugin - An example nbdkit plugin

SYNOPSIS
nbdkit example2 file=FILENAME

DESCRIPTION
"nbdkit-example2-plugin" is an example nbdkit(1) plugin.

It serves the named "FILENAME" over NBD, readonly.

Mainly this is useful for testing nbdkit and as an example for writing nbdkit plugins (nbdkit-plugin(3)).

SEE ALSO
example2.c in the nbdkit source tree, nbdkit(1), nbdkit-plugin(3)

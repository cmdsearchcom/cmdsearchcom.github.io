GORENAME(1) General Commands Manual GORENAME(1)

NAME

gorename - performs precise type-safe renaming of identifiers in Go code

SYNOPSIS

gorename

DESCRIPTION

The gorename command performs precise type-safe renaming of identifiers in Go source code.
 
Run gorename -help for more details.

NAME
changerefkeys - Search and replace PDF object numbers in the Trailer

SYNOPSIS
changerefkeys [options] infile.pdf old-objnum new-objnum
[old-objnum new-objnum ...] [outfile.pdf]

Options:
-o --order preserve the internal PDF ordering for output
-v --verbose print diagnostic messages
-h --help verbose help message
-V --version print CAM::PDF version

DESCRIPTION
Changes a PDF to alter the object numbers in the PDF Trailer. The resulting edited PDF is output to a specified file or STDOUT.

This is a very low-level utility, and is not likely useful for general users.

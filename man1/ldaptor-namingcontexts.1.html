NAME

ldaptor-namingcontexts - get the LDAP naming contexts of an LDAP server
 

SYNOPSIS

ldaptor-namingcontexts host..
 
 

DESCRIPTION

ldaptor-namingcontexts retrieves the LDAP naming contexts from a list of LDAP servers.
 

SEE ALSO

ldaptor-find-server(1), ldaptor-namingcontexts(1), ldaptor-search(1)

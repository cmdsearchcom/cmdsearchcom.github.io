INTONE(1) General Commands Manual INTONE(1)

NAME

intone - elementary audio player

SYNOPSIS

intone

DESCRIPTION

intone is an elementary based mplayer audio frontend. It has been written for Openmoko Freerunner, but may be used with any phone using the Freesmartphone.org software stack with elementary.

OPTIONS

intone takes no parameters.

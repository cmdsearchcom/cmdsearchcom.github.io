FTDUMP(1) General Commands Manual FTDUMP(1)

NAME

ftdump - simple font dumper

SYNOPSIS

ftdump [options] fontname

DESCRIPTION

ftdump lists information about a font file that is relevant for FreeType.
This program is part of the FreeType demos package.

OPTIONS

-n
Print SFNT name tables.
-u
Emit UTF-8.
-V
Be verbose.
-v
Show version.

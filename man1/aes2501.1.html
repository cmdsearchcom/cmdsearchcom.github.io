SYNOPSIS

DESCRIPTION
This manual page documents briefly the aes2501 command.

aes2501 is a command line program that scans fingerprints from an aes2501 scanner. It creates an image in local directory named fp.pnm, so assure you have write permissions in
the directory you exec the command.

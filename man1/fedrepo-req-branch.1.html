NAME

fedrepo-req-branch - manual page for fedrepo-req-branch 1.5.0

SYNOPSIS

fedrepo-req-branch [ OPTIONS] REPO BRANCH

DESCRIPTION

Request new branches on src.fedoraproject.org repositories. This adds an entry in PDC, which is required to build your package, but doesn't touch git. You must still manually create your branch in git.
Examples:
fedrepo-req-branch nethack epel7
fedrepo-req-branch nethack lts --sl security_fixes:2022-01-01
--sl
bug_fixes:2022-01-01

OPTIONS

-n, --namespace [rpms|container|modules|test-modules]
The repo's namespace.
--sl TEXT
The SLs tied to the branch. This must be in the format of "sl_name:2017-12-25"
-h, --help
Show this message and exit.

NAME
cdcat - media catalog program

SYNOPSIS
cdcat

DESCRIPTION
This manual page documents briefly the cdcat, command. This manual page was written for the Debian GNU/Linux distribution because the original program does not have a manual
page.

cdcat is a graphical, multiplatform media catalog program which scans the directories/media you specify and makes a list of the filesystem (including the tags of MP3s) and stores
the result in a gzipped XML file.

NAME
anxgrep â Search for PATTERN in each annodex FILE or standard input.

SYNOPSIS
anxgrep [-E | --extended-regexp ] [-G | --basic-regexp ] [-i | --ignore-case ] [-v | --invert-match ] [-o filename | --output filename ] [--label LABEL ] [-q |
--quiet | --silent ] [-L | --files-without-match ] [-l | --files-with-matches ] [-c | --count ] [-h | --help ] [--version ] filename

Description
anxgrep searches the CMML tracks of Annodex FILEs (or standard input if no files are named, or the file name - is given) for elements containing a match to the given PATTERN. By
default, anxgrep prints the matching elements.

Options
anxgrep accepts the following options:

Regexp selection and interpretation
-E, --extended-regexp
PATTERN is an extended regular expression

-G, --basic-regexp
PATTERN is a basic regular expression

-i, --ignore-case
Ignore case distinctions

-v, --invert-match
Select non-matching elements

CMML options
--head Search only the CMML head element of each file (ignore clips)

--meta Search only meta elements (ignore clip descriptions)

Output options
-o filename, --output filename
Write output to the specified filename instead of printing it to standard output.

--label LABEL
Print LABEL as the filename for standard input.

-q, --quiet, --silent
Suppress all normal output

-L, --files-without-match
Only print FILE names containing no match

-l, --files-with-matches
Only print FILE names containing matches

-c, --count
Only print a count of matching lines per FILE

Miscellaneous options
-h, --help
Display usage information and exit.

--version Output version information and exit.

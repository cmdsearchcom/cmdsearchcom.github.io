NAME
amule-emc - list links inside emulecollection files

SYNOPSIS
amule-emc [options] files...

DESCRIPTION
This manual page documents briefly the amule-emc command.

amule-emc is a program to list links inside emulecollection files.

OPTIONS
A summary of options is included below.

-h Show summary of options.

-v Show version of program.

ed2k(1), amule(1).

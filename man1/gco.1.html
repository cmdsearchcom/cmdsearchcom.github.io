GCO(1) General Commands Manual GCO(1)

NAME

gco - The GNOME Comics Organizer

SYNOPSIS

gco [OPTIONS]...

DESCRIPTION

This manual page documents briefly gco , a Comics Organizer for GNOME.

OPTIONS

gco accepts the standard GNOME application options. Try `gco --help' to get more information about these options.

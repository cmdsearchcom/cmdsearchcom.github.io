NAME

lurker-index-lc — wrapper for lurker-index to lowercase mailing list names

SYNOPSIS

lurker-index-lc

DESCRIPTION

lurker-index-lc is a wrapper for lurker-index which converts the mailing list name to lowercase letters.
 

SEE ALSO

lurker-index(1)

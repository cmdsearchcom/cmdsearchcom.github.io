NAME
camomilelocaledef - a localedef database translators for camomile.

SYNOPSIS
camomilelocaledef [--enc { encoding }] [--file { file }] [-help | --help] [dir]

DESCRIPTION
This manual page documents briefly the camomilelocaledef command.

This manual page was written for the Debian GNU/Linux distribution because the original program does not have a manual page.

camomilelocaledef is made to convert standard localedef file into .mar files. This convert and split a standard localedef file into the internal database format of camomile.

--enc encoding
Encoding used for reading the input file. If omitted, default to UTF-8.

--file file
Input localedef file. If ommited, use stdin.

-help, --help
Display help message.

dir
Output directory. If omitted, default to the current directory.

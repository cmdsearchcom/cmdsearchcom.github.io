NAME
Send-reminders - send email reminders for special occasions

SYNOPSIS
Send emails reminders set by users for special occasions.

DESCRIPTION
Email-reminder allows users to define events that they want to be reminded of by email. Possible events include birthdays, anniversaries and yearly events. Reminders can be
sent on the day of the event and a few days beforehand.

This script is meant to be invoked everyday by a cron job. It mails the actual reminders out.

When run by the root user, it processes all of the spooled reminders. When run by a specific user, it only processes reminders set by that user.

OPTIONS
--help
Displays basic usage message.

--simulate
Does not actually send any emails out.

--verbose
Prints out information about what the program is doing, including the full emails being sent out.

--version
Displays the version number.

FILES
~/.email-reminders, /etc/email-reminder.conf

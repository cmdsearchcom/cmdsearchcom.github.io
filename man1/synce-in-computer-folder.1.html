NAME
synce-in-computer-folder - Manages SynCE icon into the Computer folder

SYNOPSIS
synce-in-computer-folder install|uninstall|connect|disconnect

DESCRIPTION
Manages the SynCE icon in the Computer folder in Nautilus. This involves adding, removing, connecting and disconneting the shortcut. This shortcut resolves to synce:/// so the
GNOME VFS plugin must be installed correctly for this shortcut to be of any use.

OPTIONS
install
Call this program on device connect/disconnect.

uninstall
Do not call this program on device connect/disconnect.

connect
Add "Mobile Device" to Computer folder.

disconnect
Remove "Mobile Device" from Computer folder.

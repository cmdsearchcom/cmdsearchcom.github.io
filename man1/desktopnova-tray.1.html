NAME
desktopnova-tray - control the DesktopNova daemon

DESCRIPTION
DesktopNova-Tray displays a icon in the system tray. So you can control the daemon of DesktopNova with a few clicks.

To start DesktopNova-Tray you can execute desktopnova-tray in a terminal.

You can also start the program via the starter in the gnome menu.

* Applications > Accessories > DesktopNova-Tray

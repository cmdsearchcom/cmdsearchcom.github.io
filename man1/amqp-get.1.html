NAME
amqp-get â Get a message from a queue on an AMQP server

SYNOPSIS
amqp-get OPTION -q queue name

Description
amqp-get attempts to consume a single message from a queue on an AMQP server, and exits. Unless the queue was empty, the body of the resulting message is sent to standard outâ
put.

Options
-q

--queue=queue name
The name of the queue to consume messages from.

Exit Status
If the queue is not empty, and a message is successfully retrieved, the exit status is 0. If an error occurs, the exit status is 1. If the queue is found to be empty, the exit
status is 2.

Examples
Get a message from the queue ‘‘myqueue’’, and
display its body on standard output:" 10

$
amqp-get -q myqueue

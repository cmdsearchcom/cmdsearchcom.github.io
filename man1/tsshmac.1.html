TSSHMAC(1) User Commands TSSHMAC(1)

NAME

tsshmac - manual page for tsshmac v713

DESCRIPTION

hmac
Runs TPM2_HMAC
-hk key handle
-pwdk password for key (default empty)
[-halg [sha1, sha256, sha384] (default sha256)] -if input file to be HMACed -os hmac file name
-se[0-2] session handle / attributes (default PWAP)
01 continue

SEE ALSO

The full documentation for tsshmac is maintained as a Texinfo manual. If the info and tsshmac programs are properly installed at your site, the command
info tsshmac
should give you access to the complete manual.

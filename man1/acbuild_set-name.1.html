NAME
acbuild-set-name - Set the image name

SYNOPSIS
acbuild set-name [OPTIONS]

DESCRIPTION
Sets the name of the ACI in the manifest

OPTIONS INHERITED FROM PARENT COMMANDS
--debug[=false]
Print out debug information to stderr

--modify=""
Path to an ACI to modify (ignores build context)

--no-history[=false]
Don’t add annotations with the command that was run

--work-path="."
Path to place working files in

EXAMPLE
acbuild set-name quay.io/coreos/etcd

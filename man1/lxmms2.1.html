NAME
lxmms2 - control XMMS2 with a LIRC compatible remote control

SYNOPSIS
lxmms2 [lirc-config-file]

DESCRIPTION
lxmms2 is a tiny XMMS2 client to control XMMS2 with a LIRC compatible remote control. Following actions are supported:

Â· play (starts playback)

Â· pause (pauses playback)

Â· toggle_play_pause (toggles pause and starts playback if XMMS2 is not playing at all)

Â· toggle_pause (toggles pause)

Â· stop (stops playback)

Â· next (advances to the next track)

Â· prev (goes back to the previous track)

Â· volume_up (increases the volume)

Â· volume_down (decreases the volume)

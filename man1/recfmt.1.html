NAME

recfmt - apply a template to records

SYNOPSIS

recfmt [ OPTION]... [TEMPLATE]

DESCRIPTION

Apply a template to records read from standard input.
-f, --file=FILENAME
read the template to apply from a file.
--help
print a help message and exit.
--version
show version and exit.

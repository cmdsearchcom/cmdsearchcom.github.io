NAME

Cortina - a desktop wallpaper management application for the GNOME desktop

DESCRIPTION

This manual page documents briefly the cortina command.
Cortina completes (or replaces) the built-in GNOME desktop wallpapers selection tool. Cortina changes wallpapers by time. It works as an tray application with an configuration interface and works great under Ubuntu Lucid and all other GNOME using distributions.

OPTIONS

cortina

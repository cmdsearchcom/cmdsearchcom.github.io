NAME
gphoto2-config - get information about libgphoto2

SYNOPSIS
gphoto2-config [--prefix=DIR] [--libs] [--cflags] [--help] [--version]

OPTIONS
The options explained:

ââprefix=DIR change libgphoto2 prefix [default /usr]
ââlibs print library linking information
ââcflags print preâprocessor and compiler flags
ââhelp display this help and exit
ââversion output version information

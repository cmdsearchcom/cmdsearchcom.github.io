NAME
easychem - draw high-quality molecules and chemical 2D formulas

SYNOPSIS
easychem

DESCRIPTION
easychem is a program to help you to draw high-quality molecules and chemical 2D formulas, which you can then export to PDF, PS, LaTeX and fig.

OPTIONS
The program does not have any command line options.

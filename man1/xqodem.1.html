NAME

xqodem - Qodem Terminal Emulator

SYNOPSIS

xqodem qodem-options

DESCRIPTION

xqodem is a script wrapper around qodem. It tries to find a UTF-8 and color capable X11-based terminal on your system, and then runs qodem in a terminal session.

OPTIONS

See qodem (1).

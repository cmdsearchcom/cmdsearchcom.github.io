NAME
dreamchess - manual page for dreamchess 0.2.1-RC2

SYNOPSIS
dreamchess [options]

DESCRIPTION
DreamChess v0.2.1-RC2 (0203100)

An xboard-compatible chess interface.

OPTIONS
--help -h
Show help.

--list-drivers -l
List all available drivers.

--ui <drv> -u<drv>
Use user interface driver <drv>.

--fullscreen -f
Run fullscreen

--width -W<num>
Set screen width

--height -H<num>
Set screen height

--1st-engine <eng> -1<eng>
Use <eng> as first chess engine. Defaults to ’dreamer’.

--verbose <level> -v<level>
Set verbosity to <level>. Verbosity levels: 0 - Silent 1 - Errors only 2 - Errors and warnings only 3 - All Defaults to 1

SEE ALSO
The full documentation for dreamchess is maintained as a Texinfo manual. If the info and dreamchess programs are properly installed at your site, the command

info dreamchess

should give you access to the complete manual.

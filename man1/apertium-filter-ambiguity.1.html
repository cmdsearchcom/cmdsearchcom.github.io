NAME

apertium-filter-ambiguity - This application is part of ( apertium )
This tool is part of the apertium open-source machine translation toolbox: http://www.apertium.org.

SYNOPSIS

apertium-filter-ambiguity [input_file [output_file]]

DESCRIPTION

apertium-filter-ambiguity takes input from STDIN or input_file, gets tagger data, filters ambiguity classes and outputs on STDOUT or output_file, in each case.

apertium(1).

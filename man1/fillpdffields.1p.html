NAME

fillpdffields - Replace PDF form fields with specified values

SYNOPSIS

 fillpdffields [options] infile.pdf outfile.pdf field value [field value ...]
 Options:
   -b --background=val  specify a background color behind the filled text field
                        Options are: 'none', gray or 'r g b'.  Default: '1'
   -t --triggerclear    remove all of the form triggers after replacing values
   -o --order           preserve the internal PDF ordering for output
   -v --verbose         print diagnostic messages
   -h --help            verbose help message
   -V --version         print CAM::PDF version
Examples of "--background" values are: 
  --background=none
  --background=1        # white
  --background=0.5      # gray
  --background=0        # black
  --background="1 0 0"  # red

DESCRIPTION

Fill in the forms in the PDF with the specified values, identified by their field names. See listpdffields for a the names of the form fields.

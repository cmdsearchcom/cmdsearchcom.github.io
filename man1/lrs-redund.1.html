NAME
lrs-redund - remove all redundant rows from a polytope

SYNOPSIS
lrs-redund [INFILE [OUTFILE]]

DESCRIPTION
Lrs-redund removes all redundant rows from an H-representation or V-representation. If INFILE is not given, then input is read from stdin. If OUTFILE is not given, then the
result is printed on stdout.

FILE FORMAT
The input file should be in the format documented at http://cgm.cs.mcgill.ca/~avis/C/lrslib/USERGUIDE.html#File%20Formats.

EXIT STATUS
The exit status is one if an error is encountered, or zero otherwise.

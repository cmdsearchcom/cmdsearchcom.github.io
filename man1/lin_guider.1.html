NAME
lin_guider - autoguiding software for astrophotography.

SYNOPSIS
lin_guider

DESCRIPTION
Lin-guider is an astronomical autoguiding program for Linux. It supports Philips, Logitech, uvc webcams, QHY5, QHY6, DSI2PRO, QHY5L-II-M, QHY5L-II-C, QHY5-II, ATIK, Starlight
Xpress, ZWO ASI astrocams for video and FTDI chip-based, parallel port-based (LPT), GPIO-based, GPUSB devices, Nexstar-protocol based and QHY5, QHY6, QHY5L-II-M, QHY5L-II-C,
QHY5-II, ATIK, Starlight Xpress, ZWO ASI astrocams for pulse guiding.

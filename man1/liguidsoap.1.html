NAME
liguidsoap - Control GUI for liquidsoap

DESCRIPTION
This manual page documents briefly the liguidsoap control GUI for liquidsoap.

OPTIONS
-h, --host
Host where liquidsoap is running

-p, --port
Port where liquidsoap is listening

NAME

dpns-modifygrpmap - modify group entry corresponding to a given virtual gid

SYNOPSIS

dpns-modifygrpmap --gid gid --group newname [ --status status ]

DESCRIPTION

dpns-modifygrpmap modifies the group entry corresponding to a given virtual gid.
This command requires ADMIN privilege.

OPTIONS

--gid gid
specifies the Virtual Group Id.
--group newname
specifies the new group name. It must be at most 255 characters long.
--status status
status can be set to 0 or a combination of ARGUS_BAN and LOCAL_BAN. This can be either alphanumeric or the corresponding numeric value.

EXIT STATUS

This program returns 0 if the operation was successful or >0 if the operation failed.

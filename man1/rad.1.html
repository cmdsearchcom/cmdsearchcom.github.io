NAME

rad - calculate the radical series of a module

SYNOPSIS

rad [ OPTIONS] <Name>

DESCRIPTION

This program calculates the radical series of an arbitrary module Name, or the homomorphisms from the projective modules corresponding to the composition factors of the given module to the module.

OPTIONS

-Q
Quiet, no messages.
-V
Verbose, more messages.
-T <MaxTime>
Set CPU time limit
-l <Length>
Calculate the first <Length> heads.
-H <Num>
Calculate generators for the <Num>th head.

INPUT FILES

Name.1 ... Name.nbgen
Generators of a representation.
Name.cfinfo
Info file after rnning pwkond(1).

OUTPUT FILES

Name.rad
Matrix for basis change or the vectors for the homomorphisms.
Name<S>.hNum
Generators for the Numth head (with -H).

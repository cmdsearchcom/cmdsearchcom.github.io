NAME

ncid-mythtv - display Caller ID on MythTV using mythtvosd

SYNOPSIS

ncid --no-gui --program ncid-mythtv

DESCRIPTION

The ncid-mythtv output module is a MythTV display program. It displays the Caller ID or message on a TV.

CONFIGURATION

The ncid-mythtv output module is configured in ncid-mythtv.conf.
MythtvTypes
This sets the NCID message types ncid-mythtv will send. The different types are CID, OUT, HUP, BLK, WID, MSG, PID, and NOT.
 
Default: default: SambaTypes="CID OUT HUP BLK MSG PID NOT"
mythtv_bcastaddr[N]
Must be set to the Mythtv broadcast address. Multiple clients are supported. N starts at 0 and increases by one for each client. For Example, two clients might be:
 
mythtv_bcastaddr[0]=127.0.0.1
 
mythtv_bcastaddr[1]=10.0.0.1
 
Default: mythtv_bcastaddr[0]=127.0.0.1
 

REQUIREMENTS

MythTV software

FILES

/etc/ncid/conf.d/ncid-mythtv.conf
 
/usr/share/ncid/modules/ncid-mythtv

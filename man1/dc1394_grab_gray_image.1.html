NAME
grab_gray_image - grab an image using libdc1394

SYNOPSIS
grab_gray_image

DESCRIPTION
Get one gray image using libdc1394 and store it in portable grayscale map format in Image.pgm

OPTIONS
none

BUGS
The filename should be a commandline argument

NAME
lxc delete - Delete containers and snapshots

SYNOPSIS
lxc delete [<remote>:]<container>[/<snapshot>] [[<remote>:]<container>[/<snapshot>]...]

DESCRIPTION
Delete containers and snapshots.

OPTIONS
--debug
(= false)

Enable debug mode

-f, --force
(= false)

Force the removal of stopped containers

--force-local
(= false)

Force using the local unix socket

-i, --interactive
(= false)

Require user confirmation

--no-alias
(= false)

Ignore aliases when determining what command to run

--verbose
(= false)

Enable verbose mode

NAME

gfsplit - split a file into a number of shares

SYNOPSIS

gfsplit [ OPTIONS] INPUTFILE [OUTPUTSTEM]

DESCRIPTION

Generate an N-of-M share of the INPUTFILE.
-n N
the threshold for recombination
-m M
the number of shares to generate
The OUTPUTSTEM if omitted will default to the name of the INPUTFILE. The program defaults to a 3-of-5 share if not otherwise instructed.

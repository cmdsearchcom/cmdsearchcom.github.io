NAME
ffindex_from_fasta - populate index from FASTA file

DESCRIPTION
USAGE: ffindex_from_fasta -v | [-s] data_filename index_filename fasta_filename

-s sort index file

ENVIRONMENT

FFINDEX_MAX_INDEX_ENTRIES - allocate memory for this number of entries

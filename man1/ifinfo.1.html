NAME
ifinfo â Print IF (Image File) Information

SYNOPSIS
ifinfo iffile [OPTIONS]

DESCRIPTION
ifinfo prints information about an IF file

The options are as follows:

--labels
Print history labels (default)

--stats
Print image statistics (default)

--no-labels
Do not print history labels

--no-stats
Do not print image statistics

--debug
Set debug mode

--verbose
Set verbose mode

--version
Print version

--help Print terse help summary

NAME
odcs-manager - manual page for odcs-manager 0.0.3

DESCRIPTION
usage: odcs-manager [-?]

{shell,runssl,generatelocalhostcert,db,runbackend,upgradedb,runserver,cleardb}
...

positional arguments:
{shell,runssl,generatelocalhostcert,db,runbackend,upgradedb,runserver,cleardb}

shell Runs a Python shell inside Flask application context.

runssl Runs the Flask app with the HTTPS settings configured in config.py

generatelocalhostcert
Creates a public/private key pair for message signing and the frontend

db Perform database migrations

runbackend
Runs the Flask app with the HTTPS settings configured in config.py

upgradedb
Upgrades the database schema to the latest revision

runserver
Runs the Flask development server i.e. app.run()

cleardb
Clears the database

optional arguments:
-?, --help
show this help message and exit

NAME

etandem - Finds tandem repeats in a nucleotide sequence

SYNOPSIS

etandem -sequence sequence -minrepeat  integer -maxrepeat  integer -threshold  integer -mismatch  boolean -uniform boolean -outfile report -origfile outfile
etandem -help

DESCRIPTION

etandem is a command line program from EMBOSS (“the European Molecular Biology Open Software Suite”). It is part of the "Nucleic:Repeats" command group(s).

OPTIONS

Input section

-sequence sequence

Required section

-minrepeat integer
Default value: 10
-maxrepeat integer
Default value: $(minrepeat)

Advanced section

-threshold integer
Default value: 20
-mismatch boolean
-uniform boolean

Output section

-outfile report
-origfile outfile

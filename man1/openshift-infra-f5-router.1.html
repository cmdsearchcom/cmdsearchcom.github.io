NAME
openshift infra f5-router - Start an F5 route synchronizer

SYNOPSIS
openshift infra f5-router [OPTIONS]

DESCRIPTION
Start an F5 route synchronizer

This command launches a process that will synchronize an F5 to the route configuration of your master.

You may restrict the set of routes exposed to a single project (with --namespace), projects your client has access to with a set of labels (--project-labels), namespaces matching
a label (--namespace-labels), or all namespaces (no argument). You can limit the routes to those matching a --labels or --fields selector. Note that you must have a cluster-wide
administrative role to view all namespaces.

NAME
powerline-lint - Powerline configuration checker.

SYNOPSIS
powerline-lint [-pPATH]â¦ [-d]

DESCRIPTION
-p, --config-path PATH
Paths where configuration should be checked, in order. You must supply all paths necessary for powerline to work, checking partial (e.g. only user overrides) configuration
is not supported.

-d, --debug
Display additional information. Used for debugging powerline-lint itself, not for debugging configuration.

-h, --help
Display help and exit.

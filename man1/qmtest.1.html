NAME
QMTest - QMTest is an automated software test execution tool.

SYNOPSIS
qmtest [ OPTION... ] COMMAND [ COMMAND-OPTION... ] [ ARGUMENT... ]

OPTIONS
-h, --help
: Display usage summary.

--version
: Display version information.

-D, --tdb PATH
: Path to the test database.

Commands:
create : Create (or update) an extension.

create-target
: Create (or update) a target specification.

create-tdb
: Create a new test database.

gui : Start the QMTest GUI.

extensions
: List extension classes.

describe
: Describe an extension.

help : Display usage summary.

ls : List database contents.

register
: Register an extension class.

remote : Run QMTest as a remote server.

report : Generate report from one or more test results.

run : Run one or more tests.

summarize
: Summarize results from a test run.

Invoke

qmtest COMMAND --help

for information about COMMAND-OPTIONS and ARGUMENTS.

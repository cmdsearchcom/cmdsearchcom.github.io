NAME
csg_dlptopol - Part of the VOTCA package

SYNOPSIS
csg_dlptopol [OPTION] [OPTIONPARAMETERS]

DESCRIPTION
Create a dlpoly topology template based on an existing (atomistic) topology and a mapping xml-file. The created template file needs to be inspected and amended by the user!

Please visit the program site at __http://www.votca.org__.

OPTIONS
Allowed options:

-h [ --help ]
display this help and exit

-v [ --verbose ]
be loud and noisy

--top arg
atomistic topology file

--out arg
output topology in dlpoly format

Mapping options:

--cg arg
coarse graining mapping and bond definitions (xml-file)

--map-ignore arg
list of molecules to ignore separated by ;

--no-map
disable mapping and act on original trajectory

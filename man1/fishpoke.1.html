NAME

fishpoke - trigger a poll immediately instead of waiting for the configured polling interval.

SYNOPSIS

fishpoke HOST[:PORT] TOPIC [SUBJECT...]
 

DESCRIPTION

The fishpoke program is used to trigger a poll immediately instead of waiting for the configured polling interval. This man page refers to the client for the Fishpolld server.

OPTIONS

-h, --help
Shows this help message.
 

EXTRAS

More information can be found by browsing the README file shipped with the package.

NAME
hamster-indicator - A panel indicator for the Hamster time tracking application

SYNOPSIS
hamster-indicator

DESCRIPTION
This manual page documents briefly the application hamster-indicator.

hamster-indicator helps you to keep track of how much time you spend on various activities during the day. Meant as a replacement to the GNOME2 panel applet it provides the same
basic functionality under Unity on current Ubuntu releases.

It is implemented using the Indicator Applet API.

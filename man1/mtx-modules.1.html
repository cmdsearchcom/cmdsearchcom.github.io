mtx-modules(1) ConTeXt Module Documentation Generators mtx-modules(1)

NAME
mtx-modules

SYNOPSIS
mtxrun --script modules [ OPTIONS ... ] [ FILENAMES ]

DESCRIPTION
ConTeXt Module Documentation Generators

OPTIONS
--convert
convert source files (tex, mkii, mkiv, mp) to ’ted’ files

--process
process source files (tex, mkii, mkiv, mp) to ’pdf’ files

--prep use original name with suffix ’prep’ appended

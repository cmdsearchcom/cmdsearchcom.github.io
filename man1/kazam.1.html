NAME
Kazam - Screen recording and capturing program.

SYNOPSIS
kazam [-h] [--debug] [--silent] [--nosound] [--test] [-v] [-f] [-a] [-s] [-w] [-p] [-g] [-x]

DESCRIPTION
Kazam provides a well designed and easy to use interface for capturing screencasts and screenshots. It can record desktop video and multiple audio streams simultaneously with
control over audio levels and the screen region being captured.

OPTIONS
-h, --help
show this help message and exit

--debug
enable debug mode

--silent
silent start, no UI shown

--nosound
disable PulseAudio

--test generate test video signal

-v, --version
show program’s version number and exit

-f, --fullscreen
instant screenshot of a full screen

-a, --area
instant screenshot of a preselected screen area

-s, --select
instant screenshot of a preselected window

-w, --window
instant screenshot of an active window

-p, --preferences
show preferences window

-g, --godmode
god mode of capture

-x, --instance
spawn new instance

NAME

gkdebconf-term — wrapper to dpkg-reconfigure

SYNOPSIS

gkdebconf-term

DESCRIPTION

This manual page documents briefly the gkdebconf-term program.
gkdebconf-term is a wrapper to dpkg-reconfigure. It is not meant to be used alone, but by gkdebconf when running dpkg-reconfigure inside an X terminal.
 

SEE ALSO

debconf (8), dpkg-reconfigure(8), gkdebconf(1)

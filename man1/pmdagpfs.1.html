NAME
pmdagpfs - gpfs filesystem statistics performance metrics domain agent (PMDA)

DESCRIPTION
pmdagpfs is a Performance Metrics Domain Agent (PMDA) which exports metric values from the /usr/lpp/mmfs/bin/mmpmon program to provide information on mounted gpfs filesystems.

INSTALLATION
Install the gpfs PMDA by using the Install script as root:

# cd $PCP_PMDAS_DIR/gpfs
# ./Install

To uninstall, do the following as root:

# cd $PCP_PMDAS_DIR/gpfs
# ./Remove

pmdagpfs is launched by pmcd(1) and should never be executed directly. The Install and Remove scripts notify pmcd(1) when the agent is installed or removed.

FILES
$PCP_PMDAS_DIR/gpfs/Install
installation script for the pmdagpfs agent

$PCP_PMDAS_DIR/gpfs/Remove
undo installation script for the pmdagpfs agent

$PCP_LOG_DIR/pmcd/gpfs.log
default log file for error messages from pmdagpfs

NAME
ppmtoneo - convert a PPM into an Atari Neochrome .neo file

SYNOPSIS
ppmtoneo

[ppmfile]

DESCRIPTION
This program is part of Netpbm(1)

ppmtoneo reads a PPM image as input and produces an Atari Neochrome .neo file as output.

SEE ALSO
neotoppm(1) , ppm(1)

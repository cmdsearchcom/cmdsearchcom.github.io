Name
papi_avail - provides availability and detail information for PAPI preset events.

Synopsis
papi_avail [-adht] [-e event]

Description
papi_avail is a PAPI utility program that reports information about the current PAPI installation and supported preset events. Using the -e option, it will also display
information about specific native events.

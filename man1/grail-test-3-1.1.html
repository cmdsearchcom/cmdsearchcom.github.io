NAME
grail-test-3-1 - dump grail v3 events

SYNPOSIS
grail-test-3-1 window

DESCRIPTION
This tool prints grail v3 events to stdout. It requires an X11 server which supports multi-touch through XInput 2.2. It subscribes to gestures with 3 touch start and max, but
with a minimum of 1 touch. The grail-test-3-1 tool helps test continuation through touch count changes.

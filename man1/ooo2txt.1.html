NAME
ooo2txt - Convert openoffice documents to simple text

SYNOPSYS
car2file OPTIONS... file

DESCRIPTION
Output the file content in simple text.

OPTIONS
-e<encoding>
Convert output to encoding <encoding>

-h Short help summary

-v Output the version

NAME

phacter - manual page for phacter 0.2.0

DESCRIPTION

usage: phacter [-h] [-l] [-v] [phact [phact ...]]
Phacter reports facts (phacts) about a system. If a phact is not specified then all phacts will be returned. Multiple phacts can be passed. Phacter is python.

positional arguments:

phact
phact name, use --list for a list of phacts.

optional arguments:

-h, --help
show this help message and exit
-l, --list
Only list the phacts, don't evauate their values.
-v, --version
Print the version and exit.

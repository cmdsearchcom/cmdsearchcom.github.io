NAME
tssloadexternal - manual page for tssloadexternal v713

DESCRIPTION
loadexternal

Runs TPM2_LoadExternal

[-hi hierarchy (e, o, p, n) (default NULL)] [-nalg name hash algorithm [sha1, sha256, sha384] (default sha256)] [-halg [sha256, sha1] (default sha256)] [Asymmetric Key
Algorithm]

[-rsa (default)] [-ecc curve (P256)]

-ipu public key file name

-ipem PEM format public key file name

-ider DER format plaintext key pair file name

[-si signing (default)] [-st storage]

SEE ALSO
The full documentation for tssloadexternal is maintained as a Texinfo manual. If the info and tssloadexternal programs are properly installed at your site, the command

info tssloadexternal

should give you access to the complete manual.

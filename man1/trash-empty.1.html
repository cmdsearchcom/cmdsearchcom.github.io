NAME
trash-empty - Empty for Command line trash utility.

SYNOPSIS
trash-empty [arguments]

DESCRIPTION
Remove for ever any trashed file and trashed directory. This command is a part of trash-cli package that provides a command line interface trashcan utility compliant with the
FreeDesktop.org Trash Specification. It remembers the name, original path, deletion date, and permissions of each trashed file.

ARGUMENTS
To remove all trashed files, use ’trash-empty’ without arguments.

To remove files that have been in the trash more than a given number of
days, use ’trash-empty x’, ’x’ representing the number of days.

EXAMPLES
$ date
Tue Feb 19 20:26:52 CET 2008
$ trash-list
2008-02-19 20:11:34 /home/einar/today
2008-02-18 20:11:34 /home/einar/yesterday
2008-02-10 20:11:34 /home/einar/last_week
$ trash-empty 7
$ trash-list
2008-02-19 20:11:34 /home/einar/today
2008-02-18 20:11:34 /home/einar/yesterday
$ trash-empty 1
$ trash-list
2008-02-19 20:11:34 /home/einar/today
$ trash-empty
$ trash-list
<none>

NAME
gscriptor - Perl script to send commands to a smart card using a Gtk GUI

SYNOPSIS
gscriptor

DESCRIPTION
This manual page documents briefly the gscriptor command.

gscriptor is a program that sends commands to a smart card using a batch file. The syntax of the commands is the same as for scriptor(1).

scriptor(1) is in text mode. gscriptor(1) uses a Perl-Gtk graphical user interface and may be more user friendly.

See scriptor(1) for details on the comands format.

OPTIONS
none

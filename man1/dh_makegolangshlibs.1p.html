NAME
dh_makegolangshlibs - Generates golang-specific shlibs files and substvars

SYNOPSIS
dh_makegolangshlibs [debhelperÂ options]

DESCRIPTION
dh_makegolangshlibs is a debhelper program which generates a substvar and shlibs file to enable packages depending on go shared libraries to conveniently depend on the ABI hash
of the library rather than just the soversion. It replaces dh_makeshlibs in the usual dh sequences.

NOTES
The best way to invoke dh_makegolangshlibs is by using dh --with=golang.

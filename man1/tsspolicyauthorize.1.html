NAME
tsspolicyauthorize - manual page for tsspolicyauthorize v713

DESCRIPTION
policyauthorize

Runs TPM2_PolicyAuthorize

-ha session handle

-appr digest of the policy being approved

[-pref policyRef file] (default none) -skn signing key Name file name -tk ticket file name

-se[0-2] session handle / attributes (default PWAP)

01 continue

SEE ALSO
The full documentation for tsspolicyauthorize is maintained as a Texinfo manual. If the info and tsspolicyauthorize programs are properly installed at your site, the command

info tsspolicyauthorize

should give you access to the complete manual.

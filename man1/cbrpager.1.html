NAME
cbrpager - viewer for CBR and CBZ (comic book archive) files

SYNOPSIS
cbrpager [-d|-h] [file]

DESCRIPTION
cbrpager [-d] [-h]

-h print this message and quit

-d Enables extra debugging messages

file File to show

NAME

tmx2html - Converts a TMX to an HTML formated page

SYNOPSIS

   tmx2html [-icons] file.tmx > file.html

DESCRIPTION

Pretty prints a TMX as an HTML file. Icons for languages can be produced, but still buggy.

NAME
pdfsig - Portable Document Format (PDF) digital signatures tool

SYNOPSIS
pdfsig [options] [PDF-file]

DESCRIPTION
pdfsig verifies the digital signatures in a PDF document. It also displays the identity of each signer (commonName field of the signer certificate) and the time and date of the
signature.

The signer certificate validation uses the trusted certificates stored in the following locations:

OPTIONS
-nocert
Do not validate the certificate.

-v Print copyright and version information.

-h Print usage information. (-help and --help are equivalent.)

NAME

bboxx - The program outputs the page number and the Bounding Box parameters for every page in the file filename, by postprocessing the output from the gs bbox device.

SYNOPSIS

bboxx [ -help] [-letter|-a4] [-res res] [-insert ] [-bypbm ] [-pages p ] [-evenodd ] postscript.file

DESCRIPTION

This manual page documents briefly the bboxx. Run it with '-help' to see description of options
This manual page was written for the Debian(TM) distribution because the original program does not have a manual page.

SEE ALSO

impose (1), fixtd (1), psbl (1).

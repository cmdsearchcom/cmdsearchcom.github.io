NAME

apertium-gen-reformat - This application is part of ( apertium )
This tool is part of the apertium open-source machine translation architecture: http://www.apertium.org.

SYNOPSIS

apertium-gen-reformat [ -O ] <input file> <output file>

DESCRIPTION

apertium-gen-reformat is a script which generates a C++ reformatter for a particular format. The reformatter reads in a format file in XML and outputs a C++ reformatter using flex.

OPTIONS

-h, --help
Display this help.

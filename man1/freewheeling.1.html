NAME
freewheeling â live looping musical instrument

SYNOPSIS
fweelin

DESCRIPTION
This manual page documents briefly the fweelin command.

freewheeling is a live looping instrument. Using your keyboard or an external MIDI controller, you can record and trigger loops. Press ’/’ to toggle the keyboard shorcuts help.

You will need to have jackd running in order to run FreeWheeling

OPTIONS
freewheeling takes no command line arguments. The configuration file $(HOME)/.fweelin/fweelin.xml contains information on how to configure fweelin.

CONFIGURATION
The configuration file for freewheeling is now located at ~/.fweelin/.fweelin.xml. You might need to start fweelin from a terminal to update your old configuration.

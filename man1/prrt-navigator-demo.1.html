NAME

prrt-navigator-demo - Mobile robot navigation simulator

SYNOPSIS

prrt-navigator-demo

DESCRIPTION

prrt-navigator-demo is a GUI program that allows experimenting with a new robot navigation algorithm based on Rapidly-exploring Random Tree (RTT) exploration using Parameterized Trajectory Generators (PTGs).
Refer to the source code and related papers for more information about the method itself.

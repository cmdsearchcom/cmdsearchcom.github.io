NAME

fdebugdump - dumps a hierarchy of the debug info dies

SYNOPSIS

fdebugrpm { executable}

DESCRIPTION

fdebugdump runs the given program and dumps a hierarchical view of libraries loaded at attach time. Does not watch for loading of new modules, but that would be a good extension.

EXAMPLE

fdebugdump ls

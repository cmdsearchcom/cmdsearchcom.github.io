NAME
lv2ls - list all installed LV2 plugins

SYNOPSIS
lv2ls [OPTION]...

OPTIONS
-n, --names
Show names instead of URIs

--help Display help and exit

--version
Display version information and exit

SEE ALSO
lilv(3), lv2info(1)

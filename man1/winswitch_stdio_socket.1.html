NAME
winswitch_stdio_socket - Window Switch Unix Domain Socket Helper

DESCRIPTION
winswitch_stdio_socket is used for connecting to (and potentially starting) a winswitch_server instance over ssh.

It is not meant to be used directly but needs to be present on the PATH so that remote clients can find it easily.

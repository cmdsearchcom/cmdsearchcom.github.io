NAME

lxclipboard - clipboard utility for LXSession and LXDE

SYNOPSIS

lxclipboard
 

DESCRIPTION

lxclipboard is a clipboard utility, to be able to do copy/paste operation between GTK+ applications. lxcliboard is only useful when lxsession is not compile with clipboard build-in support. If lxsession is compile with clipboard build-in lxclipboard functionality will be automatically launched, without the need to launch lxclipboard.

NAME
pmlgrep â provides a grepâlike utility for PhotoML files

SYNOPSIS
pmlgrep [-h] [-e] [-p] [-x xpath] [-s regex] ... [-s regex] xmlfile [xmlfile] ...

DESCRIPTION
pmlgrep is a shell script that supports the use of xgrep on PhotoML files.

OPTIONS
pmlgrep accepts the following options:

-h Display usage information.

-e Expand defaults before searching.

-p Use perl regular expressions.

-x xpath
Find nodes matching xpath.

-s regex
Find nodes matching custom syntax used by xgrep.

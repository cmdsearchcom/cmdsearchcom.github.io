NAME
eblook - look in an electronic dictionary

SYNOPSIS
eblook [option...] [book-directory [appendix-directory]]

OPTIONS
-e, --encoding=NAME
specify input/output encoding

-q, --no-init-file
ignore user init file

-h, --help
show this message

-v, --version
show version number

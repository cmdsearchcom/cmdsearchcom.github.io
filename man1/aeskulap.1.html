NAME
aeskulap - OpenSource DICOM Viewer

SYNOPSIS
aeskulap files

DESCRIPTION
This manual page documents briefly the aeskulap command.

aeskulap is able to load a series of special images stored in the DICOM format for review. Additionally it is able to query and fetch DICOM images from archive nodes (also called
PACS) over the network. Aeskulap tries to achieve a full open source replacement for commercially available DICOM viewers.

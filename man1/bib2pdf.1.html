NAME
bib2pdf - script to convert BibTeX format files to PDF format files

SYNOPSIS
bib2pdf <input.bib> <output.pdf>

DESCRIPTION
bib2pdf is a script to convert a BibTeX file into a PDF file. The conversion is performed by first creating a small LaTeX file in which all the bibliographical records are
inserted. That LaTeX file is then latex-compiled into a DVI file which, in turn, gets converted into a PDF file using dvipdfm. Note that the script might be configured to use
pdflatex instead of latex, thus producing the PDF output file with no extra conversion step.

This script is part of the cb2bib package and is located at /usr/share/cb2bib/c2btools/bib2pdf.

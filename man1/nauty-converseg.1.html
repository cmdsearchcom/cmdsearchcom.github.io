NAME
nauty-converseg - converse digraphs

SYNOPSIS
converseg [-q] [infile [outfile]]

DESCRIPTION
Take the converse digraphs of a file of graphs.

The output file has a header if and only if the input file does. Undirected graphs are passed through without change, while directed graphs are written in digraph6 forâ
mat.

-q Suppress auxiliary information.

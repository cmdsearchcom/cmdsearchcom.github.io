NAME
multifilter - Pandoc filter to apply filters listed in field "multifilter"

DESCRIPTION
This Pandoc filter applies other filters listed in document metadata field "multifilter". The filters can be specified like with pandoc option "-F" or "--filter". For instance
"input.md":

---
filters:
- filter1 # in $PATH or ~/.pandoc/filters
- ./filter2.pl # relative path
- /path/filter3 # absolute path
...

converted via

pandoc --filter multifilter input.md

will be filtered like

pandoc -F filter1 -F ./filter2.pl -F /path/filter3 input.md

In addition, filters are searched in "~/.pandoc/filters".

OPTIONS
--help|-h
Print this help.

--to|--write|-t|-w <format>
Print this help in given Pandoc output format, e.g. "markdown".

NAME
jid - json incremental digger

SYNOPSYS
jid

DESCRIPTION
jid a very simple tool. You can drill down JSON interactively by using filtering queries like jq. Suggestion and Auto completion of this tool will provide you a very comfortable
JSON drill down.

OPTIONS
-h, -help
print a help.

-q
Output query mode

--version
print the version and exit

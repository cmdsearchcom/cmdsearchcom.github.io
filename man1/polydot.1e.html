NAME
polydot - Draw dotplots for all-against-all comparison of a sequence set

SYNOPSIS
polydot -sequences seqset -wordsize integer -gap integer [-boxit boolean] [-dumpfeat toggle] -outfeat featout -graph graph

polydot -help

DESCRIPTION
polydot is a command line program from EMBOSS (âthe European Molecular Biology Open Software Suiteâ). It is part of the "Alignment:Dot plots" command group(s).

OPTIONS
Input section
-sequences seqset
File containing a sequence alignment

Required section
-wordsize integer
Default value: 6

Output section
-gap integer
This specifies the size of the gap that is used to separate the individual dotplots in the display. The size is measured in residues, as displayed in the output. Default
value: 10

-boxit boolean
Default value: Y

-dumpfeat toggle
Default value: N

-outfeat featout

-graph graph

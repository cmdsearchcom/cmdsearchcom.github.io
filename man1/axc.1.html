NAME
axc - afnix cross compiler

SYNOPSIS
axc [options] file

OPTIONS
[h]
prints the help message

[v]
prints the program version

[i path] path
add a directory path to the resolver

DESCRIPTION
axcinvokes the AFNIX cross compiler. At this stage of development, the compiler is merely a form serializer. Syntax checking is done and a binary file is generated for each
input file. The compiled file can be read by the AFNIX interpreter or included in a librarian.

VERSION
The current version is the 2.6.3release.

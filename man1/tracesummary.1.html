NAME
tracesummary - output basic summary stats about a trace

SYNOPSIS
tracesummary inputuri ...

DESCRPTION
tracesummary provides basic summary stats about a trace.

traceconvert is a shellscript wrapper around tracestats(1)

LINKS
More details about tracesummary (and libtrace) can be found at http://www.wand.net.nz/trac/libtrace/wiki/UserDocumentation

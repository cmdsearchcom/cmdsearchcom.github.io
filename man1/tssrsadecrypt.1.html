NAME
tssrsadecrypt - manual page for tssrsadecrypt v713

DESCRIPTION
rsadecrypt

Runs TPM2_Rsadecrypt

-hk key handle

-pwdk password for key (default empty)

-ie encrypt file name

-od decrypt file name

-se[0-2] session handle / attributes (default PWAP)

01 continue

SEE ALSO
The full documentation for tssrsadecrypt is maintained as a Texinfo manual. If the info and tssrsadecrypt programs are properly installed at your site, the command

info tssrsadecrypt

should give you access to the complete manual.

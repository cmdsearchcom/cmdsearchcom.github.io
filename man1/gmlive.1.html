NAME
gmlive - live video frontend for mplayer

SYNOPSIS
gmlive

DESCRIPTION
gmlive is a mplayer frontend designed to play live video. It provides simple user interface for playing video, and is able to play several live video stream including PPStream
and PPLive when the non-free library from the service provider is available.

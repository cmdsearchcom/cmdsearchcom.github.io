NAME

imp2vs - Bible/Commentary module creation tool

SYNOPSIS

imp2vs <FILE> [PATH]

DESCRIPTION

A command line utility to convert imp formatted file Bible/Commentary SWORD module.

OPTIONS

[PATH]
Specify path for the output module if different from the input file.

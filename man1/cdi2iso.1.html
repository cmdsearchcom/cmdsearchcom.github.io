NAME
cdi2iso - convert a DiscJuggler image file into standard ISO image file

SYNOPSIS
cdi2iso FILENAME.cdi FILENAME.iso

DESCRIPTION
cdi2iso is a very simple utility to convert a DiscJuggler image to the standard ISO-9660 format.

OPTIONS
This utility doesn’t accept any options.

NAME

hrpt_decode - Decode NOAA High Rate Picture Transmission

SYNOPSIS

hrpt_decode: [ options]

OPTIONS

-h, --help
show this help message and exit
-d DECIM, --decim=DECIM
Set decim [default=32]
-p PLL_ALPHA, --pll-alpha=PLL_ALPHA
Set pll_alpha [default=10m]
-s CLOCK_ALPHA, --clock-alpha=CLOCK_ALPHA
Set clock_alpha [default=10m]
-F INPUT_FILENAME, --input-filename=INPUT_FILENAME
Set frames.hrpt [default=frames.hrpt]

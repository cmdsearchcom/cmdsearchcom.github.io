NAME

pdfrevert.plx - Removes one layer of edits from a PDF file

DESCRIPTION

Removes one layer of changes to a PDF file, trying to maximise the
size of the output file (to account for linearised PDF).

SYNOPSIS

    PDFREVERT infile

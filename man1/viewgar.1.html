NAME

viewgar - GiNaC archive file viewer

SYNPOSIS

viewgar [-d] [file...]

DESCRIPTION

viewgar displays the contents of GiNaC archive files. By default it will print the archived expressions in standard mathematical notation. If given the -d option it will output a raw dump of the archive contents).

OPTIONS

-d
print raw dump of archive instead of formatted expressions

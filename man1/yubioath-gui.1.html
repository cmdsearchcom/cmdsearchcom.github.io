NAME
yubioath-gui - Yubico Authenticator graphical interface

SYNOPSIS
yubioath-gui

DESCRIPTION
The Yubico Authenticator is a graphical desktop tool for generating Open AuTHentication (OATH) event-based HOTP and time-based TOTP one-time password codes, with the help of a
YubiKey that protects the shared secrets.

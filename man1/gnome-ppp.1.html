NAME

gnome-ppp — a PPP account configurer and dialup tool for GNOME

SYNOPSIS

gnome-ppp

DESCRIPTION

gnome-ppp is a GNOME program for configuration and control of PPP dialup accounts. It's actually a front end for wvdial. Thus PPP accounts set up using gnome-ppp can be used if you run wvdial from command line.
In the Setup dialog you can configure gnome-ppp to provide connection information in the desktop notification area, otherwise it will create a separate connection status window.

OPTIONS

--version
Show version of program.

SEE ALSO

wvdial (1), wvdialconf (1).

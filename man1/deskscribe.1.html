NAME

deskscribe - Suite of tools for recording desktop activity

SYNOPSIS

deskscribe

DESCRIPTION

Mousetweaks captures the mouse pointer at 200 frames per second, so I would advise against running it on devices powered solely by a battery, for extended periods of time. Currently, it only captures and logs mouse activity data. The log format is completely compatibile with that of LogSquare.
Logging application runs as a system tray icon. From here, you can start
and stop the logging, or quit the application. In the future, it will eventually be able to log key presses as well as capture the screen in video formats.

NAME
daemonfs - A real time file monitoring software

SYNOPSIS
daemonfs

DESCRIPTION
DaemonFS is a real time file monitoring software which lets users track modifications to the files. Every time a file gets modified, a notification launched from the tray icon
appears. This software may be used for reverse engineering, hard disk usage tracking, software analysis and more.

DaemonFS has been written with QT4 libraries, and after the installation the application is available from the Applications menu.

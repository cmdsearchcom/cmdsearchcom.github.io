NAME
maqindex_socks - maqview client-server support

DESCRIPTION
Setup Server
maqindex_socks -s <port> <map file> [<cns file> [<width> [<height>]]]

Run Client
maqindex_socks -c <host> <port> [<ref_id> <start> <end>]

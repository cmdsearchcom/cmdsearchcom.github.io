NAME

gnome-nds-thumbnailer - create thumbnails for Nintendo DS roms
 

SYNOPSIS

gnome-nds-thumbnailer [-s size] input output
 

OPTIONS

-s
Specify the size of the generated thumbnail (in pixels)

NAME

ctail - watch and colorize a logfile
 

SYNOPSIS

ctail [FILE]
 

DESCRIPTION

ctail is a very simple alias that watches a file and colorizes it. It's very useful for monitoring logfiles.
 

EXAMPLE


ctail /var/log/syslog
 

SEE ALSO

ccze(1), tail(1)

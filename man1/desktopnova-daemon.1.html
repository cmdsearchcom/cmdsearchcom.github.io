NAME

desktopnova-daemon - daemon for desktopnova

DESCRIPTION

desktopnova-daemon is the daemon for DesktopNova.
To start the daemon you can execute desktopnova-daemon in a terminal.
Normally you shouldn't run the daemon manually! Use the DesktopNova frontend instead.

OPTIONS

Currently the daemon doesn't use any options.

FILES

DesktopNova looks for the configuration at the following location:
~/.config/desktopnova/desktopnova.xml

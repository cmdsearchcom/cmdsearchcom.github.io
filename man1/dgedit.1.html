NAME

dgedit - drum kit editor for DrumGizmo

SYNOPSIS

dgedit

DESCRIPTION

DGEdit is an open source editor of drum kits for DrumGizmo, an open source drum plugin and stand-alone application.
 

BUGS

Report bugs to http://www.drumgizmo.org/wiki/doku.php?id=bugs.
 

ADDITIONAL INFORMATION

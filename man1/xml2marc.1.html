NAME

xml2marc - convert LC SlimSchema XML to MARC21

SYNOPSIS

    # convert a file 
    % xml2marc marc.xml > marc.dat.
    # or use in a pipeline
    % cat marc.xml | xml2marc > marc.dat

DESCRIPTION

xml2marc is a command line utility for converting XML that uses the Library of Congress Slim Schema to MARC21 bibliographic data. Conversion is handled using the MARC::Record and MARC::File::XML packages.

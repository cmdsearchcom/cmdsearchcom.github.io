NAME
synce-trayicon - Gnome system tray app for SynCE

SYNOPSIS
synce-trayicon [-f] [-d level]

DESCRIPTION
synce-trayicon is a Gnome system tray application for synCE, for monitoring connected Windows CE based devices.

OPTIONS
-f Run in foreground (Don’t detach from terminal), session manager no restart.

-d level
Set debug log level: 0 - No logging, 1 - Errors only (default), 2 - Errors and warnings, 3 - Everything.

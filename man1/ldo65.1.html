NAME
ldo65 - linker for o65 object files

SYNOPSIS
ldo65 [OPTION]... FILE...

DESCRIPTION
ldo65 is a linker for files in the ‘o65’ object format, formerly ld65 but renamed to avoid conflicts with the cc65 package (a separate product).

OPTIONS
-b? addr
Relocate segment ? to addr. ? must be either t, d, b or z to indicate the text, data, bss or zero segment respectively. See the xa(1) man page for an explanation.

-o filename
Set output filename. The default is a.o65.

-G Suppress writing of globals.

--help Show summary of options.

--version
Show version of program.

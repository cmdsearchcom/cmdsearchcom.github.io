NAME
un_prelink.plug - a plugin for rpmrebuild, to revert prelink

SYNOPSIS
rpmrebuild --include un_prelink.plug <packagename>

DESCRIPTION
use this plugin to revert prelink.

prelink is a program which modifies ELF shared libraries and ELF dynam- ically linked binaries, so that the time which dynamic linker needs for their relocation at startup sigâ
nificantly decreases and also due to fewer relocations the run-time memory consumption decreases too.

OPTIONS
-h, --help
output help information plugins option and exit.

-v, --version
print plugin version and exit

FILES
un_prelink.sh
contains the code for change-files

un_prelink.plug

un_prelink.plug.1
is this man page

ENVIRONMENT EXAMPLES
rpmrebuild --include un_prelink.plug apache
to build apache package with not prelinked binaries

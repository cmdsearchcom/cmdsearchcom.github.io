NAME

CGDB - curses based frontend to GDB

DESCRIPTION

CGDB Usage:

cgdb [cgdb options] [--] [gdb options]

CGDB Options:

--version
Print version information and then exit.
--help
Print help (this message) and then exit.
-d
Set debugger to use.
--
Marks the end of CGDB's options.

NAME

ldaptor-ldap2maradns - LDAPtor maradns zone file exporter
 

SYNOPSIS

ldaptor-ldap2maradns [ options]
 

OPTIONS

--base=
LDAP base dn
--service-location=
Service location, in the form BASEDN:HOST[:PORT]
 

DESCRIPTION

ldaptor-ldap2maradns exports DNS entries to A and PTR records for use by the maradns DNS server.

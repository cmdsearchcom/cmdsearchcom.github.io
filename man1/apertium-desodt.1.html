NAME

apertium-desodt - This application is part of ( apertium )
This tool is part of the apertium open-source machine translation toolbox: http://www.apertium.org.

SYNOPSIS

apertium-desodt [ -h ] [ -i ] [ -n ] [ <input file> [ <output file> ] ]

DESCRIPTION

apertium-desodt is an ODT format processor. Data should be passed through this processor before being piped to lt-proc. The program takes input in the form of an ODT document and produces output suitable for processing with lt-proc. ODT tags and other format information are enclosed in brackets so that lt-proc treats them as whitespace between words.
 

OPTIONS

-h, --help
Display this help. -i Makes the addition of trailing sentence terminator (".") unconditional, often leading to duplicates. -n Suppresses the addition of a trailing sentence terminator.

EXAMPLE

You could write the following to show how the word "gener" is analysed:
echo "<b>gener</b>" | apertium-desodt | lt-proc ca-es.automorf.bin

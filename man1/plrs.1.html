NAME

lrs - Parallel reverse search for vertex enumeration/convex hull problems

SYNOPSIS

lrs FILE

DESCRIPTION

Plrs implements the reverse search algorithm for vertex enumeration/convex hull problems. The input file format is compatible with Komei Fukuda's cdd package (cddlib); see lrslib(5). All computations are done exactly in multiple precision integer arithmetic. Output is not stored in memory, so even problems with very large output sizes can sometimes be solved.
Unlike lrs(1), this implementation works in parallel.

FILE FORMAT

The input file should be in the format documented at http://cgm.cs.mcgill.ca/~avis/C/lrslib/USERGUIDE.html#File%20Formats.

EXIT STATUS

The exit status is one if an error is encountered, or zero otherwise.

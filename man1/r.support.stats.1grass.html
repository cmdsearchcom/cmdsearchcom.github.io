NAME
r.support.stats - Update raster map statistics

KEYWORDS
raster, statistics

SYNOPSIS
r.support.stats
r.support.stats --help
r.support.stats map=name [--help] [--verbose] [--quiet] [--ui]

Flags:
--help
Print usage summary

--verbose
Verbose module output

--quiet
Quiet module output

--ui
Force launching GUI dialog

Parameters:
map=name [required]
Name of raster map

DESCRIPTION
r.support.stats allows the user to update raster map statistics information.

NOTES
None

r.support, r.report

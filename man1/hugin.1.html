NAME

hugin - Hugin panorama creator

SYNOPSIS

hugin project_file|image files ...

DESCRIPTION

Hugin is a GUI tool for panorama creation.
This man page is a stub, complete documentation of the GUI can be found in the online manual available in the Help menu. Moreover there are related CLI tools listed below in the SEE ALSO section.

OPTIONS

Hugin has no command-line options.

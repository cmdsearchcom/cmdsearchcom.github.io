NAME
apertium-gen-deformat - This application is part of ( apertium )

This tool is part of the apertium open-source machine translation toolbox: http://www.apertium.org.

SYNOPSIS
apertium-gen-deformat [ -a | -A | -m | -M ] <input file> <output file>

DESCRIPTION
apertium-gen-deformat is a script which generates a C++ deformatter for a particular format. The deformatter reads in a format specification file in XML and outputs a C++ deforâ
matter using flex.

OPTIONS
-a Runs in apertium standard mode.

-A, Runs in apertium optimised mode (default)

-m Runs in matxin standard mode (matxin is another open-source machine translation system: http://www.sourceforge.org/matxin)

-M Runs in matxin optimised mode

-h, --help
Display this help.

NAME
groo-ga.pl - Check the grammar of Irish language OpenOffice.org documents

SYNOPSIS
groo-ga.pl filetocheck[.sxw] errorfile[.sxw]

DESCRIPTION
This script checks the grammar of the Irish language OpenOffice.org document given as the first argument, and creates a new document with the errors highlighted and annotated
with appropriate messages.

REQUIRES
Perl 5.8, Lingua::GA::Gramadoir

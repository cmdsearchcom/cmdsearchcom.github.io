NAME

fbcat - takes a screenshot using the framebuffer device

SYNOPSIS

fbcat [ fbdev]

DESCRIPTION

fbcat takes a screenshot using the framebuffer device. The produced image is in PPM format. Note that redirection of standard output to a file or another process is needed since fbcat will refuse to write it's output directly to the terminal.

SEE ALSO

ppm(5), fbgrab(1)

NAME
ao-telem - Analyse a flight log (either telemetry or eeprom)

SYNOPSIS
ao-telem [-s <summary-file>] [--summary=<summary-file>] [-d <detail-file>] [--detail=<detail-file>] [-r <raw-file>] [--raw=<raw-file>] [-p <plot-file>] [--plot=<plot-file>] [-g
<gps-file] [--gps=<gps-file] [-k <kml-file] [--kml=<kml-file] {flight.eeprom|flight.telem}

DESCRIPTION
ao-telem reads the specified flight log and produces several different kinds of output.

Summary
By default, summary information is shown on stdout. With the --summary option, it can be redirected to a file.

Detail When requested with the --detail option, a filtered version of the flight position, speed and acceleration are written to the specified file.

Raw The --raw option writes the unfiltered, but converted acceleration and height data to the specified file.

Plot The --plot option writes plots of height, speed and acceleration to the specified file in .svg format

GPS The --gps option writes the recorded GPS data to the specified file in three columns.

KML The --kml option writes the recorded GPS data to the specified file in Keyhole Markup Language format, which can be displayed in Googleearth.

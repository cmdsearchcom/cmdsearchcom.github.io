NAME

ltsp-cluster-lbserver — Loadbalancer server for LTSP-Cluster.
 

SYNOPSIS

ltsp-cluster-lbserver
 

DESCRIPTION

ltsp-cluster-lbserver is a daemon retrieving server status from the loadbalancer agents and returning IP addresses based on these.
 
<configuration file> The configuration is usually stored in /etc/ltsp/lbsconfig.xml
 

AVAILABILITY

ltsp-cluster-lbserver is part of ltsp-cluster-lbserver package and the latest versions are available in source form from https://launchpad.net/ltsp-cluster.

NAME
dh_testdir - test directory before building Debian package

SYNOPSIS
dh_testdir [debhelperÂ options] [fileÂ ...]

DESCRIPTION
dh_testdir tries to make sure that you are in the correct directory when building a Debian package. It makes sure that the file debian/control exists, as well as any other files
you specify. If not, it exits with an error.

OPTIONS
file ...
Test for the existence of these files too.

debhelper(7)

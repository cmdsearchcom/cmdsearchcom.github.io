NAME
falcon-conf - Falcon makefile settings helper

SYNOPSIS
falcon-conf [options]

DESCRIPTION
The falcon-conf script can be used to determine automatically Falcon development file installation path and settings for include and library options in compilers.

falcon-conf is written mainly to work with GNU gcc and make, but it provides also options to determine solely the paths or the files to be included or linked, stripped of GNU
options.

OPTIONS
Please, refer to the inline option list, which is available calling falcon-conf without any parameter.

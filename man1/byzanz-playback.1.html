NAME
byzanz-playback - Process a byzanz debug recording

SYNOPSIS
byzanz-playback [options] INFILE OUTFILE

DESCRIPTION
Byzanz debug recording can be created by using the extension .byzanz. To convert these recordings into the other formats supported by Byzanz, byzanz-playback can be used. The
INFILE must be a byzanz debug file, the OUTFILE is the file to convert it to. Its extension determines the format to be used. See the byzanz-record(1) man page for a list of supâ
ported formats and their extensions.

OPTIONS
-h, --help
Show brief help.

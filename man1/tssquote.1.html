NAME

tssquote - manual page for tssquote v713

DESCRIPTION

quote
Runs TPM2_Quote
-hp pcr handle (may be specified more than once)
-hk quoting key handle
[-pwdk password for quoting key (default empty)] [-halg for signing [sha1, sha256, sha384] (default sha256)] [-palg for PCR bank selection [sha1, sha256, sha384] (default sha256)] [-salg signature algorithm [rsa, ecc] (default rsa)] [-qd qualifying data file name] [-os quote signature file name (default none)] [-oa attestation output file name]
-se[0-2] session handle / attributes (default PWAP)
01 continue

SEE ALSO

The full documentation for tssquote is maintained as a Texinfo manual. If the info and tssquote programs are properly installed at your site, the command
info tssquote
should give you access to the complete manual.

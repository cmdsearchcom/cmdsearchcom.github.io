NAME
stag-query - aggregare queries

SYNOPSIS
stag-query avg person/age file.xml

stag-query sum person/salary file.xml

stag-query ’sub { $agg .= ", ".shift }’ person/name file.xml

DESCRIPTION
Performs aggregate queries

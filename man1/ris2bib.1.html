NAME
ris2bib - script to convert RIS format files to BibTeX format files

SYNOPSIS
ris2bib <input.ris> <output.bib>

DESCRIPTION
ris2bib is a script to convert an RIS-format file into a PDF file. The conversion is a two step process: the RIS-format file is first converted into an XML-format file; that
XML-format file is in turn converted into the BibTeX-format file (both utilities are shipped in the bibutils package).

This script is part of the cb2bib package and is located at /usr/share/cb2bib/c2btools/ris2bib.

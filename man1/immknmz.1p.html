NAME

immknmz - make namazu index for mail folders

SYNOPSIS

immknmz [OPTIONS] [FOLDERS]

DESCRIPTION

The immknmz command makes namazu index for mail folders.
Folders contained by the ~/Mail/.folders file are used by default. If you want to specify the folders, use the `--src' option.
This command is provided by IM (Internet Message).

OPTIONS

-s, --src=FOLDER,FOLDER...
Set source folders. "--src=+xxx" is equivalent to "+xxx".
-g, --gcnmz={on,off}
Use gcnmz after mknmz (Garbage Collection for Namazu index).
-v, --verbose={on,off}
Print verbose messages when running.
-t, --timestamp={on,off}
Use checking time stamp.
-h, --help
Display help message and exit.
--version
Output version information and exit.

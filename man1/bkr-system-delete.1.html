NAME
bkr-system-delete - Delete a Beaker system permanently

SYNOPSIS
bkr system-delete [options] <fqdn>

DESCRIPTION
Deletes a Beaker system permanently, but only if the system was never referenced in any recipes.

OPTIONS
Common bkr options are described in the Options section of bkr(1).

EXIT STATUS
Non-zero on error, otherwise zero.

EXAMPLES
Delete a particular system:

bkr system-delete system1.example.invalid

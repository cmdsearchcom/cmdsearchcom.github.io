NAME
tssduplicate - manual page for tssduplicate v713

DESCRIPTION
Duplicate

Runs TPM2_Duplicate

-ho object handle

[-pwdo password for object (default empty)] [-hp new parent handle (default TPM_RH_NULL)] [-ik encryption key in file name] [-salg symmetric algorithm (aes)(default none)]

[-oek encryption key out file name] -od duplicate private area file name -oss symmetric seed file name -se[0-2] session handle / attributes (default PWAP)

01 continue 20 command decrypt 40 response encrypt

SEE ALSO
The full documentation for tssduplicate is maintained as a Texinfo manual. If the info and tssduplicate programs are properly installed at your site, the command

info tssduplicate

should give you access to the complete manual.
